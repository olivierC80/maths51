1. Reconnaissance d'une application linéaire

Cette section introduit la notion fondamentale d'application linéaire et propose des exercices pour s'entraîner à identifier si une fonction donnée entre espaces vectoriels respecte les propriétés de linéarité.

Exercice 8.1.1

Objectif : savoir reconnaître si une application est linéaire ou non.

Déterminer si les applications suivantes sont des applications linéaires :

  1. $f_1 : \mathbb{R}^2 \to \mathbb{R}^2, (x,y) \longmapsto (-2x + 3y, x + y)$
  2. $f_2 : \mathbb{R}^3 \to \mathbb{R}^2, (x,y,z) \longmapsto (2x + y - 1, x + y - 3z)$
  3. $f_3 : \mathcal{M}_{3,1}(\mathbb{R}) \to \mathcal{M}_{2,1}(\mathbb{R}), \begin{pmatrix} x \\ y \\ z \end{pmatrix} \longmapsto \begin{pmatrix} 3x + y + z \\ -4x - 2y - z \end{pmatrix}$
  4. $f_4 : \mathbb{R}^3 \to \mathbb{R}^3, (x,y,z) \longmapsto (x + y, -x^2 + y - 2z, 3xyz)$
  5. $f_5 : \mathcal{M}_{3,1}(\mathbb{R}) \to \mathcal{M}_{3,1}(\mathbb{R}), X \longmapsto AX - X$ où $A = \begin{pmatrix} 1 & 2 & -2 \\ -3 & 1 & 2 \\ 0 & 0 & 1 \end{pmatrix}$
  6. $f_6 : \mathcal{M}_{3,1} \to \mathbb{R}, X \longmapsto \ ^tXX$
  7. $f_7 : \mathcal{M}_{3,1}(\mathbb{R}) \to \mathcal{M}_3(\mathbb{R}), X \longmapsto X\ ^tX$

Voir la correction.

Exercice 8.1.2

Objectif : trouver une condition sur un paramètre apparaissant dans la définition d'une application pour qu'elle soit linéaire.

Pour quelle valeur de $a$, l'application $f$ suivante est-elle linéaire ?

$$f : \mathbb{R}^3 \to \mathbb{R}^3, (x,y,z) \longmapsto (2x + y, 3y - 2x, a)$$

Voir la correction.

2. Applications linéaires sur $\mathbb{R}^n$ ou $M_{n,1}(\mathbb{R})$

Cette section se concentre sur l'étude des applications linéaires définies sur les espaces vectoriels usuels $\mathbb{R}^n$ (vecteurs lignes ou colonnes) ou $\mathcal{M}_{n,1}(\mathbb{R})$ (matrices colonnes). On s'intéresse particulièrement à la détermination de leur noyau et de leur image, ainsi qu'à leurs propriétés d'injectivité et de surjectivité.

Exercice 8.2.1

Objectif : détermination pratique du noyau et de l'image d'une application linéaire.

Pour toutes les applications linéaires reconnues de l'exercice 8.1.1, déterminer le noyau ($\text{Ker}$) et l'image ($\text{Im}$).

Voir la correction.

Exercice 8.2.2

Objectif : après avoir déterminé le noyau et l'image de $f$, conclure sur l'éventuelle injectivité ou surjectivité de $f$.

Déterminer le noyau et l'image des applications linéaires suivantes puis dire si elles sont injectives ou surjectives :

  1. $f_1 : \mathbb{R}^3 \to \mathbb{R}^3, (x, y, z) \mapsto (-2x + y, y - x, x + y + z)$
  2. $f_2 : \mathbb{R}^3 \to \mathbb{R}^3, (x, y, z) \mapsto (x + y + 2z, x + y, x + y - 3z)$
  3. $f_3 : \mathbb{R}^3 \to \mathbb{R}^4, (x, y, z) \mapsto (x + z, y - x, z + y, x + y + 2z)$
  4. $f_4 : \mathbb{R}^4 \to \mathbb{R}^3, (x, y, z, t) \mapsto (x + y + z + t, x + z + t, x + 2y + z + t)$
  5. $f_5 : \mathbb{R}^4 \to \mathbb{R}^2, (x, y, z, t) \mapsto (x + 2y - z + t, 3x + z - y - t)$

Voir la correction.

Exercice 8.2.4

Objectif : construction d'une composée bijective à l'aide d'une application injective et d'une application surjective.

Exercice 8.2.5

Objectif : construction d'une base de $\mathbb{R}^3$ à l'aide du noyau et de l'image d'une application linéaire.

Voir la correction.

3. Applications linéaires sur d'autres espaces

Cette section élargit le champ d'étude aux applications linéaires définies sur des espaces vectoriels autres que $\mathbb{R}^n$, comme les espaces de polynômes ($\mathbb{R}_n[X]$), les espaces de matrices ($\mathcal{M}_n(\mathbb{R})$) ou les espaces de suites.

Exercice 8.3.1

Objectif : étude d'un automorphisme de $\mathbb{R}_3[X]$.

Soit $f$ l'application définie sur $\mathbb{R}_3[X]$ par :

$$f(P) = XP - \frac{1}{3}(X^2 - 1)P' $$

Montrer qu'il s'agit d'un automorphisme de $\mathbb{R}_3[X]$. (Rappel: un automorphisme est un endomorphisme bijectif).

Voir la correction.

Exercice 8.3.2

Objectif : recherche d'une bijection réciproque.

Soit $f$ l'application définie sur $\mathbb{R}_n[X]$ par :

$$f(P) = P - P' $$
  1. Montrer que $f$ est un automorphisme de $\mathbb{R}_n[X]$.
  2. Soit $g \in \mathcal{L}(\mathbb{R}_n[X])$ défini par : $g(P) = P + P' + \cdots + P^{(n)}$. Déterminer $g \circ f$ et en déduire $f^{-1}$.

Voir la correction.

Soit $f$ l'application définie sur $\mathbb{R}_2[X]$ par :

Montrer qu'il s'agit d'un isomorphisme de $\mathbb{R}_2[X]$ sur $\mathbb{R}^3$. (Rappel: un isomorphisme est une application linéaire bijective entre deux espaces vectoriels).

Voir la correction.

Correction Exercice 8.1.1

Rappel: Une application $f: E \to F$ (où $E, F$ sont des $\mathbb{K}$-espaces vectoriels) est linéaire si $\forall u, v \in E, \forall \lambda \in \mathbb{K}$, $f(u + \lambda v) = f(u) + \lambda f(v)$. Une condition nécessaire (mais non suffisante) est que $f(0_E) = 0_F$. Une condition suffisante pratique pour $f: \mathbb{R}^n \to \mathbb{R}^m$ est que chaque composante de $f(x_1, ..., x_n)$ soit une combinaison linéaire des variables $x_1, ..., x_n$.

  1. $f_1(x,y) = (-2x + 3y, x + y)$. Les deux composantes sont des combinaisons linéaires de $x$ et $y$. Donc $f_1$ est linéaire.
  2. $f_2(x,y,z) = (2x + y - 1, x + y - 3z)$. La première composante contient un terme constant $-1$. Testons $f_2(0,0,0) = (-1, 0) \neq (0,0)$. Donc $f_2$ n'est pas linéaire.
  3. $f_3\begin{pmatrix} x \\ y \\ z \end{pmatrix} = \begin{pmatrix} 3x + y + z \\ -4x - 2y - z \end{pmatrix}$. Les deux composantes de la matrice image sont des combinaisons linéaires de $x, y, z$. Donc $f_3$ est linéaire. (C'est l'application associée à la matrice $\begin{pmatrix} 3 & 1 & 1 \\ -4 & -2 & -1 \end{pmatrix}$).
  4. $f_4(x,y,z) = (x + y, -x^2 + y - 2z, 3xyz)$. La deuxième composante contient $-x^2$ et la troisième $3xyz$, qui ne sont pas des termes linéaires. Donc $f_4$ n'est pas linéaire. Par exemple, $f_4(2,0,0) = (2, -4, 0)$ mais $2 f_4(1,0,0) = 2(1, -1, 0) = (2, -2, 0)$. L'additivité ou l'homogénéité n'est pas respectée.
  5. $f_5(X) = AX - X = (A-I)X$. Soient $X_1, X_2 \in \mathcal{M}_{3,1}(\mathbb{R})$ et $\lambda \in \mathbb{R}$. $$f_5(\lambda X_1 + X_2) = A(\lambda X_1 + X_2) - (\lambda X_1 + X_2)$$ Par distributivité et associativité de la multiplication matricielle et de l'addition : $$= \lambda A X_1 + A X_2 - \lambda X_1 - X_2$$ $$= \lambda (A X_1 - X_1) + (A X_2 - X_2) = \lambda f_5(X_1) + f_5(X_2).$$ Donc $f_5$ est linéaire. (C'est l'application linéaire associée à la matrice $A-I$).
  6. $f_6(X) = {}^tXX$. Soit $X = \begin{pmatrix} x \\ y \\ z \end{pmatrix}$, alors $f_6(X) = x^2 + y^2 + z^2$. Ceci n'est pas linéaire. Par exemple, $f_6(2X) = (2x)^2 + (2y)^2 + (2z)^2 = 4(x^2+y^2+z^2) = 4 f_6(X)$, mais la condition est $f_6(\lambda X) = \lambda f_6(X)$. Ici, $4 \neq 2$ (si $X \neq 0$). Donc $f_6$ n'est pas linéaire.
  7. $f_7(X) = X {}^tX$. Soit $X = \begin{pmatrix} x \\ y \\ z \end{pmatrix}$, $f_7(X) = \begin{pmatrix} x^2 & xy & xz \\ xy & y^2 & yz \\ xz & yz & z^2 \end{pmatrix}$. Les coefficients ne sont pas linéaires en $x, y, z$. Par exemple $f_7(2X) = (2X) {}^t(2X) = 4 X {}^tX = 4 f_7(X)$. Même problème qu'en 6. Donc $f_7$ n'est pas linéaire.

Une condition nécessaire pour qu'une application $f: E \to F$ soit linéaire est que $f(0_E) = 0_F$.

Ici, $E = \mathbb{R}^3$ et $F = \mathbb{R}^3$. Le vecteur nul de $E$ est $(0,0,0)$.

$$f(0,0,0) = (2(0) + 0, 3(0) - 2(0), a) = (0, 0, a)$$

Pour que $f$ soit linéaire, il est nécessaire que $f(0,0,0) = (0,0,0)$. En comparant avec le résultat obtenu, on doit avoir $(0, 0, a) = (0, 0, 0)$, ce qui impose $a = 0$.

Les trois composantes $(2x+y)$, $(3y-2x)$, et $0$ sont des combinaisons linéaires des variables $x, y, z$. Par conséquent, l'application $f$ est linéaire si et seulement si $a=0$.

La valeur de $a$ pour laquelle $f$ est linéaire est donc $\mathbf{a = 0}$.

Posons $(x', y', z') = f(x,y,z) = (\frac{x+y+z}{2}, y, \frac{x-y+z}{2})$. $$ \begin{aligned} f(x', y', z') &= \frac{1}{2} (x' + y' + z', 2y', x' - y' + z') \\ &= \frac{1}{2} \left( \frac{x+y+z}{2} + y + \frac{x-y+z}{2}, \quad 2y, \quad \frac{x+y+z}{2} - y + \frac{x-y+z}{2} \right) \\ &= \frac{1}{2} \left( \frac{x+y+z + 2y + x-y+z}{2}, \quad 2y, \quad \frac{x+y+z - 2y + x-y+z}{2} \right) \\ &= \frac{1}{2} \left( \frac{2x+2y+2z}{2}, \quad 2y, \quad \frac{2x-2y+2z}{2} \right) \\ &= \frac{1}{2} (x+y+z, 2y, x-y+z) \\ &= f(x,y,z) \end{aligned} $$ Donc $f \circ f = f$. $f$ est bien un projecteur.
  2. Noyau ($\text{Ker } f$): $f(x, y, z) = (0,0,0)$ $$ \frac{1}{2} (x + y + z, 2y, x - y + z) = (0,0,0) \implies \left\{ \begin{array}{rcr} x + y + z & = & 0 \\ 2y & = & 0 \\ x - y + z & = & 0 \end{array} \right. $$ De la deuxième équation, $y = 0$. Le système devient: $$ \left\{ \begin{array}{rcr} x + z & = & 0 \\ y & = & 0 \\ x + z & = & 0 \end{array} \right. \implies \left\{ \begin{array}{l} z = -x \\ y = 0 \end{array} \right. $$ Les solutions sont de la forme $(x, 0, -x) = x(1, 0, -1)$. $\text{Ker } f = \text{Vect}((1, 0, -1))$. On pose $v_1 = (1, 0, -1)$.
  3. Image ($\text{Im } f$): Base canonique $e_1, e_2, e_3$. $f(e_1) = f(1,0,0) = \frac{1}{2}(1+0+0, 2(0), 1-0+0) = (\frac{1}{2}, 0, \frac{1}{2})$. $f(e_2) = f(0,1,0) = \frac{1}{2}(0+1+0, 2(1), 0-1+0) = (\frac{1}{2}, 1, -\frac{1}{2})$. $f(e_3) = f(0,0,1) = \frac{1}{2}(0+0+1, 2(0), 0-0+1) = (\frac{1}{2}, 0, \frac{1}{2})$. On remarque que $f(e_1) = f(e_3)$. $$ \begin{aligned} \text{Im } f &= \text{Vect}(f(e_1), f(e_2), f(e_3)) = \text{Vect}((\frac{1}{2}, 0, \frac{1}{2}), (\frac{1}{2}, 1, -\frac{1}{2})) \\ &= \text{Vect}((1, 0, 1), (1, 2, -1)) \quad \text{(en multipliant par 2)} \end{aligned} $$ On pose $v_2 = (1, 0, 1)$ et $v_3 = (1, 2, -1)$. Ces deux vecteurs ne sont pas colinéaires, ils forment une base de $\text{Im } f$. $\dim(\text{Im } f) = 2$. (Théorème du rang: $\dim(\text{Ker } f) + \dim(\text{Im } f) = \dim(\mathbb{R}^3) \implies 1 + \dim(\text{Im } f) = 3 \implies \dim(\text{Im } f) = 2$.)
  4. Base de $\mathbb{R}^3$: On considère la famille $(v_1, v_2, v_3) = ((1, 0, -1), (1, 0, 1), (1, 2, -1))$. C'est une famille de 3 vecteurs dans $\mathbb{R}^3$ (qui est de dimension 3). Pour montrer que c'est une base, il suffit de montrer qu'elle est libre. Soit $\lambda_1, \lambda_2, \lambda_3 \in \mathbb{R}$ tels que $\lambda_1 v_1 + \lambda_2 v_2 + \lambda_3 v_3 = 0$. $$ \lambda_1 \begin{pmatrix} 1 \\ 0 \\ -1 \end{pmatrix} + \lambda_2 \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix} + \lambda_3 \begin{pmatrix} 1 \\ 2 \\ -1 \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \\ 0 \end{pmatrix} $$ $$ \left\{ \begin{array}{rcr} \lambda_1 + \lambda_2 + \lambda_3 & = & 0 \\ 2\lambda_3 & = & 0 \\ -\lambda_1 + \lambda_2 - \lambda_3 & = & 0 \end{array} \right. $$ De la deuxième équation, $\lambda_3 = 0$. Le système devient: $$ \left\{ \begin{array}{rcr} \lambda_1 + \lambda_2 & = & 0 \\ \lambda_3 & = & 0 \\ -\lambda_1 + \lambda_2 & = & 0 \end{array} \right. $$ Additionnons la première et la troisième ligne: $(\lambda_1+\lambda_2) + (-\lambda_1+\lambda_2) = 0 \implies 2\lambda_2 = 0 \implies \lambda_2 = 0$. Alors, avec $\lambda_1 + \lambda_2 = 0$, on a $\lambda_1 = 0$. La seule solution est $\lambda_1 = \lambda_2 = \lambda_3 = 0$. La famille $(v_1, v_2, v_3)$ est libre. Étant une famille libre de 3 vecteurs dans $\mathbb{R}^3$, c'est une base de $\mathbb{R}^3$.

Remarque: Pour un projecteur $p$, on a toujours $E = \text{Ker } p \oplus \text{Im } p$. La question 4 illustre ce résultat en montrant qu'une base du noyau et une base de l'image forment ensemble une base de l'espace total.
