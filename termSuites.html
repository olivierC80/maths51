Les Suites Numériques

Les suites numériques sont des objets mathématiques fondamentaux qui modélisent des processus évoluant par étapes discrètes. Elles apparaissent dans de nombreux domaines : économie (intérêts composés), biologie (évolution de populations), informatique (algorithmes récursifs), etc.

1. Définitions et Notations

Définition : Suite Numérique

Une suite numérique est une fonction $u$ définie sur l'ensemble des entiers naturels $\mathbb{N}$ (ou une partie de $\mathbb{N}$, souvent à partir d'un certain rang $n_0$) et à valeurs dans l'ensemble des nombres réels $\mathbb{R}$.

L'image de l'entier $n$ par la suite $u$ est notée $u_n$ (plutôt que $u(n)$) et est appelée le terme de rang $n$ (ou terme général) de la suite.

La suite elle-même est notée $(u_n)_{n \in \mathbb{N}}$ ou simplement $(u_n)$.

Le terme $u_0$ (ou $u_{n_0}$) est appelé le terme initial.

Modes de définition d'une suite

Une suite peut être définie de plusieurs manières :

  1. Forme explicite : Le terme général $u_n$ est donné directement en fonction de $n$.
    Exemple : $u_n = 3n^2 - 5$ pour $n \ge 0$. On peut calculer n'importe quel terme directement, par exemple $u_{10} = 3(10)^2 - 5 = 295$.
  2. Formule de récurrence : Le premier terme (ou les premiers termes) est donné, et une relation lie un terme au terme précédent (ou aux termes précédents).
    Exemple : $\begin{cases} u_0 = 2 \\ u_{n+1} = 2u_n + 1 \text{ pour } n \ge 0 \end{cases}$.
    Pour calculer un terme, il faut calculer tous les termes précédents : $u_1 = 2u_0+1 = 2(2)+1 = 5$, $u_2 = 2u_1+1 = 2(5)+1 = 11$, etc.
  3. De manière implicite : Le terme $u_n$ est défini comme une solution d'une équation dépendant de $n$, ou par une description. (Moins courant en Terminale pour la définition initiale).

2. Sens de Variation (Monotonie)

Définitions : Monotonie

Soit $(u_n)$ une suite définie pour $n \ge n_0$.

  • $(u_n)$ est croissante si pour tout $n \ge n_0$, $u_{n+1} \ge u_n$.
  • $(u_n)$ est strictement croissante si pour tout $n \ge n_0$, $u_{n+1} > u_n$.
  • $(u_n)$ est décroissante si pour tout $n \ge n_0$, $u_{n+1} \le u_n$.
  • $(u_n)$ est strictement décroissante si pour tout $n \ge n_0$, $u_{n+1} < u_n$.
  • $(u_n)$ est constante si pour tout $n \ge n_0$, $u_{n+1} = u_n$.
  • $(u_n)$ est monotone si elle est soit croissante, soit décroissante.

Méthodes pour étudier le sens de variation

Pour déterminer la monotonie d'une suite $(u_n)$ :

  1. Étudier le signe de la différence $u_{n+1} - u_n$ :
    • Si $u_{n+1} - u_n \ge 0$ pour tout $n$, la suite est croissante.
    • Si $u_{n+1} - u_n \le 0$ pour tout $n$, la suite est décroissante.
    • (Adapter pour la stricte monotonie avec des inégalités strictes).
    C'est la méthode la plus générale.
  2. Comparer le quotient $u_{n+1} / u_n$ à 1 (si la suite est strictement positive) :
    • Si $u_n > 0$ pour tout $n$ et $u_{n+1} / u_n \ge 1$, la suite est croissante.
    • Si $u_n > 0$ pour tout $n$ et $u_{n+1} / u_n \le 1$, la suite est décroissante.
    Utile notamment pour les suites géométriques ou avec des produits/puissances.
  3. Étudier la fonction associée (si $u_n = f(n)$) : Si $u_n = f(n)$ où $f$ est une fonction définie sur $[n_0, +\infty[$, alors le sens de variation de la fonction $f$ sur cet intervalle donne le sens de variation de la suite $(u_n)$. Attention : La réciproque est fausse. Une suite peut être monotone sans que la fonction associée le soit entre les entiers.
  4. Raisonnement par récurrence : Pour les suites définies par récurrence, on peut parfois prouver par récurrence que $u_{n+1} \ge u_n$ (ou une autre inégalité).

Exemple d'étude de monotonie

1. Soit $u_n = n^2 - n$ pour $n \ge 1$.

$u_{n+1} - u_n = [(n+1)^2 - (n+1)] - [n^2 - n]$

$= (n^2 + 2n + 1 - n - 1) - n^2 + n = n^2 + n - n^2 + n = 2n$.

Pour $n \ge 1$, $2n > 0$. Donc $u_{n+1} - u_n > 0$. La suite $(u_n)$ est strictement croissante.

2. Soit $v_n = \frac{2^n}{n!}$ pour $n \ge 1$. Les termes sont strictement positifs.

$\frac{v_{n+1}}{v_n} = \frac{2^{n+1}/(n+1)!}{2^n/n!} = \frac{2^{n+1}}{(n+1)!} \times \frac{n!}{2^n} = \frac{2^{n+1}}{2^n} \times \frac{n!}{(n+1)n!} = 2 \times \frac{1}{n+1} = \frac{2}{n+1}$.

Si $n=1$, $\frac{v_2}{v_1} = \frac{2}{2} = 1$.

Si $n \ge 2$, alors $n+1 \ge 3$, donc $\frac{2}{n+1} < 1$.

Donc $\frac{v_{n+1}}{v_n} < 1$ pour $n \ge 2$. La suite $(v_n)$ est strictement décroissante à partir du rang $n=2$. ($v_1=2, v_2=4/2=2, v_3=8/6=4/3$, etc.)

3. Suites Majorées, Minorées, Bornées

Définitions

Soit $(u_n)$ une suite numérique.

  • $(u_n)$ est majorée s'il existe un réel $M$ tel que pour tout $n$, $u_n \le M$. ($M$ est un majorant).
  • $(u_n)$ est minorée s'il existe un réel $m$ tel que pour tout $n$, $u_n \ge m$. ($m$ est un minorant).
  • $(u_n)$ est bornée si elle est à la fois majorée et minorée. Cela équivaut à dire qu'il existe un réel positif $K$ tel que pour tout $n$, $|u_n| \le K$.

Exemples

  • $u_n = 1/n$ pour $n \ge 1$. $0 < u_n \le 1$. La suite est minorée par 0 et majorée par 1. Elle est bornée.
  • $v_n = (-1)^n$. $-1 \le v_n \le 1$. La suite est bornée.
  • $w_n = n^2$. $w_n \ge 0$. La suite est minorée par 0, mais n'est pas majorée. Elle n'est pas bornée.

4. Limite d'une Suite

Définitions des Limites

  • Convergence vers $L \in \mathbb{R}$ : On dit que la suite $(u_n)$ converge vers $L$ si tout intervalle ouvert contenant $L$ contient tous les termes de la suite à partir d'un certain rang. On note $\lim_{n \to +\infty} u_n = L$.
    Formellement : $\forall \epsilon > 0, \exists N \in \mathbb{N}, \forall n \ge N, |u_n - L| < \epsilon$.
  • Divergence vers $+\infty$ : On dit que la suite $(u_n)$ diverge vers $+\infty$ si tout intervalle de la forme $]A, +\infty[$ (avec $A$ réel) contient tous les termes de la suite à partir d'un certain rang. On note $\lim_{n \to +\infty} u_n = +\infty$.
    Formellement : $\forall A \in \mathbb{R}, \exists N \in \mathbb{N}, \forall n \ge N, u_n > A$.
  • Divergence vers $-\infty$ : On dit que la suite $(u_n)$ diverge vers $-\infty$ si tout intervalle de la forme $]-\infty, B[$ (avec $B$ réel) contient tous les termes de la suite à partir d'un certain rang. On note $\lim_{n \to +\infty} u_n = -\infty$.
    Formellement : $\forall B \in \mathbb{R}, \exists N \in \mathbb{N}, \forall n \ge N, u_n < B$.
  • Divergence sans limite : Une suite qui ne converge pas et ne diverge pas vers $\pm\infty$ est dite divergente sans limite (ou oscillante). Exemple : $u_n = (-1)^n$.

Une suite qui admet une limite finie est dite convergente. Sinon, elle est dite divergente.

Limites Usuelles (à connaître)

  • $\lim_{n \to +\infty} n^p = +\infty$ si $p > 0$.
  • $\lim_{n \to +\infty} \frac{1}{n^p} = 0$ si $p > 0$.
  • Comportement de $q^n$ :
    • Si $q > 1$, $\lim_{n \to +\infty} q^n = +\infty$.
    • Si $q = 1$, $\lim_{n \to +\infty} q^n = 1$.
    • Si $-1 < q < 1$, $\lim_{n \to +\infty} q^n = 0$.
    • Si $q \le -1$, la suite $(q^n)$ n'a pas de limite (divergence sans limite).
  • $\lim_{n \to +\infty} \sqrt{n} = +\infty$.
  • $\lim_{n \to +\infty} \ln(n) = +\infty$.
  • $\lim_{n \to +\infty} e^n = +\infty$.
  • $\lim_{n \to +\infty} e^{-n} = 0$.
  • Croissances comparées : $e^n$ "l'emporte" sur $n^p$, qui "l'emporte" sur $\ln(n)$ en $+\infty$.

Opérations sur les Limites

Les règles sur les limites de sommes, produits et quotients de fonctions s'appliquent aux suites.

Attention aux Formes Indéterminées (FI) :

  • Somme : $+\infty - \infty$
  • Produit : $0 \times \infty$
  • Quotient : $\frac{0}{0}$, $\frac{\infty}{\infty}$

Pour lever une indétermination, on utilise souvent la factorisation par le terme dominant (pour les polynômes ou sommes/quotients de puissances/exponentielles).

Exemple de calcul de limite

1. $u_n = \frac{3n^2 - 5n + 1}{2n^2 + n + 4}$

FI $\frac{\infty}{\infty}$. On factorise par $n^2$ au numérateur et dénominateur :

$$u_n = \frac{n^2(3 - 5/n + 1/n^2)}{n^2(2 + 1/n + 4/n^2)} = \frac{3 - 5/n + 1/n^2}{2 + 1/n + 4/n^2}$$

Quand $n \to +\infty$, $1/n \to 0$, $1/n^2 \to 0$.

Donc $\lim_{n \to +\infty} u_n = \frac{3-0+0}{2+0+0} = \frac{3}{2}$.

2. $v_n = n - 2^n$.

FI $+\infty - \infty$. On factorise par le terme dominant, $2^n$ :

$$v_n = 2^n \left( \frac{n}{2^n} - 1 \right)$$

Par croissances comparées, $\lim_{n \to +\infty} \frac{n}{2^n} = 0$ (car $2^n$ croît plus vite que $n$).

Donc $\lim_{n \to +\infty} \left( \frac{n}{2^n} - 1 \right) = 0 - 1 = -1$.

Comme $\lim_{n \to +\infty} 2^n = +\infty$, on a (par produit des limites) :

$\lim_{n \to +\infty} v_n = (+\infty) \times (-1) = -\infty$.

5. Théorèmes de Convergence

Théorème des Gendarmes (ou d'Encadrement)

Soient $(u_n)$, $(v_n)$ et $(w_n)$ trois suites telles qu'à partir d'un certain rang $N$ :

$$ v_n \le u_n \le w_n $$

Si $\lim_{n \to +\infty} v_n = L$ et $\lim_{n \to +\infty} w_n = L$ (avec $L \in \mathbb{R}$), alors :

$$ \lim_{n \to +\infty} u_n = L $$

Exemple (Gendarmes)

Soit $u_n = \frac{\sin(n)}{n+1}$ pour $n \ge 0$.

On sait que $-1 \le \sin(n) \le 1$ pour tout $n$.

Comme $n+1 > 0$, on a : $\frac{-1}{n+1} \le \frac{\sin(n)}{n+1} \le \frac{1}{n+1}$.

Soit $v_n = \frac{-1}{n+1}$ et $w_n = \frac{1}{n+1}$.

On a $\lim_{n \to +\infty} v_n = 0$ et $\lim_{n \to +\infty} w_n = 0$.

Par le théorème des gendarmes, $\lim_{n \to +\infty} u_n = 0$.

Théorèmes de Comparaison

Soient $(u_n)$ et $(v_n)$ deux suites.

  • Si $u_n \ge v_n$ à partir d'un certain rang et $\lim_{n \to +\infty} v_n = +\infty$, alors $\lim_{n \to +\infty} u_n = +\infty$.
  • Si $u_n \le v_n$ à partir d'un certain rang et $\lim_{n \to +\infty} v_n = -\infty$, alors $\lim_{n \to +\infty} u_n = -\infty$.
  • Si $|u_n - L| \le v_n$ à partir d'un certain rang et $\lim_{n \to +\infty} v_n = 0$, alors $\lim_{n \to +\infty} u_n = L$.

Théorème de la Limite Monotone

  • Toute suite croissante et majorée est convergente (vers une limite $L \le M$, où $M$ est un majorant).
  • Toute suite décroissante et minorée est convergente (vers une limite $L \ge m$, où $m$ est un minorant).
  • Toute suite croissante et non majorée diverge vers $+\infty$.
  • Toute suite décroissante et non minorée diverge vers $-\infty$.

Exemple (Limite Monotone)

Soit la suite $(u_n)$ définie par $u_0 = 0$ et $u_{n+1} = \sqrt{u_n + 6}$.

On peut montrer par récurrence (voir section suivante) :

  1. Que $0 \le u_n < 3$ pour tout $n$. (La suite est bornée : minorée par 0, majorée par 3).
  2. Que la suite $(u_n)$ est croissante. (Étudier $u_{n+1}-u_n = \sqrt{u_n+6} - u_n$).

Comme $(u_n)$ est croissante et majorée (par 3), d'après le théorème de la limite monotone, elle converge vers une limite finie $L$.

Cette limite $L$ vérifie $0 \le L \le 3$.

Comme $u_{n+1} = f(u_n)$ avec $f(x) = \sqrt{x+6}$ qui est continue, la limite $L$ vérifie l'équation $L = f(L)$ :

$$ L = \sqrt{L+6} $$

En élevant au carré (possible car $L \ge 0$) :

$$ L^2 = L+6 \implies L^2 - L - 6 = 0 $$

Les solutions de cette équation du second degré sont $L = \frac{1 \pm \sqrt{1 - 4(1)(-6)}}{2} = \frac{1 \pm \sqrt{25}}{2} = \frac{1 \pm 5}{2}$.

Soit $L = 3$ ou $L = -2$.

Comme on sait que $L \ge 0$, la seule solution possible est $L=3$.

Conclusion : $\lim_{n \to +\infty} u_n = 3$.

6. Raisonnement par Récurrence

Principe du Raisonnement par Récurrence

Pour démontrer qu'une propriété $P(n)$ est vraie pour tout entier $n \ge n_0$ :

  1. Initialisation : Vérifier que la propriété $P(n_0)$ est vraie pour le premier rang $n_0$.
  2. Hérédité : Supposer que la propriété $P(k)$ est vraie pour un entier $k \ge n_0$ quelconque fixé (c'est l'hypothèse de récurrence). Démontrer, en utilisant cette hypothèse, que la propriété $P(k+1)$ est alors vraie.
  3. Conclusion : D'après le principe de récurrence, la propriété $P(n)$ est vraie pour tout entier $n \ge n_0$.

C'est un outil essentiel pour prouver des propriétés sur les suites définies par récurrence (monotonie, bornes, forme explicite...).

Exemple de Récurrence

Montrons par récurrence que pour la suite $u_0=0, u_{n+1}=\sqrt{u_n+6}$, on a $0 \le u_n < 3$ pour tout $n \ge 0$. Soit $P(n)$ la propriété "$0 \le u_n < 3$".

1. Initialisation (n=0) : $u_0 = 0$. On a bien $0 \le 0 < 3$. Donc $P(0)$ est vraie.

2. Hérédité : Supposons que $P(k)$ est vraie pour un entier $k \ge 0$, c'est-à-dire $0 \le u_k < 3$ (Hypothèse de Récurrence - HR).

On veut montrer que $P(k+1)$ est vraie, c'est-à-dire $0 \le u_{k+1} < 3$.

On part de l'HR : $0 \le u_k < 3$.

On ajoute 6 : $6 \le u_k + 6 < 9$.

On applique la fonction racine carrée (qui est croissante sur $\mathbb{R}^+$) :

$\sqrt{6} \le \sqrt{u_k + 6} < \sqrt{9}$.

$\sqrt{6} \le u_{k+1} < 3$.

Comme $\sqrt{6} > 0$, on a a fortiori $0 \le u_{k+1} < 3$.

Donc $P(k+1)$ est vraie.

3. Conclusion : Par le principe de récurrence, la propriété $P(n)$ est vraie pour tout entier $n \ge 0$.

7. Suites Arithmétiques et Géométriques

Suite Arithmétique

Une suite $(u_n)$ est arithmétique s'il existe un réel $r$ (la raison) tel que pour tout $n$ :

$$u_{n+1} = u_n + r$$
  • Terme général : $u_n = u_0 + nr$ ou $u_n = u_p + (n-p)r$.
  • Somme des termes : $S_n = u_0 + u_1 + \dots + u_n = (n+1) \frac{u_0 + u_n}{2}$.
    Plus généralement : $\text{Somme} = (\text{Nombre de termes}) \times \frac{\text{Premier terme} + \text{Dernier terme}}{2}$.
  • Limite : Si $r>0$, $\lim u_n = +\infty$ (si $u_0$ ou un terme est fini). Si $r<0$, $\lim u_n = -\infty$. Si $r=0$, la suite est constante $\lim u_n = u_0$.

Suite Géométrique

Une suite $(u_n)$ est géométrique s'il existe un réel $q$ (la raison) tel que pour tout $n$ :

$$u_{n+1} = q \times u_n$$
  • Terme général : $u_n = u_0 \times q^n$ ou $u_n = u_p \times q^{n-p}$.
  • Somme des termes (si $q \neq 1$) : $S_n = u_0 + u_1 + \dots + u_n = u_0 \frac{1 - q^{n+1}}{1 - q}$.
    Plus généralement : $\text{Somme} = (\text{Premier terme}) \times \frac{1 - \text{raison}^{\text{Nombre de termes}}}{1 - \text{raison}}$.
  • Somme des termes (si $q = 1$) : $S_n = (n+1) u_0$.
  • Limite (si $u_0 \neq 0$) :
    • Si $q > 1$, $\lim |u_n| = +\infty$ (le signe dépend de $u_0$).
    • Si $q = 1$, $\lim u_n = u_0$ (suite constante).
    • Si $-1 < q < 1$, $\lim u_n = 0$.
    • Si $q \le -1$, pas de limite (sauf si $u_0=0$).

Exemple (Suite Géométrique)

Soit $(u_n)$ la suite géométrique de premier terme $u_0=8$ et de raison $q=1/2$.

$u_n = 8 \times (1/2)^n$.

Calculons $S_4 = u_0+u_1+u_2+u_3+u_4$.

$S_4 = u_0 \frac{1 - q^{4+1}}{1 - q} = 8 \frac{1 - (1/2)^5}{1 - 1/2} = 8 \frac{1 - 1/32}{1/2} = 16 \left( \frac{31}{32} \right) = \frac{31}{2} = 15.5$.

Limite : Comme $-1 < q=1/2 < 1$, $\lim_{n \to +\infty} u_n = 0$.

Points Clés sur les Suites

  • Définition : Explicite ($u_n=f(n)$) ou par Récurrence ($u_{n+1}=f(u_n)$).
  • Variation : Étude de $u_{n+1}-u_n$ ou $u_{n+1}/u_n$.
  • Bornes : Majorée, minorée, bornée.
  • Limite : Convergence vers $L \in \mathbb{R}$, divergence vers $\pm\infty$, ou pas de limite.
  • Théorèmes importants : Gendarmes, Comparaison, Limite Monotone.
  • Outil essentiel : Raisonnement par Récurrence.
  • Suites particulières : Arithmétiques (somme $r$) et Géométriques (produit $q$). Connaître formules du terme général, somme et limites.

Conclusion

L'étude des suites numériques est une étape importante qui développe la rigueur du raisonnement (notamment avec la récurrence) et introduit les concepts fondamentaux de convergence et de limite. Les outils et théorèmes vus ici sont essentiels pour aborder des notions plus avancées en analyse.
