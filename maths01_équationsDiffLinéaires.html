Équations Différentielles Linéaires

Introduction

Une équation différentielle est une équation dont l'inconnue est une fonction et qui fait intervenir cette fonction et ses dérivées successives. Les équations différentielles linéaires sont un type particulièrement important, apparaissant dans de nombreux modèles en physique, biologie, économie, etc. Ce chapitre se concentre sur les équations linéaires d'ordre 1 et d'ordre 2 à coefficients constants.

Le corps de base $K$ sera $\mathbb{R}$ ou $\mathbb{C}$. $I$ désigne un intervalle de $\mathbb{R}$.

1. Équation Différentielle Linéaire d'Ordre 1

Définition (Équation différentielle linéaire d'ordre 1). Soit $I$ un intervalle de $\mathbb{R}$. Soient $a: I \to K$ et $b: I \to K$ deux fonctions continues. L'équation : $$ (E) \quad y' + a(x) y = b(x) $$ est appelée équation différentielle linéaire d'ordre 1.
  • Une solution de $(E)$ sur $I$ est une fonction $y: I \to K$ dérivable sur $I$ telle que pour tout $x \in I$, $y'(x) + a(x) y(x) = b(x)$.
  • L'équation homogène associée à $(E)$ est : $$ (E_H) \quad y' + a(x) y = 0 $$
  • Si $b(x) = 0$ pour tout $x$, l'équation $(E)$ est dite homogène.
Structure de l'ensemble des solutions. Soit $\mathcal{S}$ l'ensemble des solutions de l'équation complète $(E)$ et $\mathcal{S}_H$ l'ensemble des solutions de l'équation homogène $(E_H)$.
  1. L'ensemble $\mathcal{S}_H$ est un $K$-espace vectoriel de dimension 1.
  2. Si $\mathcal{S}$ est non vide (c'est-à-dire s'il existe au moins une solution particulière $y_P$ de $(E)$), alors : $$ \mathcal{S} = \{ y_P + y_H \mid y_H \in \mathcal{S}_H \} = y_P + \mathcal{S}_H $$
Pour résoudre $(E)$, il suffit donc de trouver la forme générale des solutions de $(E_H)$ et de trouver une solution particulière de $(E)$.
Résolution de l'équation homogène $(E_H)$. Soit $A: I \to K$ une primitive de la fonction continue $a$ sur $I$ (c'est-à-dire $A'(x) = a(x)$). Les solutions de l'équation homogène $y' + a(x) y = 0$ sur $I$ sont les fonctions de la forme : $$ y_H(x) = \lambda e^{-A(x)} $$ où $\lambda$ est une constante appartenant à $K$.
Idée de la preuve : Soit $y$ une solution de $y' + a(x)y = 0$. Multiplions l'équation par $e^{A(x)}$ (qui ne s'annule pas). $y'(x)e^{A(x)} + a(x)y(x)e^{A(x)} = 0$. Reconnaissons la dérivée d'un produit : $a(x)e^{A(x)}$ est la dérivée de $e^{A(x)}$ (car $A'(x)=a(x)$). L'équation devient $y'(x)e^{A(x)} + y(x)(e^{A(x)})' = 0$. C'est exactement la dérivée de $x \mapsto y(x)e^{A(x)}$. Donc $\frac{d}{dx}(y(x)e^{A(x)}) = 0$. Comme cette dérivée est nulle sur l'intervalle $I$, la fonction $x \mapsto y(x)e^{A(x)}$ est constante sur $I$. Il existe $\lambda \in K$ tel que $y(x)e^{A(x)} = \lambda$ pour tout $x \in I$. Donc $y(x) = \lambda e^{-A(x)}$. Réciproquement, on vérifie que toute fonction de cette forme est bien solution. $\square$
Méthode de Variation de la Constante (MVC). Pour trouver une solution particulière $y_P$ de l'équation complète $y' + a(x) y = b(x)$.
  1. Résoudre l'équation homogène associée $(E_H)$. La solution générale est $y_H(x) = \lambda e^{-A(x)}$, où $A$ est une primitive de $a$.
  2. Chercher une solution particulière de $(E)$ sous la forme $y_P(x) = \lambda(x) e^{-A(x)}$, où $\lambda$ n'est plus une constante mais une fonction dérivable $\lambda: I \to K$.
  3. Dériver $y_P$: $y_P'(x) = \lambda'(x) e^{-A(x)} + \lambda(x) (-A'(x) e^{-A(x)}) = \lambda'(x) e^{-A(x)} - \lambda(x) a(x) e^{-A(x)}$.
  4. Injecter $y_P$ et $y_P'$ dans l'équation complète $(E)$: $(\lambda'(x) e^{-A(x)} - \lambda(x) a(x) e^{-A(x)}) + a(x) (\lambda(x) e^{-A(x)}) = b(x)$.
  5. Simplifier : Les termes en $\lambda(x)$ s'annulent (c'est le principe de la méthode). Il reste : $\lambda'(x) e^{-A(x)} = b(x)$.
  6. Trouver $\lambda(x)$: $\lambda'(x) = b(x) e^{A(x)}$. Comme $b$ et $A$ sont continues ($a$ est continue, $A$ est $C^1$, $e^{A(x)}$ est continue), la fonction $x \mapsto b(x) e^{A(x)}$ est continue sur $I$. Elle admet donc des primitives. On choisit une primitive $\Lambda(x)$ de $b(x) e^{A(x)}$. On peut prendre $\lambda(x) = \Lambda(x)$.
  7. La solution particulière est $y_P(x) = \Lambda(x) e^{-A(x)}$.
  8. La solution générale de $(E)$ est $y(x) = y_P(x) + y_H(x) = (\Lambda(x) + C) e^{-A(x)}$, où $C$ est une constante quelconque.
Résoudre $(E) \quad y' - \frac{1}{x} y = x$ sur $I = ]0, +\infty[$. Ici $a(x) = -1/x$ et $b(x) = x$. Elles sont continues sur $I$.
  1. Équation homogène $(E_H): y' - \frac{1}{x} y = 0$. $a(x) = -1/x$. Une primitive est $A(x) = -\ln x$ (car $x>0$). $-A(x) = \ln x$. Les solutions de $(E_H)$ sont $y_H(x) = \lambda e^{\ln x} = \lambda x$, avec $\lambda \in \mathbb{R}$.
  2. Solution particulière (MVC). On cherche $y_P(x) = \lambda(x) x$. $y_P'(x) = \lambda'(x) x + \lambda(x)$. On injecte dans $(E)$: $(\lambda'(x) x + \lambda(x)) - \frac{1}{x} (\lambda(x) x) = x$. $\lambda'(x) x + \lambda(x) - \lambda(x) = x$. $\lambda'(x) x = x$. Comme $x \in ]0, +\infty[$, on a $\lambda'(x) = 1$. On choisit une primitive $\lambda(x) = x$. La solution particulière est $y_P(x) = \lambda(x) x = x \times x = x^2$.
  3. Solution générale. Les solutions de $(E)$ sur $]0, +\infty[$ sont $y(x) = y_P(x) + y_H(x) = x^2 + \lambda x$, avec $\lambda \in \mathbb{R}$.
Théorème de Cauchy-Lipschitz (Ordre 1). Soit $a, b: I \to K$ continues sur l'intervalle $I$. Pour tout $x_0 \in I$ et tout $y_0 \in K$, il existe une unique solution $y: I \to K$ de l'équation différentielle $y' + a(x) y = b(x)$ vérifiant la condition initiale $y(x_0) = y_0$.
Le système $\begin{cases} y' + a(x)y = b(x) \\ y(x_0) = y_0 \end{cases}$ est appelé un problème de Cauchy.
Idée de la preuve : La solution générale est $y(x) = (\Lambda(x) + C) e^{-A(x)}$, où $A$ est une primitive de $a$ et $\Lambda$ est une primitive de $b(x)e^{A(x)}$. La condition $y(x_0) = y_0$ impose : $(\Lambda(x_0) + C) e^{-A(x_0)} = y_0$. Comme $e^{-A(x_0)} \neq 0$, on peut trouver $C$: $\Lambda(x_0) + C = y_0 e^{A(x_0)}$, donc $C = y_0 e^{A(x_0)} - \Lambda(x_0)$. Cette valeur de $C$ est unique, donc la solution $y(x)$ est unique. L'existence est assurée par le fait qu'on a trouvé une solution de cette forme. $\square$

2. Équation Différentielle Linéaire d'Ordre 2 à Coefficients Constants

Définition (Équation différentielle linéaire d'ordre 2 à coefficients constants). Soient $a, b \in K$ des constantes et $f: I \to K$ une fonction continue sur un intervalle $I$. L'équation : $$ (E) \quad y'' + ay' + by = f(x) $$ est appelée équation différentielle linéaire d'ordre 2 à coefficients constants.
  • Une solution de $(E)$ sur $I$ est une fonction $y: I \to K$ deux fois dérivable sur $I$ telle que pour tout $x \in I$, $y''(x) + ay'(x) + by(x) = f(x)$.
  • L'équation homogène associée à $(E)$ est : $$ (E_H) \quad y'' + ay' + by = 0 $$
  • Si $f(x) = 0$ pour tout $x$, l'équation $(E)$ est dite homogène.
Structure de l'ensemble des solutions. Soit $\mathcal{S}$ l'ensemble des solutions de $(E)$ et $\mathcal{S}_H$ l'ensemble des solutions de $(E_H)$.
  1. L'ensemble $\mathcal{S}_H$ est un $K$-espace vectoriel de dimension 2.
  2. Si $\mathcal{S}$ est non vide (c'est-à-dire s'il existe une solution particulière $y_P$ de $(E)$), alors : $$ \mathcal{S} = \{ y_P + y_H \mid y_H \in \mathcal{S}_H \} = y_P + \mathcal{S}_H $$
La stratégie est donc : 1. Trouver la solution générale de $(E_H)$. 2. Trouver une solution particulière de $(E)$.
Équation caractéristique. L'équation polynomiale du second degré en $r$ : $$ (EC) \quad r^2 + ar + b = 0 $$ est appelée équation caractéristique de l'équation différentielle homogène $(E_H): y'' + ay' + by = 0$.
Résolution de l'équation homogène $(E_H)$ dans $\mathbb{C}$. Soit $(EC): r^2 + ar + b = 0$ l'équation caractéristique (à coefficients dans $K=\mathbb{C}$).
  1. Si $(EC)$ admet deux racines distinctes $r_1, r_2 \in \mathbb{C}$, alors les solutions de $(E_H)$ sur $\mathbb{R}$ (ou $\mathbb{C}$) sont les fonctions de la forme : $$ y_H(x) = \lambda e^{r_1 x} + \mu e^{r_2 x} $$ où $\lambda, \mu \in \mathbb{C}$.
  2. Si $(EC)$ admet une racine double $r \in \mathbb{C}$ (c'est-à-dire $a^2-4b=0$ et $r=-a/2$), alors les solutions de $(E_H)$ sur $\mathbb{R}$ (ou $\mathbb{C}$) sont les fonctions de la forme : $$ y_H(x) = (\lambda x + \mu) e^{r x} $$ où $\lambda, \mu \in \mathbb{C}$.
Dans les deux cas, l'ensemble des solutions $\mathcal{S}_H$ est un $\mathbb{C}$-espace vectoriel de dimension 2. Une base est $(e^{r_1 x}, e^{r_2 x})$ dans le cas 1, et $(e^{r x}, x e^{r x})$ dans le cas 2.
Résolution de l'équation homogène $(E_H)$ dans $\mathbb{R}$. Soit $(E_H): y'' + ay' + by = 0$ avec $a, b \in \mathbb{R}$. Soit $(EC): r^2 + ar + b = 0$ l'équation caractéristique et $\Delta = a^2 - 4b$ son discriminant. On cherche les solutions réelles $y: I \to \mathbb{R}$.
  1. Cas $\Delta > 0$: $(EC)$ admet deux racines réelles distinctes $r_1, r_2 = \frac{-a \pm \sqrt{\Delta}}{2}$. Les solutions réelles de $(E_H)$ sont : $$ y_H(x) = \lambda e^{r_1 x} + \mu e^{r_2 x} $$ où $\lambda, \mu \in \mathbb{R}$.
  2. Cas $\Delta = 0$: $(EC)$ admet une racine réelle double $r = -a/2$. Les solutions réelles de $(E_H)$ sont : $$ y_H(x) = (\lambda x + \mu) e^{r x} $$ où $\lambda, \mu \in \mathbb{R}$.
  3. Cas $\Delta < 0$: $(EC)$ admet deux racines complexes conjuguées $r_{1,2} = \alpha \pm i\beta$, où $\alpha = -a/2$ et $\beta = \sqrt{-\Delta}/2$ ($\beta > 0$). Les solutions réelles de $(E_H)$ sont : $$ y_H(x) = e^{\alpha x} (\lambda \cos(\beta x) + \mu \sin(\beta x)) $$ où $\lambda, \mu \in \mathbb{R}$. (On peut aussi écrire $y_H(x) = C e^{\alpha x} \cos(\beta x - \phi)$ avec $C, \phi$ des constantes réelles).
Dans les trois cas, l'ensemble des solutions réelles $\mathcal{S}_H$ est un $\mathbb{R}$-espace vectoriel de dimension 2.
Lien entre cas réel $\Delta<0$ et cas complexe. Si les racines complexes sont $r_1 = \alpha + i\beta$ et $r_2 = \alpha - i\beta$. La solution générale complexe est $y(x) = C_1 e^{(\alpha+i\beta)x} + C_2 e^{(\alpha-i\beta)x}$ ($C_1, C_2 \in \mathbb{C}$). $y(x) = e^{\alpha x} (C_1 e^{i\beta x} + C_2 e^{-i\beta x})$ $y(x) = e^{\alpha x} (C_1(\cos \beta x + i\sin \beta x) + C_2(\cos \beta x - i\sin \beta x))$ $y(x) = e^{\alpha x} ((C_1+C_2)\cos \beta x + i(C_1-C_2)\sin \beta x)$. Pour que la solution $y(x)$ soit réelle pour tout $x$, il faut que les coefficients devant $\cos$ et $\sin$ soient réels. Posons $\lambda = C_1+C_2$ et $\mu = i(C_1-C_2)$. $y(x)$ est réelle $\iff \lambda \in \mathbb{R}$ et $\mu \in \mathbb{R}$. Cela correspond à $\overline{C_1+C_2} = C_1+C_2$ et $\overline{i(C_1-C_2)} = i(C_1-C_2)$. La deuxième condition donne $-i(\bar{C_1}-\bar{C_2}) = i(C_1-C_2)$, soit $\bar{C_2}-\bar{C_1} = C_1-C_2$. En posant $C_1 = u+iv$ et $C_2 = z+iw$, on trouve que cela équivaut à $C_2 = \bar{C_1}$. Si $C_2 = \bar{C_1}$, alors $\lambda = C_1+\bar{C_1} = 2 \text{Re}(C_1) \in \mathbb{R}$ et $\mu = i(C_1-\bar{C_1}) = i(2i \text{Im}(C_1)) = -2 \text{Im}(C_1) \in \mathbb{R}$. Donc les solutions réelles sont obtenues en choisissant $C_2=\bar{C_1}$, et elles s'écrivent bien $e^{\alpha x}(\lambda \cos(\beta x) + \mu \sin(\beta x))$ avec $\lambda, \mu \in \mathbb{R}$. $\square$
Théorème de Cauchy-Lipschitz (Ordre 2). Soient $a, b \in K$ des constantes et $f: I \to K$ une fonction continue sur l'intervalle $I$. Pour tout $x_0 \in I$ et tous $y_0, y_1 \in K$, il existe une unique solution $y: I \to K$ de l'équation différentielle $y'' + ay' + by = f(x)$ vérifiant les conditions initiales $y(x_0) = y_0$ et $y'(x_0) = y_1$.
Recherche d'une solution particulière $y_P$. Contrairement au premier ordre, il n'y a pas de méthode générale simple comme la MVC (bien qu'une version existe pour l'ordre 2, elle est plus complexe). On cherche souvent une solution particulière de la même "forme" que le second membre $f(x)$, notamment dans les cas suivants :
  • Principe de superposition : Si $f(x) = f_1(x) + f_2(x)$, et si $y_{P1}$ est une solution particulière pour le second membre $f_1$, et $y_{P2}$ pour $f_2$, alors $y_P = y_{P1} + y_{P2}$ est une solution particulière pour le second membre $f$.
  • Second membre polynomial : Si $f(x)$ est un polynôme de degré $d$, on cherche $y_P(x)$ sous forme d'un polynôme.
    • Si $b \neq 0$, on cherche $y_P$ de degré $d$.
    • Si $b=0$ et $a \neq 0$, on cherche $y_P$ de degré $d+1$.
    • Si $b=a=0$, on cherche $y_P$ de degré $d+2$.
  • Second membre exponentiel-polynôme : Si $f(x) = P(x) e^{kx}$ où $P$ est un polynôme de degré $d$ et $k \in K$. On cherche $y_P(x) = Q(x) e^{kx}$ où $Q$ est un polynôme.
    • Si $k$ n'est pas racine de l'équation caractéristique $(EC)$, on cherche $Q$ de degré $d$.
    • Si $k$ est racine simple de $(EC)$, on cherche $Q$ de degré $d+1$.
    • Si $k$ est racine double de $(EC)$, on cherche $Q$ de degré $d+2$.
  • Second membre trigonométrique (cas réel) : Si $f(x) = P(x) e^{\alpha x} \cos(\beta x)$ ou $P(x) e^{\alpha x} \sin(\beta x)$. On passe en complexe en considérant le second membre $g(x) = P(x) e^{(\alpha+i\beta)x}$. On cherche une solution particulière complexe $y_{P, \mathbb{C}}$ de $y''+ay'+by=g(x)$ par la méthode précédente (avec $k=\alpha+i\beta$). La solution particulière réelle $y_P$ pour $f(x)$ sera alors $y_P(x) = \text{Re}(y_{P, \mathbb{C}})$ si $f(x)$ contient $\cos$, ou $y_P(x) = \text{Im}(y_{P, \mathbb{C}})$ si $f(x)$ contient $\sin$.
Ces méthodes consistent à injecter la forme supposée de $y_P$ dans $(E)$ et à identifier les coefficients du polynôme $Q(x)$.
Résoudre $(E) \quad y'' - 3y' + 2y = e^{3x}$.
  1. Équation homogène $(E_H): y'' - 3y' + 2y = 0$. Équation caractéristique : $r^2 - 3r + 2 = 0$. Racines : $r = \frac{3 \pm \sqrt{9-8}}{2} = \frac{3 \pm 1}{2}$. $r_1 = 1, r_2 = 2$. Solution homogène réelle : $y_H(x) = \lambda e^x + \mu e^{2x}$ ($\lambda, \mu \in \mathbb{R}$).
  2. Solution particulière. Le second membre est $f(x) = 1 \cdot e^{3x}$. $k=3$. $P(x)=1$ (degré 0). Vérifions si $k=3$ est racine de $r^2-3r+2=0$. $3^2 - 3(3) + 2 = 9-9+2 = 2 \neq 0$. Donc 3 n'est pas racine. On cherche $y_P(x) = Q(x) e^{3x}$ avec $\deg Q = \deg P = 0$. Soit $Q(x) = C$ (une constante). $y_P(x) = C e^{3x}$. $y_P'(x) = 3C e^{3x}$. $y_P''(x) = 9C e^{3x}$. On injecte dans $(E)$: $(9C e^{3x}) - 3(3C e^{3x}) + 2(C e^{3x}) = e^{3x}$. $9C e^{3x} - 9C e^{3x} + 2C e^{3x} = e^{3x}$. $2C e^{3x} = e^{3x}$. Ceci doit être vrai pour tout $x$, donc $2C=1 \implies C = 1/2$. La solution particulière est $y_P(x) = \frac{1}{2} e^{3x}$.
  3. Solution générale. Les solutions réelles de $(E)$ sont $y(x) = y_P(x) + y_H(x) = \frac{1}{2} e^{3x} + \lambda e^x + \mu e^{2x}$ ($\lambda, \mu \in \mathbb{R}$).
Résoudre $(E) \quad y'' + y = x$.
  1. Équation homogène $(E_H): y'' + y = 0$. Équation caractéristique : $r^2 + 1 = 0$. Racines : $r = \pm i$. Cas $\Delta < 0$ avec $\alpha = 0, \beta = 1$. Solution homogène réelle : $y_H(x) = e^{0x}(\lambda \cos(1x) + \mu \sin(1x)) = \lambda \cos x + \mu \sin x$ ($\lambda, \mu \in \mathbb{R}$).
  2. Solution particulière. Le second membre est $f(x) = x$, un polynôme de degré $d=1$. Ici $a=0, b=1$. $b \neq 0$. On cherche $y_P(x)$ comme un polynôme de degré $d=1$. Soit $y_P(x) = Ax + B$. $y_P'(x) = A$. $y_P''(x) = 0$. On injecte dans $(E)$: $0 + (Ax+B) = x$. Pour que ceci soit vrai pour tout $x$, on identifie les coefficients des polynômes : Coefficient de $x$: $A = 1$. Coefficient constant: $B = 0$. La solution particulière est $y_P(x) = 1x + 0 = x$.
  3. Solution générale. Les solutions réelles de $(E)$ sont $y(x) = y_P(x) + y_H(x) = x + \lambda \cos x + \mu \sin x$ ($\lambda, \mu \in \mathbb{R}$).
