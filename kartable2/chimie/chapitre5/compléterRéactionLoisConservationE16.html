Thème : Réactions Nucléaires

Fiche : Compléter une équation de réaction nucléaire (Lois de conservation)

Instructions : Utilisez les lois de conservation de Soddy (conservation du nombre de masse A et du numéro atomique Z) pour trouver le noyau ou la particule manquant(e), ou le coefficient stœchiométrique correct dans les équations de réactions nucléaires suivantes (désintégration, fission, fusion).

Question 1 : Complétez l'équation de cette réaction de fission :

23592U + 10n → 9438Sr + ??? + 2 10n
  • a) 14054Xe
  • b) 13954Xe
  • c) 14055Cs
  • d) 14154Xe

Bonne réponse : a) 14054Xe

23592U + 10n → 9438Sr + 14054Xe + 2 10n

Précisions : Soit A'Z'Y le produit inconnu. Réactifs : A_total = 235 + 1 = 236 ; Z_total = 92 + 0 = 92. Produits : A = 94 + A' + 2*1 ; Z = 38 + Z' + 2*0.

  • Conservation de Z : 92 = 38 + Z' ⇒ Z' = 92 - 38 = 54. (Élément Xénon, Xe)
  • Conservation de A : 236 = 94 + A' + 2 ⇒ A' = 236 - 94 - 2 = 140.
Le produit inconnu est 14054Xe.

Question 2 : Le Carbone 11 (116C) est un émetteur β⁺. Quel noyau est produit ?

116C → ??? + 0+1e
  • a) 117N
  • b) 105B
  • c) 115B
  • d) 126C

Bonne réponse : c) 115B

116C → 115B + 0+1e (+ νₑ)

Précisions : Désintégration β⁺ : émission d'un positon 0+1e. Soit A'Z'Y le noyau fils. Réactif : A = 11 ; Z = 6. Produits : A = A' + 0 ; Z = Z' + 1.

  • Conservation de A : 11 = A' + 0 ⇒ A' = 11.
  • Conservation de Z : 6 = Z' + 1 ⇒ Z' = 6 - 1 = 5. (Élément Bore, B)
Le noyau produit est 115B.

Question 3 : Identifiez la particule X dans cette réaction de fusion :

21H + 21H → 31H + X
  • a) Neutron (10n)
  • b) Proton (11p)
  • c) Particule Alpha (42He)
  • d) Positon (0+1e)

Bonne réponse : b) Proton (11p)

21H + 21H → 31H + 11p

Précisions : Soit A'Z'X la particule inconnue. Réactifs : A_total = 2 + 2 = 4 ; Z_total = 1 + 1 = 2. Produits : A = 3 + A' ; Z = 1 + Z'.

  • Conservation de A : 4 = 3 + A' ⇒ A' = 1.
  • Conservation de Z : 2 = 1 + Z' ⇒ Z' = 1.
La particule X est 11X, ce qui correspond au proton (11p ou 11H).

Question 4 : L'Américium 241 (24195Am), utilisé dans les détecteurs de fumée, est radioactif α. Quel est le noyau fils ?

24195Am → ??? + 42He
  • a) 23793Np
  • b) 24193Np
  • c) 23797Bk
  • d) 23794Pu

Bonne réponse : a) 23793Np

24195Am → 23793Np + 42He

Précisions : Désintégration α : émission d'une particule alpha 42He. Soit A'Z'Y le noyau fils. Réactif : A = 241 ; Z = 95. Produits : A = A' + 4 ; Z = Z' + 2.

  • Conservation de A : 241 = A' + 4 ⇒ A' = 241 - 4 = 237.
  • Conservation de Z : 95 = Z' + 2 ⇒ Z' = 95 - 2 = 93. (Élément Neptunium, Np)
Le noyau produit est 23793Np.

Question 5 : Déterminez le nombre 'k' de neutrons émis dans cette fission :

23994Pu + 10n → 13552Te + 10242Mo + k 10n
  • a) k = 1
  • b) k = 2
  • c) k = 3
  • d) k = 4

Bonne réponse : c) k = 3

23994Pu + 10n → 13552Te + 10242Mo + 3 10n

Précisions : Réactifs : A_total = 239 + 1 = 240 ; Z_total = 94 + 0 = 94. Produits : A = 135 + 102 + k*1 ; Z = 52 + 42 + k*0.

  • Vérification Z : 94 = 52 + 42 + 0 ⇒ 94 = 94. (OK)
  • Conservation de A : 240 = 135 + 102 + k ⇒ 240 = 237 + k ⇒ k = 240 - 237 = 3.
Il y a 3 neutrons émis.
