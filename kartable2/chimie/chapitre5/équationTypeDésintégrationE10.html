Thème : Modélisation temporelle d’un système nucléaire

Fiche : Établir l'équation de désintégration radioactive

Instructions : Pour chaque noyau père et type de désintégration donnés, choisissez l'équation bilan correcte en appliquant les lois de conservation de Soddy (conservation de A et de Z).

Question 1 : L'Uranium 238 (23892U) subit une désintégration alpha (α). Quelle est l'équation correcte ?

  • a)
    23892U → 23490Th + 42He
  • b)
    23892U → 23890Th + 02?
  • c)
    23892U → 23492U + 40n
  • d)
    23892U → 23488Ra + 44Be

Bonne réponse : a)

23892U → 23490Th + 42He

Précisions : La désintégration alpha (α) correspond à l'émission d'un noyau d'hélium 42He. Soit A'Z'Y le noyau fils. L'équation est : 23892U → A'Z'Y + 42He.

  • Conservation de A : 238 = A' + 4 ⇒ A' = 238 - 4 = 234
  • Conservation de Z : 92 = Z' + 2 ⇒ Z' = 92 - 2 = 90
Le noyau fils a A'=234 et Z'=90. L'élément de Z=90 est le Thorium (Th). L'équation est donc bien celle de l'option a).

Question 2 : Le Cobalt 60 (6027Co) est un émetteur bêta moins (β⁻). Quelle équation décrit sa désintégration ? (On omet l'antineutrino)

  • a)
    6027Co → 6028Ni + 0-1e
  • b)
    6027Co → 6026Fe + 0+1e
  • c)
    6027Co → 5927Co + 10n
  • d)
    6027Co → 6027Co + γ

Bonne réponse : a)

6027Co → 6028Ni + 0-1e

Précisions : La désintégration bêta moins (β⁻) correspond à l'émission d'un électron 0-1e. Soit A'Z'Y le noyau fils. L'équation est : 6027Co → A'Z'Y + 0-1e.

  • Conservation de A : 60 = A' + 0 ⇒ A' = 60
  • Conservation de Z : 27 = Z' + (-1) ⇒ Z' = 27 - (-1) = 27 + 1 = 28
Le noyau fils a A'=60 et Z'=28. L'élément de Z=28 est le Nickel (Ni). L'équation correcte est celle de l'option a).

Question 3 : Le Fluor 18 (189F), utilisé en imagerie médicale (TEP), se désintègre par émission bêta plus (β⁺). Quelle est l'équation correspondante ? (On omet le neutrino)

  • a)
    189F → 1810Ne + 0-1e
  • b)
    189F → 178O + 11p
  • c)
    189F → 188O + 0+1e
  • d)
    189F → 147N + 42He

Bonne réponse : c)

189F → 188O + 0+1e

Précisions : La désintégration bêta plus (β⁺) correspond à l'émission d'un positon 0+1e. Soit A'Z'Y le noyau fils. L'équation est : 189F → A'Z'Y + 0+1e.

  • Conservation de A : 18 = A' + 0 ⇒ A' = 18
  • Conservation de Z : 9 = Z' + (+1) ⇒ Z' = 9 - 1 = 8
Le noyau fils a A'=18 et Z'=8. L'élément de Z=8 est l'Oxygène (O). L'équation correcte est celle de l'option c).

Question 4 : Le Strontium 90 (9038Sr), un produit de fission dangereux, est radioactif β⁻. Écrire l'équation de sa désintégration.

  • a)
    9038Sr → 9039Y + 0-1e
  • b)
    9038Sr → 9037Rb + 0+1e
  • c)
    9038Sr → 8636Kr + 42He
  • d)
    9038Sr → 8938Sr + 10n

Bonne réponse : a)

9038Sr → 9039Y + 0-1e

Précisions : La désintégration β⁻ implique l'émission d'un électron 0-1e. Soit A'Z'Y le noyau fils. L'équation est : 9038Sr → A'Z'Y + 0-1e.

  • Conservation de A : 90 = A' + 0 ⇒ A' = 90
  • Conservation de Z : 38 = Z' + (-1) ⇒ Z' = 38 - (-1) = 38 + 1 = 39
Le noyau fils a A'=90 et Z'=39. L'élément de Z=39 est l'Yttrium (Y). L'équation correcte est donc celle de l'option a).
