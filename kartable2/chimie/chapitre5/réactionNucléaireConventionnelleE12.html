Thème : Réactions Nucléaires

Fiche : Établir l'écriture d'une réaction de fusion

Instructions : Pour chaque paire de noyaux légers qui fusionnent, déterminez l'équation complète de la réaction en utilisant les lois de conservation de Soddy (conservation de A et Z) pour identifier les produits. Notez qu'une particule (comme un neutron ou un proton) peut aussi être émise.

Question 1 : Deux noyaux de Deutérium (21H) fusionnent pour former un noyau d'Hélium 3 (32He) et une autre particule. Quelle est l'équation correcte ?

  • a)
    21H + 21H → 32He + 10n
  • b)
    21H + 21H → 32He + 11p
  • c)
    21H + 21H → 32He + 0-1e
  • d)
    21H + 21H → 42He

Bonne réponse : a)

21H + 21H → 32He + 10n

Précisions : Réactifs : 21H + 21H. Somme A = 2+2=4, Somme Z = 1+1=2. Produits : 32He + A'Z'X (particule inconnue).

  • Conservation de A : 4 = 3 + A' ⇒ A' = 4 - 3 = 1
  • Conservation de Z : 2 = 2 + Z' ⇒ Z' = 2 - 2 = 0
La particule inconnue est 10X. Cette notation correspond au neutron (10n). L'équation correcte est donc celle de l'option a). (Note : La fusion D+D peut aussi produire 31H + 11p, mais cette option n'était pas proposée ici sous la forme correcte).

Question 2 : La fusion du Deutérium (21H) et du Tritium (31H) produit un noyau d'Hélium et un neutron. Quelle est l'écriture conventionnelle de cette réaction ?

  • a)
    21H + 31H → 42He + 10n
  • b)
    21H + 31H → 52He
  • c)
    21H + 31H → 32He + 20n
  • d)
    21H + 31H → 41H + 11p

Bonne réponse : a)

21H + 31H → 42He + 10n

Précisions : Réactifs : 21H + 31H. Somme A = 2+3=5, Somme Z = 1+1=2. Produits annoncés : Noyau d'Hélium (A'2He) + Neutron (10n).

  • Conservation de A : 5 = A' + 1 ⇒ A' = 5 - 1 = 4
  • Conservation de Z : 2 = 2 + 0 ⇒ Z=2 est bien conservé.
Le noyau d'Hélium produit est donc l'Hélium 4 (42He). L'équation complète est celle de l'option a), qui est la réaction de fusion D-T très étudiée.

Question 3 : Dans certaines étoiles, un noyau de proton (11H) peut fusionner avec un noyau de Lithium 7 (73Li) pour produire deux particules identiques. Quelle est l'équation de cette réaction ?

  • a)
    11H + 73Li → 84Be
  • b)
    11H + 73Li → 42He + 42He
  • c)
    11H + 73Li → 2 42? + e⁻
  • d)
    11H + 73Li → 74Be + 10n

Bonne réponse : b)

11H + 73Li → 42He + 42He

Précisions : Réactifs : 11H + 73Li. Somme A = 1+7=8, Somme Z = 1+3=4. Produits : Deux particules identiques A'Z'X. L'équation est : 11H + 73Li → 2 A'Z'X.

  • Conservation de A : 8 = 2 * A' ⇒ A' = 8 / 2 = 4
  • Conservation de Z : 4 = 2 * Z' ⇒ Z' = 4 / 2 = 2
Les deux particules identiques sont 42X. L'élément de Z=2 est l'Hélium (He). Il s'agit donc de deux noyaux d'Hélium 4 (particules alpha). L'équation est celle de l'option b). L'option a) (84Be) est un intermédiaire très instable qui se désintègre quasi instantanément en deux alphas.

Question 4 : La fusion de deux noyaux d'Hélium 3 (32He) peut produire un noyau d'Hélium 4 (42He) et deux autres particules identiques. Identifiez ces particules et écrivez l'équation.

  • a)
    32He + 32He → 42He + 2 11p
  • b)
    32He + 32He → 42He + 2 10n
  • c)
    32He + 32He → 42He + 22He
  • d)
    32He + 32He → 64Be

Bonne réponse : a)

32He + 32He → 42He + 2 11p

Précisions : Réactifs : 32He + 32He. Somme A = 3+3=6, Somme Z = 2+2=4. Produits : 42He + 2 A'Z'X (deux particules identiques).

  • Conservation de A : 6 = 4 + 2 * A' ⇒ 2 * A' = 6 - 4 = 2 ⇒ A' = 1
  • Conservation de Z : 4 = 2 + 2 * Z' ⇒ 2 * Z' = 4 - 2 = 2 ⇒ Z' = 1
Les deux particules identiques sont 11X. La particule avec A=1 et Z=1 est le proton (11p ou 11H). L'équation correcte est donc celle de l'option a). (Cette réaction fait partie de la chaîne proton-proton dans le Soleil).
