Thème : Modélisation temporelle d’un système nucléaire

Fiche : Calculer le temps de désintégration (t) à l'aide de N₀, N(t) et λ

En connaissant le nombre initial de noyaux radioactifs (N₀), le nombre de noyaux restants à un instant t (N(t)) et la constante radioactive (λ), on peut déterminer le temps t écoulé en utilisant la loi de décroissance radioactive N(t) = N₀ e-λt et la fonction logarithme népérien (ln).

Question 1 : À partir de la loi de décroissance N(t) = N₀ e-λt, quelle est l'expression correcte pour calculer le temps t ?

  • a) t = λ × ln(N₀ / N(t))
  • b) t = (1 / λ) × ln(N₀ / N(t))
  • c) t = (1 / λ) × ln(N(t) / N₀)
  • d) t = e(λ N₀ / N(t))

Bonne réponse : b) t = (1 / λ) × ln(N₀ / N(t))

Précisions : Démonstration :

  1. Partir de N(t) = N₀ e-λt.
  2. Isoler l'exponentielle : N(t) / N₀ = e-λt.
  3. Prendre le logarithme népérien (ln) des deux côtés : ln(N(t) / N₀) = ln(e-λt) = -λt.
  4. Utiliser la propriété ln(a/b) = -ln(b/a) : -ln(N₀ / N(t)) = -λt.
  5. Simplifier les signes moins : ln(N₀ / N(t)) = λt.
  6. Isoler t : t = ln(N₀ / N(t)) / λ, ce qui est équivalent à t = (1 / λ) × ln(N₀ / N(t)).
(Note : l'option c donnerait un temps négatif car N(t) < N₀, donc ln(N(t)/N₀) < 0).

Question 2 : Dans la formule t = (1 / λ) × ln(N₀ / N(t)), si λ est exprimée en an⁻¹, quelle sera l'unité du temps t calculé ?

  • a) an⁻¹
  • b) an (années)
  • c) s (secondes)
  • d) Sans unité

Bonne réponse : b) an (années)

Précisions : Le terme ln(N₀ / N(t)) est un nombre sans dimension (logarithme d'un rapport). La constante λ a la dimension de l'inverse d'un temps [T⁻¹]. Le calcul t = (1 / λ) × NombreSansDimension donne donc une grandeur t qui a la dimension de 1 / [T⁻¹] = [T], c'est-à-dire un temps. Si l'unité de λ est **an⁻¹**, alors l'unité de t sera l'**année (an)**.

Question 3 : Un échantillon contient initialement N₀ = 6,0 x 10¹⁰ noyaux radioactifs. Au bout d'un certain temps t, il n'en reste que N(t) = 1,5 x 10¹⁰. La constante radioactive est λ = 0,050 min⁻¹. Calculez le temps t écoulé. (On donne ln(4) ≈ 1,386).

  • a) Environ 13,9 min
  • b) Environ 27,7 min
  • c) Environ 0,036 min
  • d) Environ 40,0 min

Bonne réponse : b) Environ 27,7 min

Précisions : 1. Calculer le rapport N₀ / N(t) : N₀ / N(t) = (6,0 x 10¹⁰) / (1,5 x 10¹⁰) = 6,0 / 1,5 = 4. 2. Calculer le logarithme : ln(N₀ / N(t)) = ln(4) ≈ 1,386. 3. Appliquer la formule : t = ln(N₀ / N(t)) / λ t ≈ 1,386 / (0,050 min⁻¹). 4. Calculer t : t ≈ 27,72 min. L'unité de λ étant min⁻¹, le temps t est obtenu en **minutes**. La valeur la plus proche est 27,7 min. (Remarque : On voit que N(t) = N₀/4, donc il s'est écoulé 2 demi-vies. t₁/₂ = ln(2)/λ ≈ 0.693/0.050 ≈ 13.86 min. t = 2 * t₁/₂ ≈ 2 * 13.86 ≈ 27.72 min).

Question 4 : L'activité initiale d'un échantillon de Césium 137 est A₀ = 500 Bq. Sa constante radioactive est λ = 7,3 x 10⁻¹⁰ s⁻¹. Combien de temps faudra-t-il pour que son activité tombe à A(t) = 50 Bq ? (On donne ln(10) ≈ 2,30).

  • a) Environ 3,15 x 10⁸ s (≈ 10 ans)
  • b) Environ 7,3 x 10⁹ s (≈ 230 ans)
  • c) Environ 9,47 x 10⁸ s (≈ 30 ans)
  • d) Environ 3,15 x 10⁹ s (≈ 100 ans)

Bonne réponse : d) Environ 3,15 x 10⁹ s (≈ 100 ans)

Précisions : La loi de décroissance s'applique aussi à l'activité : A(t) = A₀ e-λt. La formule pour calculer le temps est donc la même en remplaçant N par A : t = ln(A₀ / A(t)) / λ. 1. Calculer le rapport A₀ / A(t) : A₀ / A(t) = 500 Bq / 50 Bq = 10. 2. Calculer le logarithme : ln(A₀ / A(t)) = ln(10) ≈ 2,30. 3. Appliquer la formule : t = ln(A₀ / A(t)) / λ t ≈ 2,30 / (7,3 x 10⁻¹⁰ s⁻¹). 4. Calculer t : t ≈ 0,315 x 10¹⁰ s = 3,15 x 10⁹ s. (Conversion en années : t ≈ (3,15 x 10⁹ s) / (3,15 x 10⁷ s/an) ≈ 100 ans).

Question 5 : Un archéologue trouve un ossement dont le rapport 14C / 12C est 1/8 de celui d'un organisme vivant actuel. Sachant que λ(14C) ≈ 1,21 x 10⁻⁴ an⁻¹, quel est l'âge approximatif de l'ossement ? (On donne ln(8) ≈ 2,08).

  • a) Environ 5730 ans
  • b) Environ 11460 ans
  • c) Environ 17190 ans
  • d) Environ 22920 ans

Bonne réponse : c) Environ 17190 ans

Précisions : Dans la datation au Carbone 14, on suppose que le nombre initial de noyaux N₀ est proportionnel à la quantité de Carbone total (principalement 12C) et que le rapport 14C/12C initial est le même que celui des organismes vivants actuels. Le rapport actuel 14C/12C dans l'échantillon est donc proportionnel à N(t)/N₀. Si le rapport est 1/8 de celui d'origine, cela signifie que N(t) / N₀ = 1/8. Par conséquent, N₀ / N(t) = 8. On applique la formule : t = ln(N₀ / N(t)) / λ = ln(8) / λ. t ≈ 2,08 / (1,21 x 10⁻⁴ an⁻¹) ≈ 17190 ans. (Remarque : 1/8 = 1/2³. Il s'est écoulé 3 demi-vies. t = 3 * t₁/₂ = 3 * 5730 ans = 17190 ans).
