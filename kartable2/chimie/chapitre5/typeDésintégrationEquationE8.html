Thème : Modélisation temporelle d’un système nucléaire

Fiche : Déterminer le type de désintégration à l'aide de l'équation

Question 1 : Observez l'équation suivante : 22688Ra → 22286Rn + 42He. De quel type de désintégration s'agit-il ?

  • a) Désintégration β⁻
  • b) Désintégration β⁺
  • c) Désintégration α
  • d) Émission γ

Bonne réponse : c) Désintégration α

Précisions : On observe l'émission d'une particule 42He, qui est la particule alpha (noyau d'hélium). De plus, les lois de conservation de Soddy sont respectées :

  • Conservation du nombre de masse (A) : 226 = 222 + 4
  • Conservation du nombre de charge (Z) : 88 = 86 + 2
Le noyau père perd 4 nucléons dont 2 protons, ce qui est caractéristique de la désintégration alpha.

Question 2 : Considérez la transformation : 146C → 147N + 0-1e (+ ν̅ₑ) Quel type de radioactivité est représenté ?

  • a) Désintégration α
  • b) Désintégration β⁻
  • c) Désintégration β⁺
  • d) Émission γ

Bonne réponse : b) Désintégration β⁻

Précisions : La particule émise est 0-1e, qui est un électron, caractéristique de la désintégration bêta moins (β⁻). Vérification des lois de Soddy :

  • Conservation de A : 14 = 14 + 0
  • Conservation de Z : 6 = 7 + (-1)
Le nombre de masse A est conservé, tandis que le numéro atomique Z augmente de 1 (un neutron s'est transformé en proton dans le noyau).

Question 3 : Identifiez le type de désintégration décrit par : 2211Na → 2210Ne + 0+1e (+ νₑ)

  • a) Désintégration α
  • b) Désintégration β⁻
  • c) Désintégration β⁺
  • d) Capture électronique

Bonne réponse : c) Désintégration β⁺

Précisions : La particule émise 0+1e est un positon (ou positron), signature de la désintégration bêta plus (β⁺). Vérifions les lois de Soddy :

  • Conservation de A : 22 = 22 + 0
  • Conservation de Z : 11 = 10 + 1
Le nombre de masse A est conservé, tandis que le numéro atomique Z diminue de 1 (un proton s'est transformé en neutron dans le noyau). (Note : La capture électronique est un autre processus pour les noyaux riches en protons, mais elle n'implique pas l'émission d'un positon).

Question 4 : L'équation suivante représente quelle étape ? 6028Ni* → 6028Ni + γ

  • a) Désintégration α
  • b) Désintégration β⁻
  • c) Désintégration β⁺
  • d) Désexcitation par émission γ

Bonne réponse : d) Désexcitation par émission γ

Précisions : Le noyau initial (6028Ni*) est dans un état excité (indiqué par l'astérisque *). Il revient à un état stable (6028Ni) en émettant un photon gamma (γ). On constate que ni le nombre de masse A (60) ni le numéro atomique Z (28) ne changent. Il ne s'agit donc pas d'une désintégration qui transforme le noyau, mais d'une libération d'énergie excédentaire sous forme de rayonnement électromagnétique.
