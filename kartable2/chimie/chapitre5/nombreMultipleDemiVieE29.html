Thème : Modélisation temporelle d’un système nucléaire

Fiche : Calculer le nombre de noyaux présents au bout d'un temps multiple du temps de demi-vie

Le temps de demi-vie t1/2 est la durée au bout de laquelle la moitié des noyaux radioactifs initiaux N₀ se sont désintégrés. On cherche à déterminer le nombre de noyaux restants N(t) après une durée t qui est un multiple entier (n) de la demi-vie : t = n × t1/2.

Question 1 : Si un échantillon contient initialement N₀ noyaux radioactifs, combien en reste-t-il après une durée t = 1 × t1/2 (une demi-vie) ?

  • a) N₀
  • b) N₀ / 2
  • c) N₀ / 4
  • d) 0

Bonne réponse : b) N₀ / 2

Précisions : C'est la définition même du temps de demi-vie. Après une durée égale à une demi-vie, la moitié de la population initiale s'est désintégrée, il reste donc l'autre moitié : N(t1/2) = N₀ / 2.

Question 2 : Combien de noyaux radioactifs reste-t-il après une durée t = 2 × t1/2 (deux demi-vies) ?

  • a) N₀ / 2
  • b) N₀ / 3
  • c) N₀ / 4
  • d) 0

Bonne réponse : c) N₀ / 4

Précisions : Après une première demi-vie (t₁ = t1/2), il reste N₁ = N₀ / 2 noyaux. Pendant la deuxième demi-vie (entre t₁ et t₂ = 2t1/2), la moitié des noyaux présents au début de cet intervalle (N₁) vont se désintégrer. Il reste donc à t₂ : N₂ = N₁ / 2 = (N₀ / 2) / 2 = N₀ / 4.
On peut aussi écrire N₀ / 4 = N₀ / 2².

Question 3 : Quelle est l'expression générale du nombre de noyaux N(t) restants après une durée t = n × t1/2, où n est un entier positif ?

  • a) N(t) = N₀ / n
  • b) N(t) = N₀ / (n+1)
  • c) N(t) = N₀ / 2n
  • d) N(t) = N₀ × (1/2)n+1

Bonne réponse : c) N(t) = N₀ / 2n

Précisions : À chaque fois qu'une durée égale à t1/2 s'écoule, le nombre de noyaux est divisé par 2.

  • Après 1 demi-vie (n=1) : N = N₀ / 2 = N₀ / 2¹
  • Après 2 demi-vies (n=2) : N = N₀ / 4 = N₀ / 2²
  • Après 3 demi-vies (n=3) : N = N₀ / 8 = N₀ / 2³
Par généralisation, après n demi-vies (t = n × t1/2), le nombre de noyaux restants est : N(t) = N₀ / 2n. On peut aussi l'écrire : N(t) = N₀ × (1/2)n.

Question 4 : Un échantillon contient initialement N₀ = 8,0 x 10⁶ noyaux radioactifs. Le temps de demi-vie est t1/2 = 15 minutes. Combien de noyaux reste-t-il après une durée t = 45 minutes ?

  • a) 4,0 x 10⁶ noyaux
  • b) 2,0 x 10⁶ noyaux
  • c) 1,0 x 10⁶ noyaux
  • d) 0 noyaux

Bonne réponse : c) 1,0 x 10⁶ noyaux

Précisions : 1. Déterminer le nombre de demi-vies écoulées (n) : n = t / t1/2 = 45 min / 15 min = 3. Il s'est écoulé 3 demi-vies. 2. Appliquer la formule N(t) = N₀ / 2n : N(45 min) = N₀ / 2³ = (8,0 x 10⁶) / 8 N(45 min) = 1,0 x 10⁶ noyaux.

Question 5 : La demi-vie d'un isotope est de 4 heures. Quelle fraction de la quantité initiale N₀ reste-t-il après une journée (24 heures) ?

  • a) 1/6
  • b) 1/16
  • c) 1/32
  • d) 1/64

Bonne réponse : d) 1/64

Précisions : 1. Déterminer le nombre de demi-vies (n) en 24 heures : n = t / t1/2 = 24 h / 4 h = 6. Il s'est écoulé 6 demi-vies. 2. Calculer la fraction restante N(t) / N₀ : N(t) / N₀ = 1 / 2n = 1 / 2⁶ 2⁶ = 2 × 2 × 2 × 2 × 2 × 2 = 64. La fraction restante est 1 / 64.

Question 6 : Un échantillon contenait initialement 6000 noyaux radioactifs. Après un certain temps t, il n'en reste plus que 750. Sachant que t est un multiple entier de la demi-vie t1/2, combien de demi-vies se sont écoulées ?

  • a) 1 demi-vie
  • b) 2 demi-vies
  • c) 3 demi-vies
  • d) 4 demi-vies

Bonne réponse : c) 3 demi-vies

Précisions : On cherche l'entier n tel que N(t) = N₀ / 2n. Ici, N₀ = 6000 et N(t) = 750. On a donc 750 = 6000 / 2n. Ce qui équivaut à 2n = 6000 / 750. 6000 / 750 = 8. On cherche donc n tel que 2n = 8. Comme 8 = 2 x 2 x 2 = 2³, on a n = 3. Il s'est écoulé 3 demi-vies.
