Connaître les caractéristiques des acide fort et base forte

Quelles sont les caractéristiques d'un acide fort et d'une base forte en solution aqueuse ?

Acide Fort (AH) :

  • Réagit totalement avec l'eau selon l'équation : AH + H2O → A- + H3O+.
  • La réaction est considérée comme quantitative (avancement final xf = avancement maximal xmax).
  • Dans une solution d'acide fort de concentration C, [H3O+] = C (si C est suffisamment élevée pour négliger l'autoprotolyse de l'eau, typiquement C > 10-6 mol/L).
  • Le pH d'une solution d'acide fort est donné par : pH = -log(C) (sous les mêmes conditions).
  • Un acide fort n'existe pas en tant que tel en solution aqueuse, il est entièrement sous forme de sa base conjuguée A- et d'ions H3O+.
  • Sa base conjuguée A- est une base spectatrice (très faible, sans caractère basique notable dans l'eau).

Base Forte (B) :

  • Réagit totalement avec l'eau selon l'équation : B + H2O → BH+ + OH-.
  • La réaction est considérée comme quantitative (xf = xmax).
  • Dans une solution de base forte de concentration C, [OH-] = C (si C est suffisamment élevée pour négliger l'autoprotolyse, C > 10-6 mol/L).
  • Le pOH d'une solution de base forte est donné par : pOH = -log(C), et le pH = 14 - pOH = 14 + log(C) (à 25°C, sous les mêmes conditions).
  • Une base forte n'existe pas en tant que telle en solution aqueuse, elle est entièrement sous forme de son acide conjugué BH+ et d'ions OH-.
  • Son acide conjugué BH+ est un acide spectateur (très faible, sans caractère acide notable dans l'eau).

Connaître les acide et base forts usuels

Citez quelques acides forts et bases fortes couramment rencontrés.

Acides Forts Usuels :

  • Acide chlorhydrique (HCl dans l'eau) : H3O+ + Cl-
  • Acide sulfurique (H2SO4 dans l'eau) : donne 2 H3O+ + SO42- (la première acidité est forte, la seconde est faible mais souvent considérée forte en première approximation au lycée pour simplifier)
  • Acide nitrique (HNO3 dans l'eau) : H3O+ + NO3-
  • Acide perchlorique (HClO4 dans l'eau) : H3O+ + ClO4-
  • Acide bromhydrique (HBr dans l'eau) : H3O+ + Br-
  • Acide iodhydrique (HI dans l'eau) : H3O+ + I-
Note : En solution aqueuse, l'espèce acide est en réalité l'ion oxonium H3O+. Les anions (Cl-, NO3-, etc.) sont des bases spectatrices.

Bases Fortes Usuelles :

  • Hydroxyde de sodium (Soude, NaOH) : Na+ + OH-
  • Hydroxyde de potassium (Potasse, KOH) : K+ + OH-
  • Hydroxyde de calcium (Ca(OH)2) : Ca2+ + 2 OH- (souvent peu soluble)
  • Les ions alcoolates (RO-), comme l'ion éthanolate (CH3CH2O-)
  • Les ions amidures (NH2-)
Note : En solution aqueuse, l'espèce basique est l'ion hydroxyde OH-. Les cations (Na+, K+, Ca2+) sont des acides spectateurs. Les alcoolates et amidures sont des bases si fortes qu'elles réagissent totalement avec l'eau pour former OH-.

Connaître les caractéristiques du produit ionique de l’eau

Qu'est-ce que le produit ionique de l'eau (Ke) et quelles sont ses caractéristiques ?

Le produit ionique de l'eau, noté Ke (ou parfois Kw pour water), est la constante d'équilibre associée à la réaction d'autoprotolyse de l'eau :

2 H2O(l) ⇌ H3O+(aq) + OH-(aq)

Caractéristiques :

  • Son expression est : Ke = [H3O+]éq × [OH-]éq
  • Il est défini pour toute solution aqueuse à l'équilibre.
  • Il ne dépend que de la température.
  • À 25 °C, Ke = 1,0 × 10-14. C'est une valeur sans unité.
  • On définit aussi le pKe = -log(Ke). À 25 °C, pKe = 14,0.
  • La relation pH + pOH = pKe est toujours vraie en solution aqueuse. À 25°C, pH + pOH = 14.
  • Le produit ionique Ke augmente lorsque la température augmente (l'autoprotolyse est une réaction endothermique).

Connaître les caractéristiques des acide et base faibles

Quelles sont les caractéristiques d'un acide faible et d'une base faible en solution aqueuse ?

Acide Faible (AH) :

  • Réagit partiellement avec l'eau selon l'équilibre : AH(aq) + H2O(l) ⇌ A-(aq) + H3O+(aq).
  • La réaction n'est pas totale (avancement final xf < avancement maximal xmax).
  • En solution, l'acide faible AH coexiste avec sa base conjuguée A- et les ions H3O+ et OH- (issus aussi de l'autoprotolyse de l'eau).
  • La force d'un acide faible est caractérisée par sa constante d'acidité Ka. Plus Ka est grand (et pKa petit), plus l'acide est fort (mais reste faible par définition).
  • Le pH d'une solution d'acide faible de concentration C ne peut pas être calculé simplement par pH = -log(C). Il dépend de C et du Ka. [H3O+] < C.
  • Sa base conjuguée A- est une base faible.

Base Faible (A- ou B) :

  • Réagit partiellement avec l'eau selon l'équilibre : A-(aq) + H2O(l) ⇌ AH(aq) + OH-(aq) (ou B(aq) + H2O(l) ⇌ BH+(aq) + OH-(aq)).
  • La réaction n'est pas totale (xf < xmax).
  • En solution, la base faible A- (ou B) coexiste avec son acide conjugué AH (ou BH+) et les ions H3O+ et OH-.
  • La force d'une base faible est caractérisée par sa constante de basicité Kb (ou implicitement par le Ka de son acide conjugué). Plus Kb est grand (et pKb petit), plus la base est forte (mais reste faible par définition).
  • Le pH d'une solution de base faible de concentration C ne peut pas être calculé simplement par pH = 14 + log(C). Il dépend de C et du Kb (ou Ka). [OH-] < C.
  • Son acide conjugué AH (ou BH+) est un acide faible.
Relation importante pour un couple acide/base faible AH/A- : Ka × Kb = Ke et pKa + pKb = pKe (soit 14 à 25°C).

Connaître les acide et base faibles usuels

Citez quelques acides faibles et bases faibles couramment rencontrés.

Acides Faibles Usuels :

  • Acide éthanoïque (ou acétique) : CH3COOH (dans le vinaigre)
  • Acide méthanoïque (ou formique) : HCOOH
  • Acide benzoïque : C6H5COOH
  • Ion ammonium : NH4+
  • Acide carbonique : H2CO3 (formé par dissolution de CO2 dans l'eau, souvent noté CO2,H2O)
  • Acide hypochloreux : HClO
  • Acide fluorhydrique : HF

Bases Faibles Usuelles :

  • Ammoniac : NH3
  • Ion éthanoate (ou acétate) : CH3COO-
  • Ion méthanoate (ou formiate) : HCOO-
  • Ion benzoate : C6H5COO-
  • Ion hydrogénocarbonate (ou bicarbonate) : HCO3-
  • Ion carbonate : CO32-
  • Ion hypochlorite : ClO-
  • Les amines (comme la méthylamine CH3NH2, l'aniline C6H5NH2)
  • Ion fluorure : F-
Notez que la base conjuguée d'un acide faible est une base faible, et l'acide conjugué d'une base faible est un acide faible.

Connaître la réaction d’autoprotolyse de l’eau

Écrire l'équation de la réaction d'autoprotolyse de l'eau et expliquer sa signification.

L'équation de la réaction d'autoprotolyse de l'eau est :

2 H2O(l) ⇌ H3O+(aq) + OH-(aq)

Ou parfois écrite sous la forme simplifiée :

H2O(l) ⇌ H+(aq) + OH-(aq)

Signification :

  • L'eau pure n'est pas constituée uniquement de molécules H2O.
  • Même en l'absence d'acide ou de base ajouté, l'eau subit une réaction acido-basique sur elle-même : une molécule d'eau cède un proton (agit comme acide) à une autre molécule d'eau (qui agit comme base).
  • Cela produit des ions oxonium (H3O+) et des ions hydroxyde (OH-) en très faibles quantités.
  • Cette réaction est un équilibre chimique, caractérisé par la constante d'équilibre Ke (produit ionique de l'eau).
  • Dans l'eau pure à 25°C, [H3O+] = [OH-] = 1,0 × 10-7 mol/L, ce qui conduit à un pH de 7,0.
  • Cette réaction a lieu dans toutes les solutions aqueuses, qu'elles soient acides, basiques ou neutres. Les concentrations de H3O+ et OH- sont toujours liées par la relation [H3O+] × [OH-] = Ke.

Connaître le lien entre le caractère fort d’un acide ou d’une base à la transformation quasi-totale de cet acide ou de cette base avec l’eau

Expliquez le lien entre le caractère fort d'un acide (ou d'une base) et le fait que sa réaction avec l'eau soit quasi-totale.

Par définition, un acide est dit "fort" si sa réaction avec l'eau est totale (ou quasi-totale).

Pour un acide AH : AH + H2O → A- + H3O+

  • Le terme "quasi-totale" signifie que l'équilibre de cette réaction est extrêmement déplacé vers la droite. La constante d'équilibre K de cette réaction est très grande (>> 1).
  • Concrètement, cela veut dire qu'à l'état final, la quasi-totalité des molécules de AH introduites initialement ont réagi pour former A- et H3O+. La concentration résiduelle de AH est négligeable.
  • Le taux d'avancement final (τ = xf / xmax) est égal à 1 (ou très proche de 1, τ ≈ 1).

De même, une base est dite "forte" si sa réaction avec l'eau est totale (ou quasi-totale).

Pour une base B : B + H2O → BH+ + OH-

  • L'équilibre est totalement déplacé vers la droite (K très grande).
  • La quasi-totalité des molécules de B introduites a réagi pour former BH+ et OH-.
  • Le taux d'avancement final τ ≈ 1.
Ce caractère "total" est la caractéristique fondamentale qui distingue les acides/bases forts des acides/bases faibles, dont la réaction avec l'eau est limitée (équilibre non total, τ < 1).

Déterminer quel ion est majoritaire entre l’ion oxonium et l’ion hydroxyde

Comment déterminer, en fonction du pH d'une solution aqueuse, si l'ion oxonium (H3O+) ou l'ion hydroxyde (OH-) est majoritaire ?

Dans toute solution aqueuse à 25°C, on a la relation [H3O+] × [OH-] = Ke = 1,0 × 10-14.

On compare les concentrations [H3O+] et [OH-] en fonction du pH :

  • Si pH < 7 : La solution est acide.
    • [H3O+] = 10-pH > 10-7 mol/L.
    • [OH-] = Ke / [H3O+] = 10-14 / 10-pH = 10pH-14 < 10-7 mol/L.
    • Donc, [H3O+] > [OH-]. L'ion oxonium est majoritaire.
  • Si pH = 7 : La solution est neutre.
    • [H3O+] = 10-7 mol/L.
    • [OH-] = 10-7 mol/L.
    • Donc, [H3O+] = [OH-]. Aucun des deux ions n'est majoritaire par rapport à l'autre.
  • Si pH > 7 : La solution est basique.
    • [H3O+] = 10-pH < 10-7 mol/L.
    • [OH-] = 10pH-14 > 10-7 mol/L.
    • Donc, [OH-] > [H3O+]. L'ion hydroxyde est majoritaire.
Cette comparaison est valable à 25°C. Si la température change, Ke change et la valeur du pH de neutralité (où [H3O+] = [OH-]) change également (elle vaut pKe / 2).

Connaître les caractéristiques de la constante d’acidité d’un couple acide-base

Qu'est-ce que la constante d'acidité (Ka) d'un couple acide-base (AH/A-) et quelles sont ses caractéristiques ?

La constante d'acidité, notée Ka, est la constante d'équilibre associée à la réaction d'un acide faible (AH) avec l'eau :

AH(aq) + H2O(l) ⇌ A-(aq) + H3O+(aq)

Caractéristiques :

  • Son expression est : Ka = ([A-]éq × [H3O+]éq) / [AH]éq
  • Elle est définie pour un couple acide/base faible en solution aqueuse.
  • Elle ne dépend que de la température.
  • Le Ka est une mesure de la force de l'acide AH dans l'eau :
    • Plus Ka est grand, plus l'équilibre est déplacé vers la droite, plus l'acide est dissocié, et donc plus l'acide est fort (parmi les acides faibles).
    • Plus Ka est petit, moins l'acide est dissocié, et donc plus l'acide est faible.
  • Le Ka est une grandeur sans unité (ou parfois exprimée en mol/L si on ne respecte pas strictement la définition thermodynamique avec les activités).
  • On définit aussi le pKa = -log(Ka).
    • Plus pKa est petit, plus l'acide est fort.
    • Plus pKa est grand, plus l'acide est faible.
  • La valeur du Ka (ou pKa) permet aussi de connaître la force de la base conjuguée A-. Un acide faible (petit Ka, grand pKa) a une base conjuguée "relativement forte" (parmi les bases faibles). Un acide "relativement fort" (grand Ka, petit pKa) a une base conjuguée faible.
Les acides forts ont des Ka tellement grands (pKa < 0) qu'on ne les utilise généralement pas pour les quantifier en solution aqueuse (ils sont nivelés par l'eau). De même, les bases fortes ont des pKa de leurs acides conjugués très élevés (> 14).

Comparer les forces d’un acide et d’une base dans l’eau à l’aide du pKa

Comment utiliser le pKa pour comparer la force de deux acides faibles ou de deux bases faibles dans l'eau ?

Le pKa est une mesure directe de la force d'un acide faible en solution aqueuse.

Comparaison de deux acides faibles (AH1 et AH2) :

  • Soient pKa1 le pKa du couple AH1/A1- et pKa2 le pKa du couple AH2/A2-.
  • L'acide le plus fort est celui qui a le plus petit pKa.
  • Si pKa1 < pKa2, alors AH1 est un acide plus fort que AH2.

Comparaison de deux bases faibles (A1- et A2-) :

  • Une base faible est d'autant plus forte que son acide conjugué est faible.
  • L'acide conjugué le plus faible est celui qui a le plus grand pKa.
  • Donc, la base la plus forte est celle dont l'acide conjugué a le plus grand pKa.
  • Si pKa1 < pKa2, alors AH1 est plus fort que AH2, ce qui implique que la base conjuguée A1- est plus faible que la base conjuguée A2-.
  • Autrement dit, la base A2- (associée au pKa le plus élevé) est la base la plus forte.
On peut aussi utiliser la relation pKa + pKb = pKe = 14 (à 25°C). Une base est d'autant plus forte que son pKb est petit. Comme pKb = 14 - pKa, un petit pKb correspond à un grand pKa.

Résumé :

  • Axe des pKa croissants : Force des acides diminue ↓, Force des bases conjuguées augmente ↑.

Comparer les forces d’un acide et d’une base dans l’eau à l’aide de la constante d’acidité

Comment utiliser la constante d'acidité (Ka) pour comparer la force de deux acides faibles ou de deux bases faibles dans l'eau ?

La constante d'acidité Ka mesure la tendance d'un acide à céder un proton à l'eau.

Comparaison de deux acides faibles (AH1 et AH2) :

  • Soient Ka1 la constante d'acidité du couple AH1/A1- et Ka2 celle du couple AH2/A2-.
  • L'acide le plus fort est celui qui se dissocie le plus dans l'eau, c'est-à-dire celui qui a la plus grande constante d'acidité Ka.
  • Si Ka1 > Ka2, alors AH1 est un acide plus fort que AH2.

Comparaison de deux bases faibles (A1- et A2-) :

  • La force d'une base faible A- est liée à la force de son acide conjugué AH par la relation Ka × Kb = Ke, où Kb est la constante de basicité de A- (Kb = [AH]éq[OH-]éq / [A-]éq).
  • Une base est d'autant plus forte que sa constante de basicité Kb est grande.
  • Comme Kb = Ke / Ka, un grand Kb correspond à un petit Ka.
  • Donc, la base la plus forte est celle dont l'acide conjugué a le plus petit Ka.
  • Si Ka1 > Ka2, alors AH1 est plus fort que AH2, et la base conjuguée A1- est plus faible que la base conjuguée A2-. La base A2- (associée au Ka le plus faible) est la base la plus forte.

Résumé :

  • Axe des Ka croissants : Force des acides augmente ↑, Force des bases conjuguées diminue ↓.

Connaître les caractéristiques du diagramme de prédominance d’un couple acide-base

Qu'est-ce qu'un diagramme de prédominance d'un couple acide-base faible (AH/A-) et quelles informations donne-t-il ?

Un diagramme de prédominance est une représentation graphique simple qui indique quelle espèce (l'acide AH ou sa base conjuguée A-) prédomine en solution aqueuse en fonction du pH.

Il est basé sur la relation de Henderson-Hasselbalch (ou directement sur l'expression du Ka) :

pH = pKa + log ( [A-]éq / [AH]éq )

Caractéristiques et lecture du diagramme :

  • Il s'agit d'un axe gradué en pH.
  • Le point clé de l'axe est la valeur du pKa du couple.
  • Si pH < pKa :
    • log ( [A-] / [AH] ) < 0 => [A-] / [AH] < 1 => [AH] > [A-].
    • L'espèce acide AH prédomine.
  • Si pH = pKa :
    • log ( [A-] / [AH] ) = 0 => [A-] / [AH] = 1 => [AH] = [A-].
    • Il y a autant d'acide que de base conjuguée. C'est un point important pour les solutions tampons.
  • Si pH > pKa :
    • log ( [A-] / [AH] ) > 0 => [A-] / [AH] > 1 => [A-] > [AH].
    • L'espèce basique A- prédomine.

Représentation schématique :

       AH prédomine       |       A⁻ prédomine
<----------------------|----------------------> pH
                      pKₐ
            [AH] > [A⁻]   [AH] = [A⁻]   [A⁻] > [AH]
Le diagramme de prédominance ne donne pas les concentrations exactes, mais indique quelle est l'espèce majoritaire du couple à un pH donné. "Prédomine" signifie simplement "est en concentration supérieure à".

Connaître les caractéristiques du diagramme de prédominance d’un indicateur coloré

Qu'est-ce qu'un indicateur coloré acido-basique et comment son diagramme de prédominance est-il lié à sa couleur ?

Un indicateur coloré acido-basique est un couple acide-base faible, noté HInd/Ind-, dont la forme acide (HInd) et la forme basique (Ind-) ont des couleurs différentes en solution aqueuse.

Caractéristiques :

  • Comme tout couple acide-base faible, il est caractérisé par un pKa (parfois noté pKi pour indicateur).
  • Son diagramme de prédominance suit la règle générale :
    • Si pH < pKa : [HInd] > [Ind-], la forme acide HInd prédomine.
    • Si pH > pKa : [Ind-] > [HInd], la forme basique Ind- prédomine.
    • Si pH = pKa : [HInd] = [Ind-].
  • La couleur observée de la solution dépend de l'espèce qui prédomine :
    • Si HInd prédomine largement (typiquement pH < pKa - 1), la solution prend la couleur de HInd.
    • Si Ind- prédomine largement (typiquement pH > pKa + 1), la solution prend la couleur de Ind-.
    • Si les concentrations sont du même ordre de grandeur (autour du pKa, typiquement dans l'intervalle [pKa - 1 ; pKa + 1]), la solution prend une couleur intermédiaire appelée teinte sensible (mélange des couleurs de HInd et Ind-).

Zone de virage :

L'intervalle de pH où la couleur de l'indicateur change visiblement est appelé la zone de virage. Elle est généralement considérée comme l'intervalle [pKa - 1 ; pKa + 1].

Représentation schématique :

      Couleur de HInd    |   Teinte Sensible   |   Couleur de Ind⁻
<---------------------|---------------------|---------------------> pH
                  pKₐ - 1                 pKₐ                pKₐ + 1
                           Zone de Virage

Connaître les caractéristiques des acides alpha-aminés

Qu'est-ce qu'un acide α-aminé et quelles sont ses propriétés acido-basiques particulières ?

Un acide α-aminé est une molécule organique qui possède deux groupes fonctionnels caractéristiques portés par le même atome de carbone (le carbone α, adjacent au groupe carboxyle) :

  • Un groupe carboxyle (-COOH), qui a des propriétés acides.
  • Un groupe amine (-NH2), qui a des propriétés basiques.

La formule générale est R-CH(NH2)-COOH, où R est une chaîne latérale (sauf pour la glycine où R = H).

Propriétés Acido-Basiques :

  • En raison de la présence d'un groupe acide et d'un groupe basique sur la même molécule, les acides α-aminés sont des ampholytes ou amphotères.
  • En solution aqueuse, ils existent principalement sous forme d'un ion dipolaire appelé zwitterion (ou amphion), où le groupe carboxyle a cédé son proton au groupe amine : R-CH(NH3+)-COO-.
  • Un acide α-aminé possède (au moins) deux pKa :
    • pKa1 (généralement entre 2 et 3) correspond à l'acidité du groupe carboxonium (-COOH ou -COOH2+ sous forme très acide) du zwitterion/forme cationique :
      R-CH(NH3+)-COOH + H2O ⇌ R-CH(NH3+)-COO- + H3O+
    • pKa2 (généralement entre 9 et 10) correspond à l'acidité du groupe ammonium (-NH3+) du zwitterion :
      R-CH(NH3+)-COO- + H2O ⇌ R-CH(NH2)-COO- + H3O+
  • Il existe un pH particulier, appelé point isoélectrique (pI), où la charge électrique nette moyenne de l'acide aminé est nulle. C'est à ce pH que la concentration du zwitterion est maximale. Pour un acide aminé simple, pI = (pKa1 + pKa2) / 2.
  • Selon le pH de la solution, l'acide α-aminé peut exister sous trois formes principales :
    • Forme cationique : R-CH(NH3+)-COOH (à pH très acide, pH < pKa1)
    • Zwitterion : R-CH(NH3+)-COO- (autour du pI, pKa1 < pH < pKa2)
    • Forme anionique : R-CH(NH2)-COO- (à pH très basique, pH > pKa2)
Certains acides aminés ont une chaîne latérale R qui contient aussi un groupe acide ou basique, ils auront alors trois pKa.

Connaître les caractéristiques d’une solution tampon

Qu'est-ce qu'une solution tampon et quelles sont ses propriétés caractéristiques ?

Une solution tampon est une solution aqueuse qui possède la propriété de maintenir un pH approximativement constant malgré :

  • L'ajout modéré d'un acide fort.
  • L'ajout modéré d'une base forte.
  • Une dilution modérée.

Composition :

Une solution tampon est typiquement constituée par le mélange :

  • D'un acide faible (AH) et de sa base conjuguée (A-)
  • En concentrations voisines et relativement élevées.

On peut la préparer :

  • En mélangeant directement l'acide faible et un sel de sa base conjuguée (ex: CH3COOH et CH3COONa).
  • En neutralisant partiellement un acide faible par une base forte (ex: excès de CH3COOH + NaOH en défaut).
  • En neutralisant partiellement une base faible par un acide fort (ex: excès de NH3 + HCl en défaut).

Propriétés et Fonctionnement :

  • Le pH d'une solution tampon est proche du pKa du couple acide/base faible qui la constitue. Il est donné par la relation de Henderson-Hasselbalch : pH = pKa + log ([A-]/[AH]).
  • Le pouvoir tampon est maximal lorsque [AH] = [A-], c'est-à-dire lorsque pH = pKa.
  • La plage de pH où le pouvoir tampon est efficace est généralement considérée comme [pKa - 1 ; pKa + 1].
  • Mécanisme :
    • Si on ajoute un acide (H3O+), il est consommé par la base faible présente : A- + H3O+ → AH + H2O.
    • Si on ajoute une base (OH-), elle est consommée par l'acide faible présent : AH + OH- → A- + H2O.
    Dans les deux cas, la variation du rapport [A-]/[AH] est limitée (car les concentrations initiales sont grandes et voisines), donc la variation de pH est faible.
Les solutions tampons sont cruciales dans de nombreux processus biologiques (ex: sang, pH ≈ 7.4) et chimiques où un pH stable est requis.

Calculer le pH d’une solution d’acide fort

Comment calculer le pH d'une solution aqueuse d'un acide fort de concentration C ?

Soit un acide fort HA de concentration molaire apportée C.

1. Écrire l'équation de réaction avec l'eau :

La réaction est totale : HA + H2O → A- + H3O+

2. Déterminer la concentration en ions oxonium [H3O+] :

  • Comme la réaction est totale, tout l'acide fort initialement présent (concentration C) s'est transformé en H3O+.
  • On a donc : [H3O+]final = C

3. Calculer le pH :

Le pH est défini par : pH = -log([H3O+])

Donc : pH = -log(C)

Conditions d'application :

  • Cette formule simple est valable si la concentration C de l'acide fort est suffisamment élevée pour que l'autoprotolyse de l'eau soit négligeable devant l'apport d'ions H3O+ par l'acide.
  • En pratique, cela est vrai si C ≥ 10-6 mol/L (ou parfois C ≥ 10 × [H3O+]eau pure = 10-6 mol/L).
  • Si C est très faible (proche de 10-7 mol/L ou moins), il faut tenir compte des H3O+ provenant de l'eau, et le pH sera très proche de 7. Par exemple, pour C = 10-8 mol/L, le pH n'est pas 8 mais environ 6,98. Le calcul exact nécessite de résoudre une équation plus complexe ([H3O+] = C + [OH-] = C + Ke/[H3O+]).
Attention : C est la concentration molaire apportée en acide fort (mol/L). La fonction log est le logarithme décimal.

Calculer le pH d’une solution de base forte

Comment calculer le pH d'une solution aqueuse d'une base forte de concentration C ?

Soit une base forte B (comme NaOH, qui libère OH-) de concentration molaire apportée C.

1. Écrire l'équation de réaction avec l'eau (ou de dissolution pour les hydroxydes) :

La réaction est totale : B + H2O → BH+ + OH- (ou NaOH(s) → Na+(aq) + OH-(aq))

2. Déterminer la concentration en ions hydroxyde [OH-] :

  • Comme la réaction est totale, toute la base forte initialement présente (concentration C) s'est transformée en OH- (ou a libéré OH-).
  • On a donc : [OH-]final = C

3. Calculer le pOH :

Le pOH est défini par : pOH = -log([OH-])

Donc : pOH = -log(C)

4. Calculer le pH :

On utilise la relation liée au produit ionique de l'eau (à 25°C) : pH + pOH = pKe = 14,0

Donc : pH = 14,0 - pOH = 14,0 - (-log(C)) = 14,0 + log(C)

Conditions d'application :

  • Cette formule est valable si la concentration C de la base forte est suffisamment élevée pour que l'autoprotolyse de l'eau soit négligeable devant l'apport d'ions OH- par la base.
  • En pratique, cela est vrai si C ≥ 10-6 mol/L.
  • Si C est très faible, il faut tenir compte des OH- provenant de l'eau, et le pH sera très proche de 7. Par exemple, pour C = 10-8 mol/L de NaOH, le pH n'est pas 6 (14+log(10-8)) mais environ 7,02. Le calcul exact est plus complexe.
Attention : C est la concentration molaire apportée en base forte (mol/L).

Calculer la constante d’acidité d’un couple acide-base

Comment calculer la constante d'acidité Ka d'un couple acide-base faible (AH/A-) à partir des concentrations à l'équilibre ?

La constante d'acidité Ka est associée à l'équilibre :

AH(aq) + H2O(l) ⇌ A-(aq) + H3O+(aq)

Définition :

Le Ka est défini comme le quotient de réaction à l'équilibre (Qr,éq) pour cette réaction :

Ka = ([A-]éq × [H3O+]éq) / [AH]éq

Méthode de calcul :

Pour calculer Ka, il faut connaître les concentrations molaires des trois espèces (AH, A-, H3O+) à l'état d'équilibre.

Ces concentrations peuvent être obtenues :

  • Expérimentalement : Par exemple, en mesurant le pH d'une solution d'acide faible de concentration initiale connue C.
    • On mesure le pH, ce qui donne [H3O+]éq = 10-pH.
    • On établit un tableau d'avancement pour la réaction AH + H2O ⇌ A- + H3O+ :
      Initialement : C | excès | 0 | ≈0
      À l'équilibre : C - xf/V | excès | xf/V | xf/V (où V est le volume, xf l'avancement final)
    • On identifie [H3O+]éq = xf/V = 10-pH.
    • On en déduit [A-]éq = xf/V = 10-pH.
    • Et [AH]éq = C - xf/V = C - 10-pH.
    • On remplace ces valeurs dans l'expression du Ka :
      Ka = (10-pH × 10-pH) / (C - 10-pH) = (10-pH)2 / (C - 10-pH)
      (Formule valable pour une solution initiale d'acide faible AH pur).
  • Par dosage/titrage : Au point de demi-équivalence d'un titrage d'un acide faible par une base forte, on a [AH]éq = [A-]éq. D'après la relation pH = pKa + log([A-]/[AH]), on a alors pH1/2 = pKa. On peut donc déterminer pKa et en déduire Ka = 10-pKa.
  • À partir de données thermodynamiques (moins courant au lycée).
Le Ka est une constante pour un couple donné à une température donnée. Il ne dépend pas des concentrations initiales utilisées pour le déterminer (même si les concentrations à l'équilibre en dépendent).

Calculer le pKa à l’aide de la constante d’acidité d’un couple acide-base

Comment calculer le pKa d'un couple acide-base à partir de sa constante d'acidité Ka ?

La relation entre le pKa et la constante d'acidité Ka est une définition mathématique utilisant le logarithme décimal.

Définition :

pKa = -log(Ka)

Où :

  • pKa est la constante d'acidité logarithmique (sans unité).
  • Ka est la constante d'acidité (sans unité ou parfois en mol/L).
  • log désigne la fonction logarithme décimal.

Exemple :

Si la constante d'acidité de l'acide éthanoïque est Ka = 1,8 × 10-5 (à 25°C).

pKa = -log(1,8 × 10-5) ≈ 4,74

Le pKa est souvent utilisé car il manipule des nombres plus simples (généralement entre 0 et 14 pour les couples usuels dans l'eau) que les Ka (qui sont souvent très petits). N'oubliez pas le signe "moins" dans la formule.

Calculer la constante d’acidité à l’aide du pKa d’un couple acide-base

Comment calculer la constante d'acidité Ka d'un couple acide-base à partir de son pKa ?

Pour passer du pKa au Ka, on utilise la fonction réciproque du logarithme décimal, qui est la puissance de 10.

Partant de la définition pKa = -log(Ka) :

1. Multiplier par -1 : -pKa = log(Ka)

2. Appliquer la fonction puissance de 10 (10x) : 10-pKa = 10log(Ka)

3. Simplifier : Ka = 10-pKa

Formule :

Ka = 10-pKa

Exemple :

Si le pKa de l'ion ammonium NH4+ est 9,2 (à 25°C).

Ka = 10-9,2 ≈ 6,3 × 10-10

Assurez-vous d'utiliser correctement la fonction puissance de 10 de votre calculatrice (souvent notée 10x ou accessible via SHIFT/2nd + log).

Estimer la valeur de la constante d’acidité d’un couple acide-base à l’aide d’une mesure de pH

Comment estimer la valeur du Ka (ou pKa) d'un acide faible AH en mesurant le pH d'une solution de cet acide de concentration initiale C connue ?

Pour estimer Ka ou pKa à partir d'une mesure de pH d'une solution d'acide faible AH de concentration C :

1. Mesurer le pH de la solution :

Soit pHmesuré la valeur obtenue.

2. Calculer [H3O+]éq :

[H3O+]éq = 10-pHmesuré

3. Utiliser le tableau d'avancement de la réaction AH + H2O ⇌ A- + H3O+ :

  • État initial : C | excès | 0 | ≈0
  • État final (éq) : C - xf/V | excès | xf/V | xf/V

4. Exprimer les concentrations à l'équilibre en fonction de [H3O+]éq :

  • [H3O+]éq = xf/V = 10-pHmesuré
  • [A-]éq = xf/V = 10-pHmesuré
  • [AH]éq = C - xf/V = C - 10-pHmesuré

5. Calculer Ka :

Utiliser la définition de Ka :

Ka = ([A-]éq × [H3O+]éq) / [AH]éq

Ka = (10-pHmesuré × 10-pHmesuré) / (C - 10-pHmesuré) = (10-pHmesuré)2 / (C - 10-pHmesuré)

6. (Optionnel) Calculer pKa :

pKa = -log(Ka)

Approximation possible : Si l'acide est très peu dissocié (c'est-à-dire si pHmesuré est significativement plus grand que -log(C/10), ou si τ << 1), on peut parfois approximer [AH]éq ≈ C.
Dans ce cas, Ka ≈ ([H3O+]éq)2 / C = (10-pHmesuré)2 / C.
Cette approximation simplifie le calcul mais est moins précise. Il est préférable d'utiliser la formule exacte.
Cette méthode fonctionne aussi pour une base faible B de concentration C, en mesurant le pH, calculant [OH-]éq = 10-(14-pH), et utilisant Kb = ([BH+]éq[OH-]éq)/[B]éq = ([OH-]éq)2 / (C - [OH-]éq). On trouve ensuite Ka = Ke / Kb.

Déduire le caractère fort ou faible d’acide ou de la base à l’aide du pH de solutions d’acide ou de base de concentration donnée

Comment peut-on utiliser la mesure du pH d'une solution d'acide (ou de base) de concentration C connue pour déterminer si cet acide (ou cette base) est fort ou faible ?

La comparaison entre le pH mesuré et le pH théorique d'un acide (ou d'une base) fort de même concentration permet de conclure sur sa force.

Pour un Acide de concentration C :

  1. Calculer le pH théorique si l'acide était fort : pHfort = -log(C) (en supposant C ≥ 10-6 mol/L).
  2. Mesurer le pH réel de la solution : pHmesuré.
  3. Comparer :
    • Si pHmesuré = pHfort (ou très proche, compte tenu des incertitudes de mesure), alors l'acide réagit totalement avec l'eau. L'acide est fort.
    • Si pHmesuré > pHfort, cela signifie que [H3O+]réelle < C. L'acide n'a pas réagi totalement avec l'eau. L'acide est faible.
    • (Note : pHmesuré ne peut pas être inférieur à pHfort).

Pour une Base de concentration C :

  1. Calculer le pH théorique si la base était forte : pOHforte = -log(C), puis pHforte = 14,0 + log(C) (à 25°C, en supposant C ≥ 10-6 mol/L).
  2. Mesurer le pH réel de la solution : pHmesuré.
  3. Comparer :
    • Si pHmesuré = pHforte (ou très proche), alors la base réagit totalement avec l'eau. La base est forte.
    • Si pHmesuré < pHforte, cela signifie que [OH-]réelle < C (car pOHréel > pOHforte). La base n'a pas réagi totalement avec l'eau. La base est faible.
    • (Note : pHmesuré ne peut pas être supérieur à pHforte).
Cette méthode est un moyen expérimental simple pour distinguer un acide/base fort d'un faible sans connaître son pKa a priori.

Prévoir la composition finale d’une solution aqueuse de concentration donnée en acide fort ou faible apporté

Comment prévoir la composition (espèces présentes et leurs concentrations) à l'état final d'une solution aqueuse obtenue en dissolvant un acide (fort ou faible) à une concentration C ?

La composition finale dépend de la force de l'acide.

Cas 1 : Acide Fort (AH) de concentration C

  • Réaction : AH + H2O → A- + H3O+ (Totale)
  • Espèces présentes à l'état final :
    • Ion H3O+ : [H3O+] = C
    • Base conjuguée spectatrice A- : [A-] = C
    • Ion OH- (issu de l'autoprotolyse) : [OH-] = Ke / [H3O+] = Ke / C (très faible si C est élevée)
    • Acide AH : [AH] ≈ 0 (négligeable car la réaction est totale)
    • Solvant H2O (largement majoritaire)
  • pH : pH = -log(C) (si C ≥ 10-6 M)

Cas 2 : Acide Faible (AH) de concentration C et de constante Ka

  • Réaction : AH + H2O ⇌ A- + H3O+ (Équilibre)
  • Espèces présentes à l'état final :
    • Acide faible AH : [AH]éq = C - [H3O+]éq
    • Base conjuguée A- : [A-]éq = [H3O+]éq
    • Ion H3O+ : [H3O+]éq
    • Ion OH- : [OH-]éq = Ke / [H3O+]éq
    • Solvant H2O
  • Calcul de [H3O+]éq : Il faut résoudre l'équation issue de Ka :
    Ka = ([H3O+]éq)2 / (C - [H3O+]éq)
    Cela mène à une équation du second degré en [H3O+]éq : ([H3O+]éq)2 + Ka[H3O+]éq - KaC = 0.
    On résout cette équation (seule la solution positive est physiquement acceptable).
  • pH : pH = -log([H3O+]éq)
  • Approximations possibles :
    • Si l'acide est peu dissocié (Ka/C << 1 ou C/Ka > 100), on peut négliger [H3O+]éq devant C, soit [AH]éq ≈ C.
      Alors Ka ≈ ([H3O+]éq)2 / C => [H3O+]éq ≈ √(KaC) => pH ≈ ½(pKa - log C).
La démarche est similaire pour une base forte ou faible, en se concentrant sur la réaction avec l'eau et le calcul de [OH-]éq. Pour une base forte, [OH-] = C et [B] ≈ 0. Pour une base faible B, on calcule [OH-]éq à partir de Kb = ([OH-]éq)2 / (C - [OH-]éq), puis on trouve le pH.

Établir un diagramme de prédominance

Comment établir le diagramme de prédominance d'un couple acide-base faible AH/A- dont on connaît le pKa ?

Pour établir le diagramme de prédominance du couple AH/A- de pKa donné :

  1. Tracer un axe horizontal gradué en unité de pH. Il doit couvrir une gamme de pH pertinente (souvent de 0 à 14).
  2. Placer le point correspondant à la valeur du pKa sur cet axe. Ce point est la frontière entre les domaines de prédominance.
  3. Identifier la zone où pH < pKa : Dans cette zone, l'espèce acide AH prédomine ([AH] > [A-]). Indiquer "AH prédomine" ou simplement "AH" dans cette partie de l'axe.
  4. Identifier la zone où pH > pKa : Dans cette zone, l'espèce basique A- prédomine ([A-] > [AH]). Indiquer "A- prédomine" ou simplement "A-" dans cette partie de l'axe.
  5. Au point pH = pKa : Les concentrations des deux espèces sont égales : [AH] = [A-]. On peut marquer ce point spécifiquement.

Exemple pour le couple CH3COOH / CH3COO- (pKa = 4,7) :

       CH₃COOH prédomine   |   CH₃COO⁻ prédomine
<------------------------|-----------------------> pH
                        4,7
Le diagramme est une représentation simple et visuelle. Il est essentiel de bien positionner le pKa et d'indiquer correctement quelle espèce prédomine de chaque côté.

Déterminer une prédominance à partir du pH

Étant donné le pKa d'un couple AH/A- et le pH d'une solution, comment déterminer quelle espèce (AH ou A-) prédomine ?

Pour déterminer l'espèce prédominante d'un couple AH/A- (de pKa connu) à un pH donné :

Comparer simplement la valeur du pH de la solution à la valeur du pKa du couple :

  • Si pH < pKa : La solution est "plus acide" que le point d'égalité des concentrations. C'est la forme acide AH qui prédomine.
  • Si pH = pKa : Les concentrations des deux formes sont égales. [AH] = [A-]. Aucune ne prédomine strictement sur l'autre.
  • Si pH > pKa : La solution est "plus basique" que le point d'égalité des concentrations. C'est la forme basique A- qui prédomine.

Exemple :

Le pKa du couple NH4+/NH3 est 9,2. Dans une solution de pH = 7,0 :

  • On compare pH = 7,0 et pKa = 9,2.
  • Comme 7,0 < 9,2 (pH < pKa), c'est la forme acide du couple, l'ion ammonium NH4+, qui prédomine.

Dans une solution de pH = 10,0 :

  • On compare pH = 10,0 et pKa = 9,2.
  • Comme 10,0 > 9,2 (pH > pKa), c'est la forme basique du couple, l'ammoniac NH3, qui prédomine.
Cette règle découle directement de l'analyse de la relation pH = pKa + log([A-]/[AH]).

Justifier le choix d’un indicateur coloré lors d’un titrage

Comment choisir un indicateur coloré approprié pour repérer l'équivalence lors d'un titrage acido-basique ?

Le but d'un indicateur coloré dans un titrage acido-basique est de signaler la fin du titrage, c'est-à-dire le point d'équivalence, par un changement de couleur net et rapide.

Critère de choix :

Un indicateur coloré est approprié si sa zone de virage (l'intervalle de pH où il change de couleur, typiquement [pKa - 1 ; pKa + 1]) contient le pH à l'équivalence (pHéq) du titrage.

Justification :

  • À l'équivalence d'un titrage acido-basique, on observe généralement un saut de pH important (une variation rapide et brutale du pH pour un faible ajout de réactif titrant).
  • Pour que l'indicateur change de couleur précisément au moment où l'équivalence est atteinte, il faut que le pH correspondant à son changement de couleur (sa zone de virage) se situe dans cette zone de saut de pH.
  • Idéalement, le pKa de l'indicateur doit être le plus proche possible du pH à l'équivalence.

Détermination du pHéq :

  • Titrage Acide Fort / Base Forte : pHéq = 7 (à 25°C). Indicateurs possibles : Bleu de Bromothymol (BBT, pKa ≈ 7,1), Rouge de phénol.
  • Titrage Acide Faible / Base Forte : À l'équivalence, on a la base faible A- en solution. La solution est basique. pHéq > 7. Indicateur possible : Phénolphtaléine (pKa ≈ 9,4).
  • Titrage Base Faible / Acide Fort : À l'équivalence, on a l'acide faible BH+ en solution. La solution est acide. pHéq < 7. Indicateur possible : Hélianthine (ou méthylorange, pKa ≈ 3,7), Rouge de méthyle (pKa ≈ 5,0).
Le choix dépend donc de la nature des réactifs titré et titrant, qui déterminent la valeur du pH à l'équivalence. Il faut choisir un indicateur dont le pKa est proche de ce pHéq.

Faire le diagramme de prédominance d’un indicateur coloré

Comment tracer le diagramme de prédominance d'un indicateur coloré HInd/Ind- de pKa donné, en indiquant les couleurs et la zone de virage ?

Pour tracer le diagramme de prédominance d'un indicateur coloré HInd/Ind- (avec pKa, couleur de HInd et couleur de Ind- connues) :

  1. Tracer un axe horizontal gradué en pH.
  2. Placer le pKa de l'indicateur sur l'axe.
  3. Délimiter la zone de virage : Marquer les points pKa - 1 et pKa + 1 sur l'axe. L'intervalle [pKa - 1 ; pKa + 1] est la zone de virage.
  4. Indiquer la couleur pour pH < pKa - 1 : Dans cette zone, HInd prédomine très largement. La couleur observée est celle de HInd. Écrire "Couleur de HInd" (ou le nom de la couleur) dans cette zone.
  5. Indiquer la couleur pour pH > pKa + 1 : Dans cette zone, Ind- prédomine très largement. La couleur observée est celle de Ind-. Écrire "Couleur de Ind-" (ou le nom de la couleur) dans cette zone.
  6. Indiquer la couleur dans la zone de virage [pKa - 1 ; pKa + 1] : Dans cet intervalle, les deux formes coexistent en proportions notables. La couleur est un mélange des deux couleurs précédentes. Écrire "Teinte sensible" ou "Mélange des couleurs" ou décrire la couleur résultante (ex: "Orange" si HInd est rouge et Ind- est jaune).

Exemple pour le Bleu de Bromothymol (BBT) : pKa = 7,1 ; HInd (jaune) / Ind- (bleu)

        Jaune (HInd)      |    Vert (Teinte sensible)    |      Bleu (Ind⁻)
<----------------------|------------------------------|----------------------> pH
                      6,1                             7,1                            8,1
                                   Zone de Virage [6,1 ; 8,1]
Ce diagramme visualise directement la plage de pH où l'indicateur change de couleur et les couleurs observées selon le pH.

Déterminer l’espèce prédominante dans une solution à l’aide de son diagramme de prédominance

Comment utiliser un diagramme de prédominance déjà établi pour un couple AH/A- afin de déterminer quelle espèce prédomine à un pH donné ?

Pour utiliser un diagramme de prédominance afin de trouver l'espèce majoritaire à un certain pH :

  1. Repérer la valeur du pH de la solution sur l'axe des pH du diagramme.
  2. Observer dans quelle zone du diagramme se situe cette valeur de pH :
    • Si le pH se situe dans la zone où est indiquée l'espèce AH (généralement à gauche du pKa), alors c'est AH qui prédomine.
    • Si le pH se situe dans la zone où est indiquée l'espèce A- (généralement à droite du pKa), alors c'est A- qui prédomine.
    • Si le pH correspond exactement à la valeur du pKa (la frontière), alors les deux espèces sont en concentrations égales : [AH] = [A-].

Exemple :

Soit le diagramme de prédominance du couple acide hypochloreux/ion hypochlorite (HClO/ClO-) :

           HClO prédomine     |     ClO⁻ prédomine
<------------------------|------------------------> pH
                         7,5

Si on a une solution à pH = 5,0 :

  • On place 5,0 sur l'axe. C'est à gauche de 7,5.
  • Donc, l'espèce HClO prédomine.

Si on a une solution à pH = 9,0 :

  • On place 9,0 sur l'axe. C'est à droite de 7,5.
  • Donc, l'espèce ClO- prédomine.

Si on a une solution à pH = 7,5 :

  • On est exactement sur la frontière.
  • Donc, [HClO] = [ClO-].
Le diagramme de prédominance est un outil visuel rapide pour répondre à cette question sans calcul.

Déterminer le diagramme de prédominance d’un acide alpha-aminé

Comment établir le diagramme de prédominance d'un acide α-aminé simple (avec deux pKa, notés pKa1 et pKa2) ?

Un acide α-aminé simple (type R-CH(NH2)-COOH) possède deux fonctions acido-basiques et donc deux pKa :

  • pKa1 : associé au groupe carboxyle (-COOH/-COO-).
  • pKa2 : associé au groupe ammonium (-NH3+/-NH2).

Généralement, pKa1 < pKa2.

Les trois formes possibles sont :

  • Forme cationique (acide) : H2A+ (ou AH2+) : R-CH(NH3+)-COOH
  • Zwitterion (amphion) : HA (ou AH+/-) : R-CH(NH3+)-COO-
  • Forme anionique (basique) : A- : R-CH(NH2)-COO-

Le diagramme de prédominance comportera deux frontières (pKa1 et pKa2) et trois zones :

  1. Tracer un axe horizontal gradué en pH.
  2. Placer les deux valeurs pKa1 et pKa2 sur l'axe (avec pKa1 à gauche de pKa2).
  3. Identifier la zone où pH < pKa1 : Dans cette zone très acide, les deux groupes sont sous leur forme acide. C'est la forme cationique H2A+ qui prédomine.
  4. Identifier la zone où pKa1 < pH < pKa2 : Dans cette zone, le groupe carboxyle est sous sa forme basique (-COO-) et le groupe amine est sous sa forme acide (-NH3+). C'est le zwitterion HA qui prédomine.
  5. Identifier la zone où pH > pKa2 : Dans cette zone très basique, les deux groupes sont sous leur forme basique. C'est la forme anionique A- qui prédomine.

Représentation schématique :

       H₂A⁺ prédomine   |     HA (Zwitterion) prédomine     |     A⁻ prédomine
<---------------------|---------------------------------|---------------------> pH
                    pKₐ₁                              pKₐ₂

Points particuliers :

  • À pH = pKa1 : [H2A+] = [HA]
  • À pH = pKa2 : [HA] = [A-]
  • Au point isoélectrique pI = (pKa1 + pKa2) / 2 : la concentration du zwitterion [HA] est maximale.
Pour les acides aminés avec une chaîne latérale R acido-basique, il y aura un troisième pKa (noté pKR) et donc quatre formes et trois zones de prédominance sur le diagramme.
