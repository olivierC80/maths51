L'Évolution Spontanée d'un Système Chimique

Ce cours aborde la manière dont un système chimique évolue naturellement depuis un état initial jusqu'à un état final, qu'il s'agisse d'une transformation totale ou d'un équilibre. Nous verrons comment décrire quantitativement cette évolution et comment prévoir son sens.

Partie 1 : Description d'une Transformation Chimique

1.1 L'Équation de Réaction

Une transformation chimique est modélisée par une équation de réaction qui indique les réactifs (espèces consommées) et les produits (espèces formées), ainsi que leurs proportions.

Exemple : Combustion du méthane

CH₄(g) + 2 O₂(g) → CO₂(g) + 2 H₂O(l)

Les nombres placés devant les formules chimiques sont les coefficients stœchiométriques. Ils indiquent les proportions dans lesquelles les espèces réagissent et se forment (en moles).

1.1.1 Équilibrer une Équation de Réaction

Équilibrer une équation consiste à ajuster les coefficients stœchiométriques pour respecter les lois de conservation :

  • Conservation des éléments chimiques : Le nombre d'atomes de chaque élément doit être le même dans les réactifs et les produits.
  • Conservation de la charge électrique : La charge électrique totale doit être la même dans les réactifs et les produits (essentiel pour les réactions ioniques et rédox).

Exemple 1 : Conservation des éléments

Équilibrer : Fe(s) + O₂(g) → Fe₂O₃(s)

  1. Équilibrer Fe : 2 Fe(s) + O₂(g) → 1 Fe₂O₃(s)
  2. Équilibrer O : Il faut 3 O à gauche. On a O₂. On utilise un coefficient fractionnaire temporaire : 2 Fe(s) + 3/2 O₂(g) → 1 Fe₂O₃(s)
  3. Rendre les coefficients entiers (multiplier par 2) : 4 Fe(s) + 3 O₂(g) → 2 Fe₂O₃(s)

Vérification : Gauche (4 Fe, 6 O) ; Droite (2x2=4 Fe, 2x3=6 O). C'est équilibré.

Exemple 2 : Conservation de la charge (demi-équation rédox)

Équilibrer en milieu acide : MnO₄⁻(aq) → Mn²⁺(aq)

  1. Élément principal (Mn) : Déjà équilibré. 1 MnO₄⁻ → 1 Mn²⁺
  2. Oxygène (O) : Ajouter H₂O. 1 MnO₄⁻ → 1 Mn²⁺ + 4 H₂O
  3. Hydrogène (H) : Ajouter H⁺ (milieu acide). 1 MnO₄⁻ + 8 H⁺ → 1 Mn²⁺ + 4 H₂O
  4. Charge : Gauche (-1 + 8x(+1) = +7) ; Droite (+2 + 4x0 = +2). Ajouter des e⁻ du côté le plus positif (gauche) pour équilibrer. (+7) - (+2) = 5 e⁻.
    5. MnO₄⁻(aq) + 8 H⁺(aq) + 5 e⁻ → Mn²⁺(aq) + 4 H₂O(l)

Vérification : Éléments (1 Mn, 4 O, 8 H de chaque côté). Charge (Gauche: -1+8-5 = +2 ; Droite: +2). C'est équilibré.

1.2 Suivi Quantitatif : L'Avancement

L'avancement de la réaction, noté x, est une grandeur exprimée en moles (mol) qui permet de suivre l'évolution des quantités de matière des réactifs et des produits au cours de la transformation.

Pour une réaction aA + bB → cC + dD :

  • Quantité de réactif A à l'instant t : n(A)t = n(A)₀ - ax
  • Quantité de réactif B à l'instant t : n(B)t = n(B)₀ - bx
  • Quantité de produit C à l'instant t : n(C)t = n(C)₀ + cx (souvent n(C)₀ = 0)
  • Quantité de produit D à l'instant t : n(D)t = n(D)₀ + dx (souvent n(D)₀ = 0)

Où n₀ désigne la quantité de matière initiale.

1.3 Le Tableau d'Avancement

Le tableau d'avancement est un outil essentiel pour organiser le bilan de matière d'une réaction chimique.

Exemple : Réaction 2 H₂(g) + O₂(g) → 2 H₂O(l)

État initial : 3 mol de H₂, 2 mol de O₂, 0 mol de H₂O.

Équation 2 H₂(g) + O₂(g) → 2 H₂O(l)
État (Avancement) Quantités de matière (mol)
État Initial (EI)
(x = 0)		 n(H₂)₀ = 3 n(O₂)₀ = 2 n(H₂O)₀ = 0
État Intermédiaire
(x)		 n(H₂) = 3 - 2x n(O₂) = 2 - x n(H₂O) = 0 + 2x
État Final (EF)
(x = xfinal)		 n(H₂)f = 3 - 2xf n(O₂)f = 2 - xf n(H₂O)f = 2xf

Ce tableau permet de calculer les quantités de chaque espèce à n'importe quel stade de la réaction si l'on connaît l'avancement x.

Partie 2 : État Final d'un Système Chimique

2.1 Réactif Limitant et Avancement Maximal (xmax)

Lors d'une transformation chimique, la réaction s'arrête lorsque l'un des réactifs est entièrement consommé (sauf cas d'équilibre).

Le réactif limitant est le réactif qui est entièrement consommé en premier et qui provoque l'arrêt de la réaction (dans le cas d'une transformation considérée comme totale).

L'avancement maximal, noté xmax, est la valeur de l'avancement lorsque le réactif limitant est entièrement consommé.

Comment trouver le réactif limitant et xmax ?

Méthode 1 : Tester les hypothèses

  1. Faire l'hypothèse que le réactif A est limitant : calculer l'avancement xmax,A qui annule sa quantité (n(A)₀ - ax = 0 ⇒ xmax,A = n(A)₀ / a).
  2. Faire l'hypothèse que le réactif B est limitant : calculer l'avancement xmax,B qui annule sa quantité (n(B)₀ - bx = 0 ⇒ xmax,B = n(B)₀ / b).
  3. Comparer les valeurs : L'avancement maximal réel xmax est la plus petite des valeurs calculées (xmax = min(xmax,A, xmax,B, ...)).
  4. Le réactif qui correspond à cette plus petite valeur est le réactif limitant.

Méthode 2 : Comparer les rapports n₀/ν

  1. Calculer pour chaque réactif le rapport entre sa quantité initiale et son coefficient stœchiométrique : n(A)₀ / a, n(B)₀ / b, ...
  2. Le réactif qui a le plus petit rapport est le réactif limitant.
  3. La valeur de ce plus petit rapport est égale à l'avancement maximal xmax.

Reprenons l'exemple : 2 H₂(g) + O₂(g) → 2 H₂O(l) avec n(H₂)₀ = 3 mol et n(O₂)₀ = 2 mol.

Méthode 1 :

  • Si H₂ limitant : 3 - 2x = 0 ⇒ x = 3/2 = 1,5 mol.
  • Si O₂ limitant : 2 - x = 0 ⇒ x = 2 mol.

Le plus petit est 1,5 mol. Donc xmax = 1,5 mol et H₂ est le réactif limitant.

Méthode 2 :

  • Rapport pour H₂ : n(H₂)₀ / 2 = 3 / 2 = 1,5 mol.
  • Rapport pour O₂ : n(O₂)₀ / 1 = 2 / 1 = 2 mol.

Le plus petit rapport est 1,5 mol (pour H₂). Donc xmax = 1,5 mol et H₂ est le réactif limitant.

2.1.2 Mélange Stœchiométrique

Un mélange initial de réactifs est dit stœchiométrique si les quantités initiales sont exactement dans les proportions indiquées par les coefficients stœchiométriques.

Condition : n(A)₀ / a = n(B)₀ / b = ...

Dans un mélange stœchiométrique, si la réaction est totale, tous les réactifs sont entièrement consommés à l'état final (il n'y a pas de réactif limitant unique, ou on peut dire qu'ils sont tous limitants).

2.1.3 Composition de l'État Final (Réaction Totale)

Une fois xmax déterminé, on peut calculer les quantités de matière de toutes les espèces à l'état final en remplaçant x par xmax dans la ligne "État Final" du tableau d'avancement.

nfinal = n₀ ± ν * xmax

La quantité finale du réactif limitant est nulle (par définition). Les autres réactifs (en excès) ont une quantité finale non nulle.

Pour l'exemple précédent (xmax = 1,5 mol) :

  • n(H₂)f = 3 - 2 * 1,5 = 3 - 3 = 0 mol (limitant)
  • n(O₂)f = 2 - 1 * 1,5 = 0,5 mol (en excès)
  • n(H₂O)f = 2 * 1,5 = 3 mol (produit formé)

2.2 Transformations Non Totales et État d'Équilibre

Toutes les réactions ne sont pas totales. Certaines s'arrêtent avant l'épuisement du réactif limitant, atteignant un état d'équilibre.

2.2.1 Avancement Final (xf) et Taux d'Avancement (τ)

L'avancement final, noté xf (ou parfois xéq à l'équilibre), est la valeur réelle de l'avancement atteinte à l'état final du système.

Le taux d'avancement final, noté τ (tau), compare l'avancement final à l'avancement maximal :

τ = xf / xmax
  • 0 ≤ τ ≤ 1
  • Si τ = 1 (soit xf = xmax), la transformation est totale.
  • Si τ < 1 (soit xf < xmax), la transformation est non totale ou limitée. Elle conduit à un état d'équilibre.

Le taux d'avancement final peut être déterminé expérimentalement en mesurant la quantité d'un réactif restant ou d'un produit formé à l'état final.

Estérification : CH₃COOH + C₂H₅OH ⇌ CH₃COOC₂H₅ + H₂O

Initial : 1 mol acide, 1 mol alcool (mélange stœchiométrique, xmax = 1 mol).

Final : On mesure 0,67 mol d'ester formé.

Calcul de xf : n(ester)f = 1 * xf ⇒ xf = 0,67 mol.

Calcul de τ : τ = xf / xmax = 0,67 mol / 1,0 mol = 0,67 (ou 67%).

Comme τ < 1, la réaction est non totale.

2.2.2 L'État d'Équilibre Chimique

Un système chimique atteint l'état d'équilibre lorsque les quantités de matière (et donc les concentrations) des réactifs et des produits n'évoluent plus macroscopiquement au cours du temps.

Caractéristiques :

  • Dynamique : À l'échelle microscopique, les réactions directe (réactifs → produits) et inverse (produits → réactifs) continuent de se produire, mais à des vitesses égales.
  • Coexistence : Tous les réactifs et produits de l'équation bilan coexistent dans le système à l'état final. Aucun réactif (même le limitant théorique) n'est totalement consommé.

2.3 Quotient de Réaction (Qr)

Le quotient de réaction permet de caractériser l'état d'un système chimique à un instant donné, qu'il soit ou non à l'équilibre.

Pour une réaction générique : a A + b B ⇌ c C + d D

Le quotient de réaction, noté Qr, est une grandeur sans dimension définie par :

Qr = (aCc * aDd) / (aAa * aBb)

Où ai représente l'activité de l'espèce i.

Approximations usuelles pour l'activité :

  • Soluté en solution aqueuse diluée : ai = [i] / C⁰, où [i] est la concentration molaire (mol/L) et C⁰ = 1 mol/L (concentration standard).
  • Gaz : ai = Pi / P⁰, où Pi est la pression partielle du gaz (bar ou Pa) et P⁰ = 1 bar ou 10⁵ Pa (pression standard).
  • Solide pur : ai = 1.
  • Liquide pur (y compris le solvant eau si en large excès) : ai = 1.

Les solides et les liquides purs (ou solvant majoritaire comme l'eau) n'apparaissent jamais dans l'expression de Qr car leur activité vaut 1.

Exemple 1 : Réaction en solution aqueuse

Fe³⁺(aq) + SCN⁻(aq) ⇌ [Fe(SCN)]²⁺(aq)

Qr = [ [Fe(SCN)]²⁺ ] / ( [Fe³⁺] * [SCN⁻] )
(En supposant C⁰ = 1 mol/L)

Exemple 2 : Réaction avec solide et gaz

CaCO₃(s) ⇌ CaO(s) + CO₂(g)

Qr = (aCaO * aCO₂) / aCaCO₃ = (1 * (PCO₂/P⁰)) / 1 = PCO₂ / P⁰

Exemple 3 : Réaction où l'eau est produit (mais pas solvant majoritaire) ou réactif

CH₃COOH(aq) + C₂H₅OH(aq) ⇌ CH₃COOC₂H₅(aq) + H₂O(l)

Si réalisée en solution aqueuse diluée, H₂O est le solvant :

Qr = [CH₃COOC₂H₅] / ([CH₃COOH] * [C₂H₅OH])

Si réalisée avec peu ou pas d'eau initialement (ex: estérification directe), l'eau formée est un produit dont l'activité (ou fraction molaire) intervient. L'expression peut être plus complexe ou approximée différemment selon le contexte.

2.4 Constante d'Équilibre (K)

La constante d'équilibre, notée K (ou K(T) pour souligner sa dépendance à la température), est la valeur que prend le quotient de réaction Qr lorsque le système a atteint l'état d'équilibre.

K = Qr, éq

Caractéristiques de K :

  • Elle est sans dimension.
  • Elle est associée à une équation de réaction écrite dans un sens donné. Si on inverse l'équation, K' = 1/K.
  • Elle ne dépend que de la température pour une réaction donnée. Elle est indépendante des quantités initiales, de la pression totale (sauf influence sur T), de la présence d'un catalyseur.

Interprétation de la valeur de K :

  • K >> 1 (conventionnellement K > 10⁴) : L'équilibre est fortement déplacé dans le sens direct (formation des produits). La réaction est considérée comme quasi-totale. τ ≈ 1.
  • K << 1 (conventionnellement K < 10⁻⁴) : L'équilibre est fortement déplacé dans le sens inverse (formation des réactifs). La réaction dans le sens direct est très limitée. τ ≈ 0.
  • K proche de 1 (ex: entre 10⁻⁴ et 10⁴) : L'équilibre correspond à un mélange significatif de réactifs et de produits. La réaction est limitée et nettement réversible.

2.5 Critère d'Évolution Spontanée

La comparaison de Qr à un instant t avec la constante d'équilibre K permet de prévoir le sens d'évolution spontanée du système.

Critère d'évolution spontanée :

  • Si Qr < K : Le système n'est pas à l'équilibre. Il évolue spontanément dans le sens direct (→) pour augmenter Qr jusqu'à K.
  • Si Qr > K : Le système n'est pas à l'équilibre. Il évolue spontanément dans le sens inverse (←) pour diminuer Qr jusqu'à K.
  • Si Qr = K : Le système est à l'état d'équilibre. Il n'évolue plus macroscopiquement.

Réaction N₂(g) + 3 H₂(g) ⇌ 2 NH₃(g). K = 6,0 x 10⁻² à 500°C.

À t, P(N₂) = 10 bar, P(H₂) = 5 bar, P(NH₃) = 1 bar. (P⁰ = 1 bar).

Calcul de Qr :

Qr = (PNH₃/P⁰)² / [ (PN₂/P⁰)¹ * (PH₂/P⁰)³ ] = (1)² / [ (10)¹ * (5)³ ] = 1 / (10 * 125) = 1 / 1250 ≈ 8,0 x 10⁻⁴

Comparaison : Qr (8,0 x 10⁻⁴) < K (6,0 x 10⁻²).

Conclusion : Le système va évoluer spontanément dans le sens direct (formation de NH₃).

Partie 3 : Les Réactions d'Oxydoréduction

3.1 Définitions Clés

  • Une réaction d'oxydoréduction (rédox) est une réaction chimique caractérisée par un transfert d'électrons entre les réactifs.
  • Un oxydant (Ox) est une espèce chimique capable de capter un ou plusieurs électrons. Il est réduit au cours de la réaction.
  • Un réducteur (Red) est une espèce chimique capable de céder un ou plusieurs électrons. Il est oxydé au cours de la réaction.
  • Un couple oxydant/réducteur (Ox/Red) est constitué des deux espèces qui se transforment l'une en l'autre par gain ou perte d'électrons. Il est associé à une demi-équation électronique :
Ox + n e⁻ ⇌ Red

Oxydation et réduction sont toujours simultanées : les électrons cédés par le réducteur sont captés par l'oxydant.

Oxydants et Réducteurs Usuels :

  • Oxydants : O₂, Cl₂, Br₂, I₂, MnO₄⁻, Cr₂O₇²⁻, H₂O₂, Fe³⁺, Cu²⁺, H⁺...
  • Réducteurs : Métaux (Zn, Fe, Cu, Al, Na...), H₂, C, I⁻, Br⁻, Cl⁻, Fe²⁺, S₂O₃²⁻...
  • Caractère réducteur des métaux du bloc s (Groupes 1 & 2) : Ces métaux (Na, K, Mg, Ca...) ont peu d'électrons de valence (1 ou 2) qu'ils cèdent très facilement pour former des cations stables (configuration de gaz noble), d'où leur fort pouvoir réducteur.

3.2 Établir l'Équation d'une Réaction Rédox

La méthode consiste à combiner les deux demi-équations électroniques des couples mis en jeu.

  1. Identifier les réactifs (l'oxydant d'un couple et le réducteur de l'autre couple).
  2. Écrire les demi-équations correspondantes (réduction pour l'oxydant, oxydation pour le réducteur). Assurez-vous qu'elles sont bien équilibrées (éléments, O, H, charge).
  3. Multiplier chaque demi-équation par un coefficient entier approprié pour que le nombre d'électrons cédés soit égal au nombre d'électrons captés.
  4. Additionner membre à membre les deux demi-équations ajustées.
  5. Simplifier les espèces (notamment les électrons, H⁺, H₂O) qui apparaissent des deux côtés de l'équation bilan.

Réaction entre l'ion permanganate MnO₄⁻ (couple MnO₄⁻/Mn²⁺) et l'ion fer(II) Fe²⁺ (couple Fe³⁺/Fe²⁺) en milieu acide.

  1. Réactifs : Oxydant = MnO₄⁻ ; Réducteur = Fe²⁺.
  2. Demi-équations :
    • Réduction : MnO₄⁻ + 8 H⁺ + 5 e⁻ → Mn²⁺ + 4 H₂O
    • Oxydation : Fe²⁺ → Fe³⁺ + 1 e⁻
  3. Ajustement des électrons : PPCM(5, 1) = 5. Multiplier la 2ème demi-équation par 5.
    • MnO₄⁻ + 8 H⁺ + 5 e⁻ → Mn²⁺ + 4 H₂O (x 1)
    • 5 Fe²⁺ → 5 Fe³⁺ + 5 e⁻ (x 5)
  4. Addition : (MnO₄⁻ + 8 H⁺ + 5 e⁻) + (5 Fe²⁺) → (Mn²⁺ + 4 H₂O) + (5 Fe³⁺ + 5 e⁻)
  5. Simplification (des 5 e⁻) :
MnO₄⁻(aq) + 5 Fe²⁺(aq) + 8 H⁺(aq) → Mn²⁺(aq) + 5 Fe³⁺(aq) + 4 H₂O(l)

3.3 Sens du Transfert d'Électrons

Le transfert d'électrons se fait toujours du réducteur (qui s'oxyde) vers l'oxydant (qui se réduit).

On peut déterminer ce sens :

  • À partir de l'équation bilan : Identifier l'espèce qui perd des électrons (son nombre d'oxydation augmente ou elle passe de Red à Ox) et celle qui en gagne (son nombre d'oxydation diminue ou elle passe de Ox à Red).
  • À partir de données expérimentales : Observer quelle espèce disparaît et quelle nouvelle espèce apparaît (ex: dépôt métallique, changement de couleur...). Voir Fiche TP 41 & 45.
  • À partir des potentiels standard (E⁰) : L'oxydant du couple ayant le potentiel standard le plus élevé réagit spontanément avec le réducteur du couple ayant le potentiel standard le moins élevé. (Règle du gamma).

Partie 4 : Les Piles Électrochimiques

Les piles sont des dispositifs qui exploitent l'énergie d'une réaction d'oxydoréduction spontanée pour générer un courant électrique.

4.1 Constitution et Principe

  • Deux demi-piles : Chaque demi-pile contient un couple Ox/Red (une électrode métallique ou inerte plongeant dans une solution électrolytique).
  • Séparation physique : Les réactifs sont séparés pour forcer les électrons à passer par un circuit extérieur.
  • Jonction ionique (pont salin ou paroi poreuse) : Assure la fermeture du circuit électrique par migration des ions et maintient l'électroneutralité des solutions.
  • Circuit extérieur : Relie les deux électrodes et permet la circulation des électrons (production de courant).
Dans une pile (générateur) :
  • L'Anode est le siège de l'oxydation. C'est le Pôle Négatif (-).
  • La Cathode est le siège de la réduction. C'est le Pôle Positif (+).
  • Les électrons circulent de l'anode (-) vers la cathode (+) dans le circuit extérieur.
  • Le courant conventionnel circule de la cathode (+) vers l'anode (-) dans le circuit extérieur.
  • Dans le pont salin, les cations migrent vers la cathode (+), les anions migrent vers l'anode (-).

Pile Daniell : Zn(s) | Zn²⁺(aq) || Cu²⁺(aq) | Cu(s)

  • Couple Zn²⁺/Zn (E⁰ = -0,76 V) ; Couple Cu²⁺/Cu (E⁰ = +0,34 V)
  • E⁰(Cu²⁺/Cu) > E⁰(Zn²⁺/Zn)
  • Anode (-) : Électrode de Zinc. Oxydation : Zn(s) → Zn²⁺(aq) + 2 e⁻
  • Cathode (+) : Électrode de Cuivre. Réduction : Cu²⁺(aq) + 2 e⁻ → Cu(s)
  • Réaction globale spontanée : Zn(s) + Cu²⁺(aq) → Zn²⁺(aq) + Cu(s)
  • Électrons : Zn → Cu (circuit extérieur)
  • Pont salin (ex: K⁺, NO₃⁻) : K⁺ → compartiment Cu ; NO₃⁻ → compartiment Zn.

4.2 Caractéristiques d'une Pile

  • Polarité : Déterminée par la comparaison des potentiels des électrodes (le plus élevé est le pôle +).
  • Tension à vide (Force électromotrice, f.e.m.) : Différence de potentiel entre la cathode et l'anode lorsque la pile ne débite pas de courant.
    Epile = Ecathode - Eanode
    Dans les conditions standard : E⁰pile = E⁰cathode - E⁰anode. Une f.e.m. positive correspond à une réaction globale spontanée.
  • Capacité électrique (Qmax) : Quantité maximale d'électricité (en Coulombs, C) que la pile peut fournir avant d'être usée (épuisement du réactif limitant).
    Qmax = ne⁻ * xmax * F
    Où ne⁻ est le nombre d'électrons échangés dans l'équation bilan pour un avancement de 1 mole, xmax est l'avancement maximal de la pile, et F est la constante de Faraday (F ≈ 96500 C/mol).
  • Usure : La pile s'use car les réactifs sont consommés et les produits formés. Sa tension diminue au cours de son fonctionnement. Lorsque Qmax est atteint (x = xmax), la pile est "plate".

Conclusion

L'évolution spontanée d'un système chimique tend vers un état de plus grande stabilité, qui est l'état d'équilibre. La comparaison du quotient de réaction Qr à la constante d'équilibre K permet de prédire ce sens d'évolution.

Les transformations peuvent être totales (τ=1) ou limitées (τ<1). Le suivi quantitatif via le tableau d'avancement est essentiel pour déterminer l'état final.

Les réactions d'oxydoréduction sont un cas majeur de transformations spontanées, dont l'énergie peut être convertie en énergie électrique dans les piles.

La compréhension de ces concepts est fondamentale pour maîtriser et prévoir le comportement des systèmes chimiques.
