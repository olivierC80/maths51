Connaître les caractéristiques des acide et base de Brönsted

1. Dans la théorie de Brönsted, comment appelle-t-on l'espèce chimique susceptible de céder un ion hydrogène \( H^+ \)?

Un acide
Une base
Un cation
Un anion

Réponse correcte : Un acide

Dans la théorie de Brönsted, un acide est une espèce chimique pouvant céder un ion hydrogène \( H^+ \).

2. Dans la théorie de Brönsted, comment appelle-t-on l'espèce chimique susceptible de capter un ion hydrogène \( H^+ \)?

Une base
Un acide
Un cation
Un anion

Réponse correcte : Une base

Dans la théorie de Brönsted, une base est une espèce chimique pouvant capter un ion hydrogène \( H^+ \).

3. Vrai ou faux ? Un ion \( H^+ \) est un proton.

Vrai
Faux

Réponse correcte : Vrai

4. Vrai ou faux ? Un acide est nécessairement un cation.

Faux
Vrai

Réponse correcte : Faux

Faux. Un acide peut être un cation (ex: \( NH_4^+ \)) ou une espèce neutre (ex: \( CH_3COOH \)) ou même un anion (ex: \( HSO_4^- \)).

5. Vrai ou faux ? Une base est nécessairement un anion.

Faux
Vrai

Réponse correcte : Faux

Faux. Une base peut être un anion (ex: \( OH^- \), \( CH_3COO^- \)) ou une espèce neutre (ex: \( NH_3 \), \( H_2O \)).
