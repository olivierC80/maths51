Déterminer si une espèce chimique est amphotère à l'aide de ses réactions acidobasiques

1. L'ion \\( HSO_4^- \\) intervient dans les deux demi-équations acide/base suivantes :
\\[ H_2SO_4(aq) \longrightarrow HSO_4^-(aq) + H^+(aq) \\] \\[ HSO_4^-(aq) \longrightarrow SO_4^{2-}(aq) + H^+(aq) \\] Quelle affirmation caractérise l'ion \\( HSO_4^- \\) ?

\\( HSO_4^- \\) joue uniquement le rôle d'un acide.
\\( HSO_4^- \\) joue uniquement le rôle d'une base.
\\( HSO_4^- \\) est une espèce amphotère car elle peut jouer le rôle d'un acide et d'une base.
\\( HSO_4^- \\) n'est ni un acide ni une base.

Réponse correcte : \\( HSO_4^- \\) est une espèce amphotère car elle peut jouer le rôle d'un acide et d'une base.

Dans la demi-équation \\( H_2SO_4(aq) \longrightarrow HSO_4^-(aq) + H^+(aq) \\), \\( HSO_4^- \\) est la base conjuguée de l'acide sulfurique dans le couple acide base \\( H_2SO_4/HSO_4^- \\).
Dans la demi-équation \\( HSO_4^-(aq) \longrightarrow SO_4^{2-}(aq) + H^+(aq) \\), \\( HSO_4^- \\) est l'acide du couple \\( HSO_4^-/SO_4^{2-} \\).
Cette espèce chimique peut donc jouer le rôle d'un acide et le rôle d'une base. C'est donc une espèce amphotère.
