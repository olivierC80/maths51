Déterminer le schéma de Lewis d'un acide conjugué d'amine

1. Quel est le schéma de Lewis de l'acide conjugué de l'éthanamine (ion éthylammonium) ?

  H   H   H
  |   |   | 
H-C---C---N̈⁺-H
  |   |   |
  H   H   H
            
  H   H   H
  |   |   |
H-C---C---N-H
  |   |   |
  H   H   H
            
  H   H   H   H
  |   |   |   |
H-C---C---C---N⁺-H ?
  |   |   |   |
  H   H   H   H
            
      H   H
      |   | 
    H-C---N⁺-H
      |   |
      H   H
Réponse correcte : 
  H   H   H
  |   |   | +
H-C---C---N-H
  |   |   |
  H   H   H
La formule d'une amine est R–NH₂, R étant le radical alkyle d'une chaîne carbonée. Dans le cas de l'éthanamine, il s'agit d'un groupe éthyle, c'est-à-dire CH₃–CH₂–, donc la formule de l'éthanamine est : CH₃–CH₂–NH₂.
De plus, toutes les amines R–NH₂ ont des propriétés basiques et leurs acides conjugués sont les ions ammonium de formule R–NH₃⁺. Donc l'acide associé à l'éthanamine a pour formule : CH₃–CH₂–NH₃⁺.
Par conséquent, son schéma de Lewis est celui présenté ci-dessus (avec une charge positive formelle sur l'azote).

4. Quel est le schéma de Lewis de l'acide conjugué de la 3-méthyl-1-butylamine ?

Réponse correcte : 
      H
      |
  H H-C-H H   H
  |   |   |   | +
H-C---C---C---C-N-H
  |   |   |   | |
  H   H   H   H H
La formule d'une amine est R–NH₂. Dans le cas de la 3-méthyl-1-butylamine, le groupe R est 3-méthylbutyl (CH₃–CH(CH₃)–CH₂–CH₂–). La formule de l'amine est CH₃–CH(CH₃)–CH₂–CH₂–NH₂.
L'acide conjugué (ion ammonium) a pour formule R–NH₃⁺, soit CH₃–CH(CH₃)–CH₂–CH₂–NH₃⁺.
Par conséquent, son schéma de Lewis est celui présenté ci-dessus.
