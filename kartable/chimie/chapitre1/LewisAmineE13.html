Déterminer le schéma de Lewis d'une amine

1. Quel est le schéma de Lewis de l'éthanamine ?

  H   H
  |   |
H-C---C---N-H
  |   |   |
  H   H   H
            
  H   H   H
  |   |   |
H-C---C---N=O-H ?
  |   |
  H   H
            
  H   H   H
  |   |   |
H-C---C---C---N-H ?
  |   |   |   |
  H   H   H   H
            
  H   H
  |   |
H-C---C=N: ?
  |   |
  H   H
Réponse correcte : 
  H   H
  |   |
H-C---C---N̈-H
  |   |   |
  H   H   H
Une amine présente un groupe fonctionnel –NH₂.
Ici, le préfixe éthan- implique que la molécule contient une chaîne avec 2 atomes de carbone.
De plus, on sait que, dans les schémas de Lewis, l'atome d'azote porte un doublet non liant (représenté par N̈ ou :) et que les atomes de carbone et d'hydrogène n'en portent pas.
Le schéma correspondant est celui présenté ci-dessus.

2. Quel est le schéma de Lewis du propan-2-amine ?

  H       H
            
  H H-N̈-H H
  | | | |

  | | |

    | | |
  H-C-C-C-H
    | | |
    H H H

  H   H   H
Réponse correcte : 
  H H-N̈-H H
  |   |   |
H-C---C---C-H
  |       |
  H       H
Une amine présente un groupe fonctionnel –NH₂.
Ici, le préfixe propan- implique que la molécule contient une chaîne avec 3 atomes de carbone.
Ici, le chiffre 2 devant le suffixe amine indique que le groupement –NH₂ se situe sur le carbone numéro 2.
De plus, on sait que, dans les schémas de Lewis, l'atome d'azote porte un doublet non liant et que les atomes de carbone et d'hydrogène n'en portent pas.
Le schéma correspondant est le suivant : 
  H H-N̈-H H
  |   |   |
H-C---C---C-H
  |       |
  H       H
