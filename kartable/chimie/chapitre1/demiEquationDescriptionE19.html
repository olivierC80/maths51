Déterminer la demi-équation acidobasique à l'aide d'une description physique de la réaction

1. On étudie une réaction durant laquelle l'acide éthanoïque (\\( \ce{CH3COOH} \\)) cède un ion hydrogène pour former l'ion éthanoate.

Quelle est la demi-équation acidobasique correspondante ?

\\( \ce{CH3COOH + H+ = CH3COO^-} \\)
\\( \ce{CH3COOH + CH3COO^- = H+} \\)
\\( \ce{H+ + CH3COO^- = CH3COOH} \\)
\\( \ce{CH3COOH = H+ + CH3COO^-} \\)

Réponse correcte : \\( \ce{CH3COOH = H+ + CH3COO^-} \\)

À chaque couple acide-base correspond une demi-équation acidobasique illustrant le transfert d'ions hydrogène \\( \ce{H^+} \\) entre les deux espèces chimiques conjuguées. Pour un couple acide-base noté \\( \ce{AH / A^-} \\), la demi-équation acidobasique s'écrit (généralement avec une double flèche d'équilibre) :
\\( \ce{AH <=> A^- + H+} \\) Dans le cas présent, l'acide est l'acide éthanoïque \\( \ce{CH3COOH} \\) et sa base conjuguée est l'ion éthanoate \\( \ce{CH3COO^-} \\).
La demi-équation correspondante est \\( \ce{CH3COOH = H+ + CH3COO^-} \\) (ou \\( \ce{CH3COOH <=> H+ + CH3COO^-} \\)).

2. On étudie une réaction durant laquelle la méthylamine (\\( \ce{CH3-NH2} \\)) capte un ion hydrogène pour former l'ion méthylammonium.

Quelle est la demi-équation acidobasique correspondante ?

\\( \ce{CH3-NH2 + H+ = CH3-NH3^+} \\)
\\( \ce{CH3-NH2 = CH3-NH3^+ - H+} \\)
\\( \ce{CH3-NH2 = CH3-NH3^+ + H+} \\)
\\( \ce{CH3-NH2 - H+ = CH3-NH3^+} \\)

Réponse correcte : \\( \ce{CH3-NH2 + H+ = CH3-NH3^+} \\)

À chaque couple acide-base correspond une demi-équation acidobasique. Pour un couple acide-base noté \\( \ce{AH^+ / A} \\), la demi-équation peut s'écrire sous la forme :
\\( \ce{A + H^+ <=> AH^+} \\) Dans le cas présent, la base est la méthylamine (car on sait que cette molécule capte un ion hydrogène) \\( \ce{CH3-NH2} \\) et son acide conjugué est l'ion méthylammonium \\( \ce{CH3-NH3^+} \\).
La demi-équation correspondante est \\( \ce{CH3-NH2 + H+ = CH3-NH3^+} \\) (ou \\( \ce{CH3-NH2 + H^+ <=> CH3-NH3^+} \\)). (Note : La forme standard est Acide = Base + H⁺, soit \\( \ce{CH3NH3+ = CH3NH2 + H+} \\)).

3. On étudie une réaction durant laquelle l'acide sulfurique (\\( \ce{H2SO4} \\)) cède deux ions hydrogène pour former l'ion sulfate.

Quelle est la demi-équation acidobasique correspondante ?

\\( \ce{H2SO4 = SO4^{2-} + 2H+} \\)
\\( \ce{H2SO4 = SO4^{2-} + H+} \\)
\\( \ce{H2SO4 + 2H+ = SO4^{2-}} \\)
\\( \ce{HSO4^- = SO4^{2-} + 2H+} \\)

Réponse correcte : \\( \ce{H2SO4 = SO4^{2-} + 2H+} \\)

À chaque couple acide-base correspond une demi-équation acidobasique. Pour un couple acide-base noté \\( \ce{AH2 / A^{2-}} \\) (diacide), la demi-équation globale s'écrit :
\\( \ce{AH2 <=> A^{2-} + 2H+} \\) Dans le cas présent, l'acide est l'acide sulfurique \\( \ce{H2SO4} \\) et sa base conjuguée (après perte de 2 protons) est l'ion sulfate \\( \ce{SO4^{2-}} \\).
La demi-équation correspondante est \\( \ce{H2SO4 = SO4^{2-} + 2H+} \\) (ou \\( \ce{H2SO4 <=> SO4^{2-} + 2H+} \\)). (Note: Ceci est une simplification, la déprotonation se fait en deux étapes via \\( \ce{HSO4^-} \\)).

4. On étudie une réaction durant laquelle l'ion ammonium (\\( \ce{NH4^+} \\)) cède un ion hydrogène pour former l'ammoniaque.

Quelle est la demi-équation acidobasique correspondante ?

\\( \ce{NH4+ + H+ = NH3} \\)
\\( \ce{NH3+ + H+ = NH4} \\)
\\( \ce{NH4+ = NH3 + H+} \\)
\\( \ce{2NH3 + H+ = 2NH4+} \\)

Réponse correcte : \\( \ce{NH4+ = NH3 + H+} \\)

À chaque couple acide-base correspond une demi-équation acidobasique illustrant le transfert d'ions hydrogène \\( \ce{H^+} \\). Pour un couple acide-base noté \\( \ce{AH^+ / A} \\), la demi-équation acidobasique s'écrit :
\\( \ce{AH^+ <=> A + H+} \\) Dans le cas présent, l'acide est l'ion ammonium \\( \ce{NH4^+} \\) et sa base conjuguée est l'ammoniaque \\( \ce{NH3} \\).
La demi-équation correspondante est \\( \ce{NH4+ = NH3 + H+} \\) (ou \\( \ce{NH4+ <=> NH3 + H+} \\)).

5. On étudie une réaction durant laquelle l'ion hydrogénocarbonate (\\( \ce{HCO3^-} \\)) cède un ion hydrogène pour former l'ion carbonate.

Quelle est la demi-équation acidobasique correspondante ?

\\( \ce{HCO3^- + H+ = CO3^{2-}} \\)
\\( \ce{H2CO3 = CO3^{2-} + H+} \\)
\\( \ce{HCO3 = CO3^{2-} + H+} \\)
\\( \ce{HCO3^- = CO3^{2-} + H+} \\)

Réponse correcte : \\( \ce{HCO3^- = CO3^{2-} + H+} \\)

À chaque couple acide-base correspond une demi-équation acidobasique. Pour un couple acide-base noté \\( \ce{AH^- / A^{2-}} \\), la demi-équation acidobasique s'écrit :
\\( \ce{AH^- <=> A^{2-} + H+} \\) Dans le cas présent, l'acide est l'ion hydrogénocarbonate \\( \ce{HCO3^-} \\) et sa base conjuguée est l'ion carbonate \\( \ce{CO3^{2-}} \\).
La demi-équation correspondante est \\( \ce{HCO3^- = CO3^{2-} + H+} \\) (ou \\( \ce{HCO3^- <=> CO3^{2-} + H+} \\)).
