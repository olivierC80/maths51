Déterminer le schéma de Lewis d'un ion carboxylate

1. Quel est le schéma de Lewis de l'ion propanoate ?

  H   H   O
  |   |  //
H-C---C---C
  |   |   \
  H   H    O-H
            
  H   H   O
  |   |  //
H-C---C---C
  |   |   \
  H   H    O(-)
            
  H   H   O(-)
  |   |  /
H-C---C---C
  |   |  \\
  H   H   O
            
  H   H   O(-)
  |   |   |
H-C---C---C=O
  |   |
  H   H
Réponse correcte : 
  H   H   O
  |   |  //
H-C---C---C
  |   |   \
  H   H    O(-)
Un ion carboxylate est la base conjuguée d'un acide carboxylique et présente un groupe fonctionnel \\( -COO^- \\).
Ici, le préfixe propan- implique que la molécule contient une chaîne avec 3 atomes de carbone.
De plus, on sait que dans les schémas de Lewis :
  • les atomes d'hydrogène établissent une seule liaison covalente et ne portent pas de doublets non liants;
  • les atomes de carbone établissent quatre liaisons covalentes et ne portent pas de doublets non liants;
  • les atomes d'oxygène établissent normalement deux liaisons (ou une liaison double) et portent deux doublets non liants;
  • l'atome d'oxygène qui établit une seule liaison covalente (dans l'ion) porte trois doublets non liants et une charge négative. (Note: la charge est délocalisée entre les deux O).
Le schéma correspondant est celui présenté ci-dessus (montrant une forme de résonance).

2. Quel est le schéma de Lewis de l'ion méthanoate ?

  H   H   O
  |   |  //
H-C---C---C
  |   |   \
  H   H    O(-)
             
  O
 //
H-C
  \
   O(-)
             
   O(-)
 //
H-C=O
             
   O
 //

 |
 O-H
Réponse correcte : 
  O
 //
H-C
  \
   O(-)
Un ion carboxylate présente un groupe fonctionnel \\( -COO^- \\).
Ici, le préfixe méthan- implique que la molécule contient une chaîne avec 1 atome de carbone.
En respectant les règles de valence et la charge négative sur l'un des oxygènes (délocalisée), le schéma correct est celui présenté ci-dessus.
