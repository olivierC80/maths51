EXERCICE: Identifier un type de catalyse

Identifier un type de catalyse

1. En faisant circuler de l'éthanol gazeux sur du cuivre solide qui est un catalyseur, on constate la formation d'éthanal.
Quel type de catalyse est mis en jeu ici ?

  • Une catalyse hétérogène
  • Une catalyse homogène
  • Une catalyse enzymatique et hétérogène
  • Une catalyse enzymatique
Bonne réponse : Une catalyse hétérogène
L'éthanol est gazeux alors que le catalyseur est solide. Ils ne sont pas dans la même phase. Le type de catalyse mis en jeu ici est une catalyse hétérogène.

2. En faisant réagir de l'acide éthanoïque avec de l'éthanol en solution, et en présence d'acide sulfurique aussi en solution comme catalyseur, on constate que la réaction est plus rapide.
Quel type de catalyse est mis en jeu ici ?

  • Une catalyse hétérogène
  • Une catalyse homogène
  • Une catalyse enzymatique et hétérogène
  • Une catalyse enzymatique
Bonne réponse : Une catalyse homogène
Les réactifs ainsi que le catalyseur sont dans la même solution : ils sont dans la même phase. Le type de catalyse mis en jeu ici est une catalyse homogène.

3. Le (E)-11-rétinal se transforme en (Z)-11-rétinal sous l'action de la rétinal isomérase. Cette transformation permet la régénération des pigments visuels.
Quel type de catalyse est mis en jeu ici ?

  • Une catalyse hétérogène
  • Une catalyse homogène
  • Une catalyse enzymatique et hétérogène
  • Une catalyse enzymatique
Bonne réponse : Une catalyse enzymatique
Le rétinal isomérase est responsable de la transformation et c'est une enzyme. Le type de catalyse mis en jeu ici est une catalyse enzymatique.

4. Dans les pots d'échappement équipés de pots catalytiques, les gaz comme le monoxyde de carbone ou le monoxyde d'azote sont oxydés grâce à, entre autres, du platine ou du rhodium solide.
Quel type de catalyse est mis en jeu ici ?

  • Une catalyse hétérogène
  • Une catalyse homogène
  • Une catalyse enzymatique et hétérogène
  • Une catalyse enzymatique
Bonne réponse : Une catalyse hétérogène
Le platine ou le rhodium sont solides, alors que les monoxyde de carbone ou d'azote sont gazeux. Ils ne sont pas dans la même phase. Le type de catalyse mis en jeu ici est une catalyse hétérogène.

5. Afin de décomposer l'eau oxygénée utilisée pour décontaminer les lentilles de contact, on peut y tremper un fil de platine.
Quel type de catalyse est mis en jeu ici ?

  • Une catalyse hétérogène
  • Une catalyse homogène
  • Une catalyse enzymatique et hétérogène
  • Une catalyse enzymatique
Bonne réponse : Une catalyse hétérogène
Le fil de platine est solide, alors que l'eau oxygénée est liquide. Ils ne sont pas dans la même phase. Le type de catalyse mis en jeu ici est une catalyse hétérogène.

6. Afin de décomposer l'eau oxygénée utilisée pour décontaminer les lentilles de contact, on peut y ajouter une enzyme, la catalase.
Quel type de catalyse est mis en jeu ici ?

  • Une catalyse hétérogène
  • Une catalyse homogène
  • Une catalyse enzymatique et hétérogène
  • Une catalyse enzymatique
Bonne réponse : Une catalyse enzymatique
Le catalyseur utilisé ici est une enzyme. Le type de catalyse mis en jeu ici est une catalyse enzymatique.

7. La réaction entre les ions iodure I-(aq) et les ions peroxodisulfate S2O82-(aq) est catalysée par les ions fer III Fe3+(aq).
Quel type de catalyse est mis en jeu ici ?

  • Une catalyse hétérogène
  • Une catalyse homogène
  • Une catalyse enzymatique et hétérogène
  • Une catalyse enzymatique
Bonne réponse : Une catalyse homogène
Tous les ions ici sont en solution aqueuse : ils sont dans la même phase. Le type de catalyse mis en jeu ici est une catalyse homogène.
