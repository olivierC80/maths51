Différencier les types de catalyse

Kartable

1. Combien y a-t-il de types de catalyses ?

3
2
1
6

Réponse correcte : 3

Il y a trois types de catalyses.

2. Parmi les propositions suivantes, laquelle ne correspond pas à un type de catalyse ?

Micrométrique
Homogène
Hétérogène
Enzymatique

Réponse correcte : Micrométrique

Il n'existe pas de catalyse micrométrique.

3. Qu'est-ce qu'une catalyse homogène ?

Le catalyseur et les réactifs ne forment qu'une seule phase.
Le catalyseur et les réactifs ne forment des phases complémentaires.
Le catalyseur et les réactifs sont dans le même état de matière.
Le catalyseur et les réactifs ont la même concentration.

Réponse correcte : Le catalyseur et les réactifs ne forment qu'une seule phase.

Dans une catalyse homogène, le catalyseur et les réactifs ne forment qu'une seule phase. Cette phase est souvent liquide ou gazeuse.

4. Qu'est-ce qu'une catalyse hétérogène ?

Le catalyseur et les réactifs forment deux phases distinctes.
Le catalyseur et les réactifs forment deux phases non complémentaires.
Le catalyseur et les réactifs sont dans différents états de matière.
Le catalyseur et les réactifs ont des concentrations différentes.

Réponse correcte : Le catalyseur et les réactifs forment deux phases distinctes.

Dans une catalyse hétérogène, le catalyseur et les réactifs forment deux phases distinctes. Le catalyseur est souvent un solide.

5. Qu'est-ce qu'une catalyse enzymatique ?

Le catalyseur est une macromolécule organique.
Le catalyseur est une nanomolécule organique.
Le catalyseur est un alcane.
Le catalyseur est une micromolécule non organique.

Réponse correcte : Le catalyseur est une macromolécule organique.

Dans une catalyse enzymatique, le catalyseur est une macromolécule organique. Généralement, la macromolécule est une protéine appelée enzyme.
