Déterminer le comportement des électrons dans une liaison à l'aide de l'électronégativité

1. Comment se comportent les électrons de la liaison simple carbone carbone \( C_{(1)}–C_{(2)} \) ?

Donnée : On connaît l'électronégativité du carbone : \( E(C) = 2,5 \).

a) Les électrons de chaque carbone se repoussent.

b) Les électrons restent à égale distance des deux atomes.

c) Les électrons se rapprochent de \( C_{(1)} \).

d) Les électrons se rapprochent de \( C_{(2)} \).

Réponse correcte : b) Les électrons restent à égale distance des deux atomes.

L'électronégativité est l'aptitude à attirer les électrons. Or, les atomes ont ici les mêmes électronégativités (\( E(C_{(1)}) = E(C_{(2)}) = 2,5 \)). Les électrons restent à égale distance des deux atomes.

2. Comment se comportent les électrons de la liaison simple \( H–Cl \) ?

Données : On connaît les électronégativités suivantes : \( E(H) = 2,2 \) et \( E(Cl) = 3,2 \).

a) Les électrons se rapprochent du chlore.

b) Les électrons se rapprochent de l'hydrogène.

c) Les électrons restent à égale distance des deux atomes.

d) Les électrons se repoussent.

Réponse correcte : a) Les électrons se rapprochent du chlore.

L'électronégativité est l'aptitude à attirer les électrons. Or, le chlore (\( E(Cl) = 3,2 \)) a une électronégativité plus élevée que l'hydrogène (\( E(H) = 2,2 \)). Les électrons se rapprocheront du chlore.

3. Comment se comportent les électrons de la liaison simple \( H–C \) ?

Données : On connaît les électronégativités suivantes : \( E(H) = 2,2 \) et \( E(C) = 2,5 \).

a) Les électrons se rapprochent du carbone.

b) Les électrons se rapprochent de l'hydrogène.

c) Les électrons restent à égale distance des deux atomes.

d) Un électron va vers le carbone et l'autre vers l'hydrogène.

Réponse correcte : c) Les électrons restent à égale distance des deux atomes.

L'électronégativité est l'aptitude à attirer les électrons. Le carbone (\( E(C) = 2,5 \)) est légèrement plus électronégatif que l'hydrogène (\( E(H) = 2,2 \)). La différence d'électronégativité est \( \Delta E = 2,5 - 2,2 = 0,3 \). Or, le carbone et l'hydrogène n'ont pas assez d'écart d'électronégativité pour que les électrons les distinguent significativement (souvent, une liaison est considérée comme peu polarisée ou quasi apolaire si \( \Delta E \le 0,4 \)). Les électrons restent (approximativement) à égale distance des deux atomes. (Note: Bien que le carbone soit légèrement plus électronégatif, la liaison C-H est souvent considérée comme très peu polaire en chimie organique, d'où cette réponse dans ce contexte pédagogique.)

4. Comment se comportent les électrons de la liaison simple \( O–H \) ?

Données : On connaît les électronégativités suivantes : \( E(H) = 2,2 \) et \( E(O) = 3,4 \).

a) Les électrons se rapprochent de l'hydrogène.

b) Les électrons se rapprochent de l'oxygène.

c) Les électrons restent à égale distance des deux atomes.

d) Un électron va vers l'oxygène et l'autre vers l'hydrogène.

Réponse correcte : b) Les électrons se rapprochent de l'oxygène.

L'électronégativité est l'aptitude à attirer les électrons. Or, l'oxygène (\( E(O) = 3,4 \)) a une électronégativité plus élevée que l'hydrogène (\( E(H) = 2,2 \)). Les électrons se rapprochent de l'oxygène.

b) Les électrons se rapprochent de l'azote.

c) Les électrons restent à égale distance des deux atomes.

Réponse correcte : b) Les électrons se rapprochent de l'azote.

L'électronégativité est l'aptitude à attirer les électrons. Or, l'azote (\( E(N) = 3,0 \)) a une électronégativité plus élevée que l'hydrogène (\( E(H) = 2,2 \)). Les électrons se rapprochent de l'azote.
