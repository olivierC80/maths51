Kartable

EXERCICE

Identifier le catalyseur d'une transformation chimique à l'aide de données expérimentales

1. Quel est le catalyseur de la transformation suivante ?

CH3CO2H(aq) + CH3CH2OH(aq)    H+   
chaleur → CH3CO2CH2CH3(aq) + H2O(l)
  • La chaleur
  • H+
  • CH3CO2H
  • CH3CO2CH2CH3

Réponse correcte : H+

Explication : Un catalyseur permet d'accélérer une transformation chimique mais n'intervient pas dans le bilan de matière : il n'est pas un réactif dans l'équation globale de la transformation. La chaleur n'est pas de la matière mais un mode d'échange d'énergie. Ce n'est donc pas un catalyseur. Les ions H+ sont le catalyseur.

2. Quel est le catalyseur de la transformation suivante ?

CH3CH2OH(l) chaleur
Cu → CH3CHO(g) + H2(g)
  • La chaleur
  • CH3CO2H
  • Cu
  • H2

Réponse correcte : Cu

Explication : Un catalyseur permet d'accélérer une transformation chimique mais n'intervient pas dans le bilan de matière: il n'est pas un réactif dans l'équation globale de la transformation. La chaleur n'est pas de la matière mais un mode d'échange d'énergie. Ce n'est donc pas un catalyseur. Le cuivre est le catalyseur.

3. H2O2, eau oxygénée, se décompose en dioxygène et en eau avec le temps. En présence de fer solide, cette décomposition est plus rapide. En refroidissant et à l'abri de la lumière, la transformation est plus lente. Quel est le catalyseur de la transformation évoquée ?

  • L'eau oxygénée
  • Le fer solide
  • Le froid
  • La lumière

Réponse correcte : Le fer solide

Explication : Un catalyseur permet d'accélérer une transformation chimique mais n'intervient pas dans le bilan de matière: il n'est pas un réactif dans l'équation globale de la transformation. L'eau oxygénée (H2O2) est le réactif. La lumière n'est pas de la matière mais un mode d'échange d'énergie. Le froid est en réalité une perte d'énergie thermique. Ce ne sont donc pas des catalyseurs. Le fer est donc le catalyseur car sa présence accélère la réaction.

4. En présence d'eau, la rouille se forme assez facilement après réaction entre le fer et le dioxygène de l'air. Quel est le catalyseur de la transformation évoquée (formation de rouille) ?

  • La rouille
  • Le fer
  • L'eau
  • Le dioxygène

Réponse correcte : L'eau

Explication : La formation de rouille est la réaction entre le fer (Fe) et le dioxygène (O2) pour former des oxydes de fer hydratés (rouille). Le fer et le dioxygène sont les réactifs. La rouille est le produit. L'eau, bien que parfois incorporée dans la formule de la rouille, n'est pas consommée de manière stœchiométrique majeure dans la réaction d'oxydation principale mais elle est essentielle pour que la réaction se produise à une vitesse notable dans les conditions ambiantes ; elle agit comme un catalyseur (ou plus précisément, elle facilite le processus électrochimique de la corrosion). Un catalyseur permet d'accélérer une transformation chimique mais n'intervient pas (ou est régénéré) dans le bilan de matière global. L'eau est le catalyseur ici.

5. Un pot catalytique enrichi en palladium, platine ou rhodium permet par exemple la réaction suivante (simplifiée) : 2 NO(g) + ... → ...
(Le pot aide à convertir les polluants). Quel est le catalyseur de la transformation évoquée ?

  • Le rhodium (ou platine, palladium)
  • Le monoxyde d'azote (NO)
  • Le dioxygène (O2)
  • Le dioxyde d'azote (NO2)

Réponse correcte : Le rhodium (ou platine, palladium)

Explication : Le pot catalytique contient des métaux (palladium, platine, rhodium) qui accélèrent la conversion des gaz polluants (comme le NO) en gaz moins nocifs (comme N2 et O2 ou CO2 et H2O pour d'autres réactions). Ces métaux ne sont pas consommés dans la réaction globale. Un catalyseur permet d'accélérer une transformation chimique mais n'intervient pas dans le bilan de matière : il n'est pas un réactif dans l'équation globale de la transformation. Le monoxyde d'azote est un réactif. Le dioxygène peut être un réactif ou produit selon la réaction spécifique. Le dioxyde d'azote peut être un intermédiaire ou produit. Le rhodium (ainsi que le platine et le palladium) est le catalyseur.
