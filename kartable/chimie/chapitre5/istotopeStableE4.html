Déterminer si un isotope est stable

Exercice

1. D'après le diagramme de Segré ci-dessous, lequel des isotopes du carbone est stable ?
[Diagramme de Segré Image Placeholder]
Axes: Z (protons) vs N (neutrons)
Légende: Décroissance β-, β+, α, Fission, Inconnue
  • 126C
  • 116C
  • 136C
  • 146C

Réponse correcte : 126C

Explication :

Le nombre de neutrons (N) d'un isotope est égal à la différence entre son nombre de masse (A) et son numéro atomique (Z). Pour les isotopes du carbone (Z=6) proposés :

  • 116C : N = 11 - 6 = 5
  • 126C : N = 12 - 6 = 6
  • 136C : N = 13 - 6 = 7
  • 146C : N = 14 - 6 = 8

En regardant le diagramme de Segré pour Z = 6, l'isotope stable (souvent représenté en noir ou au centre de la "vallée de stabilité") est celui pour lequel N = 6.

Seul l'isotope possédant 6 neutrons et dont le numéro atomique vaut 6 se situe dans la zone des noyaux stables, par conséquent c'est le carbone 126C qui est l'isotope stable.

2. D'après le diagramme de Segré ci-dessous, lequel des isotopes du cuivre est stable ?
[Diagramme de Segré Image Placeholder]
Axes: Z (protons) vs N (neutrons)
Légende: Décroissance β-, β+, α, Fission, Inconnue
  • 6329Cu
  • 2029Cu
  • 6319Cu
  • 5329Cu

Réponse correcte : 6329Cu

Explication :

Le nombre de neutrons (N) est A - Z. Le cuivre a Z = 29.

  • 6329Cu : Z = 29 (Cuivre), N = 63 - 29 = 34
  • 2029Cu : Z = 29 (Cuivre), A = 20. Impossible car A doit être supérieur ou égal à Z.
  • 6319Cu : Z = 19 (Potassium, K), pas du Cuivre.
  • 5329Cu : Z = 29 (Cuivre), N = 53 - 29 = 24

En regardant le diagramme de Segré pour Z = 29, l'isotope stable est celui avec N = 34 (ou 36, le cuivre a deux isotopes stables).

Seul l'isotope possédant 34 neutrons et dont le numéro atomique vaut 29 se situe dans la zone des noyaux stables, par conséquent c'est le cuivre 6329Cu qui est l'isotope stable (le 6529Cu, N=36, est aussi stable mais non proposé).

3. D'après le diagramme de Segré ci-dessous, lequel des isotopes de l'argent est stable ?
[Diagramme de Segré Image Placeholder]
Axes: Z (protons) vs N (neutrons)
Légende: Décroissance β-, β+, α, Fission, Inconnue
  • 10747Ag
  • 5747Ag
  • 10737Ag
  • 3737Ag

Réponse correcte : 10747Ag

Explication :

Le nombre de neutrons (N) est A - Z. L'argent a Z = 47.

  • 10747Ag : Z = 47 (Argent), N = 107 - 47 = 60
  • 5747Ag : Z = 47 (Argent), N = 57 - 47 = 10. (A=57 est trop faible pour être stable à Z=47).
  • 10737Ag : Z = 37 (Rubidium, Rb), pas de l'Argent.
  • 3737Ag : Z = 37 (Rubidium, Rb), pas de l'Argent. (De plus, A=Z=37 correspondrait à 37Rb, très instable).

En regardant le diagramme de Segré pour Z = 47, l'isotope stable est celui avec N = 60 (ou 62, l'argent a deux isotopes stables).

Seul l'isotope possédant 60 neutrons et dont le numéro atomique vaut 47 se situe dans la zone des noyaux stables, par conséquent c'est l'argent 10747Ag qui est l'isotope stable (le 10947Ag, N=62, est aussi stable mais non proposé).

4. D'après le diagramme de Segré ci-dessous, lequel des isotopes du xénon est stable ?
[Diagramme de Segré Image Placeholder]
Axes: Z (protons) vs N (neutrons)
Légende: Décroissance β-, β+, α, Fission, Inconnue
  • 13054Xe
  • 11054Xe
  • 13034Xe
  • 5754Xe

Réponse correcte : 13054Xe

Explication :

Le nombre de neutrons (N) est A - Z. Le xénon a Z = 54.

  • 13054Xe : Z = 54 (Xénon), N = 130 - 54 = 76
  • 11054Xe : Z = 54 (Xénon), N = 110 - 54 = 56
  • 13034Xe : Z = 34 (Sélénium, Se), pas du Xénon.
  • 5754Xe : Z = 54 (Xénon), A = 57. Impossible car A doit être supérieur ou égal à Z.

Le Xénon (Z=54) a plusieurs isotopes stables avec N entre 70 et 82.

L'isotope possédant 76 neutrons et dont le numéro atomique vaut 54 (13054Xe) se situe dans la zone des noyaux stables, par conséquent c'est le xénon 13054Xe qui est l'isotope stable parmi les choix valides.

*(Note: L'explication originale contenait une erreur de calcul pour N, 130-54 = 76)*
5. D'après le diagramme de Segré ci-dessous, lequel des isotopes du plomb est stable ?
[Diagramme de Segré Image Placeholder]
Axes: Z (protons) vs N (neutrons)
Légende: Décroissance β-, β+, α, Fission, Inconnue
  • 20482Pb
  • 10482Pb
  • 20452Pb
  • 30482Pb

Réponse correcte : 20482Pb

Explication :

Le nombre de neutrons (N) est A - Z. Le plomb a Z = 82.

  • 20482Pb : Z = 82 (Plomb), N = 204 - 82 = 122
  • 10482Pb : Z = 82 (Plomb), N = 104 - 82 = 22. (A=104 est beaucoup trop faible pour Z=82).
  • 20452Pb : Z = 52 (Tellure, Te), pas du Plomb.
  • 30482Pb : Z = 82 (Plomb), N = 304 - 82 = 222. (A=304 est beaucoup trop élevé pour Z=82).

Le Plomb (Z=82) a des isotopes stables avec N = 122, 124, 125, 126.

Seul l'isotope possédant 122 neutrons et dont le numéro atomique vaut 82 se situe dans la zone des noyaux stables parmi les choix valides, par conséquent c'est le plomb 20482Pb qui est l'isotope stable (206Pb, 207Pb, 208Pb sont les autres stables).
