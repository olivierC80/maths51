KaTeX Example

This is an example of using KaTeX to render math in HTML.

Inline math example: $c = \sqrt{a^2 + b^2}$

Another inline example: \(e^{i\pi} + 1 = 0\)

Display math example:

$$\int_0^\infty e^{-x} \, dx = 1$$

Another display example:

\[ \frac{d}{dx}\left( \int_{a}^{x} f(t)\, dt \right) = f(x) \]
