Généralités et Structure des Solutions

Quelle est la structure de l'ensemble des solutions $\mathcal{S}$ d'un système linéaire $(S)$, en supposant qu'il admette au moins une solution $x_P$ (solution particulière) ?
D'après le théorème sur la structure de l'ensemble des solutions, si $(S)$ est compatible et $x_P$ est une solution particulière de $(S)$, alors l'ensemble $\mathcal{S}$ de toutes les solutions de $(S)$ est donné par : $$ \mathcal{S} = \{ x_P + x_H \mid x_H \in \mathcal{S}_H \} = x_P + \mathcal{S}_H $$ où $\mathcal{S}_H$ est l'ensemble des solutions du système homogène associé $(S_H)$. Autrement dit, toute solution s'obtient en ajoutant à une solution particulière $x_P$ une solution quelconque du système homogène. Géométriquement, si $\mathcal{S}_H$ est un sous-espace vectoriel (de $K^p$), alors $\mathcal{S}$ est un sous-espace affine, obtenu par translation de $\mathcal{S}_H$ par le vecteur $x_P$.
Pourquoi l'ensemble des solutions $\mathcal{S}_H$ d'un système linéaire homogène $(S_H)$ est-il toujours un sous-espace vectoriel de $K^p$ ?
Pour montrer que $\mathcal{S}_H$ est un sous-espace vectoriel de $K^p$, il faut vérifier trois points :
  1. Non vide : Le vecteur nul $(0, \dots, 0)$ vérifie trivialement toutes les équations $\sum a_{i,j} x_j = 0$. Donc $0_{K^p} \in \mathcal{S}_H$, et $\mathcal{S}_H$ n'est pas vide.
  2. Stabilité par addition : Soient $x = (x_1, \dots, x_p)$ et $y = (y_1, \dots, y_p)$ deux solutions de $(S_H)$. Montrons que $x+y = (x_1+y_1, \dots, x_p+y_p)$ est aussi solution. Pour chaque équation $i$ ($1 \le i \le n$), on a $\sum_{j=1}^p a_{i,j} x_j = 0$ et $\sum_{j=1}^p a_{i,j} y_j = 0$. Alors $\sum_{j=1}^p a_{i,j} (x_j + y_j) = \sum_{j=1}^p a_{i,j} x_j + \sum_{j=1}^p a_{i,j} y_j = 0 + 0 = 0$. Donc $x+y \in \mathcal{S}_H$.
  3. Stabilité par multiplication par un scalaire : Soit $x = (x_1, \dots, x_p) \in \mathcal{S}_H$ et $\lambda \in K$. Montrons que $\lambda x = (\lambda x_1, \dots, \lambda x_p)$ est aussi solution. Pour chaque équation $i$, on a $\sum_{j=1}^p a_{i,j} x_j = 0$. Alors $\sum_{j=1}^p a_{i,j} (\lambda x_j) = \lambda \sum_{j=1}^p a_{i,j} x_j = \lambda \times 0 = 0$. Donc $\lambda x \in \mathcal{S}_H$.
Ces trois propriétés montrent que $\mathcal{S}_H$ est un sous-espace vectoriel de $K^p$.

Algorithme du Pivot de Gauss

Quelles sont les trois opérations élémentaires sur les lignes qui permettent de transformer un système linéaire en un système équivalent ?
Les opérations élémentaires sur les lignes qui préservent l'ensemble des solutions sont :
  1. Échange de deux lignes : $L_i \leftrightarrow L_j$.
  2. Multiplication d'une ligne par un scalaire non nul : $L_i \leftarrow \lambda L_i$, avec $\lambda \in K^*$.
  3. Ajout à une ligne d'un multiple d'une autre ligne : $L_i \leftarrow L_i + \lambda L_j$, avec $j \neq i$ et $\lambda \in K$.
Qu'est-ce qu'un système échelonné ? Donner les caractéristiques principales.
Un système est dit échelonné s'il a une structure "en escalier". Plus précisément :
  • Les équations éventuelles de la forme $0=b'_k$ sont regroupées à la fin.
  • Dans chaque équation non triviale (non de la forme $0=b'_k$), le premier coefficient non nul (appelé pivot) est situé strictement à droite du pivot de la ligne précédente.
Cela implique que les coefficients sous chaque pivot sont nuls. Les caractéristiques principales sont :
  • L'existence de $r$ pivots non nuls (où $r$ est le rang du système).
  • Les indices des colonnes $j_1 < j_2 < \dots < j_r$ où se trouvent les pivots.
  • Les $r$ inconnues principales $x_{j_1}, \dots, x_{j_r}$.
  • Les $p-r$ inconnues secondaires (ou variables libres).
  • Les équations de compatibilité éventuelles ($0 = b'_k$ pour $k > r$).
Appliquer l'algorithme du pivot de Gauss pour échelonner le système suivant : $$ \begin{cases} x + 2y - z = 3 \\ 2x + 4y + z = 0 \\ -x - y + 3z = -1 \end{cases} $$
Système initial : 
(L1)  x + 2y -  z =  3
(L2) 2x + 4y +  z =  0
(L3) -x -  y + 3z = -1
Le premier pivot est $a_{1,1}=1$ (coefficient de $x$ dans L1). On élimine $x$ dans L2 et L3 :
  • $L_2 \leftarrow L_2 - 2 L_1$: $(2x+4y+z) - 2(x+2y-z) = 0 - 2(3) \implies 3z = -6$.
  • $L_3 \leftarrow L_3 + L_1$: $(-x-y+3z) + (x+2y-z) = -1 + 3 \implies y + 2z = 2$.
Le système devient : 
(L1)   x + 2y - z =  3
(L'2)         3z = -6
(L'3)      y + 2z =  2
Le sous-système est formé par L'2 et L'3. La première colonne non nulle est celle de $y$. Le pivot se trouve en L'3 ($a'_{3,2}=1$). On échange L'2 et L'3 pour placer le pivot en deuxième position : $L'_2 \leftrightarrow L'_3$. 
(L1)   x + 2y - z =  3
(L''2)     y + 2z =  2
(L''3)        3z = -6
Le système est maintenant échelonné. Les pivots sont 1 (pour $x$), 1 (pour $y$), 3 (pour $z$). Le rang est $r=3$.

Résolution

Résoudre le système échelonné obtenu à la question précédente : $$ \begin{cases} x + 2y - z = 3 \\ \phantom{0x} y + 2z = 2 \\ \phantom{0x + 0y} 3z = -6 \end{cases} $$
Le système est échelonné, de rang $r=3$. Il y a $p=3$ inconnues. Il n'y a pas d'équation de compatibilité (type $0=b \neq 0$). Le système est compatible. Les inconnues principales sont $x, y, z$. Il n'y a pas d'inconnue libre ($p-r=0$). Le système admet donc une unique solution. On utilise la méthode de remontée :
  • $L''_3$: $3z = -6 \implies z = -2$.
  • $L''2$: $y + 2z = 2 \implies y = 2 - 2z = 2 - 2(-2) = 2 + 4 = 6$.
  • $L_1$: $x + 2y - z = 3 \implies x = 3 - 2y + z = 3 - 2(6) + (-2) = 3 - 12 - 2 = -11$.
L'unique solution est $(x, y, z) = (-11, 6, -2)$. L'ensemble des solutions est $\mathcal{S} = \{ (-11, 6, -2) \}$.
Considérer le système échelonné suivant (à 4 inconnues $x,y,z,t$) : $$ \begin{cases} x - y + 2z - t = 1 \\ \phantom{0x + } 3z + t = 6 \\ \phantom{0x + 0y + 0z + } 0 = 0 \end{cases} $$ Est-il compatible ? Quel est son rang ? Identifier les inconnues principales et libres. Décrire l'ensemble des solutions.
Le système est échelonné.
  • Compatibilité : La dernière ligne est $0=0$, ce qui est toujours vrai. Le système est donc compatible.
  • Rang : Il y a deux pivots non nuls (le 1 devant $x$ et le 3 devant $z$). Le rang est $r=2$.
  • Inconnues principales/libres : Les inconnues correspondant aux pivots (colonnes 1 et 3) sont les inconnues principales : $x$ et $z$. Les autres inconnues (colonnes 2 et 4) sont les inconnues libres (ou secondaires) : $y$ et $t$. Il y a $p=4$ inconnues et $r=2$, donc $p-r=2$ inconnues libres.
  • Description des solutions : On exprime les inconnues principales en fonction des inconnues libres en utilisant la remontée.
    • $L_2$: $3z + t = 6 \implies 3z = 6 - t \implies z = 2 - \frac{1}{3}t$.
    • $L_1$: $x - y + 2z - t = 1 \implies x = 1 + y - 2z + t$. On remplace $z$ par son expression : $x = 1 + y - 2(2 - \frac{1}{3}t) + t = 1 + y - 4 + \frac{2}{3}t + t = -3 + y + \frac{5}{3}t$.
    Les inconnues libres $y$ et $t$ peuvent prendre n'importe quelle valeur dans $K$. L'ensemble des solutions est : $$ \mathcal{S} = \{ (-3 + y + \frac{5}{3}t, \; y, \; 2 - \frac{1}{3}t, \; t) \mid y \in K, t \in K \} $$ C'est un sous-espace affine de dimension 2 dans $K^4$. On peut l'écrire sous la forme $x_P + \mathcal{S}_H$ : Une solution particulière $x_P$ (pour $y=0, t=0$) est $x_P = (-3, 0, 2, 0)$. Les solutions du système homogène associé sont de la forme $(y + \frac{5}{3}t, \; y, \; -\frac{1}{3}t, \; t)$ avec $y,t \in K$. $\mathcal{S}_H = \{ y(1, 1, 0, 0) + t(\frac{5}{3}, 0, -\frac{1}{3}, 1) \mid y,t \in K \} = \text{Vect}((1, 1, 0, 0), (5/3, 0, -1/3, 1))$. On a bien $\mathcal{S} = x_P + \mathcal{S}_H$.
