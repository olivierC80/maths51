Questions : Primitives

Cliquez sur une question pour voir apparaître la réponse.

1. Qu'est-ce qu'une primitive d'une fonction $f$ sur un intervalle $I$ ?

Une fonction $F$ est une primitive de la fonction $f$ sur un intervalle $I$ si $F$ est dérivable sur $I$ et si pour tout $x \in I$ :

$$ F'(x) = f(x) $$

Exemple : $F(x) = x^2$ est une primitive de $f(x) = 2x$ sur $\mathbb{R}$ car $(x^2)' = 2x$.

2. Si une fonction $f$ admet une primitive $F$ sur un intervalle $I$, admet-elle d'autres primitives ? Si oui, comment sont-elles liées ?

Oui. Si $F$ est une primitive de $f$ sur $I$, alors toutes les primitives de $f$ sur $I$ sont les fonctions de la forme :

$$ G(x) = F(x) + C $$

où $C$ est une constante réelle quelconque.

Deux primitives d'une même fonction sur un intervalle diffèrent d'une constante.

3. Quelle condition suffisante garantit qu'une fonction $f$ admet des primitives sur un intervalle $I$ ?

Toute fonction continue sur un intervalle $I$ admet des primitives sur cet intervalle.

(Théorème fondamental de l'analyse - partie existence, admise en Terminale).

4. Donner une primitive des fonctions usuelles suivantes : $f(x)=k$ (constante), $g(x)=x^n$ ($n \in \mathbb{Z}, n \neq -1$), $h(x)=1/x$ (sur $]0, +\infty[$), $k(x)=e^x$, $l(x)=\cos x$, $m(x)=\sin x$.

(On donne une primitive, sans la constante $C$)

  • Primitive de $f(x)=k$ : $F(x) = kx$
  • Primitive de $g(x)=x^n$ ($n \neq -1$) : $G(x) = \frac{x^{n+1}}{n+1}$
  • Primitive de $h(x)=1/x$ (sur $]0, +\infty[$) : $H(x) = \ln(x)$
  • Primitive de $k(x)=e^x$ : $K(x) = e^x$
  • Primitive de $l(x)=\cos x$ : $L(x) = \sin x$
  • Primitive de $m(x)=\sin x$ : $M(x) = -\cos x$
5. Quelle est la propriété concernant la primitive d'une somme de fonctions et la primitive du produit d'une fonction par une constante ?

Si $F$ est une primitive de $f$ et $G$ une primitive de $g$ sur un intervalle $I$, et si $k$ est une constante réelle :

  • Une primitive de $f+g$ est $F+G$.
  • Une primitive de $k \times f$ est $k \times F$.

C'est la propriété de linéarité de la recherche de primitives.

6. Donner une primitive de $f(x) = 3x^2 - \frac{4}{x^3} + 5e^x$.

On cherche une primitive de chaque terme :

  • Primitive de $3x^2$ : $3 \times \frac{x^{2+1}}{2+1} = 3 \frac{x^3}{3} = x^3$.
  • Primitive de $-\frac{4}{x^3} = -4x^{-3}$ : $-4 \times \frac{x^{-3+1}}{-3+1} = -4 \frac{x^{-2}}{-2} = 2x^{-2} = \frac{2}{x^2}$.
  • Primitive de $5e^x$ : $5e^x$.

Par linéarité, une primitive de $f(x)$ est :

$$ F(x) = x^3 + \frac{2}{x^2} + 5e^x $$

(Les primitives sont $F(x) + C$).

7. Comment trouver une primitive d'une fonction de la forme $u'(x) [u(x)]^n$ (avec $n \in \mathbb{Z}, n \neq -1$) ? Donner un exemple.

Une primitive de $f(x) = u'(x) [u(x)]^n$ est :

$$ F(x) = \frac{[u(x)]^{n+1}}{n+1} $$

Exemple : Trouver une primitive de $f(x) = (2x+1)(x^2+x+3)^4$.

On pose $u(x) = x^2+x+3$. Alors $u'(x) = 2x+1$.

La fonction est de la forme $u'(x)[u(x)]^4$ (avec $n=4$).

Une primitive est $F(x) = \frac{[u(x)]^{4+1}}{4+1} = \frac{(x^2+x+3)^5}{5}$.

8. Comment trouver une primitive d'une fonction de la forme $\frac{u'(x)}{u(x)}$ sur un intervalle où $u(x) \neq 0$ ? Donner un exemple.

Une primitive de $f(x) = \frac{u'(x)}{u(x)}$ est :

$$ F(x) = \ln(|u(x)|) $$

Si l'on sait que $u(x) > 0$ sur l'intervalle considéré, on peut simplement écrire $F(x) = \ln(u(x))$.

Exemple : Trouver une primitive de $f(x) = \frac{2x}{x^2+1}$ sur $\mathbb{R}$.

On pose $u(x) = x^2+1$. Alors $u'(x) = 2x$. De plus, $u(x) > 0$ pour tout $x \in \mathbb{R}$.

La fonction est de la forme $\frac{u'(x)}{u(x)}$ avec $u(x)>0$.

Une primitive est $F(x) = \ln(u(x)) = \ln(x^2+1)$.

9. Comment trouver une primitive d'une fonction de la forme $u'(x) e^{u(x)}$ ? Donner un exemple.

Une primitive de $f(x) = u'(x) e^{u(x)}$ est :

$$ F(x) = e^{u(x)} $$

Exemple : Trouver une primitive de $f(x) = \cos(x) e^{\sin(x)}$.

On pose $u(x) = \sin(x)$. Alors $u'(x) = \cos(x)$.

La fonction est de la forme $u'(x)e^{u(x)}$.

Une primitive est $F(x) = e^{u(x)} = e^{\sin(x)}$.

10. Comment trouver une primitive d'une fonction de la forme $\frac{u'(x)}{\sqrt{u(x)}}$ sur un intervalle où $u(x) > 0$ ? Donner un exemple.

On peut écrire $f(x) = \frac{u'(x)}{\sqrt{u(x)}} = u'(x) [u(x)]^{-1/2}$.

C'est la forme $u'u^n$ avec $n = -1/2 \neq -1$.

Une primitive est $F(x) = \frac{[u(x)]^{-1/2 + 1}}{-1/2 + 1} = \frac{[u(x)]^{1/2}}{1/2} = 2\sqrt{u(x)}$.

Une primitive de $f(x) = \frac{u'(x)}{\sqrt{u(x)}}$ est :

$$ F(x) = 2\sqrt{u(x)} $$

Exemple : Trouver une primitive de $f(x) = \frac{x}{\sqrt{x^2+4}}$ sur $\mathbb{R}$.

On pose $u(x) = x^2+4$. Alors $u'(x) = 2x$. $u(x) > 0$ sur $\mathbb{R}$.

On écrit $f(x) = \frac{1}{2} \times \frac{2x}{\sqrt{x^2+4}} = \frac{1}{2} \frac{u'(x)}{\sqrt{u(x)}}$.

Une primitive est $F(x) = \frac{1}{2} \times (2\sqrt{u(x)}) = \sqrt{u(x)} = \sqrt{x^2+4}$.

11. Comment déterminer l'unique primitive $F$ d'une fonction $f$ qui vérifie une condition initiale $F(x_0) = y_0$ ?

La méthode est la suivante :

  1. Trouver l'expression générale des primitives de $f$. Elle est de la forme $G(x) = F_0(x) + C$, où $F_0$ est une primitive particulière et $C$ une constante réelle.
  2. Utiliser la condition initiale pour déterminer $C$. On résout l'équation $G(x_0) = y_0$, c'est-à-dire $F_0(x_0) + C = y_0$.
  3. On en déduit la valeur de $C = y_0 - F_0(x_0)$.
  4. On remplace $C$ par sa valeur dans l'expression générale pour obtenir l'unique primitive recherchée.

Exemple : Trouver la primitive $F$ de $f(x) = e^{2x}$ telle que $F(0) = 3$.

  1. Une primitive de $e^{ax}$ est $\frac{1}{a}e^{ax}$. Ici $a=2$. Les primitives générales sont $G(x) = \frac{1}{2}e^{2x} + C$.
  2. On veut $G(0)=3$. Or $G(0) = \frac{1}{2}e^{2 \times 0} + C = \frac{1}{2}e^0 + C = \frac{1}{2} + C$.
  3. L'équation est $\frac{1}{2} + C = 3$, donc $C = 3 - \frac{1}{2} = \frac{5}{2}$.
  4. L'unique primitive recherchée est $F(x) = \frac{1}{2}e^{2x} + \frac{5}{2}$.
