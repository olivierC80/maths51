Généralités, Degré, Dérivation

Soient $P(X) = X^3 - 2X + 1$ et $Q(X) = -X^3 + X^2$. Calculer $\deg(P)$, $\deg(Q)$, $\deg(P+Q)$ et $\deg(PQ)$.
$P(X) = X^3 - 2X + 1$. Le coefficient de $X^3$ est $1 \neq 0$, c'est le terme de plus haut degré. Donc $\deg(P) = 3$.
$Q(X) = -X^3 + X^2$. Le coefficient de $X^3$ est $-1 \neq 0$, c'est le terme de plus haut degré. Donc $\deg(Q) = 3$.
$P(X) + Q(X) = (X^3 - 2X + 1) + (-X^3 + X^2) = X^2 - 2X + 1$. Le terme de plus haut degré est $X^2$ (coefficient $1 \neq 0$). Donc $\deg(P+Q) = 2$.
Remarque : ici $\deg(P) = \deg(Q) = 3$, mais les termes de degré 3 se sont annulés. On a bien $\deg(P+Q) = 2 \le \max(\deg P, \deg Q) = 3$.
Pour le produit : $\deg(PQ) = \deg(P) + \deg(Q) = 3 + 3 = 6$. (Le coefficient dominant de $PQ$ est le produit des coefficients dominants de $P$ et $Q$, soit $1 \times (-1) = -1$, qui est non nul).
Donner la formule de Taylor pour le polynôme $P(X) = X^4 - X^2 + 3$ au point $a=1$.
La formule de Taylor pour un polynôme $P$ de degré $N$ en $a$ est : $$ P(X) = \sum_{k=0}^N \frac{P^{(k)}(a)}{k!} (X-a)^k $$ Ici $P(X) = X^4 - X^2 + 3$, $N=4$ et $a=1$. Il faut calculer $P$ et ses dérivées successives en $a=1$.
  • $P(X) = X^4 - X^2 + 3 \implies P(1) = 1 - 1 + 3 = 3$. ($P^{(0)}(1)=3$)
  • $P'(X) = 4X^3 - 2X \implies P'(1) = 4 - 2 = 2$.
  • $P''(X) = 12X^2 - 2 \implies P''(1) = 12 - 2 = 10$.
  • $P'''(X) = 24X \implies P'''(1) = 24$.
  • $P^{(4)}(X) = 24 \implies P^{(4)}(1) = 24$.
  • $P^{(k)}(X) = 0$ pour $k \ge 5$.
Les coefficients de la formule sont :
  • $\frac{P^{(0)}(1)}{0!} = \frac{3}{1} = 3$.
  • $\frac{P^{(1)}(1)}{1!} = \frac{2}{1} = 2$.
  • $\frac{P^{(2)}(1)}{2!} = \frac{10}{2} = 5$.
  • $\frac{P^{(3)}(1)}{3!} = \frac{24}{6} = 4$.
  • $\frac{P^{(4)}(1)}{4!} = \frac{24}{24} = 1$.
La formule de Taylor en $a=1$ est donc : $$ P(X) = 3 + 2(X-1) + 5(X-1)^2 + 4(X-1)^3 + 1(X-1)^4 $$

Division Euclidienne, Racines, Multiplicité

Effectuer la division euclidienne de $A(X) = X^5 + X + 1$ par $B(X) = X^2 + 1$.
On pose la division euclidienne : 

          X^3   - X      (Quotient Q)
        _______________
 X^2+1 | X^5 + 0X^4 + 0X^3 + 0X^2 + X + 1
       -(X^5       + X^3)
       _________________
            - X^3 + 0X^2 + X
           -(- X^3       - X)
           _______________
                  2X + 1     (Reste R)
On obtient : $$ X^5 + X + 1 = (X^2+1)(X^3 - X) + (2X + 1) $$ Le quotient est $Q(X) = X^3 - X$. Le reste est $R(X) = 2X + 1$. On vérifie que $\deg(R) = 1$ est strictement inférieur à $\deg(B) = 2$.
Montrer que $a=2$ est une racine du polynôme $P(X) = X^3 - 5X^2 + 8X - 4$. Quelle est sa multiplicité ?
On utilise la caractérisation de la multiplicité par les dérivées successives. $P(X) = X^3 - 5X^2 + 8X - 4$.
  • $P(2) = 2^3 - 5(2^2) + 8(2) - 4 = 8 - 5(4) + 16 - 4 = 8 - 20 + 16 - 4 = 24 - 24 = 0$. Donc $a=2$ est bien racine (multiplicité $\ge 1$).
  • $P'(X) = 3X^2 - 10X + 8$. $P'(2) = 3(2^2) - 10(2) + 8 = 3(4) - 20 + 8 = 12 - 20 + 8 = 20 - 20 = 0$. Donc la multiplicité est $\ge 2$.
  • $P''(X) = 6X - 10$. $P''(2) = 6(2) - 10 = 12 - 10 = 2$. Comme $P''(2) \neq 0$, la multiplicité est exactement $m=2$.
Conclusion : $a=2$ est une racine de multiplicité 2 pour $P(X)$. Cela signifie que $(X-2)^2$ divise $P(X)$, mais $(X-2)^3$ ne divise pas $P(X)$.
Soit $P(X) = X^3 + aX^2 + bX + c$ un polynôme de $\mathbb{C}[X]$ dont les racines sont $z_1, z_2, z_3$. Exprimer $a, b, c$ en fonction de $z_1, z_2, z_3$.
Le polynôme $P(X)$ est unitaire (coefficient dominant $a_N=1$, ici $N=3$). D'après les relations coefficients-racines (avec $a_3=1, a_2=a, a_1=b, a_0=c$):
  • $\sigma_1 = z_1 + z_2 + z_3 = -\frac{a_{N-1}}{a_N} = -\frac{a_2}{a_3} = -\frac{a}{1} = -a$.
  • $\sigma_2 = z_1 z_2 + z_1 z_3 + z_2 z_3 = (-1)^2 \frac{a_{N-2}}{a_N} = \frac{a_1}{a_3} = \frac{b}{1} = b$.
  • $\sigma_3 = z_1 z_2 z_3 = (-1)^3 \frac{a_{N-3}}{a_N} = -\frac{a_0}{a_3} = -\frac{c}{1} = -c$.
On en déduit :
  • $a = -(z_1 + z_2 + z_3)$
  • $b = z_1 z_2 + z_1 z_3 + z_2 z_3$
  • $c = -z_1 z_2 z_3$
On retrouve le développement : $P(X) = (X-z_1)(X-z_2)(X-z_3) = X^3 - (z_1+z_2+z_3)X^2 + (z_1z_2+z_1z_3+z_2z_3)X - z_1z_2z_3$.

Arithmétique, Factorisation

Calculer le PGCD de $A(X) = X^4 - 1$ et $B(X) = X^3 - 1$ par l'algorithme d'Euclide.
On applique l'algorithme d'Euclide pour les polynômes. 1. Division de $A$ par $B$: $X^4 - 1 = X(X^3 - 1) + X - 1$. Le reste est $R_1(X) = X-1$. 2. Division de $B$ par $R_1$: $X^3 - 1 = (X^2 + X + 1)(X - 1) + 0$. Le reste est $R_2(X) = 0$. L'algorithme s'arrête. Le dernier reste non nul est $R_1(X) = X-1$. Le PGCD est le dernier reste non nul rendu unitaire. $R_1(X)$ est déjà unitaire. Donc, $PGCD(A, B) = X-1$.
Donner la forme de la décomposition en produit de facteurs irréductibles sur $\mathbb{R}[X]$ et sur $\mathbb{C}[X]$ du polynôme $P(X) = (X^2+4)^3 (X-1)^2 (X^2+X+1)$.
On analyse chaque facteur :
  • $(X-1)^2$ : Facteur réel de degré 1, puissance 2. (Racine réelle $z=1$ de multiplicité 2).
  • $X^2+X+1$ : Facteur réel de degré 2. Discriminant $\Delta = 1^2 - 4(1)(1) = 1-4 = -3 < 0$. Ce facteur est donc irréductible sur $\mathbb{R}[X]$. Ses racines complexes sont $j = e^{i2\pi/3}$ et $j^2 = e^{i4\pi/3}$ (racines cubiques de l'unité).
  • $X^2+4$ : Facteur réel de degré 2. Discriminant $\Delta = 0^2 - 4(1)(4) = -16 < 0$. Ce facteur est irréductible sur $\mathbb{R}[X]$. Ses racines complexes sont $2i$ et $-2i$. Le facteur est $(X^2+4)^3$.
Décomposition sur $\mathbb{R}[X]$ : On regroupe les facteurs irréductibles sur $\mathbb{R}$: $$ P(X) = (X-1)^2 (X^2+X+1)^1 (X^2+4)^3 $$ Les facteurs sont de degré 1 ($X-1$) ou de degré 2 à discriminant négatif ($X^2+X+1$, $X^2+4$). C'est la décomposition en produit d'irréductibles sur $\mathbb{R}[X]$. Le coefficient dominant est $c=1$. Décomposition sur $\mathbb{C}[X]$ : Les seuls irréductibles sur $\mathbb{C}[X]$ sont de degré 1. Il faut factoriser les termes de degré 2.
  • $X^2+X+1 = (X-j)(X-j^2)$ où $j=e^{i2\pi/3}$.
  • $X^2+4 = (X-2i)(X+2i)$.
On substitue dans l'expression de $P(X)$: $$ P(X) = (X-1)^2 [(X-j)(X-j^2)]^1 [(X-2i)(X+2i)]^3 $$ En développant les puissances : $$ P(X) = (X-1)^2 (X-j)^1 (X-j^2)^1 (X-2i)^3 (X+2i)^3 $$ C'est la décomposition en produit d'irréductibles (de degré 1) sur $\mathbb{C}[X]$. Le coefficient dominant est $c=1$. Les racines distinctes sont $1, j, j^2, 2i, -2i$ avec les multiplicités respectives $2, 1, 1, 3, 3$. (La somme des multiplicités est $2+1+1+3+3=10$, ce qui correspond bien au degré de $P$, $\deg(P) = 2+1+2\times 3 = 9$. Attendez, $\deg(P)=2 \times 1 + 1 \times 1 + 2 \times 3 = 2+1+6 = 9$? Non, $\deg(P) = \deg((X-1)^2) + \deg(X^2+X+1) + \deg((X^2+4)^3) = 2 + 2 + (2 \times 3) = 2+2+6=10$. Correct).

Interpolation

Soient $a, b, c$ trois réels distincts. Écrire le polynôme de Lagrange $L_1(X)$ associé aux points $a, b, c$. Quel est son degré ? Que valent $L_1(a)$, $L_1(b)$, $L_1(c)$ ?
On a $n=3$ points d'abscisses distinctes $a_1=a, a_2=b, a_3=c$. Le premier polynôme de Lagrange $L_1(X)$ (associé au point $a_1=a$) est défini par : $$ L_1(X) = \prod_{\substack{1 \le j \le 3 \\ j \neq 1}} \frac{X - a_j}{a_1 - a_j} $$ Ce qui donne : $$ L_1(X) = \frac{X - a_2}{a_1 - a_2} \times \frac{X - a_3}{a_1 - a_3} = \frac{X - b}{a - b} \times \frac{X - c}{a - c} $$ C'est un produit de deux polynômes de degré 1, donc $L_1(X)$ est de degré $n-1 = 3-1 = 2$. Par construction (ou par vérification directe) :
  • $L_1(a) = \frac{a - b}{a - b} \times \frac{a - c}{a - c} = 1 \times 1 = 1$.
  • $L_1(b) = \frac{b - b}{a - b} \times \frac{b - c}{a - c} = 0 \times \frac{b - c}{a - c} = 0$.
  • $L_1(c) = \frac{c - b}{a - b} \times \frac{c - c}{a - c} = \frac{c - b}{a - b} \times 0 = 0$.
On retrouve bien la propriété $L_i(a_k) = \delta_{i,k}$.
