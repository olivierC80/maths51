Primitives et Équations Différentielles

Ce chapitre aborde deux notions fondamentales et liées : la recherche de primitives (l'opération "inverse" de la dérivation) et la résolution d'équations différentielles, qui modélisent de nombreux phénomènes d'évolution.

Partie 1 : Primitives

Définition : Primitive d'une fonction

Soit $f$ une fonction définie sur un intervalle $I$.

On appelle primitive de $f$ sur $I$, toute fonction $F$ dérivable sur $I$ telle que pour tout $x \in I$ :

$$F'(x) = f(x)$$

Exemple introductif

Soit $f(x) = 2x$ définie sur $\mathbb{R}$.

La fonction $F(x) = x^2$ est une primitive de $f$ sur $\mathbb{R}$ car $F'(x) = 2x = f(x)$.

La fonction $G(x) = x^2 + 3$ est aussi une primitive de $f$ sur $\mathbb{R}$ car $G'(x) = 2x = f(x)$.

La fonction $H(x) = x^2 - \pi$ est aussi une primitive de $f$ sur $\mathbb{R}$ car $H'(x) = 2x = f(x)$.

Cela suggère qu'une fonction n'admet pas une unique primitive.

Théorème (Existence de primitives - Admis)

Toute fonction continue sur un intervalle $I$ admet des primitives sur $I$.

Propriété : Ensemble des primitives

Soit $f$ une fonction admettant une primitive $F$ sur un intervalle $I$.

Alors l'ensemble des primitives de $f$ sur $I$ est l'ensemble des fonctions $G$ définies sur $I$ par :

$$G(x) = F(x) + C$$

où $C$ est une constante réelle quelconque.

Preuve

Soit $F$ une primitive de $f$ sur $I$.

1. Montrons que toute fonction $G(x) = F(x) + C$ est une primitive de $f$.

$G$ est dérivable sur $I$ comme somme de $F$ (dérivable) et d'une constante.

Pour tout $x \in I$, $G'(x) = (F(x) + C)' = F'(x) + 0 = f(x)$. Donc $G$ est bien une primitive de $f$.

2. Montrons que toute primitive $H$ de $f$ est de la forme $F(x) + C$.

Soit $H$ une autre primitive de $f$ sur $I$. Considérons la fonction $K(x) = H(x) - F(x)$.

$K$ est dérivable sur $I$ et pour tout $x \in I$ :

$$K'(x) = H'(x) - F'(x) = f(x) - f(x) = 0$$

Une fonction dont la dérivée est nulle sur un intervalle est constante sur cet intervalle. Donc il existe une constante réelle $C$ telle que $K(x) = C$ pour tout $x \in I$.

Ainsi, $H(x) - F(x) = C$, ce qui signifie $H(x) = F(x) + C$.

Conclusion : L'ensemble des primitives est bien $\{ F(x) + C \mid C \in \mathbb{R} \}$.

Notation Intégrale Indéfinie

On note parfois l'ensemble des primitives de $f$ par $\int f(x) \, dx$. Par exemple :

$$\int 2x \, dx = x^2 + C$$

Attention : En Terminale, on utilise plus souvent la notion d'intégrale définie $\int_a^b f(x) \, dx$, qui est un nombre. L'écriture $\int f(x) \, dx$ représente une famille de fonctions.

Opérations sur les primitives (Linéarité)

Soient $f$ et $g$ deux fonctions admettant des primitives $F$ et $G$ respectivement sur un intervalle $I$. Soit $k$ un réel.

  • Une primitive de $f+g$ sur $I$ est $F+G$.
  • Une primitive de $k \times f$ sur $I$ est $k \times F$.

On peut résumer par : $\int (a f(x) + b g(x)) \, dx = a \int f(x) \, dx + b \int g(x) \, dx$ (à une constante près).

Preuve (Linéarité)

1. $(F+G)' = F' + G' = f + g$. Donc $F+G$ est une primitive de $f+g$.

2. $(kF)' = k F' = k f$. Donc $kF$ est une primitive de $kf$.

Exemple de calcul de primitive

Trouver une primitive de $f(x) = 3x^2 - \frac{4}{x^3} + 5e^x$ sur $]0, +\infty[$.

On utilise la linéarité et le tableau des primitives usuelles :

  • Une primitive de $x^2$ est $\frac{x^3}{3}$.
  • Une primitive de $\frac{1}{x^3} = x^{-3}$ est $\frac{x^{-3+1}}{-3+1} = \frac{x^{-2}}{-2} = -\frac{1}{2x^2}$.
  • Une primitive de $e^x$ est $e^x$.

Donc, une primitive de $f(x)$ est :

$$F(x) = 3 \times \left(\frac{x^3}{3}\right) - 4 \times \left(-\frac{1}{2x^2}\right) + 5 \times e^x$$ $$F(x) = x^3 + \frac{2}{x^2} + 5e^x$$

L'ensemble des primitives est $G(x) = x^3 + \frac{2}{x^2} + 5e^x + C$, où $C \in \mathbb{R}$.

Méthode : Pour trouver une primitive, on essaie souvent de faire apparaître l'une de ces formes $u' \times (\text{qqch} \circ u)$.

Exemple avec forme composée

Trouver une primitive de $g(x) = \frac{x}{x^2+1}$ sur $\mathbb{R}$.

On pose $u(x) = x^2+1$. Alors $u'(x) = 2x$.

On réécrit $g(x)$ pour faire apparaître $u'(x)$ :

$$g(x) = \frac{1}{2} \times \frac{2x}{x^2+1} = \frac{1}{2} \times \frac{u'(x)}{u(x)}$$

Comme $u(x) = x^2+1 > 0$ pour tout $x \in \mathbb{R}$, une primitive de $\frac{u'(x)}{u(x)}$ est $\ln(u(x)) = \ln(x^2+1)$.

Donc, par linéarité, une primitive de $g(x)$ est :

$$G(x) = \frac{1}{2} \ln(x^2+1)$$

Partie 2 : Équations Différentielles Linéaires du Premier Ordre

Définition : Équation différentielle

Une équation différentielle est une équation dont l'inconnue est une fonction (souvent notée $y$) et qui fait intervenir cette fonction et ses dérivées successives ($y', y'', \dots$).

Résoudre une équation différentielle sur un intervalle $I$, c'est trouver toutes les fonctions $y$ définies et dérivables sur $I$ qui vérifient l'équation.

Nous nous concentrons ici sur les équations différentielles linéaires du premier ordre à coefficients constants.

2.1 Équation homogène $y' = ay$

Théorème : Solution de $y' = ay$

Soit $a$ un réel.

Les solutions sur $\mathbb{R}$ de l'équation différentielle $y' = ay$ sont les fonctions $y$ définies par :

$$y(x) = C e^{ax}$$

où $C$ est une constante réelle quelconque.

Preuve (Partielle - Vérification et unicité admise)

1. Vérification : Soit $y(x) = C e^{ax}$. $y$ est dérivable sur $\mathbb{R}$ et $y'(x) = C \times (a e^{ax}) = a \times (C e^{ax}) = a y(x)$. Donc toute fonction de cette forme est solution.

2. Unicité (Idée) : Montrons qu'il n'y a pas d'autres solutions. Soit $y$ une solution quelconque de $y' = ay$. Considérons la fonction $z(x) = y(x) e^{-ax}$. $z$ est dérivable sur $\mathbb{R}$ et

$$z'(x) = y'(x) e^{-ax} + y(x) (-a e^{-ax})$$

Comme $y'(x) = ay(x)$, on remplace :

$$z'(x) = (ay(x)) e^{-ax} - a y(x) e^{-ax} = 0$$

Puisque $z'(x)=0$ pour tout $x \in \mathbb{R}$, la fonction $z$ est constante. Il existe donc $C \in \mathbb{R}$ tel que $z(x) = C$.

D'où $y(x) e^{-ax} = C$, ce qui signifie $y(x) = C e^{ax}$.

Exemple : Résolution de $y' = -2y$

Il s'agit d'une équation de la forme $y' = ay$ avec $a = -2$.

Les solutions sont donc les fonctions définies sur $\mathbb{R}$ par :

$$y(x) = C e^{-2x}$$

où $C \in \mathbb{R}$.

Théorème : Problème de Cauchy pour $y' = ay$

Soient $a$, $x_0$, $y_0$ trois réels.

L'équation différentielle $y' = ay$ admet une unique solution $y$ telle que $y(x_0) = y_0$.

Cette solution est donnée par :

$$y(x) = y_0 e^{a(x-x_0)}$$

Preuve

On sait que les solutions sont de la forme $y(x) = C e^{ax}$.

On cherche la valeur de $C$ telle que $y(x_0) = y_0$.

$$y(x_0) = C e^{ax_0} = y_0$$

Comme $e^{ax_0} \neq 0$, on peut diviser : $C = y_0 e^{-ax_0}$.

En remplaçant $C$ dans l'expression de $y(x)$, on obtient :

$$y(x) = (y_0 e^{-ax_0}) e^{ax} = y_0 e^{ax - ax_0} = y_0 e^{a(x-x_0)}$$

Cette expression définit bien une fonction unique.

Exemple : Problème de Cauchy

Résoudre le problème de Cauchy suivant :

$$\begin{cases} y' = 3y \\ y(1) = 5 \end{cases}$$

Ici, $a=3$, $x_0=1$, $y_0=5$.

La solution générale de $y'=3y$ est $y(x) = C e^{3x}$.

En utilisant la condition initiale : $y(1) = C e^{3 \times 1} = C e^3 = 5$.

Donc $C = 5 e^{-3}$.

L'unique solution est $y(x) = (5 e^{-3}) e^{3x} = 5 e^{3x-3} = 5 e^{3(x-1)}$.

Vérification : $y(1) = 5 e^{3(1-1)} = 5 e^0 = 5 \times 1 = 5$. C'est correct.

Alternative (formule directe) : $y(x) = y_0 e^{a(x-x_0)} = 5 e^{3(x-1)}$.

2.2 Équation complète $y' = ay + b$

Théorème : Solution de $y' = ay + b$ ($a \neq 0$)

Soit $a$ un réel non nul et $b$ un réel.

L'équation différentielle $y' = ay + b$ admet pour solutions sur $\mathbb{R}$ les fonctions $y$ définies par :

$$y(x) = C e^{ax} - \frac{b}{a}$$

où $C$ est une constante réelle quelconque.

Structure de la solution : La solution générale est la somme de :

  • La solution générale de l'équation homogène associée $y' = ay$ (qui est $y_h(x) = C e^{ax}$).
  • Une solution particulière de l'équation complète $y' = ay + b$ (ici, la fonction constante $y_p(x) = -b/a$).
$$y(x) = y_h(x) + y_p(x)$$

Preuve (Structure et Vérification)

1. Cherchons une solution particulière constante. Soit $y_p(x) = k$ (constante). On veut que $y_p$ vérifie $y' = ay + b$.

La dérivée de $y_p$ est $y_p'(x) = 0$. L'équation devient :

$$0 = a k + b$$

Comme $a \neq 0$, on peut résoudre : $k = -b/a$. Donc $y_p(x) = -b/a$ est une solution particulière constante.

2. Montrons que $y = y_h + y_p$ est solution. Soit $y(x) = C e^{ax} - b/a$.

$y'(x) = (C e^{ax})' - (-b/a)' = a (C e^{ax}) - 0 = a (y(x) + b/a) = a y(x) + b$.

Donc $y(x) = C e^{ax} - b/a$ est bien solution.

3. Montrons que toute solution est de cette forme (admis via unicité du pb de Cauchy, ou similaire à $y'=ay$). Soit $z$ une solution quelconque de $y'=ay+b$. Considérons $u(x) = z(x) - y_p(x) = z(x) + b/a$.

$u'(x) = z'(x) - y_p'(x) = (az(x)+b) - 0 = a(u(x) - b/a) + b = a u(x) - b + b = a u(x)$.

Donc $u$ est solution de l'équation homogène $u' = au$. Ainsi, $u(x) = C e^{ax}$ pour une certaine constante $C$.

Il s'ensuit que $z(x) = u(x) + y_p(x) = C e^{ax} - b/a$.

Exemple : Résolution de $y' = -0.5y + 1$

C'est une équation de la forme $y' = ay + b$ avec $a = -0.5$ et $b = 1$.

1. Solution particulière constante : $y_p = -b/a = -1 / (-0.5) = 2$.

2. Solution générale de l'équation homogène associée $y' = -0.5y$ : $y_h(x) = C e^{-0.5x}$.

3. Solution générale de l'équation complète :

$$y(x) = y_h(x) + y_p(x) = C e^{-0.5x} + 2$$

où $C \in \mathbb{R}$.

Notez que lorsque $x \to +\infty$, $e^{-0.5x} \to 0$, donc $y(x) \to 2$. La droite $y=2$ est une asymptote horizontale.

Théorème : Problème de Cauchy pour $y' = ay + b$ ($a \neq 0$)

Soient $a \neq 0$, $b$, $x_0$, $y_0$ quatre réels.

L'équation différentielle $y' = ay + b$ admet une unique solution $y$ telle que $y(x_0) = y_0$.

Cette solution est donnée par :

$$y(x) = (y_0 + \frac{b}{a}) e^{a(x-x_0)} - \frac{b}{a}$$

Preuve

On sait que les solutions sont de la forme $y(x) = C e^{ax} - b/a$.

On cherche $C$ tel que $y(x_0) = y_0$.

$$y(x_0) = C e^{ax_0} - \frac{b}{a} = y_0$$ $$C e^{ax_0} = y_0 + \frac{b}{a}$$ $$C = (y_0 + \frac{b}{a}) e^{-ax_0}$$

On remplace $C$ dans l'expression de $y(x)$ :

$$y(x) = \left( (y_0 + \frac{b}{a}) e^{-ax_0} \right) e^{ax} - \frac{b}{a}$$ $$y(x) = (y_0 + \frac{b}{a}) e^{ax - ax_0} - \frac{b}{a}$$ $$y(x) = (y_0 + \frac{b}{a}) e^{a(x-x_0)} - \frac{b}{a}$$

Cette expression définit bien une fonction unique.

Exemple : Problème de Cauchy (complet)

Résoudre le problème de Cauchy :

$$\begin{cases} y' = 2y + 6 \\ y(0) = 1 \end{cases}$$

Ici, $a=2$, $b=6$, $x_0=0$, $y_0=1$.

La solution générale est $y(x) = C e^{2x} - b/a = C e^{2x} - 6/2 = C e^{2x} - 3$.

On utilise la condition initiale $y(0) = 1$ :

$$y(0) = C e^{2 \times 0} - 3 = C e^0 - 3 = C - 3 = 1$$

Donc $C = 1 + 3 = 4$.

L'unique solution est $y(x) = 4 e^{2x} - 3$.

Vérification : $y(0) = 4e^0 - 3 = 4 - 3 = 1$. C'est correct.

Alternative (formule directe) : $-b/a = -3$.

$$y(x) = (y_0 + \frac{b}{a}) e^{a(x-x_0)} - \frac{b}{a} = (1 + (-3)) e^{2(x-0)} - (-3) = -2 e^{2x} + 3$$

Oups ! Erreur dans l'application de la formule directe. Refaisons :

$$y(x) = (y_0 - (-\frac{b}{a})) e^{a(x-x_0)} + (-\frac{b}{a})$$

Soit $y_p = -b/a = -3$.

$$y(x) = (y_0 - y_p) e^{a(x-x_0)} + y_p$$ $$y(x) = (1 - (-3)) e^{2(x-0)} + (-3) = (1+3) e^{2x} - 3 = 4 e^{2x} - 3$$

La formule correcte redonne bien le même résultat.

Conclusion

La recherche de primitives est l'opération inverse de la dérivation et est essentielle pour le calcul d'intégrales. Les équations différentielles linéaires du premier ordre, $y'=ay$ et $y'=ay+b$, sont fondamentales pour modéliser des phénomènes d'évolution proportionnelle (croissance/décroissance exponentielle) avec ou sans apport/retrait constant. La connaissance des solutions générales et la capacité à déterminer la solution unique via une condition initiale (problème de Cauchy) sont des compétences clés.
