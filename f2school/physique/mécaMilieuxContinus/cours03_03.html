Chapitre 3 : Déformation d'un milieu continu

I Description de la transformation

Considérons la figure suivante schématisant la transformation d'un domaine d'un milieu continu entre les instants $t_0$ et $t$ :

Placeholder: Figure II-1 Transformation d'un domaine (D0 -> Dt).

Configuration de référence à $t_0$: ($D_0$). Configuration courante à t : ($D_t$).

Vecteurs position :

$\vec{OM}_0 = X_1 \vec{e}_1 + X_2 \vec{e}_2 + X_3 \vec{e}_3 = X_i \vec{e}_i = \vec{X}$ (Lagrangien)
$\vec{OM}_t = x_1 \vec{e}_1 + x_2 \vec{e}_2 + x_3 \vec{e}_3 = x_i \vec{e}_i = \vec{x}$ (Eulérien)

Vecteur déplacement :

\vec{u} = \vec{OM}_t - \vec{OM}_0 = \vec{x} - \vec{X}

Description lagrangienne : Variables de Lagrange : ($\vec{X}, t$). Pour toute autre grandeur physique G, nous avons $G = G(\vec{X}, t)$.

Description eulérienne : Variables d'Euler : ($\vec{x}, t$). Pour toute autre grandeur physique G, nous avons $G = G(\vec{x}, t)$.

II Tenseurs de la transformation

II-1 Tenseur gradient déplacement (H):

En description lagrangienne : $\vec{u}(\vec{X}, t) \implies$ à t donné : $d\vec{u} = \text{grad}_{\!X} \vec{u} \, d\vec{X} = \boldsymbol{H} \, d\vec{X}$ avec $H_{ij} = \frac{\partial u_i}{\partial X_j}$.

$\boldsymbol{H}$ tenseur gradient déplacement en description lagrangienne.

En description eulérienne : $\vec{u}(\vec{x}, t) \implies$ à t donné : $d\vec{u} = \text{grad}_{\!x} \vec{u} \, d\vec{x} = \boldsymbol{H'} \, d\vec{x}$ avec $H'_{ij} = \frac{\partial u_i}{\partial x_j}$.

$\boldsymbol{H'}$ tenseur gradient déplacement en description eulérienne.

II-2-1 Tenseur gradient (F):

II-2-1 Définition

Au cours de la transformation le vecteur infinitésimal $d\vec{X}$ se transforme en $d\vec{x}$.

Nous avons $x_i = x_i(X_j, t)$ à t donné : $dx_i = \frac{\partial x_i}{\partial X_j} dX_j$.

Soit encore $d\vec{x} = \boldsymbol{F} d\vec{X}$, avec $F_{ij} = \frac{\partial x_i}{\partial X_j}$.

Le tenseur $\boldsymbol{F}$ est appelé tenseur gradient de la transformation.

La transformation inverse donne : $d\vec{X} = \boldsymbol{F}^{-1} d\vec{x} = \boldsymbol{G} d\vec{x}$ avec $G_{ij} (=F^{-1}_{ij}) = \frac{\partial X_i}{\partial x_j}$.

II-2-3 Expression de F en fonction de H :

$\vec{u}$ peut être exprimé en fonction du tenseur déplacement: $\vec{x} = \vec{X} + \vec{u} \implies d\vec{x} = d\vec{X} + d\vec{u}$. En description Lagrangienne, $d\vec{u} = H d\vec{X}$. Donc $d\vec{x} = d\vec{X} + H d\vec{X} = (I + H) d\vec{X}$.

D'où : $\boldsymbol{F = I + H}$.

D'où $\boldsymbol{G = F^{-1} = (I+H)^{-1}}$.

III Tenseurs des dilatations

II-3-1 Définition

Considérons la transformation des vecteurs infinitésimaux $d\vec{X}$ et $d\vec{X}'$, schématisée sur la figure suivante:

Placeholder: Figure III-1 Transformation de dX et dX'.

Nous avons: $d\vec{x} = \boldsymbol{F} d\vec{X}$ et $d\vec{x}' = \boldsymbol{F} d\vec{X}'$.
Le produit scalaire des vecteurs transformés est $d\vec{x} \cdot d\vec{x}' = (\boldsymbol{F} d\vec{X})^T (\boldsymbol{F} d\vec{X}') = d\vec{X}^T \boldsymbol{F}^T \boldsymbol{F} d\vec{X}'$.

Soit encore : $d\vec{x} \cdot d\vec{x}' = d\vec{X}^T \boldsymbol{C} d\vec{X}'$.

$\boldsymbol{C = F^T F}$ : tenseur des dilatations de Cauchy-Green Droit en description lagrangienne.

Par ailleurs $d\vec{X} \cdot d\vec{X}' = (\boldsymbol{F}^{-1} d\vec{x})^T (\boldsymbol{F}^{-1} d\vec{x}') = d\vec{x}^T (\boldsymbol{F}^{-1})^T \boldsymbol{F}^{-1} d\vec{x}' = d\vec{x}^T \boldsymbol{G} d\vec{x}'$.

avec $\boldsymbol{G = C^{-1} = F^{-T} F^{-1}}$ : tenseur des dilatations de Cauchy-Green Gauche ou Finger en description eulérienne.

II-3-2 Expression de C en fonction de H :

$C = F^T F = (I+H)^T (I+H) = (I+H^T)(I+H) = I + H + H^T + H^T H$.

II-3-3 Expression de G en fonction de H' :

$G = F^{-T} F^{-1}$. On a $F^{-1} = I - H'$. Donc $G = (I-H')^T (I-H') = (I-H'^T)(I-H') = I - H' - H'^T + H'^T H'$.

3.5 Tenseurs des déformations

II-4-1 Tenseur de Green-Lagrange (E)

La variation du produit scalaire (ou du carré de la longueur si $d\vec{X}'=d\vec{X}$) est :
$d\vec{x} \cdot d\vec{x}' - d\vec{X} \cdot d\vec{X}' = d\vec{X}^T C d\vec{X}' - d\vec{X}^T I d\vec{X}' = d\vec{X}^T (C-I) d\vec{X}'$.

On pose $2\boldsymbol{E} = \boldsymbol{C} - \boldsymbol{I}$.

Avec : $\boldsymbol{E = \frac{1}{2} (C - I)}$ : Tenseur des déformations de Green-Lagrange en description lagrangienne.

II-4-3 Expression de E en fonction de H :

E = \frac{1}{2} (C - I) = \frac{1}{2} (I + H + H^T + H^T H - I) = \frac{1}{2} (H + H^T + H^T H)

Soit encore en composantes:

E_{ij} = \frac{1}{2} \left( \frac{\partial u_i}{\partial X_j} + \frac{\partial u_j}{\partial X_i} + \frac{\partial u_k}{\partial X_i} \frac{\partial u_k}{\partial X_j} \right)

II-4-2 Tenseur d'Euler-Almansi (e)

La variation peut aussi s'écrire : $d\vec{x} \cdot d\vec{x}' - d\vec{X} \cdot d\vec{X}' = d\vec{x}^T I d\vec{x}' - d\vec{x}^T G d\vec{x}' = d\vec{x}^T (I-G) d\vec{x}'$.

On pose $2\boldsymbol{e} = \boldsymbol{I} - \boldsymbol{G}$.

Avec : $\boldsymbol{e = \frac{1}{2} (I - G)}$ : Tenseur des déformations d'Euler-Almansi en description eulérienne.

II-4-4 Expression de e en fonction de H' :

e = \frac{1}{2}(I-G) = \frac{1}{2} (I - (I - H' - H'^T + H'^T H')) = \frac{1}{2} (H' + H'^T - H'^T H')

Soit encore en composantes:

e_{ij} = \frac{1}{2} \left( \frac{\partial u_i}{\partial x_j} + \frac{\partial u_j}{\partial x_i} - \frac{\partial u_k}{\partial x_i} \frac{\partial u_k}{\partial x_j} \right)

3.6 Transformation des longueurs et des angles

III-1 Dilatation

Soit $d\vec{X}=dX \vec{e}$ un vecteur infinitésimal en $M_0$ porté par le vecteur unitaire $\vec{e}$. On définit la dilatation (stretch) dans la direction $\vec{e}$ par:

\lambda(M_0, \vec{e}) = \frac{ds}{dX} = \sqrt{\frac{d\vec{x} \cdot d\vec{x}}{d\vec{X} \cdot d\vec{X}}} = \sqrt{\frac{d\vec{X}^T C d\vec{X}}{d\vec{X}^T I d\vec{X}}} = \sqrt{\vec{e}^T C \vec{e}}

Si $\vec{e} = \vec{e}_i$ alors : $\lambda(M_0, \vec{e}_i) = \sqrt{C_{ii}} = \sqrt{1 + 2 E_{ii}}$.

III-2 Allongement unitaire

L'allongement unitaire (engineering strain) selon la direction $\vec{e}$ est :

\delta(M_0, \vec{e}) = \frac{ds-dX}{dX} = \lambda(M_0, \vec{e}) - 1 = \sqrt{\vec{e}^T C \vec{e}} - 1

Si $\vec{e} = \vec{e}_i$, alors $\delta(M_0, \vec{e}_i) = \sqrt{C_{ii}} - 1 = \sqrt{1+2E_{ii}} - 1$.

III-3 Glissement ou distorsion angulaire

Soient $d\vec{X}=dX \vec{e}$ et $d\vec{X}'=dX' \vec{e}'$ deux vecteurs issus de $M_0$, initialement orthogonaux ($\vec{e} \perp \vec{e}'$). Après déformation, ils deviennent $d\vec{x}$ et $d\vec{x}'$, formant un angle $\phi$.

Placeholder: Figure VI-1 Glissement angulaire (angle phi entre dx et dx').

On définit le glissement ou la distorsion angulaire au point $M_0$ entre les directions $\vec{e}$ et $\vec{e}'$ par : $\gamma(M_0, \vec{e}, \vec{e}') = \frac{\pi}{2} - \phi$.

Nous avons $d\vec{x} \cdot d\vec{x}' = ds ds' \cos \phi$. Nous avons aussi $d\vec{x} \cdot d\vec{x}' = d\vec{X}^T C d\vec{X}' = (dX \vec{e})^T C (dX' \vec{e}') = dX dX' (\vec{e}^T C \vec{e}')$.

Donc $ds ds' \cos \phi = dX dX' (\vec{e}^T C \vec{e}')$. Or $\cos \phi = \cos(\pi/2 - \gamma) = \sin \gamma$. Et $ds = \lambda(M_0, \vec{e}) dX$, $ds' = \lambda(M_0, \vec{e}') dX'$.

Alors $\lambda(M_0, \vec{e}) dX \lambda(M_0, \vec{e}') dX' \sin \gamma = dX dX' (\vec{e}^T C \vec{e}')$.

\sin \gamma(M_0, \vec{e}, \vec{e}') = \frac{\vec{e}^T C \vec{e}'}{\lambda(M_0, \vec{e}) \lambda(M_0, \vec{e}')}

Dans le cas où $\vec{e}=\vec{e}_1$ et $\vec{e}'=\vec{e}_2$, nous avons $\vec{e}^T C \vec{e}' = C_{12}$.

\sin \gamma_{12} = \frac{C_{12}}{\sqrt{C_{11} C_{22}}} = \frac{2 E_{12}}{\sqrt{(1+2E_{11})(1+2E_{22})}}

IV Transformation d'un volume infinitésimal

Considérons le volume infinitésimal $dV_0$ formé par 3 vecteurs $d\vec{A}, d\vec{B}, d\vec{C}$ issus de $M_0$. Ce volume est $dV_0 = | (d\vec{A} \wedge d\vec{B}) \cdot d\vec{C} | = | \det(d\vec{A}, d\vec{B}, d\vec{C}) |$.

Après transformation, ces vecteurs deviennent $d\vec{a} = F d\vec{A}$, $d\vec{b} = F d\vec{B}$, $d\vec{c} = F d\vec{C}$. Le volume transformé est $dV = | \det(d\vec{a}, d\vec{b}, d\vec{c}) |$.

$dV = | \det(F d\vec{A}, F d\vec{B}, F d\vec{C}) | = | \det(F) \det(d\vec{A}, d\vec{B}, d\vec{C}) | = |\det F| dV_0$.

Comme $J = \det F > 0$ pour une transformation physiquement admissible :

dV = J dV_0

La dilatation volumique est donnée par:

\theta = \frac{dV - dV_0}{dV_0} = J - 1 = \det F - 1

Applications directes

V Transformation d'une surface infinitésimale orientée

Considérons une surface élémentaire $dS_0$ orientée par le vecteur unitaire $\vec{N}$ dans la configuration initiale. Son vecteur surface est $d\vec{S}_0 = dS_0 \vec{N}$. Les vecteurs $d\vec{A}$ et $d\vec{B}$ sont contenus dans cette surface, $d\vec{S}_0 = d\vec{A} \wedge d\vec{B}$.

Après transformation, cette surface devient $dS$ orientée par $\vec{n}$, de vecteur surface $d\vec{S} = dS \vec{n}$. Les vecteurs $d\vec{a} = F d\vec{A}$ et $d\vec{b} = F d\vec{B}$ sont contenus dans la surface transformée. $d\vec{S} = d\vec{a} \wedge d\vec{b}$.

On montre alors la formule de Nanson :

d\vec{S} = J F^{-T} d\vec{S}_0 \quad \text{ou} \quad n_i dS = J \frac{\partial x_i}{\partial X_j} N_j dS_0

VI Dilatations principales

$\boldsymbol{C} = F^T F$ est un tenseur symétrique défini positif. Il existe alors une base orthonormée (formée à partir des vecteurs propres $\vec{N}_i$) : base principale $(\vec{N}_1, \vec{N}_2, \vec{N}_3)$. Dans cette base, C est diagonal :

[C]_{\text{princ}} = \begin{pmatrix} c_1 & 0 & 0 \\ 0 & c_2 & 0 \\ 0 & 0 & c_3 \end{pmatrix}

Où $c_i$ sont les valeurs propres de C ($c_i > 0$). Les valeurs propres $c_i$ sont les carrés des dilatations principales $\lambda_i = \lambda(M_0, \vec{N}_i)$, qui sont les dilatations dans les directions principales $\vec{N}_i$.

c_i = \lambda_i^2

Nous avons $\det C = (\det F)^2 = J^2$. Et $\det C = c_1 c_2 c_3 = \lambda_1^2 \lambda_2^2 \lambda_3^2$. D'où :

J = \lambda_1 \lambda_2 \lambda_3

VII Déformations principales

Nous avons : $\boldsymbol{E = \frac{1}{2}(C-I)}$. Alors : $E \vec{N}_i = \frac{1}{2}(C-I) \vec{N}_i = \frac{1}{2}(C\vec{N}_i - I\vec{N}_i) = \frac{1}{2}(c_i \vec{N}_i - \vec{N}_i) = \frac{1}{2}(c_i-1) \vec{N}_i$.

On déduit alors que la base principale de E est la même que celle de C ($\vec{N}_1, \vec{N}_2, \vec{N}_3$) et les valeurs propres de E sont $e_i = \frac{1}{2}(c_i - 1) = \frac{1}{2}(\lambda_i^2-1)$. Ce sont les déformations principales.

[E]_{\text{princ}} = \begin{pmatrix} e_1 & 0 & 0 \\ 0 & e_2 & 0 \\ 0 & 0 & e_3 \end{pmatrix} = \begin{pmatrix} \frac{\lambda_1^2-1}{2} & 0 & 0 \\ 0 & \frac{\lambda_2^2-1}{2} & 0 \\ 0 & 0 & \frac{\lambda_3^2-1}{2} \end{pmatrix}

VIII Décomposition polaire

La relation $d\vec{x}=\boldsymbol{F} d\vec{X}$ montre que $\boldsymbol{F}$ transforme $d\vec{X}$ en $d\vec{x}$. Cette transformation peut être décomposée en un mouvement de corps rigide (translation + rotation) suivi d'une déformation pure.

Mathématiquement, $\boldsymbol{F}$ étant inversible ($\det F \ne 0$), peut être écrit sous la forme (Décomposition polaire droite et gauche) :

$\boldsymbol{F = R U = V R}$

Où :

  • $\boldsymbol{R}$ tenseur orthogonal ($R^T R = R R^T = I$) et propre ($\det R = 1$, rotation pure).
  • $\boldsymbol{U}, \boldsymbol{V}$ tenseurs symétriques définis positifs ($U=U^T, V=V^T$).

$\boldsymbol{R}$ représente le mouvement d'un corps indéformable (rotation). $\boldsymbol{U}$ (tenseur d'étirement droit) et $\boldsymbol{V}$ (tenseur d'étirement gauche) représentent le mouvement d'une déformation pure.

$U = ?$ Nous avons $C = F^T F = (RU)^T RU = U^T R^T R U = U^T I U = U^2$. D'où $\boldsymbol{U = \sqrt{C}}$ (racine carrée symétrique définie positive de C).

$V = ?$ Nous avons $B = FF^T = VR R^T V^T = V I V^T = V^2$. D'où $\boldsymbol{V = \sqrt{B}}$ (où $B=FF^T$ est le tenseur de Cauchy-Green gauche).

$R = ?$ Nous avons $F=RU \implies \boldsymbol{R = F U^{-1}}$ ou $F=VR \implies \boldsymbol{R = V^{-1} F}$.

$C = U^2 \implies$ les vecteurs propres de $U$ sont identiques à ceux de $C (\vec{N}_i)$. Les valeurs propres de U sont $\lambda_i = \sqrt{c_i}$.

$B = V^2 \implies$ les vecteurs propres de $V$ sont identiques à ceux de $B (\vec{n}_i=R\vec{N}_i)$. Les valeurs propres de V sont $\lambda_i = \sqrt{b_i}$. (On montre $b_i=c_i=\lambda_i^2$).

[U]_{\text{princ } (\vec{N}_i)} = \begin{pmatrix} \lambda_1 & 0 & 0 \\ 0 & \lambda_2 & 0 \\ 0 & 0 & \lambda_3 \end{pmatrix}
[V]_{\text{princ } (\vec{n}_i)} = \begin{pmatrix} \lambda_1 & 0 & 0 \\ 0 & \lambda_2 & 0 \\ 0 & 0 & \lambda_3 \end{pmatrix}

Par ailleurs, $B = FF^T = (RU)(RU)^T = R U U^T R^T = R U^2 R^T = R C R^T$. Donc $V = \sqrt{B} = R \sqrt{C} R^T = R U R^T$.

Finalement : $\boldsymbol{F = RU = VR}$ avec $\boldsymbol{U = \sqrt{F^T F}}$ et $\boldsymbol{V = \sqrt{FF^T}} = R U R^T$.
