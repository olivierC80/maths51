MECANIQUE DES MILIEUX CONTINUS

COURS ET APPLICATIONS - EXERCICES CORRIGES

Exercices Corrigés (Golay)

1. Calcul tensoriel

Exercice A. Montrer que la symétrie est une propriété tensorielle.

C'est-à-dire que si un tenseur $\boldsymbol{S}$ est symétrique ($S_{ij} = S_{ji}$) dans une base orthonormée donnée $B = (\vec{e}_1, \vec{e}_2, \vec{e}_3)$, alors cette propriété est également vraie dans toute autre base orthonormée $B' = (\vec{e}'_1, \vec{e}'_2, \vec{e}'_3)$.

Corrigé A:

Soit $\boldsymbol{S}$ symétrique. Dans la base $B$, on peut donc écrire $S_{ij} = S_{ji}$. Soit une base orthonormée quelconque $B' = (\vec{e}'_1, \vec{e}'_2, \vec{e}'_3)$ différente de $B$. On appelle $\alpha$ la matrice de passage de $B$ à $B'$ ($\vec{e}'_i = \alpha_{ij} \vec{e}_j$). Si on appelle $S'_{ij}$ les termes de la matrice de $\boldsymbol{S}$ dans la base $B'$, on a d'après le cours (formule de changement de base pour tenseur d'ordre 2, Eq 1.2.5 dans Golay):

S'_{ij} = \alpha_{ip} \alpha_{jq} S_{pq}

Calculons $S'_{ji}$:

S'_{ji} = \alpha_{jp} \alpha_{iq} S_{pq}

Or la matrice $S$ est symétrique, on a donc $S_{pq} = S_{qp}$. On écrit donc :

S'_{ji} = \alpha_{jp} \alpha_{iq} S_{qp}

Les indices $p$ et $q$ du second membre sont muets, on pourrait donc les remplacer par n'importe quelle lettre, et on peut aussi les intervertir :

S'_{ji} = \alpha_{jq} \alpha_{ip} S_{pq} = \alpha_{ip} \alpha_{jq} S_{pq}

On en déduit que $S'_{ij} = S'_{ji}$, ce qu'il fallait démontrer.

Exercice B. Montrer que l'antisymétrie est également une propriété tensorielle.

Corrigé B:

Soit un tenseur $\boldsymbol{S}$ antisymétrique. Dans la base $B = (\vec{e}_1, \vec{e}_2, \vec{e}_3)$ on peut donc écrire $S_{ij} = -S_{ji}$. Soit une base orthonormée quelconque $B' = (\vec{e}'_1, \vec{e}'_2, \vec{e}'_3)$ différente de $B$. On appelle $\alpha$ la matrice de passage de $B$ à $B'$. Si on appelle $S'_{ij}$ les termes de la matrice de $\boldsymbol{S}$ dans la base $B'$, on a d'après le cours :

S'_{ij} = \alpha_{ip} \alpha_{jq} S_{pq}

Calculons $S'_{ji}$:

S'_{ji} = \alpha_{jp} \alpha_{iq} S_{pq}

Or la matrice $S$ est antisymétrique, on a donc $S_{pq} = -S_{qp}$. On écrit donc :

S'_{ji} = \alpha_{jp} \alpha_{iq} (-S_{qp}) = - \alpha_{jp} \alpha_{iq} S_{qp}

Les indices $p$ et $q$ du second membre sont muets, on pourrait donc les remplacer par n'importe quelle lettre, et on peut aussi les intervertir :

S'_{ji} = - \alpha_{jq} \alpha_{ip} S_{pq} = - (\alpha_{ip} \alpha_{jq} S_{pq})

On en déduit que $S'_{ji} = -S'_{ij}$, ce qu'il fallait démontrer.

Exercice C. Montrer qu'un tenseur $\boldsymbol{T}$ quelconque peut toujours se décomposer en une partie symétrique et une partie antisymétrique.

Corrigé C:

On reprend les formules du cours. Soient les tenseurs $\boldsymbol{T}^S$ et $\boldsymbol{T}^A$ donnés par les formules suivantes :

\boldsymbol{T}^S = \frac{1}{2} (\boldsymbol{T} + \boldsymbol{T}^T)
\boldsymbol{T}^A = \frac{1}{2} (\boldsymbol{T} - \boldsymbol{T}^T)

On va démontrer que $\boldsymbol{T}^S$ est symétrique et que $\boldsymbol{T}^A$ est antisymétrique. Pour cela on se place dans une base $B = (\vec{e}_1, \vec{e}_2, \vec{e}_3)$, pour laquelle le tenseur $\boldsymbol{T}$ s'exprime sous la forme d'une matrice de terme générale $T_{ij}$. Explicitions les termes des matrices de $\boldsymbol{T}^S$ et $\boldsymbol{T}^A$ :

T^S_{ij} = \frac{1}{2} (T_{ij} + T_{ji}) = \frac{1}{2} (T_{ji} + T_{ij}) = T^S_{ji}
T^A_{ij} = \frac{1}{2} (T_{ij} - T_{ji}) = - \frac{1}{2} (T_{ji} - T_{ij}) = -T^A_{ji}

On a donc démontré que $\boldsymbol{T}^S$ est symétrique et $\boldsymbol{T}^A$ antisymétrique. De plus, $\boldsymbol{T}^S + \boldsymbol{T}^A = \frac{1}{2}(\boldsymbol{T} + \boldsymbol{T}^T) + \frac{1}{2}(\boldsymbol{T} - \boldsymbol{T}^T) = \boldsymbol{T}$.

Exercice D. Soit $\boldsymbol{S}$ un tenseur symétrique et $\boldsymbol{A}$ un tenseur antisymétrique, montrer que l'on a toujours $\boldsymbol{S} : \boldsymbol{A} = 0$.

Corrigé D:

Dans une base orthonormée $B = (\vec{e}_1, \vec{e}_2, \vec{e}_3)$, on peut écrire que $S_{ij} = S_{ji}$ et également que $A_{ij} = -A_{ji}$. On en déduit que, comme pour tout tenseur antisymétrique, les termes diagonaux de la matrice de $\boldsymbol{A}$ sont nuls dans toute base ($A_{ii}=-A_{ii} \implies A_{ii}=0$). D'après le cours, le produit doublement contracté de ces deux tenseurs est un scalaire égal à :

\boldsymbol{S} : \boldsymbol{A} = S_{ij} A_{ji}

Développons cette notation d'Einstein sous forme explicite :

$\boldsymbol{S} : \boldsymbol{A} = S_{11}A_{11} + S_{12}A_{21} + S_{13}A_{31}$
$+ S_{21}A_{12} + S_{22}A_{22} + S_{23}A_{32}$
$+ S_{31}A_{13} + S_{32}A_{23} + S_{33}A_{33}$

On sait que l'on a $A_{11} = A_{22} = A_{33} = 0$, et par ailleurs :

$S_{12} = S_{21}$ ; $S_{13} = S_{31}$ ; $S_{23} = S_{32}$
$A_{21} = -A_{12}$ ; $A_{31} = -A_{13}$ ; $A_{32} = -A_{23}$

En substituant :

$\boldsymbol{S} : \boldsymbol{A} = S_{12}(-A_{12}) + S_{13}(-A_{13}) + S_{12}A_{12} + S_{23}(-A_{23}) + S_{13}A_{13} + S_{23}A_{23} = 0$.

On en déduit directement $\boldsymbol{S} : \boldsymbol{A} = 0$.

Exercice E. Soit $\boldsymbol{S}$ un tenseur symétrique et $\boldsymbol{T}$ un tenseur quelconque, montrer que l'on a toujours : $\boldsymbol{S} : \boldsymbol{T} = \boldsymbol{S} : \boldsymbol{T}^S$, où $\boldsymbol{T}^S$ est la partie symétrique de $\boldsymbol{T}$.

Corrigé E:

Le résultat découle directement de celui de l'exercice précédent et de la distributivité de l'opérateur produit doublement contracté :

\boldsymbol{S} : \boldsymbol{T} = \boldsymbol{S} : (\boldsymbol{T}^S + \boldsymbol{T}^A) = \boldsymbol{S} : \boldsymbol{T}^S + \boldsymbol{S} : \boldsymbol{T}^A = \boldsymbol{S} : \boldsymbol{T}^S

Car $\boldsymbol{S} : \boldsymbol{T}^A = 0$ puisque S est symétrique et TA est antisymétrique.

Exercice F. Soit $\vec{A}$ un champ vectoriel. Montrer que l'on a toujours $\text{div}(\text{rot } \vec{A}) = 0$.

Corrigé F:

Posons le résultat intermédiaire $\vec{B} = \text{rot } \vec{A}$. Dans une base orthonormée donnée $B = (\vec{e}_1, \vec{e}_2, \vec{e}_3)$, on a par définition :

\vec{B} = \begin{pmatrix} \partial A_3 / \partial x_2 - \partial A_2 / \partial x_3 \\ \partial A_1 / \partial x_3 - \partial A_3 / \partial x_1 \\ \partial A_2 / \partial x_1 - \partial A_1 / \partial x_2 \end{pmatrix}_B

Par ailleurs, on a par définition : $\text{div } \vec{B} = \frac{\partial B_1}{\partial x_1} + \frac{\partial B_2}{\partial x_2} + \frac{\partial B_3}{\partial x_3}$.

On en déduit que $\text{div}(\text{rot } \vec{A})$ vaut :

$\text{div}(\text{rot } \vec{A}) = \frac{\partial}{\partial x_1}(\frac{\partial A_3}{\partial x_2} - \frac{\partial A_2}{\partial x_3}) + \frac{\partial}{\partial x_2}(\frac{\partial A_1}{\partial x_3} - \frac{\partial A_3}{\partial x_1}) + \frac{\partial}{\partial x_3}(\frac{\partial A_2}{\partial x_1} - \frac{\partial A_1}{\partial x_2})$
\text{div}(\text{rot } \vec{A}) = \frac{\partial^2 A_3}{\partial x_1 \partial x_2} - \frac{\partial^2 A_2}{\partial x_1 \partial x_3} + \frac{\partial^2 A_1}{\partial x_2 \partial x_3} - \frac{\partial^2 A_3}{\partial x_2 \partial x_1} + \frac{\partial^2 A_2}{\partial x_3 \partial x_1} - \frac{\partial^2 A_1}{\partial x_3 \partial x_2}

L'ordre des dérivations partielles successives d'une fonction de plusieurs variables est quelconque (Théorème de Schwarz, si les dérivées secondes sont continues), on peut donc en déduire directement : $\text{div}(\text{rot } \vec{A}) = 0$.

Exercice G. Soit A un champ scalaire. Montrer que l'on a toujours $\text{rot}(\text{grad } A) = \vec{0}$.

Corrigé G:

On se place également dans une base orthonormée $B = (\vec{e}_1, \vec{e}_2, \vec{e}_3)$. Soit le résultat intermédiaire $\vec{B} = \text{grad } A$ donné par la formule du cours :

\vec{B} = \begin{pmatrix} \partial A / \partial x_1 \\ \partial A / \partial x_2 \\ \partial A / \partial x_3 \end{pmatrix}_B

Par ailleurs, on peut écrire, toujours dans la base $B$ :

$\text{rot } \vec{B} = \begin{pmatrix} \partial B_3 / \partial x_2 - \partial B_2 / \partial x_3 \\ \partial B_1 / \partial x_3 - \partial B_3 / \partial x_1 \\ \partial B_2 / \partial x_1 - \partial B_1 / \partial x_2 \end{pmatrix}_B$

On en déduit :

\text{rot}(\text{grad } A) = \begin{pmatrix} \frac{\partial^2 A}{\partial x_2 \partial x_3} - \frac{\partial^2 A}{\partial x_3 \partial x_2} \\ \frac{\partial^2 A}{\partial x_3 \partial x_1} - \frac{\partial^2 A}{\partial x_1 \partial x_3} \\ \frac{\partial^2 A}{\partial x_1 \partial x_2} - \frac{\partial^2 A}{\partial x_2 \partial x_1} \end{pmatrix}_B

Pour la même raison que dans l'exercice précédent, on peut donc écrire directement la formule classique $\text{rot}(\text{grad } A) = \vec{0}$.

Exercice H. Coordonnées sphériques.

Soit la base curviligne polaire $(\vec{e}_r, \vec{e}_\theta, \vec{e}_\phi)$, telle que :

$\begin{cases} x = R \sin\phi \cos\theta \\ y = R \sin\phi \sin\theta \\ z = R \cos\phi \end{cases}$
Placeholder: Diagram of spherical coordinates basis.

1. Calculer la surface extérieure d'une sphère de rayon R, sachant qu'un élément de surface infinitésimal peut s'écrire $dS = R^2 \sin\phi d\theta d\phi$.

Corrigé H.1:

On cherche à calculer $S = \int_{\text{sphère}} dS$. En paramétrant la surface en fonction de $\theta$ et $\phi$, on peut expliciter cette intégrale :

S = \int_{\phi=0}^{\pi} \int_{\theta=0}^{2\pi} R^2 \sin\phi d\theta d\phi

Le rayon R est indépendant de $\theta$ et $\phi$, et le terme $\sin\phi$ est indépendant de $\theta$, on peut donc écrire :

S = R^2 \int_{\phi=0}^{\pi} \sin\phi \left( \int_{\theta=0}^{2\pi} d\theta \right) d\phi
S = R^2 \int_{\phi=0}^{\pi} \sin\phi [\theta]_0^{2\pi} d\phi = 2\pi R^2 \int_{\phi=0}^{\pi} \sin\phi d\phi
S = 2\pi R^2 [-\cos\phi]_0^\pi = 2\pi R^2 [-(-1) - (-1)] = 2\pi R^2 [1+1] = 4\pi R^2

Finalement, on obtient le résultat classique $S = 4\pi R^2$.

2. Calculer l'intégrale du champ $\cos\phi \vec{e}_R$ sur cette sphère, sachant que le vecteur élémentaire dans la direction radiale vaut $\vec{e}_R = \sin\phi \cos\theta \vec{e}_x + \sin\phi \sin\theta \vec{e}_y + \cos\phi \vec{e}_z$.

Corrigé H.2:

On doit calculer :

\vec{A} = \iint_{\text{sphère}} \cos\phi \vec{e}_R dS = \int_{\phi=0}^{\pi} \int_{\theta=0}^{2\pi} \cos\phi \vec{e}_R R^2 \sin\phi d\theta d\phi

Avec : $\vec{e}_R = \sin\phi \cos\theta \vec{e}_x + \sin\phi \sin\theta \vec{e}_y + \cos\phi \vec{e}_z$.

On est en coordonnées curvilignes, donc on ne peut pas sortir le vecteur $\vec{e}_R$ car il n'est pas indépendant du point d'intégration. En revanche, les trois vecteurs de la base cartésienne $(\vec{e}_x, \vec{e}_y, \vec{e}_z)$ ont cette propriété, et peuvent être sortis de l'intégrale. On peut donc remplacer $\vec{e}_R$ par son expression, et écrire dans la base $B = (\vec{e}_x, \vec{e}_y, \vec{e}_z)$ :

$\vec{A} = \int_{\phi=0}^{\pi} \int_{\theta=0}^{2\pi} \cos\phi (\sin\phi \cos\theta \vec{e}_x + \sin\phi \sin\theta \vec{e}_y + \cos\phi \vec{e}_z) R^2 \sin\phi d\theta d\phi$
\vec{A} = \begin{pmatrix} R^2 \int \int \cos\phi \sin^2\phi \cos\theta d\theta d\phi \\ R^2 \int \int \cos\phi \sin^2\phi \sin\theta d\theta d\phi \\ R^2 \int \int \cos^2\phi \sin\phi d\theta d\phi \end{pmatrix}_B
\vec{A} = R^2 \begin{pmatrix} (\int_{\phi=0}^{\pi} \cos\phi \sin^2\phi d\phi) (\int_{\theta=0}^{2\pi} \cos\theta d\theta) \\ (\int_{\phi=0}^{\pi} \cos\phi \sin^2\phi d\phi) (\int_{\theta=0}^{2\pi} \sin\theta d\theta) \\ (\int_{\phi=0}^{\pi} \cos^2\phi \sin\phi d\phi) (\int_{\theta=0}^{2\pi} d\theta) \end{pmatrix}_B

Du fait de la périodicité des fonctions trigonométriques, on a :

$\int_0^{2\pi} \cos\theta d\theta = [\sin\theta]_0^{2\pi} = 0$
$\int_0^{2\pi} \sin\theta d\theta = [-\cos\theta]_0^{2\pi} = 0$

Par ailleurs, on a :

$\int_0^{2\pi} d\theta = 2\pi$

Donc :

\vec{A} = 2\pi R^2 \begin{pmatrix} 0 \\ 0 \\ \int_{\phi=0}^{\pi} \cos^2\phi \sin\phi d\phi \end{pmatrix}_B

La primitive de la fonction $\cos^2\phi \sin\phi$ est la fonction $-\frac{\cos^3\phi}{3}$, donc :

$\int_0^{\pi} \cos^2\phi \sin\phi d\phi = [-\frac{\cos^3\phi}{3}]_0^\pi = (-\frac{(-1)^3}{3}) - (-\frac{1^3}{3}) = \frac{1}{3} + \frac{1}{3} = \frac{2}{3}$.
\vec{A} = 2\pi R^2 \begin{pmatrix} 0 \\ 0 \\ 2/3 \end{pmatrix}_B = \begin{pmatrix} 0 \\ 0 \\ \frac{4\pi R^2}{3} \end{pmatrix}_B

3. Déformations

Exercice A.

Soit un milieu soumis à un tenseur de déformation $\boldsymbol{E}$, dont la matrice dans une base $B = (\vec{e}_1, \vec{e}_2, \vec{e}_3)$ donnée est :

[\boldsymbol{E}]_B = \begin{pmatrix} 3 & 2 & 4 \\ 2 & 3 & 4 \\ 4 & 4 & 1 \end{pmatrix}_B

1. Calculer la trace de $\boldsymbol{E}$, ainsi que son déterminant. Calculer la matrice du tenseur $\boldsymbol{E}^2$, et en déduire le deuxième invariant du tenseur $\boldsymbol{E}$ donné par l'expression classique $E_{II} = \frac{1}{2}( (Tr \boldsymbol{E})^2 - Tr(\boldsymbol{E}^2) )$. En déduire sans calcul le polynôme caractéristique du tenseur $\boldsymbol{E}$.

2. Développer le déterminant du tenseur $\boldsymbol{E} - \lambda \boldsymbol{I}$ et retrouver l'expression du polynôme caractéristique.

3. Développer différemment $\det(\boldsymbol{E} - \lambda \boldsymbol{I})$ pour l'exprimer sous la forme suivante, beaucoup plus intéressante : $\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = -(\lambda - E_1)(\lambda - E_2)(\lambda - E_3)$. En déduire les valeurs des déformations principales $E_1, E_2, E_3$ et la forme de la matrice de $\boldsymbol{E}$ dans sa base principale.

4. Ordonner les déformations principales, puis calculer les coordonnées du vecteur propre $\vec{b}_2$ correspondant à la déformation principale intermédiaire $E_2$.

Corrigé A:

1. La trace de $\boldsymbol{E}$ s'obtient immédiatement :

$\text{tr}(\boldsymbol{E}) = 3 + 3 + 1 = 7 = E_I$.

Le déterminant d'une matrice 3*3 se calcule par la formule classique : somme des produits des termes selon les diagonales descendantes moins somme des produits des termes selon les diagonales montantes :

$\det(\boldsymbol{E}) = 3 \times 3 \times 1 + 2 \times 4 \times 4 + 4 \times 2 \times 4 - 4 \times 3 \times 4 - 4 \times 4 \times 3 - 1 \times 2 \times 2 = 9 + 32 + 32 - 48 - 48 - 4 = -27 = E_{III}$.

Le tenseur $\boldsymbol{E}^2$ est le produit contracté de $\boldsymbol{E}$ par lui-même. Sous forme matricielle dans la base $B = (\vec{e}_1, \vec{e}_2, \vec{e}_3)$, on a :

[\boldsymbol{E}]^2 = \begin{pmatrix} 3 & 2 & 4 \\ 2 & 3 & 4 \\ 4 & 4 & 1 \end{pmatrix}_B \cdot \begin{pmatrix} 3 & 2 & 4 \\ 2 & 3 & 4 \\ 4 & 4 & 1 \end{pmatrix}_B = \begin{pmatrix} 29 & 28 & 24 \\ 28 & 29 & 24 \\ 24 & 24 & 33 \end{pmatrix}_B

La trace de ce tenseur vaut $\text{tr}(\boldsymbol{E}^2) = 29 + 29 + 33 = 91$.

Le second invariant principal de $\boldsymbol{E}$ s'obtient directement par la formule de l'énoncé :

E_{II} = \frac{1}{2}( (Tr \boldsymbol{E})^2 - Tr(\boldsymbol{E}^2) ) = \frac{1}{2}(7^2 - 91) = \frac{1}{2}(49 - 91) = -21

Par application de la formule du cours, on en déduit le polynôme caractéristique du tenseur des déformations linéarisées :

$\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = -\lambda^3 + E_I \lambda^2 - E_{II} \lambda + E_{III} = -\lambda^3 + 7 \lambda^2 + 21 \lambda - 27$.

2. On cherche le déterminant du tenseur $\boldsymbol{E} - \lambda \boldsymbol{I}$ qui s'exprime sous forme matricielle dans la base $B = (\vec{e}_1, \vec{e}_2, \vec{e}_3)$ par :

\boldsymbol{E} - \lambda \boldsymbol{I} = \begin{pmatrix} 3 & 2 & 4 \\ 2 & 3 & 4 \\ 4 & 4 & 1 \end{pmatrix}_B - \lambda \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{pmatrix}_B = \begin{pmatrix} 3-\lambda & 2 & 4 \\ 2 & 3-\lambda & 4 \\ 4 & 4 & 1-\lambda \end{pmatrix}_B

Donc :

$\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = \begin{vmatrix} 3-\lambda & 2 & 4 \\ 2 & 3-\lambda & 4 \\ 4 & 4 & 1-\lambda \end{vmatrix}$
$\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = (3-\lambda)[(3-\lambda)(1-\lambda)-16] - 2[2(1-\lambda)-16] + 4[8-4(3-\lambda)]$
$= (3-\lambda)(\lambda^2-4\lambda-13) - 2(-2\lambda-14) + 4(4\lambda-4)$
$= (3\lambda^2-12\lambda-39 - \lambda^3+4\lambda^2+13\lambda) + 4\lambda+28 + 16\lambda-16$
$= -\lambda^3 + 7\lambda^2 + \lambda - 39 + 20\lambda + 12$
\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = -\lambda^3 + 7\lambda^2 + 21\lambda - 27

On retrouve donc par le calcul détaillé le résultat de la question précédente.

3. On cherche à obtenir un développement du polynôme caractéristique faisant intervenir directement ses racines, sous la forme $\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = -(\lambda - E_1)(\lambda - E_2)(\lambda - E_3)$. Cette opération est peu aisée si on démarre de l'expression développée du polynôme, et il est plus judicieux de chercher cette expression dès le calcul du déterminant. On prend donc comme point de départ :

\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = \begin{vmatrix} 3-\lambda & 2 & 1 \\ 2 & 3-\lambda & 3 \\ 4 & 4 & 1-\lambda \end{vmatrix} \quad (\text{Erreur dans l'enoncé original, 3eme colonne corrigée})

(Avec la matrice originale, $\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = \begin{vmatrix} 3-\lambda & 2 & 4 \\ 2 & 3-\lambda & 4 \\ 4 & 4 & 1-\lambda \end{vmatrix}$). Selon les formules classiques de calcul de déterminant, il est possible de remplacer une colonne (respectivement une ligne) par une combinaison linéaire des différentes colonnes (respectivement des différentes lignes). On remplace donc la colonne C1 par la combinaison C1-C2 :

\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = \begin{vmatrix} (3-\lambda)-2 & 2 & 4 \\ 2-(3-\lambda) & 3-\lambda & 4 \\ 4-4 & 4 & 1-\lambda \end{vmatrix} = \begin{vmatrix} 1-\lambda & 2 & 4 \\ -1+\lambda & 3-\lambda & 4 \\ 0 & 4 & 1-\lambda \end{vmatrix}

Il est également autorisé de "sortir" une constante multiplicative d'une ligne ou d'une colonne du déterminant, donc on peut simplifier la première colonne :

\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = (1-\lambda) \begin{vmatrix} 1 & 2 & 4 \\ -1 & 3-\lambda & 4 \\ 0 & 4 & 1-\lambda \end{vmatrix}

On remplace ensuite la ligne L2 par la combinaison linéaire L1+L2 :

\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = (1-\lambda) \begin{vmatrix} 1 & 2 & 4 \\ 0 & 5-\lambda & 8 \\ 0 & 4 & 1-\lambda \end{vmatrix}

On peut donc écrire :

$\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = (1-\lambda) [1 \times ((5-\lambda)(1-\lambda) - 4 \times 8)]$
\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = (1-\lambda) (\lambda^2 - 6\lambda + 5 - 32) = (1-\lambda) (\lambda^2 - 6\lambda - 27)

Il reste quelques calculs, mais on sait qu'on a atteint le but principal, car $\lambda^2 - 6\lambda - 27$ est un polynôme de degré 2 dont on sait très bien retrouver les racines. Le discriminant du polynôme de second degré est : $\Delta = (-6)^2 - 4 \times 1 \times (-27) = 36 + 108 = 144$. Et les deux racines de ce polynôme sont : $\frac{6 \pm \sqrt{144}}{2} = 3 \pm \frac{12}{2} = 3 \pm 6$. Soit 9 et -3.

Finalement, le polynôme caractéristique du tenseur des déformations linéarisées est donné par :

\det(\boldsymbol{E} - \lambda \boldsymbol{I}) = -(\lambda - 1)(\lambda - 9)(\lambda + 3)

Les déformations principales sont les racines de ce polynôme, et on peut écrire la matrice de $\boldsymbol{E}$ dans sa base principale :

[\boldsymbol{E}]_{\text{Base principale}} = \begin{pmatrix} E_1 & 0 & 0 \\ 0 & E_2 & 0 \\ 0 & 0 & E_3 \end{pmatrix} = \begin{pmatrix} 9 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & -3 \end{pmatrix}_{\text{Base principale}}

4. On ordonne les déformations principales dans l'ordre décroissant, et on a : $E_1 = 9$, $E_2 = 1$, $E_3 = -3$.

On pourrait calculer les coordonnées dans $B = (\vec{e}_1, \vec{e}_2, \vec{e}_3)$ des trois vecteurs $\vec{b}_1, \vec{b}_2, \vec{b}_3$ de la base principale de déformation. On va le faire uniquement pour $\vec{b}_2$ car la méthode est toujours la même.

D'après le cours, la direction principale de déformation $\vec{b}_2$ est définie par :

\boldsymbol{E} \vec{b}_2 = E_2 \vec{b}_2

En développant de système, on obtient :

$\begin{pmatrix} 3 & 2 & 4 \\ 2 & 3 & 4 \\ 4 & 4 & 1 \end{pmatrix}_B \begin{pmatrix} b_{2,1} \\ b_{2,2} \\ b_{2,3} \end{pmatrix}_B = E_2 \begin{pmatrix} b_{2,1} \\ b_{2,2} \\ b_{2,3} \end{pmatrix}_B = 1 \begin{pmatrix} b_{2,1} \\ b_{2,2} \\ b_{2,3} \end{pmatrix}_B$

Dans cette expression, on a fait apparaître les trois coordonnées $(b_{2,1}, b_{2,2}, b_{2,3})$ du vecteur $\vec{b}_2$ dans la base $B = (\vec{e}_1, \vec{e}_2, \vec{e}_3)$. En mettant ce système matriciel sous forme d'un système d'équations linéaires, on a :

$\begin{cases} 3 b_{2,1} + 2 b_{2,2} + 4 b_{2,3} = b_{2,1} \\ 2 b_{2,1} + 3 b_{2,2} + 4 b_{2,3} = b_{2,2} \\ 4 b_{2,1} + 4 b_{2,2} + b_{2,3} = b_{2,3} \end{cases}$

Soit :

$\begin{cases} 2 b_{2,1} + 2 b_{2,2} + 4 b_{2,3} = 0 \\ 2 b_{2,1} + 2 b_{2,2} + 4 b_{2,3} = 0 \\ 4 b_{2,1} + 4 b_{2,2} = 0 \end{cases}$

On remarque immédiatement que $b_{2,1} = -b_{2,2}$ et que $b_{2,3} = 0$. Or on sait que le vecteur $\vec{b}_2$ est unitaire (de norme égale à 1), donc on peut écrire :

$b_{2,1}^2 + b_{2,2}^2 + b_{2,3}^2 = 1 \implies b_{2,1}^2 + (-b_{2,1})^2 + 0^2 = 1 \implies 2 b_{2,1}^2 = 1$
\implies \begin{cases} b_{2,1} = \pm \frac{\sqrt{2}}{2} \\ b_{2,2} = \mp \frac{\sqrt{2}}{2} \\ b_{2,3} = 0 \end{cases}

On note une indétermination dans le signe de $b_{2,1}$ et $b_{2,2}$. Tous les signes sont possibles, à condition que l'on vérifie $b_{2,1} = -b_{2,2}$.

Corrigé A:

On recherche les termes de la matrice de $\boldsymbol{\sigma}$ à partir d'un certain nombre de mesures de forces effectuées sur les faces d'un domaine matériel. L'état de contrainte est supposé homogène et stationnaire, et le tenseur de Cauchy est toujours symétrique, on cherche donc en tout et pour tout six inconnues scalaires : $\sigma_{11}, \sigma_{22}, \sigma_{33}, \sigma_{12}, \sigma_{13}, \sigma_{23}$.

On va d'abord travailler sur la force appliquée sur la face arrière du tétraèdre (plan $x_1=0$). L'énoncé dit que la composante normale à cette force est égale à $F_1$. On peut également remarquer que la facette en question (face triangulaire arrière OCB) est de normale sortante $-\vec{e}_1$.

On va chercher à exprimer le vecteur contrainte appliqué sur cette facette. On peut l'énoncer ainsi à partir de la formule du cours :

$\vec{T}(-\vec{e}_1) = \boldsymbol{\sigma} \cdot (-\vec{e}_1)$

Avec : $\vec{n} = \begin{pmatrix} -1 \\ 0 \\ 0 \end{pmatrix}_B$ et $[\sigma] = \begin{pmatrix} \sigma_{11} & \sigma_{12} & \sigma_{13} \\ \sigma_{12} & \sigma_{22} & \sigma_{23} \\ \sigma_{13} & \sigma_{23} & \sigma_{33} \end{pmatrix}_B$.

Donc on a :

$\vec{T}(-\vec{e}_1) = \begin{pmatrix} \sigma_{11} & \sigma_{12} & \sigma_{13} \\ \sigma_{12} & \sigma_{22} & \sigma_{23} \\ \sigma_{13} & \sigma_{23} & \sigma_{33} \end{pmatrix}_B \begin{pmatrix} -1 \\ 0 \\ 0 \end{pmatrix}_B = \begin{pmatrix} -\sigma_{11} \\ -\sigma_{12} \\ -\sigma_{13} \end{pmatrix}_B$.

Par ailleurs, on sait que la composante normale de la force appliquée à la facette vaut $F_1$. Si on note $S$ la surface de la facette triangulaire (aire = $c^2/2$), on peut intégrer la contrainte $\vec{T}(-\vec{e}_1)$ sur toute la surface de la facette (pour obtenir une force $\vec{F}_{OCB} = S \vec{T}(-\vec{e}_1)$ car état homogène), et considérer sa composante normale (par projection sur la normale $-\vec{e}_1$). On a donc :

$F_1 = \vec{F}_{OCB} \cdot (-\vec{e}_1) = S \vec{T}(-\vec{e}_1) \cdot (-\vec{e}_1) = S \begin{pmatrix} -\sigma_{11} \\ -\sigma_{12} \\ -\sigma_{13} \end{pmatrix}_B \cdot \begin{pmatrix} -1 \\ 0 \\ 0 \end{pmatrix}_B = S \sigma_{11}$.

La surface $S$ de la facette OCB est donnée par $S = c^2/2$, et on peut finalement écrire :

\sigma_{11} = \frac{2F_1}{c^2}

Par des raisonnements identiques sur les facettes de normales $-\vec{e}_2$ et $-\vec{e}_3$, on obtient de la même manière :

\sigma_{22} = \frac{2F_2}{c^2} \quad \text{et} \quad \sigma_{33} = \frac{2F_3}{c^2}

On a déterminé les trois termes diagonaux de la matrice de $\boldsymbol{\sigma}$. Il nous reste à déterminer les termes non-diagonaux. On va donc s'intéresser à la facette restante (ABC).

Sa normale sortante est un vecteur unitaire, colinéaire à la direction $\begin{pmatrix} 1 \\ 1 \\ 1 \end{pmatrix}$. Il s'agit donc d'un vecteur d'allure générale $\begin{pmatrix} a \\ a \\ a \end{pmatrix}$, où $a$ est un réel à déterminer. Par ailleurs, ce vecteur doit être unitaire, donc sa norme doit être égale à 1. On va donc avoir : $\sqrt{a^2+a^2+a^2}=1$. et donc : $a = \frac{1}{\sqrt{3}}$.

La normale sortante à la facette est donc le vecteur $\vec{n}' = \begin{pmatrix} 1/\sqrt{3} \\ 1/\sqrt{3} \\ 1/\sqrt{3} \end{pmatrix}_B$.

On a également besoin de la surface $S'$ de cette facette. Il s'agit d'un triangle équilatéral de côté $c\sqrt{2}$, et on obtient après quelques calculs trigonométriques et/ou Pythagore : $S' = \frac{c^2\sqrt{3}}{2}$.

On peut maintenant écrire le vecteur contrainte subi par cette facette en n'importe lequel de ses points (puisque l'état de contrainte est homogène) : $\vec{T}(\vec{n}')$.

\vec{T}(\vec{n}') = \boldsymbol{\sigma} \vec{n}' = \begin{pmatrix} \sigma_{11} & \sigma_{12} & \sigma_{13} \\ \sigma_{12} & \sigma_{22} & \sigma_{23} \\ \sigma_{13} & \sigma_{23} & \sigma_{33} \end{pmatrix}_B \begin{pmatrix} 1/\sqrt{3} \\ 1/\sqrt{3} \\ 1/\sqrt{3} \end{pmatrix}_B = \frac{1}{\sqrt{3}} \begin{pmatrix} \sigma_{11} + \sigma_{12} + \sigma_{13} \\ \sigma_{12} + \sigma_{22} + \sigma_{23} \\ \sigma_{13} + \sigma_{23} + \sigma_{33} \end{pmatrix}_B

L'énoncé précise que la force subie par la facette est mesurée comme étant égale à : $\vec{G} = G_1 \vec{e}_1 + G_2 \vec{e}_2 + G_3 \vec{e}_3$. Or cette force est également le produit du vecteur contrainte (homogène) par la surface de la facette, on a donc :

$\vec{G} = S' \vec{T}(\vec{n}') = \frac{c^2\sqrt{3}}{2} \frac{1}{\sqrt{3}} \begin{pmatrix} \sigma_{11} + \sigma_{12} + \sigma_{13} \\ \sigma_{12} + \sigma_{22} + \sigma_{23} \\ \sigma_{13} + \sigma_{23} + \sigma_{33} \end{pmatrix}_B = \frac{c^2}{2} \begin{pmatrix} \sigma_{11} + \sigma_{12} + \sigma_{13} \\ \sigma_{12} + \sigma_{22} + \sigma_{23} \\ \sigma_{13} + \sigma_{23} + \sigma_{33} \end{pmatrix}_B$.

Cette équation matricielle nous fournit trois équations scalaires :

$G_1 = \frac{c^2}{2}(\sigma_{11} + \sigma_{12} + \sigma_{13})$
$G_2 = \frac{c^2}{2}(\sigma_{12} + \sigma_{22} + \sigma_{23})$
$G_3 = \frac{c^2}{2}(\sigma_{13} + \sigma_{23} + \sigma_{33})$

On peut alors remplacer les termes diagonaux de la matrice par leurs valeurs trouvées en début d'exercice ($\sigma_{11}=2F_1/c^2$, etc.) :

$\sigma_{12} + \sigma_{13} = \frac{2G_1 - 2F_1}{c^2}$
$\sigma_{12} + \sigma_{23} = \frac{2G_2 - 2F_2}{c^2}$
$\sigma_{13} + \sigma_{23} = \frac{2G_3 - 2F_3}{c^2}$

On dispose alors d'un système de trois équations à trois inconnues : $\sigma_{12}, \sigma_{13}, \sigma_{23}$.
