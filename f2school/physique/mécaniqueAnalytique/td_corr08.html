    D'après le cours, $$ H(x,y,z, p_x, p_y, p_z) = \frac{1}{2m} |\mathbf{p} - e\mathbf{A}|^2 $$ $$ = \frac{1}{2m} \left( (p_x - eA_x)^2 + (p_y - eA_y)^2 + (p_z - eA_z)^2 \right) $$ $$ = \frac{1}{2m} \left( (p_x - e(-\frac{1}{2}By))^2 + (p_y - e(\frac{1}{2}Bx))^2 + (p_z - e(0))^2 \right) $$ $$ = \frac{1}{2m} \left( (p_x + \frac{eB}{2}y)^2 + (p_y - \frac{eB}{2}x)^2 + p_z^2 \right) $$
  2. $\{Q, P\} = \frac{\partial Q}{\partial x}\frac{\partial P}{\partial p_x} + \frac{\partial Q}{\partial y}\frac{\partial P}{\partial p_y} - \frac{\partial Q}{\partial p_x}\frac{\partial P}{\partial x} - \frac{\partial Q}{\partial p_y}\frac{\partial P}{\partial y}$

    Calcul détaillé : $\frac{\partial Q}{\partial x} = 0$, $\frac{\partial Q}{\partial y} = \frac{1}{\sqrt{eB}}\frac{eB}{2} = \frac{\sqrt{eB}}{2}$, $\frac{\partial Q}{\partial p_x} = \frac{1}{\sqrt{eB}}$, $\frac{\partial Q}{\partial p_y} = 0$. $\frac{\partial P}{\partial x} = -\frac{1}{\sqrt{eB}}\frac{eB}{2} = -\frac{\sqrt{eB}}{2}$, $\frac{\partial P}{\partial y} = 0$, $\frac{\partial P}{\partial p_x} = 0$, $\frac{\partial P}{\partial p_y} = \frac{1}{\sqrt{eB}}$. $\{Q, P\} = (0)(0) + (\frac{\sqrt{eB}}{2})(\frac{1}{\sqrt{eB}}) - (\frac{1}{\sqrt{eB}})(-\frac{\sqrt{eB}}{2}) - (0)(0)$ $= 0 + \frac{1}{2} - (-\frac{1}{2}) - 0 = 1$.

    Autre méthode: $\{Q, P\} = \frac{1}{eB} \{ p_x + \frac{eB}{2}y, p_y - \frac{eB}{2}x \}$ $= \frac{1}{eB} [ \underbrace{\{p_x, p_y\}}_{0} - \underbrace{\{p_x, \frac{eB}{2}x\}}_{-\frac{eB}{2}\{p_x, x\} = \frac{eB}{2}} + \underbrace{\{\frac{eB}{2}y, p_y\}}_{\frac{eB}{2}\{y, p_y\} = \frac{eB}{2}} - \underbrace{\{\frac{eB}{2}y, \frac{eB}{2}x\}}_{0} ]$ $= \frac{1}{eB} [ \frac{eB}{2} + \frac{eB}{2} ] = 1$.

    (Attention aux signes: $\{p_x, x\} = -\{x, p_x\} = -1$.)

    $\{Q, P\} = \frac{1}{eB} [ 0 - \frac{eB}{2}(-1) + \frac{eB}{2}(1) - 0 ] = \frac{1}{eB}[\frac{eB}{2} + \frac{eB}{2}] = 1$.
