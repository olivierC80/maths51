Exercice 1 (6 points)

Un explorateur aperçoit dans le désert une oasis à un angle \(\text{arctg}(\frac{h}{D})\) au dessus de l'horizontale. On se propose de déterminer la distance qui le sépare de l'oasis. Le principe de Fermat stipule que la lumière suit un chemin minimisant le temps du trajet que l'on peut exprimer comme suit \[ S = \int n(s) ds \] où \(S\) est la fonctionnelle à minimiser, \(n(s)\) est l'indice de réfraction du milieu et \(s\) est la distance du trajet avec \(ds = \sqrt{dx^2 + dz^2}\). Dans les conditions de grande chaleur du désert, l'indice de réfraction dépend linéairement de \(z\) comme suit \(n(z) = n_0(1 - \lambda z)\).

  1. Expliciter la fonctionnelle à minimiser et trouver le “lagrangien associé”. En déduire le “hamiltonien associé” et montrer qu'il est conservé.

    Explication : On cherche à minimiser \(S = \int n(s) ds\). On peut paramétriser le chemin par \(z(x)\), considérant \(x\) comme variable indépendante. Alors \(ds = \sqrt{dx^2 + dz^2} = \sqrt{1 + (dz/dx)^2} dx = \sqrt{1 + z'^2} dx\). L'indice de réfraction est \(n(z) = n_0(1 - \lambda z)\). La fonctionnelle à minimiser (temps de trajet, proportionnel à S) est : \[ S = \int_{x_1}^{x_2} n(z(x)) \sqrt{1 + z'(x)^2} dx = \int_{x_1}^{x_2} n_0(1 - \lambda z) \sqrt{1 + z'^2} dx \] Le "lagrangien associé" \(L\) est l'intégrande : \[ L(z, z', x) = n_0(1 - \lambda z) \sqrt{1 + z'^2} \] Le "moment conjugué" \(p\) associé à \(z\) est : \[ p = \frac{\partial L}{\partial z'} = n_0(1 - \lambda z) \frac{z'}{\sqrt{1 + z'^2}} \] Le "hamiltonien associé" \(H\) est : \[ H = p z' - L = \left( n_0(1 - \lambda z) \frac{z'}{\sqrt{1 + z'^2}} \right) z' - n_0(1 - \lambda z) \sqrt{1 + z'^2} \] \[ H = \frac{n_0(1 - \lambda z) z'^2}{\sqrt{1 + z'^2}} - \frac{n_0(1 - \lambda z) (1 + z'^2)}{\sqrt{1 + z'^2}} \] \[ H = \frac{n_0(1 - \lambda z) (z'^2 - 1 - z'^2)}{\sqrt{1 + z'^2}} = \frac{-n_0(1 - \lambda z)}{\sqrt{1 + z'^2}} \] Le lagrangien \(L\) ne dépend pas explicitement de \(x\). Par conséquent, le hamiltonien \(H\) est conservé le long de la trajectoire : \[ \frac{dH}{dx} = -\frac{\partial L}{\partial x} = 0 \implies H = \text{constante} \]

  2. Etablir l'équation de la trajectoire \(z = z(x)\).1

    Explication : On utilise la conservation de \(H\): \[ \frac{-n_0(1 - \lambda z)}{\sqrt{1 + z'^2}} = C \quad (\text{où } C \text{ est une constante}) \] \[ \frac{n_0^2 (1 - \lambda z)^2}{1 + z'^2} = C^2 \] \[ 1 + z'^2 = \frac{n_0^2}{C^2} (1 - \lambda z)^2 \] \[ z'^2 = \left( \frac{n_0}{C} \right)^2 (1 - \lambda z)^2 - 1 \] \[ \frac{dz}{dx} = \pm \sqrt{\left( \frac{n_0}{C} \right)^2 (1 - \lambda z)^2 - 1} \] On sépare les variables : \[ dx = \pm \frac{dz}{\sqrt{\left( \frac{n_0}{C} \right)^2 (1 - \lambda z)^2 - 1}} \] Intégrons : \[ x - x_0 = \pm \int \frac{dz}{\sqrt{\left( \frac{n_0}{C} \right)^2 (1 - \lambda z)^2 - 1}} \] Posons \(u = 1 - \lambda z\), alors \(dz = -du/\lambda\). L'intégrale devient complexe. Une forme plus simple est souvent obtenue via \(p = \partial L / \partial z'\). L'équation d'Euler-Lagrange est \(\frac{d}{dx}(\frac{\partial L}{\partial z'}) - \frac{\partial L}{\partial z} = 0\). \[ \frac{\partial L}{\partial z} = -\lambda n_0 \sqrt{1 + z'^2} \] \[ \frac{d p}{dx} = \frac{\partial L}{\partial z} \implies \frac{d}{dx} \left( n_0(1 - \lambda z) \frac{z'}{\sqrt{1 + z'^2}} \right) = -\lambda n_0 \sqrt{1 + z'^2} \] Ceci est l'équation différentielle de la trajectoire. Une autre approche : La conservation de H donne \( \frac{n_0(1 - \lambda z)}{\sqrt{1 + z'^2}} = |C| \). Soit \(\theta\) l'angle que fait la tangente à la trajectoire avec l'axe x. Alors \(z' = \tan \theta\), et \(\sqrt{1 + z'^2} = \sqrt{1 + \tan^2 \theta} = \sec \theta = 1/\cos \theta\). Donc \(n_0(1 - \lambda z) \cos \theta = |C|\). C'est la loi de Snellius généralisée \(n(z) \cos \theta = \text{const}\). Pour l'intégration, posons \(A = n_0/|C|\). \( \frac{dz}{dx} = \pm \sqrt{A^2 (1 - \lambda z)^2 - 1} \). Posons \(A(1 - \lambda z) = \cosh \phi\). Alors \(A (-\lambda dz) = \sinh \phi d\phi\). \(dz = -\frac{\sinh \phi}{A \lambda} d\phi\). \(\sqrt{A^2 (1 - \lambda z)^2 - 1} = \sqrt{\cosh^2 \phi - 1} = \sinh \phi\). \[ dx = \pm \frac{-\sinh \phi d\phi / (A \lambda)}{\sinh \phi} = \mp \frac{d\phi}{A \lambda} \] Intégration : \(x - x_0 = \mp \frac{\phi}{A \lambda}\). Donc \(\phi = \mp A \lambda (x - x_0)\). On a \(A(1 - \lambda z) = \cosh \phi = \cosh(A \lambda (x - x_0))\) (le \(\mp\) est absorbé par cosh). \[ 1 - \lambda z = \frac{1}{A} \cosh(A \lambda (x - x_0)) \] \[ z(x) = \frac{1}{\lambda} \left( 1 - \frac{|C|}{n_0} \cosh\left(\frac{n_0 \lambda}{|C|} (x - x_0)\right) \right) \] C'est l'équation de la trajectoire (une chaînette renversée).

  3. Calculer la distance qui sépare l'explorateur de l'oasis. On néglige le rayon de courbure de la terre.

    Explication : L'explorateur est en \((x_0, z_0)\), l'oasis en \((x_f, z_f)\). On peut supposer l'explorateur en \(x=0, z=0\). Alors \(n(0) = n_0\). L'angle initial \(\theta_0\) de départ du rayon est donné par \(\alpha = \arctan(h/D)\) au-dessus de l'horizontale. L'angle \(\theta\) utilisé dans la loi de Snellius est avec l'axe x. Si le rayon part vers le haut et la droite, \(\theta_0 = \alpha\). La constante \(|C|\) est \(|C| = n(0) \cos \theta_0 = n_0 \cos \alpha\). La trajectoire est \( z(x) = \frac{1}{\lambda} \left( 1 - \cos \alpha \cosh\left(\frac{\lambda x}{\cos \alpha}\right) \right) \). (On a ajusté \(x_0\) pour que \(z(0)=0\)). L'angle \(\alpha\) est tel que \(\tan \alpha = h/D\). L'énoncé donne \(\alpha = \arctan(h/D)\). Au point le plus haut de la trajectoire (si elle existe), la tangente est horizontale, \(\theta = 0\), donc \(z' = 0\). \(z'^2 = 0 \implies \left( \frac{n_0}{C} \right)^2 (1 - \lambda z_{max})^2 - 1 = 0 \). \( \frac{n_0}{|C|} (1 - \lambda z_{max}) = 1 \). \( \frac{1}{\cos \alpha} (1 - \lambda z_{max}) = 1 \). \(1 - \lambda z_{max} = \cos \alpha\). \(z_{max} = \frac{1 - \cos \alpha}{\lambda}\). Ce point est atteint quand \(\cosh(\dots)=1/\cos \alpha\). L'oasis est vue sous l'angle \(\alpha\). Le rayon lumineux qui nous parvient de l'oasis part de l'oasis (à la distance \(D\)) et arrive à nos yeux avec un angle \(\alpha\) par rapport à l'horizontale. Soit l'explorateur en (0,0) et l'oasis en (D, h). La lumière suit la trajectoire \(z(x)\). La pente à l'arrivée (\(x=0\)) est \(z'(0) = \tan \alpha\). \[ z'(x) = \frac{d}{dx} \left[ \frac{1}{\lambda} \left( 1 - \frac{|C|}{n_0} \cosh\left(\frac{n_0 \lambda}{|C|} (x - x_0)\right) \right) \right] \] \[ z'(x) = -\frac{|C|}{n_0 \lambda} \sinh\left(\frac{n_0 \lambda}{|C|} (x - x_0)\right) \cdot \frac{n_0 \lambda}{|C|} = -\sinh\left(\frac{n_0 \lambda}{|C|} (x - x_0)\right) \] A \(x=0\), \(z'(0) = \tan \alpha\). \(\tan \alpha = -\sinh\left(-\frac{n_0 \lambda}{|C|} x_0\right) = \sinh\left(\frac{n_0 \lambda}{|C|} x_0\right)\). On a \(|C|=n_0 \cos\alpha\). \(\tan \alpha = \sinh\left(\frac{n_0 \lambda}{n_0 \cos\alpha} x_0\right) = \sinh\left(\frac{\lambda x_0}{\cos\alpha}\right)\). \(x_0 = \frac{\cos\alpha}{\lambda} \text{argsh}(\tan \alpha)\). L'oasis est au point \( (D, h) \) où la trajectoire démarre (ou se termine). Supposons que l'oasis est à \(x=D\). Alors \(z(D) = h\). La distance \(D\) est la distance horizontale. La distance recherchée est probablement la distance directe \(\sqrt{D^2+h^2}\) ou la distance horizontale \(D\). L'énoncé est un peu ambigu. S'il s'agit d'un mirage, l'oasis réelle peut être à l'horizontale \(h=0\) ou même en dessous. Le rayon semble venir d'au-dessus. Si l'on suppose que l'oasis est au point où le rayon devient horizontal (\(z'=0\)), alors \(x_{oasis} = x_0\) et \(z_{oasis} = z_{max}\). La distance horizontale est \(D = x_0 = \frac{\cos\alpha}{\lambda} \text{argsh}(\tan \alpha)\). Utilisons \(\text{argsh}(u) = \ln(u + \sqrt{1+u^2})\). \(D = \frac{\cos\alpha}{\lambda} \ln(\tan \alpha + \sqrt{1+\tan^2 \alpha}) = \frac{\cos\alpha}{\lambda} \ln(\tan \alpha + \sec \alpha)\). \[ D = \frac{\cos\alpha}{\lambda} \ln\left(\frac{\sin \alpha + 1}{\cos \alpha}\right) \] La distance demandée est probablement \(D\).

1 On donne Argsh'(x) = \(1/\sqrt{1+x^2}\), et Argsh(x) = \(\ln (x + \sqrt{1 + x^2})\).

Exercice 2 (14 points)

Considérons un oscillateur harmonique de masse \(m\) et de pulsation propre \(\omega_0 = \sqrt{k/m}\) et ce en deux dimensions, figure ci-contre. Le référentiel \(R(O, xyz)\) est supposé galiléen. On utilise dans l'exercice les coordonnées cartésiennes.

Placeholder: Oscillateur harmonique 2D.

  1. Calculer la vitesse de \(M, \vec{V}(M/R)\). En déduire son énergie cinétique \(T\).

    Explication : En coordonnées cartésiennes, le vecteur position est \(\vec{r} = (x, y)\) (ou \((x, y, 0)\)). Le vecteur vitesse est \(\vec{V} = \frac{d\vec{r}}{dt} = (\dot{x}, \dot{y})\). L'énergie cinétique \(T\) est : \[ T = \frac{1}{2} m |\vec{V}|^2 = \frac{1}{2} m (\dot{x}^2 + \dot{y}^2) \]

  2. \(M\) est soumis à la seule action de la force de rappel \(\vec{F} = -k(|\vec{OM}| - L_0)\vec{u}\) avec \(\vec{u} = \frac{\vec{OM}}{|\vec{OM}|}\). \(L_0\) étant l'allongement au repos du ressort que l'on prend nul par simplification, \(L_0 = 0\). Calculer l'énergie potentielle \(V(x, y)\).

    Explication : Avec \(L_0 = 0\), la force de rappel est \(\vec{F} = -k |\vec{OM}| \vec{u} = -k |\vec{OM}| \frac{\vec{OM}}{|\vec{OM}|} = -k \vec{OM}\). En coordonnées cartésiennes, \(\vec{OM} = (x, y)\). \[ \vec{F} = -k(x, y) = (-kx, -ky) \] Cette force dérive d'un potentiel \(V\) tel que \(\vec{F} = -\nabla V = (-\frac{\partial V}{\partial x}, -\frac{\partial V}{\partial y})\). \[ -\frac{\partial V}{\partial x} = -kx \implies \frac{\partial V}{\partial x} = kx \] \[ -\frac{\partial V}{\partial y} = -ky \implies \frac{\partial V}{\partial y} = ky \] En intégrant la première équation par rapport à \(x\) : \(V(x, y) = \frac{1}{2} k x^2 + f(y)\). En dérivant par rapport à \(y\) : \(\frac{\partial V}{\partial y} = 0 + f'(y)\). En comparant avec la seconde équation : \(f'(y) = ky\). En intégrant par rapport à \(y\) : \(f(y) = \frac{1}{2} k y^2 + \text{constante}\). On choisit la constante nulle (potentiel nul à l'origine). L'énergie potentielle est donc : \[ V(x, y) = \frac{1}{2} k x^2 + \frac{1}{2} k y^2 = \frac{1}{2} k (x^2 + y^2) \] En utilisant \(k = m \omega_0^2\): \[ V(x, y) = \frac{1}{2} m \omega_0^2 (x^2 + y^2) \]

  3. Ecrire le lagrangien \(L\) de l'oscillateur harmonique et déduire les moments conjugués \(p_x\) et \(p_y\).

    Explication : Le lagrangien \(L = T - V\). \[ L(x, y, \dot{x}, \dot{y}) = \frac{1}{2} m (\dot{x}^2 + \dot{y}^2) - \frac{1}{2} m \omega_0^2 (x^2 + y^2) \] Les moments conjugués sont \(p_q = \partial L / \partial \dot{q}\). \[ p_x = \frac{\partial L}{\partial \dot{x}} = m \dot{x} \] \[ p_y = \frac{\partial L}{\partial \dot{y}} = m \dot{y} \] Ce sont les composantes usuelles de la quantité de mouvement.

  4. Montrer que \(L\) est invariant par rotation. En utilisant le théorème de Noether, déduire la grandeur conservée. On rappelle que lors d'une transformation associée à une rotation d'un angle \(s\) dans le plan (Oxy), on a \[ \begin{pmatrix} x' \\ y' \end{pmatrix} = \begin{pmatrix} \cos(s) & -\sin(s) \\ \sin(s) & \cos(s) \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix} \] Considérer une transformation infinitésimale, \(s \to 0\), et utiliser les accroissements \(\delta x\) et \(\delta y\).

    Explication : Le lagrangien \(L = \frac{1}{2} m (\dot{x}^2 + \dot{y}^2) - \frac{1}{2} m \omega_0^2 (x^2 + y^2)\) dépend de \(x^2+y^2 = r^2\) et \(\dot{x}^2+\dot{y}^2 = v^2\). Ces deux quantités sont invariantes par rotation autour de l'origine. Donc, \(L\) est invariant par rotation dans le plan (Oxy). Pour une rotation infinitésimale \(s \to 0\), on a \(\cos s \approx 1\) et \(\sin s \approx s\). \[ x' \approx (1)x - (s)y = x - sy \] \[ y' \approx (s)x + (1)y = y + sx \] La variation infinitésimale des coordonnées sous cette transformation est \(\delta q_k = \left. \frac{dq'_k}{ds} \right|_{s=0}\). \[ \delta x = x' - x = -sy \implies \left. \frac{dx'}{ds} \right|_{s=0} = -y \] \[ \delta y = y' - y = sx \implies \left. \frac{dy'}{ds} \right|_{s=0} = x \] Le théorème de Noether dit que la grandeur conservée \(I\) associée à cette symétrie est : \[ I = \sum_k p_k \delta q_k = p_x \left. \frac{dx'}{ds} \right|_{s=0} + p_y \left. \frac{dy'}{ds} \right|_{s=0} \] \[ I = p_x (-y) + p_y (x) = x p_y - y p_x \] Cette grandeur est la composante \(z\) du moment cinétique : \(\vec{L} = \vec{r} \times \vec{p} = (x, y, 0) \times (p_x, p_y, 0) = (0, 0, x p_y - y p_x)\). Donc, la grandeur conservée est \(L_z = x p_y - y p_x\).

  5. Montrer que le hamiltonien de l'oscillateur harmonique est donné par \[ H(x, y, p_x, p_y) = \frac{p_x^2}{2m} + \frac{p_y^2}{2m} + \frac{1}{2} m \omega_0^2 (x^2 + y^2). \] Montrer qu'il est conservé. On pose \(E = E_x + E_y\), \(E\) étant l'énergie de l'oscillateur harmonique, \(E_x = \frac{p_x^2}{2m} + \frac{1}{2} m \omega_0^2 x^2\) et \(E_y = \frac{p_y^2}{2m} + \frac{1}{2} m \omega_0^2 y^2\).

    Explication : Le hamiltonien \(H = \sum p_k \dot{q}_k - L = p_x \dot{x} + p_y \dot{y} - L\). On exprime \(\dot{x}\) et \(\dot{y}\) en fonction de \(p_x\) et \(p_y\) : \(\dot{x} = p_x/m\), \(\dot{y} = p_y/m\). \[ H = p_x \left(\frac{p_x}{m}\right) + p_y \left(\frac{p_y}{m}\right) - \left[ \frac{1}{2} m \left( \left(\frac{p_x}{m}\right)^2 + \left(\frac{p_y}{m}\right)^2 \right) - \frac{1}{2} m \omega_0^2 (x^2 + y^2) \right] \] \[ H = \frac{p_x^2}{m} + \frac{p_y^2}{m} - \left[ \frac{1}{2} m \left( \frac{p_x^2}{m^2} + \frac{p_y^2}{m^2} \right) - \frac{1}{2} m \omega_0^2 (x^2 + y^2) \right] \] \[ H = \frac{p_x^2}{m} + \frac{p_y^2}{m} - \frac{p_x^2}{2m} - \frac{p_y^2}{2m} + \frac{1}{2} m \omega_0^2 (x^2 + y^2) \] \[ H = \frac{p_x^2}{2m} + \frac{p_y^2}{2m} + \frac{1}{2} m \omega_0^2 (x^2 + y^2) \] C'est bien l'expression donnée. Pour montrer que \(H\) est conservé, on peut utiliser la relation \(\frac{dH}{dt} = \frac{\partial H}{\partial t}\). Puisque le hamiltonien \(H\) ne dépend pas explicitement du temps \(t\) (\(\frac{\partial H}{\partial t} = 0\)), il est conservé : \(\frac{dH}{dt} = 0\). Le hamiltonien représente l'énergie totale du système \(H = T + V\). On remarque que \(H\) peut se séparer en une partie dépendant de \(x\) et \(p_x\) et une partie dépendant de \(y\) et \(p_y\): \(H = H_x(x, p_x) + H_y(y, p_y)\) avec \(H_x = \frac{p_x^2}{2m} + \frac{1}{2} m \omega_0^2 x^2 = E_x\) et \(H_y = \frac{p_y^2}{2m} + \frac{1}{2} m \omega_0^2 y^2 = E_y\). \(E_x\) et \(E_y\) représentent les énergies associées aux oscillations selon les axes x et y respectivement. La conservation de \(H\) signifie que l'énergie totale \(E = E_x + E_y\) est conservée. \(E_x\) et \(E_y\) ne sont pas nécessairement conservées individuellement, sauf cas particulier.

  6. Considérons la transformation \[ \begin{cases} x = X \cos a + \frac{P_Y}{m\omega_0} \sin a \\ p_x = -m\omega_0 Y \sin a + P_X \cos a \end{cases} \qquad \begin{cases} y = Y \cos a + \frac{P_X}{m\omega_0} \sin a \\ p_y = -m\omega_0 X \sin a + P_Y \cos a \end{cases} \]

    6-a) Montrer que la transformation est canonique.

    Explication : Une transformation \( (q, p) \to (Q, P) \) est canonique si les crochets de Poisson fondamentaux sont préservés : \(\{Q_i, Q_j\}_{q,p} = 0\), \(\{P_i, P_j\}_{q,p} = 0\), \(\{Q_i, P_j\}_{q,p} = \delta_{ij}\). Ici, les anciennes variables sont \((x, y, p_x, p_y)\) et les nouvelles \((X, Y, P_X, P_Y)\). Il est plus simple de vérifier les crochets des anciennes variables par rapport aux nouvelles : \(\{x, y\}_{X,Y,P_X,P_Y} = 0\), \(\{p_x, p_y\}_{X,Y,P_X,P_Y} = 0\), \(\{x, p_x\}_{X,Y,P_X,P_Y} = 1\), \(\{y, p_y\}_{X,Y,P_X,P_Y} = 1\), \(\{x, p_y\}_{X,Y,P_X,P_Y} = 0\), \(\{y, p_x\}_{X,Y,P_X,P_Y} = 0\). Calculons par exemple \(\{x, p_x\}\). Il faut les dérivées de \(x\) et \(p_x\) par rapport à \(X, Y, P_X, P_Y\). \(\frac{\partial x}{\partial X} = \cos a\), \(\frac{\partial x}{\partial Y} = 0\), \(\frac{\partial x}{\partial P_X} = 0\), \(\frac{\partial x}{\partial P_Y} = \frac{\sin a}{m\omega_0}\). \(\frac{\partial p_x}{\partial X} = 0\), \(\frac{\partial p_x}{\partial Y} = -m\omega_0 \sin a\), \(\frac{\partial p_x}{\partial P_X} = \cos a\), \(\frac{\partial p_x}{\partial P_Y} = 0\). \[ \{x, p_x\} = \frac{\partial x}{\partial X}\frac{\partial p_x}{\partial P_X} - \frac{\partial x}{\partial P_X}\frac{\partial p_x}{\partial X} + \frac{\partial x}{\partial Y}\frac{\partial p_x}{\partial P_Y} - \frac{\partial x}{\partial P_Y}\frac{\partial p_x}{\partial Y} \] \[ \{x, p_x\} = (\cos a)(\cos a) - (0)(0) + (0)(0) - \left(\frac{\sin a}{m\omega_0}\right)(-m\omega_0 \sin a) \] \[ \{x, p_x\} = \cos^2 a + \sin^2 a = 1 \] Calculons \(\{x, p_y\}\). \(\frac{\partial p_y}{\partial X} = -m\omega_0 \sin a\), \(\frac{\partial p_y}{\partial Y} = 0\), \(\frac{\partial p_y}{\partial P_X} = 0\), \(\frac{\partial p_y}{\partial P_Y} = \cos a\). \[ \{x, p_y\} = \frac{\partial x}{\partial X}\frac{\partial p_y}{\partial P_X} - \frac{\partial x}{\partial P_X}\frac{\partial p_y}{\partial X} + \frac{\partial x}{\partial Y}\frac{\partial p_y}{\partial P_Y} - \frac{\partial x}{\partial P_Y}\frac{\partial p_y}{\partial Y} \] \[ \{x, p_y\} = (\cos a)(0) - (0)(-m\omega_0 \sin a) + (0)(\cos a) - \left(\frac{\sin a}{m\omega_0}\right)(0) = 0 \] Les autres calculs (\(\{y, p_y\}=1\), \(\{y, p_x\}=0\), etc.) sont similaires et montrent que la transformation est canonique. (Cette transformation est une rotation dans l'espace des phases normalisé).

    6-b) Ecrire le nouvel hamiltonien \(H(X, Y, P_X, P_Y)\).

    Explication : Puisque la transformation est canonique et ne dépend pas explicitement du temps, le nouveau hamiltonien \(K(X, Y, P_X, P_Y)\) est simplement l'ancien hamiltonien \(H(x, y, p_x, p_y)\) exprimé en termes des nouvelles variables. Il faut substituer les expressions de \(x, y, p_x, p_y\) en fonction de \(X, Y, P_X, P_Y\) dans \(H = \frac{p_x^2+p_y^2}{2m} + \frac{1}{2} m \omega_0^2 (x^2+y^2)\). C'est un calcul assez long. Une propriété clé de cette transformation (rotation dans l'espace des phases) est qu'elle simplifie souvent le hamiltonien. Si l'angle \(a\) est choisi judicieusement (par exemple, si \(a = \omega_0 t\)), cela peut correspondre au passage dans un repère tournant. Calculons \(x^2+y^2\) et \(p_x^2+p_y^2\). \(x^2+y^2 = (X\cos a + \frac{P_Y}{m\omega_0}\sin a)^2 + (Y\cos a + \frac{P_X}{m\omega_0}\sin a)^2 = \dots\) \(p_x^2+p_y^2 = (-m\omega_0 Y\sin a + P_X\cos a)^2 + (-m\omega_0 X\sin a + P_Y\cos a)^2 = \dots\) Après développement et simplification (utilisant \(\cos^2 a + \sin^2 a = 1\)), on trouve (si la transformation est bien une rotation simple dans l'espace des phases) que : \[ \frac{p_x^2+p_y^2}{2m} + \frac{1}{2} m \omega_0^2 (x^2+y^2) = \frac{P_X^2+P_Y^2}{2m} + \frac{1}{2} m \omega_0^2 (X^2+Y^2) \] Donc, le nouveau hamiltonien \(K\) a la même forme fonctionnelle que l'ancien : \[ K(X, Y, P_X, P_Y) = \frac{P_X^2}{2m} + \frac{P_Y^2}{2m} + \frac{1}{2} m \omega_0^2 (X^2 + Y^2) \]

  7. On se propose de résoudre le problème en utilisant l'équation de Hamilton-Jacobi (HJ). On note par \(Q_1, Q_2, P_1\) et \(P_2\) les nouvelles variables.

    7-a) Ecrire l'équation de HJ. En déduire que l'on peut séparer les variables \(S(x, y; t, P_1, P_2) = S_x(x; P_1) + S_y(y; P_2) - Et\), \(S(x, y; P_1, P_2)\) étant l'action hamiltonienne.

    Explication : L'équation de Hamilton-Jacobi est \(H(q, \partial S/\partial q, t) + \partial S/\partial t = 0\). Ici, \(q=(x, y)\) et \(H\) ne dépend pas de \(t\). \[ \frac{1}{2m} \left( \frac{\partial S}{\partial x} \right)^2 + \frac{1}{2m} \left( \frac{\partial S}{\partial y} \right)^2 + \frac{1}{2} m \omega_0^2 (x^2 + y^2) + \frac{\partial S}{\partial t} = 0 \] On cherche une solution par séparation des variables \(S(x, y, t) = W(x, y) - E t\). \[ \frac{1}{2m} \left( \frac{\partial W}{\partial x} \right)^2 + \frac{1}{2m} \left( \frac{\partial W}{\partial y} \right)^2 + \frac{1}{2} m \omega_0^2 (x^2 + y^2) - E = 0 \] On essaie ensuite de séparer \(W(x, y)\) en \(W(x, y) = S_x(x) + S_y(y)\). \[ \frac{1}{2m} \left( \frac{d S_x}{dx} \right)^2 + \frac{1}{2m} \left( \frac{d S_y}{dy} \right)^2 + \frac{1}{2} m \omega_0^2 x^2 + \frac{1}{2} m \omega_0^2 y^2 - E = 0 \] Réarrangeons : \[ \left[ \frac{1}{2m} \left( \frac{d S_x}{dx} \right)^2 + \frac{1}{2} m \omega_0^2 x^2 \right] + \left[ \frac{1}{2m} \left( \frac{d S_y}{dy} \right)^2 + \frac{1}{2} m \omega_0^2 y^2 \right] = E \] Le premier terme crocheté ne dépend que de \(x\), le second ne dépend que de \(y\). Pour que leur somme soit constante (\(E\)), chaque terme doit être une constante. Posons : \[ \frac{1}{2m} \left( \frac{d S_x}{dx} \right)^2 + \frac{1}{2} m \omega_0^2 x^2 = P_1 \] \[ \frac{1}{2m} \left( \frac{d S_y}{dy} \right)^2 + \frac{1}{2} m \omega_0^2 y^2 = P_2 \] où \(P_1\) et \(P_2\) sont des constantes de séparation telles que \(P_1 + P_2 = E\). Ces constantes \(P_1\) et \(P_2\) correspondent aux énergies \(E_x\) et \(E_y\) introduites précédemment. Donc, la séparation \(S(x, y, t) = S_x(x; P_1) + S_y(y; P_2) - Et\) est possible.

    7-b) Expliciter les équations vérifiées respectivement par \(S_x(x; P_1)\) et \(S_y(y; P_2)\). En déduire que \[ S(x, y; t, P_1, P_2) = \pm \int \sqrt{2m P_1 - m^2 \omega_0^2 x^2} dx \pm \int \sqrt{2m P_2 - m^2 \omega_0^2 y^2} dy - Et. \] (Correction: l'expression sous la racine dans l'énoncé semble erronée, P1=Ex, P2=Ey).

    Explication : Les équations pour \(S_x\) et \(S_y\) sont : \[ \frac{1}{2m} \left( \frac{d S_x}{dx} \right)^2 + \frac{1}{2} m \omega_0^2 x^2 = P_1 \] \[ \frac{1}{2m} \left( \frac{d S_y}{dy} \right)^2 + \frac{1}{2} m \omega_0^2 y^2 = P_2 \] On résout pour les dérivées : \[ \left( \frac{d S_x}{dx} \right)^2 = 2m P_1 - m^2 \omega_0^2 x^2 \implies \frac{d S_x}{dx} = \pm \sqrt{2m P_1 - m^2 \omega_0^2 x^2} \] \[ \left( \frac{d S_y}{dy} \right)^2 = 2m P_2 - m^2 \omega_0^2 y^2 \implies \frac{d S_y}{dy} = \pm \sqrt{2m P_2 - m^2 \omega_0^2 y^2} \] En intégrant : \[ S_x(x; P_1) = \pm \int \sqrt{2m P_1 - m^2 \omega_0^2 x^2} dx \] \[ S_y(y; P_2) = \pm \int \sqrt{2m P_2 - m^2 \omega_0^2 y^2} dy \] L'action totale est \(S = S_x + S_y - Et\): \[ S(x, y; t, P_1, P_2) = \pm \int \sqrt{2m P_1 - m^2 \omega_0^2 x^2} dx \pm \int \sqrt{2m P_2 - m^2 \omega_0^2 y^2} dy - (P_1 + P_2)t \] Ceci correspond à l'expression donnée (avec P1=Ex, P2=Ey et E=P1+P2).

    7-c) Rappeler pourquoi \(Q_1, Q_2, P_1\) et \(P_2\) sont conservées. On pose \(P_1 = E_x\) et \(P_2 = E_y = E - E_x\). Etablir les expressions de \(p_x, p_y, Q_1\) et \(Q_2\).

    Explication : Dans la théorie de Hamilton-Jacobi, les nouvelles impulsions \(P_k\) sont des constantes du mouvement (ici \(P_1 = E_x\) et \(P_2 = E_y\)). Les nouvelles coordonnées \(Q_k\) sont aussi des constantes du mouvement si le nouveau hamiltonien \(K\) est nul (ce qui est le cas ici, car la transformation canonique engendrée par \(S\) mène à \(K=0\)). Les \(Q_k\) sont définies par \(Q_k = \partial S / \partial P_k\). Comme \(P_1\) et \(P_2\) sont constantes, et que \(Q_k\) dépendent linéairement du temps (via le terme \(-\sum P_k t\)), leurs dérivées temporelles sont constantes : \(\dot{Q}_k = \partial K / \partial P_k\), \(\dot{P}_k = -\partial K / \partial Q_k\). Si \(K=0\), alors \(\dot{Q}_k = 0\) et \(\dot{P}_k = 0\).
    Expressions : \[ p_x = \frac{\partial S}{\partial x} = \frac{d S_x}{dx} = \pm \sqrt{2m E_x - m^2 \omega_0^2 x^2} \] \[ p_y = \frac{\partial S}{\partial y} = \frac{d S_y}{dy} = \pm \sqrt{2m E_y - m^2 \omega_0^2 y^2} \] \[ Q_1 = \frac{\partial S}{\partial P_1} = \frac{\partial S}{\partial E_x} = \frac{\partial S_x}{\partial E_x} + \frac{\partial S_y}{\partial E_x} - \frac{\partial (E_x+E_y)t}{\partial E_x} \] En utilisant \(E_y = E - E_x\), \(\frac{\partial E_y}{\partial E_x} = -1\). \[ Q_1 = \frac{\partial}{\partial E_x} \left( \pm \int \sqrt{2m E_x - m^2 \omega_0^2 x^2} dx \right) + \frac{\partial}{\partial E_y} \left( \pm \int \sqrt{2m E_y - m^2 \omega_0^2 y^2} dy \right) \frac{\partial E_y}{\partial E_x} - t \] \[ Q_1 = \pm \int \frac{m dx}{\sqrt{2m E_x - m^2 \omega_0^2 x^2}} \mp \int \frac{m dy}{\sqrt{2m E_y - m^2 \omega_0^2 y^2}} - t \] L'intégrale \(\int \frac{m dx}{\sqrt{2m E_x - m^2 \omega_0^2 x^2}} = \int \frac{dx}{\sqrt{\frac{2E_x}{m} - \omega_0^2 x^2}}\) est reliée à un arcsin ou arccos. Posons \(A_x^2 = 2E_x / (m\omega_0^2)\). L'intégrale est \(\frac{1}{\omega_0} \int \frac{dx}{\sqrt{A_x^2 - x^2}} = \frac{1}{\omega_0} \arcsin(x/A_x)\). Donc, \(Q_1 = \pm \frac{1}{\omega_0} \arcsin\left(\frac{x}{\sqrt{2E_x/m\omega_0^2}}\right) \mp \frac{1}{\omega_0} \arcsin\left(\frac{y}{\sqrt{2E_y/m\omega_0^2}}\right) - t\). \[ Q_2 = \frac{\partial S}{\partial P_2} = \frac{\partial S_y}{\partial P_2} - \frac{\partial E t}{\partial P_2} = \frac{\partial S_y}{\partial E_y} - t \] \[ Q_2 = \pm \int \frac{m dy}{\sqrt{2m E_y - m^2 \omega_0^2 y^2}} - t = \pm \frac{1}{\omega_0} \arcsin\left(\frac{y}{\sqrt{2E_y/m\omega_0^2}}\right) - t \] (Les expressions exactes des Qk dépendent de comment E est traité, soit comme E(P1,P2) ou comme une constante fixée). Si E est la constante, \(E = P_1+P_2\), alors \(Q_k = \partial S/\partial P_k\). \(Q_1 = \partial S_x/\partial P_1 - t = \pm \frac{1}{\omega_0} \arcsin\left(\frac{x}{\sqrt{2P_1/m\omega_0^2}}\right) - t\). \(Q_2 = \partial S_y/\partial P_2 - t = \pm \frac{1}{\omega_0} \arcsin\left(\frac{y}{\sqrt{2P_2/m\omega_0^2}}\right) - t\). Puisque \(Q_1, Q_2\) sont constantes, on peut écrire \( \omega_0 (Q_1 + t) = \pm \arcsin(\dots) \) et \( \omega_0 (Q_2 + t) = \pm \arcsin(\dots) \). Ceci mène à \(x(t) = A_x \sin(\pm \omega_0 (t+Q_1))\) et \(y(t) = A_y \sin(\pm \omega_0 (t+Q_2))\), qui sont les solutions de l'oscillateur.

    7-d) En utilisant les conditions initiales suivantes \[ \begin{pmatrix} \dot{x}(0) \\ \dot{y}(0) \end{pmatrix} = \begin{pmatrix} 0 \\ v \end{pmatrix} \quad \text{et} \quad \begin{pmatrix} x(0) \\ y(0) \end{pmatrix} = \begin{pmatrix} L \\ 0 \end{pmatrix} \] établir les trajectoires dans les nouvelles et les anciennes coordonnées, \(Q_1, Q_2, x(t)\) et \(y(t)\).

    Explication : Conditions initiales : \(x(0)=L\), \(y(0)=0\), \(\dot{x}(0)=0\), \(\dot{y}(0)=v\). Calculons les constantes \(P_1=E_x\) et \(P_2=E_y\) : \(p_x(0) = m\dot{x}(0) = 0\). \(p_y(0) = m\dot{y}(0) = mv\). \[ E_x = P_1 = \frac{p_x(0)^2}{2m} + \frac{1}{2} m \omega_0^2 x(0)^2 = 0 + \frac{1}{2} m \omega_0^2 L^2 \] \[ E_y = P_2 = \frac{p_y(0)^2}{2m} + \frac{1}{2} m \omega_0^2 y(0)^2 = \frac{(mv)^2}{2m} + 0 = \frac{1}{2} m v^2 \] Les nouvelles impulsions sont constantes : \(P_1 = \frac{1}{2} m \omega_0^2 L^2\) et \(P_2 = \frac{1}{2} m v^2\). Utilisons les relations \(Q_k = \partial S/\partial P_k\). À \(t=0\) : \(Q_1 = \pm \frac{1}{\omega_0} \arcsin\left(\frac{x(0)}{\sqrt{2P_1/m\omega_0^2}}\right)\) \(Q_1 = \pm \frac{1}{\omega_0} \arcsin\left(\frac{L}{\sqrt{2(\frac{1}{2}m\omega_0^2 L^2)/m\omega_0^2}}\right) = \pm \frac{1}{\omega_0} \arcsin\left(\frac{L}{L}\right) = \pm \frac{1}{\omega_0} \arcsin(1)\) \(Q_1 = \pm \frac{1}{\omega_0} \frac{\pi}{2}\). Choisissons le signe + (convention). \(Q_1 = \frac{\pi}{2\omega_0}\). \(Q_2 = \pm \frac{1}{\omega_0} \arcsin\left(\frac{y(0)}{\sqrt{2P_2/m\omega_0^2}}\right)\) \(Q_2 = \pm \frac{1}{\omega_0} \arcsin\left(\frac{0}{\dots}\right) = \pm \frac{1}{\omega_0} \arcsin(0) = 0\). Les nouvelles coordonnées sont constantes : \(Q_1 = \frac{\pi}{2\omega_0}\) et \(Q_2 = 0\). Trajectoires dans les anciennes coordonnées : De \(Q_1 = \pm \frac{1}{\omega_0} \arcsin(x/A_x) - t\) et \(Q_2 = \pm \frac{1}{\omega_0} \arcsin(y/A_y) - t\), on a : \(\pm \arcsin(x/A_x) = \omega_0 (t + Q_1)\). \(\pm \arcsin(y/A_y) = \omega_0 (t + Q_2)\). \(A_x = \sqrt{2P_1/m\omega_0^2} = L\). \(A_y = \sqrt{2P_2/m\omega_0^2} = \sqrt{2(\frac{1}{2}mv^2)/m\omega_0^2} = \sqrt{v^2/\omega_0^2} = |v|/\omega_0\). Pour \(x(t)\): \(\pm \arcsin(x/L) = \omega_0 (t + \pi/(2\omega_0)) = \omega_0 t + \pi/2\). \(x(t) = L \sin(\pm (\omega_0 t + \pi/2))\). Puisque \(\sin(\alpha+\pi/2)=\cos\alpha\) et \(\sin(-\alpha-\pi/2)=\cos\alpha\), on a : \(x(t) = L \cos(\omega_0 t)\). Vérifions: \(x(0)=L\cos(0)=L\). \(\dot{x}(t) = -L\omega_0 \sin(\omega_0 t)\). \(\dot{x}(0)=0\). Correct. Pour \(y(t)\): \(\pm \arcsin(y/A_y) = \omega_0 (t + 0) = \omega_0 t\). \(y(t) = A_y \sin(\pm \omega_0 t)\). On a \(\dot{y}(t) = \pm A_y \omega_0 \cos(\pm \omega_0 t)\). \(\dot{y}(0) = v\). Donc \(\pm A_y \omega_0 \cos(0) = v\). \(\pm (v/\omega_0) \omega_0 = v \implies \pm v = v\). Il faut choisir le signe +. Donc \(y(t) = A_y \sin(\omega_0 t) = \frac{v}{\omega_0} \sin(\omega_0 t)\). Vérifions: \(y(0) = (v/\omega_0)\sin(0)=0\). \(\dot{y}(t) = (v/\omega_0)\omega_0 \cos(\omega_0 t) = v \cos(\omega_0 t)\). \(\dot{y}(0)=v\cos(0)=v\). Correct. Les trajectoires sont : \[ x(t) = L \cos(\omega_0 t) \] \[ y(t) = \frac{v}{\omega_0} \sin(\omega_0 t) \] C'est une ellipse centrée à l'origine.
