Exercice 1. Modes normaux d'oscillations

On considère deux pendules identiques reliés entre eux. Voir figure.

[Placeholder: Diagramme montrant deux pendules simples suspendus côte à côte, avec leurs masses reliées par un ressort horizontal.]

(Variables q1, q2 indiquent les déplacements horizontaux des masses)

La position des pendules est notée $q_1, q_2$ respectivement, et l'on suppose que $q_1 = 0$, et $q_2 = 0$ est la position d'équilibre stable. On s'intéresse aux petites oscillations, c'est à dire $q_1, q_2 \ll 1$. Le Hamiltonien est (dans des variables sans unité)

$$ H(q_1, p_1, q_2, p_2) = \frac{1}{2}(p_1^2 + p_2^2) + U(q_1, q_2) $$ $$ U(q_1, q_2) = \frac{1}{2}q_1^2 + \frac{1}{2}q_2^2 + \frac{1}{2}\alpha(q_1 - q_2)^2, \quad \alpha \ge 0 $$
  1. Commenter le sens physique de chaque terme. Où est intervenue l'hypothèse des petites oscillations?
    • $\frac{1}{2}(p_1^2 + p_2^2)$: Énergie cinétique. Correspond à $\frac{1}{2m}(p_x^2+p_y^2)$ avec $m=1$. $p_1, p_2$ sont les moments conjugués à $q_1, q_2$.
    • $U(q_1, q_2)$: Énergie potentielle.
      • $\frac{1}{2}q_1^2$: Énergie potentielle du pendule 1 (approximation harmonique pour petites oscillations, $\propto \theta^2 \propto q^2$, avec $mgl/2=1/2$ ?).
      • $\frac{1}{2}q_2^2$: Énergie potentielle du pendule 2 (idem).
      • $\frac{1}{2}\alpha(q_1 - q_2)^2$: Énergie potentielle stockée dans le ressort de couplage, proportionnelle au carré de son allongement $(q_1-q_2)$. $\alpha$ est lié à la raideur du ressort.
    L'hypothèse des petites oscillations est intervenue pour approximer l'énergie potentielle gravitationnelle de chaque pendule ($mg l (1-\cos\theta) \approx \frac{1}{2} mgl \theta^2 \approx \frac{1}{2} mg/l q^2$) par un terme quadratique $\frac{1}{2}q_i^2$ (après mise à l'échelle appropriée des variables pour avoir un coefficient 1/2).

  2. D'après la théorie mathématique des formes quadratiques, il existe une transformation linéaire orthogonale qui “diagonalise” la forme quadratique $U$ et par conséquent $H$. Soit la transformation de variables :

    $$ Q_1 = \frac{1}{\sqrt{2}}(q_1 + q_2), \quad P_1 = \frac{1}{\sqrt{2}}(p_1 + p_2) $$ $$ Q_2 = \frac{1}{\sqrt{2}}(q_1 - q_2), \quad P_2 = \frac{1}{\sqrt{2}}(p_1 - p_2) $$

    Montrer que $(Q_1, P_1, Q_2, P_2)$ sont des variables canoniques. Exprimer l'énergie potentielle $U$ puis le Hamiltonien $H$ dans ces nouvelles variables.

    Vérification canonicité : On calcule les crochets de Poisson. $\{Q_1, P_1\} = \{ \frac{1}{\sqrt{2}}(q_1 + q_2), \frac{1}{\sqrt{2}}(p_1 + p_2) \}$. $= \frac{1}{2} [ \{q_1, p_1\} + \{q_1, p_2\} + \{q_2, p_1\} + \{q_2, p_2\} ]$. Sachant que $\{q_i, p_j\} = \delta_{ij}$, on a : $= \frac{1}{2} [ 1 + 0 + 0 + 1 ] = \frac{1}{2}(2) = 1$. De même, $\{Q_2, P_2\} = \frac{1}{2} [ \{q_1, p_1\} - \{q_1, p_2\} - \{q_2, p_1\} + \{q_2, p_2\} ] = \frac{1}{2}[1 - 0 - 0 + 1] = 1$. Calculons un crochet croisé: $\{Q_1, P_2\} = \frac{1}{2} [ \{q_1, p_1\} - \{q_1, p_2\} + \{q_2, p_1\} - \{q_2, p_2\} ] = \frac{1}{2}[1 - 0 + 0 - 1] = 0$. Les autres crochets $\{Q_1, Q_2\}, \{P_1, P_2\}, \{Q_2, P_1\}$ sont aussi nuls car ils ne contiennent que des termes $\{q_i, q_j\}$ ou $\{p_i, p_j\}$ ou des termes croisés $\{q_i, p_j\}$ qui s'annulent. La transformation est donc canonique.
    Expression de U : Il faut exprimer $q_1, q_2$ en fonction de $Q_1, Q_2$. $Q_1 + Q_2 = \frac{1}{\sqrt{2}}(2q_1) \implies q_1 = \frac{1}{\sqrt{2}}(Q_1 + Q_2)$. $Q_1 - Q_2 = \frac{1}{\sqrt{2}}(2q_2) \implies q_2 = \frac{1}{\sqrt{2}}(Q_1 - Q_2)$. $U = \frac{1}{2}q_1^2 + \frac{1}{2}q_2^2 + \frac{1}{2}\alpha(q_1 - q_2)^2$. $q_1^2 = \frac{1}{2}(Q_1^2 + 2Q_1Q_2 + Q_2^2)$. $q_2^2 = \frac{1}{2}(Q_1^2 - 2Q_1Q_2 + Q_2^2)$. $q_1 - q_2 = \frac{1}{\sqrt{2}}(Q_1+Q_2 - (Q_1-Q_2)) = \frac{1}{\sqrt{2}}(2Q_2) = \sqrt{2}Q_2$. $(q_1 - q_2)^2 = (\sqrt{2}Q_2)^2 = 2Q_2^2$. $U = \frac{1}{2} \left[ \frac{1}{2}(Q_1^2 + 2Q_1Q_2 + Q_2^2) + \frac{1}{2}(Q_1^2 - 2Q_1Q_2 + Q_2^2) \right] + \frac{1}{2}\alpha(2Q_2^2)$. $U = \frac{1}{2} \left[ \frac{1}{2}(2Q_1^2 + 2Q_2^2) \right] + \alpha Q_2^2 = \frac{1}{2}(Q_1^2 + Q_2^2) + \alpha Q_2^2$. $$ U(Q_1, Q_2) = \frac{1}{2}Q_1^2 + \frac{1}{2}(1 + 2\alpha)Q_2^2 $$ L'énergie potentielle est diagonalisée.
    Expression de H : Il faut exprimer $p_1^2+p_2^2$ en fonction de $P_1, P_2$. $p_1 = \frac{1}{\sqrt{2}}(P_1 + P_2)$. $p_2 = \frac{1}{\sqrt{2}}(P_1 - P_2)$. $p_1^2 = \frac{1}{2}(P_1^2 + 2P_1P_2 + P_2^2)$. $p_2^2 = \frac{1}{2}(P_1^2 - 2P_1P_2 + P_2^2)$. $p_1^2 + p_2^2 = \frac{1}{2}(2P_1^2 + 2P_2^2) = P_1^2 + P_2^2$. L'énergie cinétique $T = \frac{1}{2}(p_1^2 + p_2^2) = \frac{1}{2}(P_1^2 + P_2^2)$. Le Hamiltonien $H = T + U$ devient : $$ H(Q_1, P_1, Q_2, P_2) = \left(\frac{1}{2}P_1^2 + \frac{1}{2}Q_1^2\right) + \left(\frac{1}{2}P_2^2 + \frac{1}{2}(1 + 2\alpha)Q_2^2\right) $$
  3. Déduire que le problème est constitué de deux oscillateurs harmoniques découplés, appelés modes normaux. Trouver leur fréquence respective $\omega_1(\alpha), \omega_2(\alpha)$ et les tracer en fonction de $\alpha$.
    Le Hamiltonien est la somme de deux Hamiltoniens indépendants : $H = H_1(Q_1, P_1) + H_2(Q_2, P_2)$. $H_1 = \frac{1}{2}P_1^2 + \frac{1}{2}Q_1^2$. C'est l'Hamiltonien d'un oscillateur harmonique de masse $M=1$ et de raideur $K=1$. Sa pulsation est $\omega_1 = \sqrt{K/M} = \sqrt{1/1} = 1$. $H_2 = \frac{1}{2}P_2^2 + \frac{1}{2}(1 + 2\alpha)Q_2^2$. C'est l'Hamiltonien d'un oscillateur harmonique de masse $M=1$ et de raideur $K=(1+2\alpha)$. Sa pulsation est $\omega_2 = \sqrt{K/M} = \sqrt{1+2\alpha}$.
    Les fréquences (pulsations) des modes normaux sont : $$ \omega_1(\alpha) = 1 \quad \text{(constante)} $$ $$ \omega_2(\alpha) = \sqrt{1 + 2\alpha} $$
    Tracé :
    [Placeholder: Graphe des pulsations omega vs alpha (pour alpha >= 0). omega_1 est une ligne horizontale à 1. omega_2 commence à 1 pour alpha=0 et augmente comme sqrt(1+2*alpha).]
  4. On étudie le premier mode (en supposant que le deuxième mode est au repos $Q_2 = 0, P_2 = 0$) : décrire le mouvement $q_1(t), q_2(t)$ des pendules. De même pour le deuxième mode.
    Premier mode ($Q_2=0, P_2=0$) : Le système évolue selon $H_1 = \frac{1}{2}P_1^2 + \frac{1}{2}Q_1^2$. C'est un oscillateur harmonique de pulsation $\omega_1=1$. $Q_1(t) = A_1 \cos(t + \phi_1)$. Puisque $Q_2=0$, on a $q_1 - q_2 = 0 \implies q_1 = q_2$. Et $Q_1 = \frac{1}{\sqrt{2}}(q_1 + q_2) = \frac{1}{\sqrt{2}}(2q_1) = \sqrt{2}q_1$. Donc $q_1(t) = q_2(t) = \frac{1}{\sqrt{2}} Q_1(t) = \frac{A_1}{\sqrt{2}} \cos(t + \phi_1)$. Dans ce mode, les deux pendules oscillent en phase avec la même amplitude et la pulsation $\omega_1=1$. C'est le mode symétrique.
    Deuxième mode ($Q_1=0, P_1=0$) : Le système évolue selon $H_2 = \frac{1}{2}P_2^2 + \frac{1}{2}(1 + 2\alpha)Q_2^2$. C'est un oscillateur harmonique de pulsation $\omega_2=\sqrt{1+2\alpha}$. $Q_2(t) = A_2 \cos(\omega_2 t + \phi_2)$. Puisque $Q_1=0$, on a $q_1 + q_2 = 0 \implies q_1 = -q_2$. Et $Q_2 = \frac{1}{\sqrt{2}}(q_1 - q_2) = \frac{1}{\sqrt{2}}(q_1 - (-q_1)) = \sqrt{2}q_1$. Donc $q_1(t) = \frac{1}{\sqrt{2}} Q_2(t) = \frac{A_2}{\sqrt{2}} \cos(\omega_2 t + \phi_2)$ et $q_2(t) = -q_1(t) = -\frac{A_2}{\sqrt{2}} \cos(\omega_2 t + \phi_2)$. Dans ce mode, les deux pendules oscillent en opposition de phase avec la même amplitude et la pulsation $\omega_2=\sqrt{1+2\alpha}$. C'est le mode antisymétrique.

  5. (Optionnel) On suppose que à $t=0$, le pendule 2 est immobile $q_2=0, p_2=0$. Décrire le mouvement des pendules $q_1(t), q_2(t)$. Dans le cas d'un couplage faible $\alpha \ll 1$, cela s'appelle "effet de résonance" et est similaire à l'effet tunnel en mécanique quantique.

    Conditions initiales : $q_2(0)=0, p_2(0)=0$. Supposons $q_1(0)=A_0, p_1(0)=0$. Il faut exprimer ces conditions initiales en termes de $Q_1, P_1, Q_2, P_2$. $Q_1(0) = \frac{1}{\sqrt{2}}(q_1(0)+q_2(0)) = \frac{1}{\sqrt{2}}(A_0+0) = A_0/\sqrt{2}$. $Q_2(0) = \frac{1}{\sqrt{2}}(q_1(0)-q_2(0)) = \frac{1}{\sqrt{2}}(A_0-0) = A_0/\sqrt{2}$. $P_1(0) = \frac{1}{\sqrt{2}}(p_1(0)+p_2(0)) = \frac{1}{\sqrt{2}}(0+0) = 0$. $P_2(0) = \frac{1}{\sqrt{2}}(p_1(0)-p_2(0)) = \frac{1}{\sqrt{2}}(0-0) = 0$.
    Évolution des modes : Mode 1: $Q_1(0)=A_0/\sqrt{2}, P_1(0)=0$. Solution $Q_1(t) = (A_0/\sqrt{2}) \cos(\omega_1 t) = (A_0/\sqrt{2}) \cos(t)$. Mode 2: $Q_2(0)=A_0/\sqrt{2}, P_2(0)=0$. Solution $Q_2(t) = (A_0/\sqrt{2}) \cos(\omega_2 t)$.
    Mouvement des pendules : $q_1(t) = \frac{1}{\sqrt{2}}(Q_1(t) + Q_2(t)) = \frac{1}{\sqrt{2}} \left( \frac{A_0}{\sqrt{2}}\cos(t) + \frac{A_0}{\sqrt{2}}\cos(\omega_2 t) \right) = \frac{A_0}{2}(\cos(t) + \cos(\omega_2 t))$. $q_2(t) = \frac{1}{\sqrt{2}}(Q_1(t) - Q_2(t)) = \frac{1}{\sqrt{2}} \left( \frac{A_0}{\sqrt{2}}\cos(t) - \frac{A_0}{\sqrt{2}}\cos(\omega_2 t) \right) = \frac{A_0}{2}(\cos(t) - \cos(\omega_2 t))$.
    Utilisant $\cos a + \cos b = 2\cos(\frac{a+b}{2})\cos(\frac{a-b}{2})$ et $\cos a - \cos b = -2\sin(\frac{a+b}{2})\sin(\frac{a-b}{2})$: $$ q_1(t) = A_0 \cos\left(\frac{\omega_2-1}{2}t\right) \cos\left(\frac{\omega_2+1}{2}t\right) $$ $$ q_2(t) = A_0 \sin\left(\frac{\omega_2-1}{2}t\right) \sin\left(\frac{\omega_2+1}{2}t\right) $$ Cas faible couplage $\alpha \ll 1$: $\omega_2 = \sqrt{1+2\alpha} \approx 1 + \alpha$. $\frac{\omega_2-1}{2} \approx \alpha/2$ (basse fréquence, pulsation de battement). $\frac{\omega_2+1}{2} \approx 1 + \alpha/2 \approx 1$ (haute fréquence, pulsation moyenne). $q_1(t) \approx A_0 \cos(\frac{\alpha}{2}t) \cos(t)$. $q_2(t) \approx A_0 \sin(\frac{\alpha}{2}t) \sin(t)$. C'est un phénomène de battement. L'énergie oscille entre les deux pendules. Initialement, $q_1$ oscille avec une grande amplitude $A_0\cos(t)$ et $q_2$ est au repos. Lentement (sur une échelle de temps $1/\alpha$), l'amplitude de $q_1$ diminue (car $\cos(\alpha t/2)$ diminue) tandis que l'amplitude de $q_2$ augmente (car $\sin(\alpha t/2)$ augmente). Quand $\alpha t/2 = \pi/2$, l'amplitude de $q_1$ est nulle et celle de $q_2$ est maximale. Puis le processus s'inverse. L'énergie est transférée périodiquement du pendule 1 au pendule 2 et vice-versa. C'est l'"effet de résonance" ou transfert d'énergie entre oscillateurs couplés.

Exercice 2. Particule en champ magnétique

On considère une particule de charge $e$ (un proton par exemple) de masse $m$ dans un champ magnétique uniforme et constant $B$ selon l'axe $z$ : $\mathbf{B} = (0, 0, B)$ (on négligera les forces autres que la force de Lorentz).

  1. Montrer que

    $$ \mathbf{A} = (A_x, A_y, A_z) = \left(-\frac{1}{2}By, \frac{1}{2}Bx, 0\right) $$

    est une expression possible pour le potentiel vecteur, appelée "Jauge symétrique", et que le potentiel scalaire est $\phi = 0$. Écrire (sans développer) le Hamiltonien de la particule $H(x, y, z, p_x, p_y, p_z)$.

    Vérification du potentiel vecteur : On calcule $\nabla \wedge \mathbf{A}$. $(\nabla \wedge \mathbf{A})_x = \frac{\partial A_z}{\partial y} - \frac{\partial A_y}{\partial z} = \frac{\partial 0}{\partial y} - \frac{\partial (\frac{1}{2}Bx)}{\partial z} = 0 - 0 = 0$. $(\nabla \wedge \mathbf{A})_y = \frac{\partial A_x}{\partial z} - \frac{\partial A_z}{\partial x} = \frac{\partial (-\frac{1}{2}By)}{\partial z} - \frac{\partial 0}{\partial x} = 0 - 0 = 0$. $(\nabla \wedge \mathbf{A})_z = \frac{\partial A_y}{\partial x} - \frac{\partial A_x}{\partial y} = \frac{\partial (\frac{1}{2}Bx)}{\partial x} - \frac{\partial (-\frac{1}{2}By)}{\partial y} = \frac{1}{2}B - (-\frac{1}{2}B) = B$. Donc $\nabla \wedge \mathbf{A} = (0, 0, B) = \mathbf{B}$. C'est un potentiel vecteur correct.
    Potentiel scalaire : Le champ $\mathbf{B}$ est constant, donc $\mathbf{E} = -\nabla\phi - \frac{\partial\mathbf{A}}{\partial t} = 0$. Comme $\mathbf{A}$ ne dépend pas du temps, on peut choisir $\phi=0$.
    Hamiltonien : Pour une particule chargée dans un champ EM, $H = \frac{1}{2m}(\mathbf{p} - e\mathbf{A})^2 + e\phi$. Avec $\phi=0$ : $$ H = \frac{1}{2m} [ (p_x - eA_x)^2 + (p_y - eA_y)^2 + (p_z - eA_z)^2 ] $$ $$ H(x, y, z, p_x, p_y, p_z) = \frac{1}{2m} \left[ \left(p_x + \frac{eB}{2}y\right)^2 + \left(p_y - \frac{eB}{2}x\right)^2 + p_z^2 \right] $$

  2. On propose d'effectuer le changement de variables suivant $(x, p_x, y, p_y) \to (Q, P, q, p)$ :

    $$ Q = \frac{1}{\sqrt{eB}}(p_x + \frac{eB}{2}y) \quad P = \frac{1}{\sqrt{eB}}(p_y - \frac{eB}{2}x) $$ $$ q = \frac{1}{\sqrt{eB}}(p_x - \frac{eB}{2}y) \quad p = -\frac{1}{\sqrt{eB}}(p_y + \frac{eB}{2}x) $$

    (Note: Il y a probablement une erreur de signe ou de définition dans q ou p. Vérifions la canonicité).

    Calculer les crochets de Poisson des fonctions $(Q, P, q, p)$ deux à deux, pour vérifier que ce sont bien des variables canoniques.

    Vérifions les crochets fondamentaux $\{Q,P\}, \{q,p\}, \{Q,q\}, \{Q,p\}, \{P,q\}, \{P,p\}$. $\{Q, P\} = \{ \frac{1}{\sqrt{eB}}(p_x + \frac{eB}{2}y), \frac{1}{\sqrt{eB}}(p_y - \frac{eB}{2}x) \}$. $= \frac{1}{eB} [ \{p_x, p_y\} - \{p_x, \frac{eB}{2}x\} + \{\frac{eB}{2}y, p_y\} - \{\frac{eB}{2}y, \frac{eB}{2}x\} ]$ $= \frac{1}{eB} [ 0 - \frac{eB}{2}\{p_x, x\} + \frac{eB}{2}\{y, p_y\} - 0 ]$ $= \frac{1}{eB} [ -\frac{eB}{2}(-1) + \frac{eB}{2}(1) ] = \frac{1}{eB} [\frac{eB}{2} + \frac{eB}{2}] = \frac{1}{eB}[eB] = 1$.
    $\{q, p\} = \{ \frac{1}{\sqrt{eB}}(p_x - \frac{eB}{2}y), -\frac{1}{\sqrt{eB}}(p_y + \frac{eB}{2}x) \}$. $= -\frac{1}{eB} [ \{p_x, p_y\} + \{p_x, \frac{eB}{2}x\} - \{\frac{eB}{2}y, p_y\} - \{\frac{eB}{2}y, \frac{eB}{2}x\} ]$ $= -\frac{1}{eB} [ 0 + \frac{eB}{2}\{p_x, x\} - \frac{eB}{2}\{y, p_y\} - 0 ]$ $= -\frac{1}{eB} [ \frac{eB}{2}(-1) - \frac{eB}{2}(1) ] = -\frac{1}{eB} [-\frac{eB}{2} - \frac{eB}{2}] = -\frac{1}{eB}[-eB] = 1$.
    $\{Q, q\} = \{ \frac{1}{\sqrt{eB}}(p_x + \frac{eB}{2}y), \frac{1}{\sqrt{eB}}(p_x - \frac{eB}{2}y) \}$. $= \frac{1}{eB} [ \{p_x, p_x\} - \{p_x, \frac{eB}{2}y\} + \{\frac{eB}{2}y, p_x\} - \{\frac{eB}{2}y, \frac{eB}{2}y\} ]$ $= \frac{1}{eB} [ 0 - 0 + 0 - 0 ] = 0$.
    $\{Q, p\} = \{ \frac{1}{\sqrt{eB}}(p_x + \frac{eB}{2}y), -\frac{1}{\sqrt{eB}}(p_y + \frac{eB}{2}x) \}$. $= -\frac{1}{eB} [ \{p_x, p_y\} + \{p_x, \frac{eB}{2}x\} + \{\frac{eB}{2}y, p_y\} + \{\frac{eB}{2}y, \frac{eB}{2}x\} ]$ $= -\frac{1}{eB} [ 0 + \frac{eB}{2}(-1) + \frac{eB}{2}(1) + 0 ] = 0$.
    De même $\{P, q\}=0$ et $\{P, p\}=0$. La transformation est canonique, $(Q,P)$ et $(q,p)$ forment deux paires de variables conjuguées indépendantes.
  3. En introduisant la fréquence cyclotron $\omega = eB/m$, donner l'expression de $H$ en fonction des nouvelles variables $(Q, P, q, p, z, p_z)$. Quel modèle standard de dynamique obtient t-on?
    Exprimons les termes dans $H$ initial : $p_x + \frac{eB}{2}y = \sqrt{eB} Q$. $p_y - \frac{eB}{2}x = \sqrt{eB} P$. Le Hamiltonien contient $H_{xy} = \frac{1}{2m} [ (p_x + \frac{eB}{2}y)^2 + (p_y - \frac{eB}{2}x)^2 ]$. $H_{xy} = \frac{1}{2m} [ (\sqrt{eB} Q)^2 + (\sqrt{eB} P)^2 ] = \frac{eB}{2m} (Q^2 + P^2)$. $H_{z} = \frac{p_z^2}{2m}$. Le Hamiltonien total est $H = H_{xy} + H_z$. En utilisant $\omega = eB/m$: $$ H(Q, P, q, p, z, p_z) = \frac{1}{2} \omega (Q^2 + P^2) + \frac{p_z^2}{2m} $$

    Il semble que les variables $(q,p)$ n'apparaissent pas. Vérifions la transformation inverse ou l'expression de H.

    Reconsidérons $H = \frac{1}{2m} [ (p_x + \frac{eB}{2}y)^2 + (p_y - \frac{eB}{2}x)^2 + p_z^2 ]$. C'est correct.

    Le modèle obtenu est celui d'un oscillateur harmonique 1D (variables $Q, P$) avec la pulsation $\omega$, et d'une particule libre selon $z$. Les variables $(q, p)$ définies dans l'énoncé semblent ne pas correspondre directement à une partie du Hamiltonien, ou sont mal définies (peut-être devraient-elles être liées aux coordonnées $x,y$?)

    Alternative: Les variables $Q, P$ sont proportionnelles aux termes dans $H$. Il manque peut-être des termes dans la transformation pour $x,y$. Si on considère $(Q,P)$ et $(x,y)$ comme canoniques? Non. Peut-être que $H = H_1(Q,P) + H_2(q,p) + H_z$? Si $Q, P, q, p$ sont définis comme dans l'énoncé, alors $Q^2+P^2 = \frac{1}{eB}[(p_x+\frac{eB}{2}y)^2 + (p_y-\frac{eB}{2}x)^2]$ et $q^2+p^2 = \frac{1}{eB}[(p_x-\frac{eB}{2}y)^2 + (p_y+\frac{eB}{2}x)^2]$. Le Hamiltonien $H = \frac{eB}{2m}(Q^2+P^2) + \frac{p_z^2}{2m}$.

    Le modèle obtenu est celui d'un oscillateur harmonique (pour les variables $Q, P$) de pulsation $\omega$, et d'une particule libre (pour la variable $z$).
  4. Ecrire les équations de mouvement de Hamilton pour ces nouvelles variables canoniques et déduire l'allure des trajectoires en variables $(Q, P, q, p, z, p_z)$ puis en variables $(x, y, z, p_x, p_y, p_z)$. Que se passe t-il si la charge $e$ est négative (électron par exemple)?
    Équations pour $(Q, P)$: $\dot{Q} = \partial H / \partial P = \frac{1}{2}\omega(2P) = \omega P$. $\dot{P} = -\partial H / \partial Q = -\frac{1}{2}\omega(2Q) = -\omega Q$. C'est $\ddot{Q} = \omega \dot{P} = \omega(-\omega Q) = -\omega^2 Q$. Oscillateur harmonique. $Q(t) = A \cos(\omega t + \phi_Q)$, $P(t) = -\frac{1}{\omega}\dot{Q} = A \sin(\omega t + \phi_Q)$. La trajectoire dans le plan $(Q,P)$ est un cercle parcouru avec la pulsation $\omega$.
    Équations pour $(q, p)$: $H$ ne dépend pas de $q, p$. $\dot{q} = \partial H / \partial p = 0 \implies q = \text{constante}$. $\dot{p} = -\partial H / \partial q = 0 \implies p = \text{constante}$.
    Équations pour $(z, p_z)$: $H$ ne dépend pas de $z$. $\dot{z} = \partial H / \partial p_z = p_z/m$. $\dot{p}_z = -\partial H / \partial z = 0 \implies p_z = \text{constante}$. $\ddot{z} = \dot{p}_z/m = 0$. Mouvement rectiligne uniforme selon $z$. $z(t) = z_0 + (p_z/m)t$.
    Trajectoires en $(Q, P, q, p, z, p_z)$: Mouvement circulaire dans le plan $(Q,P)$, position fixe dans le plan $(q,p)$, mouvement rectiligne uniforme selon $z$.
    Trajectoires en $(x, y, z, p_x, p_y, p_z)$: Le mouvement selon $z$ est inchangé. Le mouvement dans le plan $(x,y)$ est une combinaison des oscillations de $(Q,P)$ et des valeurs constantes de $(q,p)$. Il faut inverser la transformation. Le mouvement projeté sur le plan $(x,y)$ est circulaire (projection du mouvement hélicoïdal). La vitesse selon $z$ est constante. La trajectoire 3D est une hélice d'axe parallèle à $z$.
    Si $e$ est négative : $\omega = eB/m$ devient négatif. Les équations pour $Q, P$ sont $\dot{Q} = \omega P$ et $\dot{P} = -\omega Q$. Cela correspond toujours à un oscillateur harmonique, mais le sens de rotation dans le plan $(Q, P)$ est inversé. Physiquement, la force de Lorentz $e(\vec{v} \wedge \vec{B})$ change de signe, donc le sens de giration de la particule autour des lignes de champ $\mathbf{B}$ est inversé. La trajectoire reste une hélice, mais de chiralité opposée.
