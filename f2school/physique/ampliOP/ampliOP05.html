Bornes : tensions et courants

Bornes

L'ampli-op possède cinq bornes qui sont d'intérêt pour ce cours :

  1. Entrée inversante $v_n$ (l'entrée avec le signe $-$)
  2. Entrée non-inversante $v_p$ (l'entrée avec le signe $+$)
  3. Sortie $v_o$
  4. Alimentation positive $V_{CC+}$
  5. Alimentation négative $V_{CC-}$

Propriétés

  • Gain très élevé : de l'ordre de 200 000 pour des amplis pratiques (en basse fréquence).
  • Équation du gain (idéalisée) : $v_o = A(v_p - v_n)$, où $A$ est le gain différentiel en boucle ouverte.
  • Sortie limitée : La tension de sortie $v_o$ est limitée par les tensions d'alimentation : $V_{CC-} < v_o < V_{CC+}$ (en réalité, $V_{sat-} < v_o < V_{sat+}$ où $V_{sat}$ est légèrement inférieure à $V_{CC}$).

Deux modes de fonctionnement :

  1. Mode amplificateur
  2. Mode comparateur

Dans le mode amplificateur, il y a toujours un parcours (une boucle) entre la sortie et l'entrée inversante : c'est du feedback négatif (contre-réaction). Sans ce parcours (ou avec un feedback positif), l'ampli-op fonctionne en mode comparateur (saturation).

Ampli-op en mode amplificateur

Mode amplificateur

Deux suppositions importantes (pour l'AOP idéal en régime linéaire) :

Règles importantes :
  1. $v_p = v_n$ (Les tensions aux deux entrées sont égales)
  2. $i_p = i_n = 0$ (Les courants d'entrée sont nuls)

Le reste de l'analyse se fait avec les techniques des chapitres précédents : lois de Kirchhoff (lois des nœuds et des mailles), méthode des tensions de nœud, etc., en appliquant ces deux règles.

Ampli-op en mode amplificateur

Exemple (suite)

Mode amplificateur : il y a un parcours entre la sortie et la borne $-$ (à travers la résistance de 100k$\Omega$). C'est une contre-réaction.

Appliquer les suppositions de l'AOP idéal en mode amplificateur : $v_n = v_p$. Puisque $v_p$ est mis à terre, $v_p = 0$, donc $v_n = 0$. Et $i_p = i_n = 0$.

Puisque $i_n = 0$, le courant traversant la résistance de 25k$\Omega$ ($i_{25k\Omega}$) est le même que celui traversant la résistance de 100k$\Omega$ ($i_{100k\Omega}$). On a donc (en appliquant la loi d'Ohm pour exprimer ces courants) :

\[ \underbrace{\frac{v_{in} - v_n}{25000}}_{i_{25k\Omega}} = \underbrace{\frac{v_n - v_o}{100000}}_{i_{100k\Omega}} \]

et puisque $v_n = 0$ et $v_{in} = 1V$ :

\[ \frac{1 - 0}{25000} = \frac{0 - v_o}{100000} \] \[ \frac{1}{25000} = -\frac{v_o}{100000} \implies v_o = - \frac{100000}{25000} \times 1 = -4V \]

La sortie est $v_o = -4V$. Cette valeur est comprise entre les tensions d'alimentation (-10V et +10V), donc l'AOP fonctionne bien en régime linéaire.

Amplificateur inversant

Amplificateur inversant (suite)

Conditions de l'AOP idéal en mode amplificateur :

  • $v_n = v_p = 0$ (car $v_p$ est à la masse)
  • $i_p = i_n = 0$

On peut dire que le courant $i_s$ (traversant $R_s$) est égal au courant $i_f$ (traversant $R_f$), car $i_n=0$.

Expression du courant $i_s$ :

\[ i_s = \frac{v_i - v_n}{R_s} = \frac{v_i}{R_s} \quad (1) \]

On applique le même raisonnement pour trouver le courant $i_f$ :

\[ i_f = \frac{v_n - v_o}{R_f} = \frac{0 - v_o}{R_f} = -\frac{v_o}{R_f} \quad (2) \]

Avec ces deux équations, et $i_s = i_f$, on obtient :

\[ \frac{v_i}{R_s} = -\frac{v_o}{R_f} \] \[ v_o = -\frac{R_f}{R_s} v_i \quad (3) \]
Gabriel Cormier (UdeM) GELE2112 Chapitre 4 Hiver 2009 11 / 28
Amplificateur inversant

Amplificateur inversant (suite)

  • La tension de sortie $v_o$ sera une version amplifiée (par $|-R_f/R_s|$) de la tension à l'entrée $v_i$.
  • Le signe négatif est la raison pour laquelle on appelle cette configuration inversante.
  • L'équation (3) est seulement valide si la sortie est comprise entre les limites de saturation : $V_{CC-} < v_o < V_{CC+}$.

Sommateur

Sommateur (suite)

Mode amplificateur, donc :

  • $v_n = v_p = 0$
  • $i_p = i_n = 0$

On fait la somme des courants (Loi des Nœuds de Kirchhoff) à la borne $v_n$ :

\[ \frac{v_a - v_n}{R_a} + \frac{v_b - v_n}{R_b} + \frac{v_c - v_n}{R_c} + \frac{v_o - v_n}{R_f} = 0 \]

Avec $v_n=0$ :

\[ \frac{v_a}{R_a} + \frac{v_b}{R_b} + \frac{v_c}{R_c} + \frac{v_o}{R_f} = 0 \quad (4) \]

La sortie est :

\[ v_o = - \left(\frac{R_f}{R_a} v_a + \frac{R_f}{R_b} v_b + \frac{R_f}{R_c} v_c\right) \quad (5) \]
Gabriel Cormier (UdeM) GELE2112 Chapitre 4 Hiver 2009 14 / 28
Sommateur

Sommateur (suite)

Si les trois résistances à l'entrée sont égales ($R_a = R_b = R_c = R_s$), on peut simplifier :

\[ v_o = -\frac{R_f}{R_s}(v_a + v_b + v_c) \quad (6) \]

On voit bien que la sortie est proportionnelle à la somme des trois entrées (avec inversion).

Le nombre d'entrées n'est pas limité à 3 : on peut additionner autant de signaux que l'on veut. Par exemple, on pourrait additionner les 16 canaux d'un signal audio.

Gabriel Cormier (UdeM) GELE2112 Chapitre 4 Hiver 2009 16 / 28
Amplificateur non-inversant

Amplificateur non-inversant (suite)

Mode amplificateur, donc (AOP idéal) :

  • $v_n = v_p$
  • $i_p = i_n = 0$

La tension à l'entrée non-inverseuse est $v_p = v_i$. Donc $v_n = v_i$.

On fait la somme des courants (KCL) à la borne négative ($v_n$) de l'ampli-op :

\[ \underbrace{\frac{v_n - 0}{R_s}}_{\text{courant dans } R_s} + \underbrace{\frac{v_n - v_o}{R_f}}_{\text{courant dans } R_f} + i_n = 0 \]

Comme $i_n=0$ et $v_n=v_i$ :

\[ \frac{v_i}{R_s} + \frac{v_i - v_o}{R_f} = 0 \quad (7) \]

ce qui donne :

\[ \frac{v_i}{R_s} + \frac{v_i}{R_f} = \frac{v_o}{R_f} \quad (8) \]

qu'on simplifie :

\[ v_o = v_i \left(\frac{R_f}{R_s} + \frac{R_f}{R_f}\right) = v_i \left(1 + \frac{R_f}{R_s}\right) \quad (9) \]

Le gain est $A_v = 1 + R_f/R_s$, toujours $\ge 1$.

Amplificateur différentiel

Amplificateur différentiel (suite)

Mode amplificateur, donc (AOP idéal) :

  • $i_p = i_n = 0$
  • $v_n = v_p$

Tension $v_p$ (diviseur de tension $R_c, R_d$ car $i_p=0$) :

\[ v_p = v_b \frac{R_d}{R_c + R_d} \quad (10) \]

Somme des courants (KCL) à la borne négative ($v_n$) de l'ampli-op (avec $i_n=0$) :

\[ \frac{v_a - v_n}{R_a} + \frac{v_o - v_n}{R_b} = 0 \] \[ \frac{v_a}{R_a} + \frac{v_o}{R_b} = v_n \left( \frac{1}{R_a} + \frac{1}{R_b} \right) = v_n \frac{R_a+R_b}{R_a R_b} \] \[ v_o = - \frac{R_b}{R_a} v_a + \left( \frac{R_a+R_b}{R_a} \right) v_n \quad (11) \]
Gabriel Cormier (UdeM) GELE2112 Chapitre 4 Hiver 2009 19 / 28
Amplificateur différentiel

Amplificateur différentiel (suite)

On combine les équations (10) et (11) (puisque $v_n = v_p$), ce qui donne :

\[ -v_o = \frac{R_b}{R_a}v_a - \left(\frac{R_a+R_b}{R_a}\right) \left(\frac{R_d}{R_c+R_d}\right) v_b \quad (12) \]

et donc,

\[ v_o = \left( \frac{(R_a+R_b)R_d}{R_a(R_c+R_d)} \right) v_b - \frac{R_b}{R_a} v_a \quad (13) \]
Gabriel Cormier (UdeM) GELE2112 Chapitre 4 Hiver 2009 20 / 28
Amplificateur différentiel

Amplificateur différentiel (suite)

Si on veut que la sortie soit proportionnelle à $v_b - v_a$, il faut que les coefficients de $v_b$ et $v_a$ soient égaux en valeur absolue. Il faut que :

\[ \frac{(R_a+R_b)R_d}{R_a(R_c+R_d)} = \frac{R_b}{R_a} \] Ce qui se simplifie (voir #slide20 derivation) en $R_a R_d = R_b R_c$, ou : \[ \frac{R_a}{R_b} = \frac{R_c}{R_d} \quad (15) \]

Dans ce cas-là, on obtient :

\[ v_o = \frac{R_b}{R_a} (v_b - v_a) \quad (16) \]

Procédure générale

Procédure générale

La méthode générale à suivre lorsqu'on solutionne des problèmes d'un ampli-op en mode amplificateur est la suivante :

  1. Calculer la tension à la borne positive, $v_p$. Il ne faut pas oublier que $i_p = 0$. (Souvent $v_p$ est directement la tension d'entrée ou 0V, ou issue d'un diviseur de tension).
  2. Faire la somme des courants (KCL) à la borne négative ($v_n$), tout en n'oubliant pas que $i_n = 0$.
  3. Appliquer le principe $v_n = v_p$ (car AOP idéal en régime linéaire).
  4. Solutionner l'équation obtenue pour trouver $v_o$ en fonction des tensions d'entrée ($v_i$).
  5. Vérifier que la tension de sortie $v_o$ calculée est bien comprise dans les limites de fonctionnement non saturé : $V_{CC-} < v_o < V_{CC+}$. Si ce n'est pas le cas, l'AOP est saturé et $v_o = V_{sat+}$ ou $V_{sat-}$.

Exemple (suite)

1. Calculer $v_p$ : La tension à la borne positive $v_p$ est déterminée par la source de 2V et la résistance de 5k$\Omega$. Puisque le courant $i_p = 0$, il n'y a pas de chute de tension dans la résistance de 5k$\Omega$. Donc, $v_p = 2V$.

2. Somme des courants à $v_n$ : À la borne négative $v_n$, en supposant que les courants vont vers la sortie (ou sortent du nœud via les résistances), et $i_n=0$ :

\[ \frac{1 - v_n}{20k} + \frac{v_o - v_n}{160k} = 0 \]

3. Appliquer $v_n = v_p$ : et puisque $v_p = 2V$, on a $v_n = 2V$.

\[ \frac{1 - 2}{20k} = \frac{2 - v_o}{160k} \]

4. Solutionner pour $v_o$ :

\[ \frac{-1}{20k} = \frac{2 - v_o}{160k} \] \[ -1 \times \frac{160k}{20k} = 2 - v_o \] \[ -8 = 2 - v_o \] \[ v_o = 2 + 8 = 10V \]

5. Vérifier la saturation : La sortie est $v_o = 10V$. Les alimentations sont $\pm 18V$. La sortie est comprise entre -18V et +18V (en supposant $V_{sat}$ proche de $V_{cc}$). Donc, l'AOP n'est pas saturé. Le résultat $v_o=10V$ est valide.

Le comparateur

Le comparateur

Il n'y a pas de chemin de feedback (contre-réaction) entre la sortie et la borne négative. L'AOP fonctionne donc en boucle ouverte.

Le fonctionnement est assez simple (basé sur la saturation) :

  • Si $v_p > v_n$, alors $v_o = V_{cc+}$ (ou $V_{sat+}$)
  • Si $v_n > v_p$ (ou $v_p < v_n$), alors $v_o = V_{cc-}$ (ou $V_{sat-}$)

L'AOP compare les deux tensions d'entrée et sature sa sortie en fonction du signe de leur différence.

Exemple (suite)

La tension $v_p$ est obtenue en appliquant un diviseur de tension sur $v_i$, puisque le courant $i_p = 0$.

\[ v_p = v_i \frac{R_2}{R_1 + R_2} = v_i \frac{40k\Omega}{10k\Omega + 40k\Omega} = v_i \frac{40}{50} = 0.8 v_i \]

La tension à l'entrée inversante est $v_n = v_{ref} = 5V$.

La sortie sera $V_{cc+}$ lorsque $v_p > v_n$, ou $0.8 v_i > v_{ref}$ :

\[ 0.8 v_i > 5V \implies v_i > \frac{5}{0.8} = 6.25 V \]

Si $v_i < 6.25 V$, alors $v_p < v_n$ et la sortie sera $V_{cc-}$.
