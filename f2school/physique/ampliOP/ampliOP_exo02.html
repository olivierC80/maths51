AMPLIFICATEUR OPERATIONNEL

1 - PROPRIETES DE L'AOP IDEAL

1.1 Description

L'amplificateur opérationnel se présente sous la forme d'un circuit intégré (en général boîtier DIL - Dual In Line); il possède 2 entrées notées $E^+$ (entrée non inverseuse) et $E^-$ (entrée inverseuse). La polarisation du circuit est réalisée, en général, à l'aide d'une alimentation continue symétrique (par exemple +15V/-15V) dont le point médian (0V) constitue la référence des tensions (MASSE).

1.2 Amplificateur différentiel

C'est un amplificateur différentiel intégré (A.D.I.) ; il amplifie la différence entre la tension $u^+$ appliquée à l'entrée non inverseuse et la tension $u^-$ appliquée à l'entrée inverseuse.

La relation sortie-entrée (en boucle ouverte) est :

\[ u_S = A_d (u^+ - u^-) = A_d u_d \]
  • $A_d$ : amplification différentielle en boucle ouverte (très grande)
  • $u_d$ : tension différentielle ($u_d = u^+ - u^-$)

1.3 Régimes de fonctionnement

La tension de polarisation ($\pm V$) impose une limite à la tension de sortie $u_S$ (tension de saturation $V_{SAT}$), généralement un peu inférieure aux tensions d'alimentation :

\[ -V_{SAT} \leq u_S \leq +V_{SAT} \]

On distingue donc deux régimes de fonctionnement :

  • Régime linéaire : Si $|u_S| \leq V_{SAT}$. Alors la relation $u_S = A_d (u^+ - u^-) = A_d u_d$ est vérifiée.
  • Régime de saturation : Si $|A_d \cdot u_d| \geq V_{SAT}$. Alors $u_S = +V_{SAT}$ (si $u_d > 0$) ou $u_S = -V_{SAT}$ (si $u_d < 0$).
AM - 2002/2003 - Page 1

1.4 AOP idéal

Le modèle de l'AOP idéal simplifie l'analyse en considérant les caractéristiques suivantes :

  • Résistance d'entrée $R_E$ infinie : $\implies i^+ = i^- = 0$ (aucun courant n'entre dans l'AOP).
  • Résistance de sortie $R_S$ nulle ($R_S = 0$) : $\implies u_S$ est indépendant de la charge connectée en sortie.
  • Amplification différentielle $A_d$ infinie ($A_d \rightarrow \infty$).

Conséquences pour les régimes de fonctionnement :

  • ☞ En régime linéaire : Puisque $u_S$ est finie et $A_d = \infty$, la tension différentielle doit être nulle: $u_d = (u^+ - u^-) = 0 \implies u^+ = u^-$.
  • ☞ En régime de saturation : Si $u_d > 0 \implies u_S = +V_{SAT}$
    Si $u_d < 0 \implies u_S = -V_{SAT}$

1.5 Caractéristique de transfert de l'AOP idéal

Montage équivalent en régime linéaire :

Le modèle de l'amplificateur idéal est justifié pour la plupart des applications étudiées cette année. Nous verrons en travaux pratiques les limites de ce modèle, en particulier pour des signaux de fréquences élevées.

Pour la suite du cours, l'AOP est supposé idéal.

2 - FONCTIONS ANALOGIQUES : régime linéaire

Hypothèses de l'AOP idéal en régime linéaire :

  • $R_E$ infinie : $i^+ = i^- = 0$
  • Régime linéaire : $u_d = u^+ - u^- = 0 \implies u^+ = u^-$

2.1 Montage amplificateur inverseur

Analyse :

  • $u^+ = 0$ (relié à la masse). Donc $u^- = u^+ = 0$ (masse virtuelle).
  • Loi des nœuds à l'entrée $E^-$ : $i_1 = i_2 + i^-$. Comme $i^-=0$, $i_1 = i_2$.
  • Lois des mailles (ou loi d'Ohm) :
    • Maille d'entrée : $u_E - R_1 i_1 + u_d = 0$. Avec $u_d=0$, $u_E = R_1 i_1$.
    • Maille de sortie : $u_S + R_2 i_2 + u_d = 0$. Avec $u_d=0$, $u_S = -R_2 i_2$.
  • En combinant ($i_1=i_2$): $u_S = -R_2 (u_E / R_1)$.

Relation finale :

\[ u_S = -\frac{R_2}{R_1} u_E \]

2.2 Montage amplificateur non inverseur

Analyse :

  • $u^+ = u_E$. Donc $u^- = u^+ = u_E$.
  • $i^-=0$. R1 et R2 forment un diviseur de tension alimenté par $u_S$ et donnant $u^-$: \[ u^- = u_S \frac{R_1}{R_1 + R_2} \]
  • En égalant les deux expressions de $u^-$ ($u^-=u_E$): \[ u_E = u_S \frac{R_1}{R_1 + R_2} \]

Relation finale :

\[ u_S = \left(1 + \frac{R_2}{R_1}\right) u_E \]
AM - 2002/2003 - Page 3

2.3 Montage suiveur

Analyse :

  • $u^+ = u_E$.
  • $u^-$ est directement connectée à $u_S$, donc $u^- = u_S$.
  • En régime linéaire, $u^- = u^+$.

Relation finale :

\[ u_S = u_E \]

Ce montage a un gain de 1, une impédance d'entrée infinie et une impédance de sortie nulle. Il sert d'adaptateur d'impédance (buffer).

2.4 Montage sommateur inverseur

Analyse :

  • $u^+ = 0 \implies u^- = 0$ (masse virtuelle).
  • Loi des nœuds à $E^-$ ($i^-=0$) : $i_1 + i_2 = i_3$.
  • Lois des mailles / Ohm :
    • $u_1 - R_1 i_1 = u^- = 0 \implies i_1 = u_1 / R_1$.
    • $u_2 - R_2 i_2 = u^- = 0 \implies i_2 = u_2 / R_2$.
    • $u^- - u_S = R_3 i_3 \implies -u_S = R_3 i_3$.
  • En combinant : $-u_S = R_3 (i_1 + i_2) = R_3 (u_1/R_1 + u_2/R_2)$.

Relation finale :

\[ u_S = -R_3 \left(\frac{u_1}{R_1} + \frac{u_2}{R_2}\right) \]
AM - 2002/2003 - Page 4

2.5 Montage soustracteur (Amplificateur différentiel)

Analyse (régime linéaire $u^+=u^-$):

  • Diviseur de tension à vide sur $E^+$ (car $i^+=0$) : \[ u^+ = u_2 \frac{R_4}{R_3 + R_4} \]
  • Théorème de superposition (ou Millman) sur $E^-$ (car $i^-=0$) : \[ u^- = \frac{u_1/R_1 + u_S/R_2}{1/R_1 + 1/R_2} = \frac{R_2 u_1 + R_1 u_S}{R_1 + R_2} \]
  • Égalité $u^+ = u^-$ : \[ u_2 \frac{R_4}{R_3 + R_4} = \frac{R_2 u_1 + R_1 u_S}{R_1 + R_2} \]
  • En isolant $u_S$ (après réarrangement) : \[ u_S = u_2 \frac{R_4 (R_1+R_2)}{R_1 (R_3+R_4)} - u_1 \frac{R_2}{R_1} \]

Cas particuliers : Si la condition $\frac{R_4}{R_3} = \frac{R_2}{R_1}$ (ou $R_1 R_4 = R_2 R_3$) est respectée :

\[ u_S = \frac{R_2}{R_1} (u_2 - u_1) \]

Si $R_1 = R_2 = R_3 = R_4 = R$, alors $u_S = u_2 - u_1$.

Rappels

  • Diviseur de tension à vide : $U_2 = U_1 \frac{R_2}{R_1 + R_2}$.
  • Diviseur de courant : $I_2 = I \frac{G_2}{G_1 + G_2} = I \frac{R_1}{R_1 + R_2}$.
  • Théorème de superposition : La tension (ou le courant) en un point d'un circuit linéaire comportant plusieurs sources est égale à la somme des tensions (ou courants) obtenues lorsque chaque source agit seule (les autres sources de tension étant court-circuitées et les sources de courant ouvertes).
AM - 2002/2003 - Page 5

2.6 Montage intégrateur

Analyse :

  • $u^+ = 0 \implies u^- = 0$.
  • $i = i_R = i_C$ car $i^-=0$.
  • $i = (u_E - u^-)/R = u_E / R$.
  • $i = C \frac{d(u^- - u_S)}{dt} = -C \frac{du_S}{dt}$.
  • En égalant : $u_E / R = -C du_S/dt$.

Relation finale :

\[ u_S(t) = -\frac{1}{RC} \int u_E(t) dt + K \]

(Note: $u_c$ dans le schéma original est la tension aux bornes de C, $u_c = u^- - u_S = -u_S$).

Rappel Condensateur

$q = C u_c$ et $i = \frac{dq}{dt}$. Donc $i = C \frac{du_c}{dt}$.

2.7 Montage dérivateur

Analyse :

  • $u^+ = 0 \implies u^- = 0$.
  • $i = i_C = i_R$ car $i^-=0$.
  • $i = C \frac{d(u_E - u^-)}{dt} = C \frac{du_E}{dt}$. (Note: $u_E = u_c$ dans le schéma original).
  • $i = (u^- - u_S) / R = -u_S / R$.
  • En égalant : $C du_E/dt = -u_S / R$.

Relation finale :

\[ u_S = -RC \frac{du_E}{dt} \]
AM - 2002/2003 - Page 6

3 - MONTAGES COMPARATEURS : régime de saturation

3.1 Fonction comparateur

La fonction "comparateur" simple consiste à comparer une grandeur d'entrée $u_E$ à une valeur constante $U_o$.

Exemple :

  • si $u_E < U_o$ alors $u_S = U_{S1}$ (par ex. niveau bas)
  • si $u_E > U_o$ alors $u_S = U_{S2}$ (par ex. niveau haut)

La tension de sortie ne peut donc prendre que deux états ($U_{S1}$ et $U_{S2}$). Le passage de l'un des états à l'autre est appelé basculement et la tension constante $U_o$ seuil de basculement.

Caractéristique de transfert d'un comparateur à un seuil

3.2 Montage comparateur à un seuil

Exemple de réalisation avec un amplificateur opérationnel sans circuit de contre-réaction :

L'AOP fonctionne en régime de saturation.

Tension différentielle : $u_d = u^+ - u^- = U_o - u_E$.

  • Si $u_d > 0$ (soit $u_E < U_o$) $\implies u_S = +V_{SAT}$
  • Si $u_d < 0$ (soit $u_E > U_o$) $\implies u_S = -V_{SAT}$
