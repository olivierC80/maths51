Correction Détaillée

Correction Question de cours

La force de Laplace $\vec{F}_L = I d\vec{l} \times \vec{B}$ exercée sur un élément de courant $I d\vec{l}$ est nulle si :

  • Le courant $I$ est nul ($I=0$).
  • L'élément de circuit est ponctuel ($d\vec{l} = \vec{0}$).
  • Le champ magnétique $\vec{B}$ est nul ($\vec{B}=\vec{0}$).
  • Le vecteur élément de courant $d\vec{l}$ est parallèle au vecteur champ magnétique $\vec{B}$ ($d\vec{l} \parallel \vec{B}$).

La force de Lorentz $\vec{F} = q(\vec{E} + \vec{v} \times \vec{B})$ sur une charge $q$ est nulle (en absence de $\vec{E}$) si $q=0$, $\vec{v}=0$, $\vec{B}=0$ ou $\vec{v} \parallel \vec{B}$.

Correction Exercice 2 (Onde)

Champ: $\vec{E} = E_0 \cos(\omega t + kz) \vec{i}$.

Conditions d'existence (dans le vide):

  1. $\nabla \cdot \vec{E} = \frac{\partial E_x}{\partial x} = 0$. OK.
  2. $\nabla \times \vec{E} = -\frac{\partial \vec{B}}{\partial t}$. $\nabla \times \vec{E} = \vec{e}_y \frac{\partial E_x}{\partial z} = \vec{e}_y [E_0 (-\sin(\omega t+kz)) (k)] = -k E_0 \sin(\omega t+kz) \vec{e}_y$. $-\frac{\partial \vec{B}}{\partial t} = -k E_0 \sin(\omega t+kz) \vec{e}_y$. $\frac{\partial \vec{B}}{\partial t} = k E_0 \sin(\omega t+kz) \vec{e}_y$. Intégration / t: $\vec{B} = k E_0 (-\frac{\cos(\omega t+kz)}{\omega}) \vec{e}_y = -\frac{k E_0}{\omega} \cos(\omega t+kz) \vec{e}_y$. Soit $B_0 = k E_0 / \omega$. $\vec{B} = -B_0 \cos(\omega t+kz) \vec{e}_y$.
  3. $\nabla \cdot \vec{B} = \frac{\partial B_y}{\partial y} = 0$. OK.
  4. $\nabla \times \vec{B} = \mu_0 \epsilon_0 \frac{\partial \vec{E}}{\partial t}$. $\nabla \times \vec{B} = \vec{e}_x (-\frac{\partial B_y}{\partial z})$. $\frac{\partial B_y}{\partial z} = -B_0 ( \sin(\omega t+kz)) (k) = -k B_0 \sin(\omega t+kz)$. $\nabla \times \vec{B} = -(-k B_0 \sin(\omega t+kz))\vec{e}_x = k B_0 \sin(\omega t+kz) \vec{e}_x$. $\frac{\partial \vec{E}}{\partial t} = E_0 (-\sin(\omega t+kz)) (\omega) \vec{i} = -\omega E_0 \sin(\omega t+kz) \vec{i}$. $\mu_0 \epsilon_0 \frac{\partial \vec{E}}{\partial t} = -\mu_0 \epsilon_0 \omega E_0 \sin(\omega t+kz) \vec{i}$. Egalité: $k B_0 = -\mu_0 \epsilon_0 \omega E_0$. Substituer $B_0 = k E_0 / \omega$: $k (k E_0 / \omega) = -\mu_0 \epsilon_0 \omega E_0$. $k^2/\omega = -\mu_0 \epsilon_0 \omega$. $k^2 = -\omega^2 \mu_0 \epsilon_0 = -\omega^2/c^2$.

La condition $k^2 = -\omega^2/c^2$ ne peut pas être satisfaite si k et $\omega$ sont réels. Ce champ $\vec{E}$ ne peut pas exister comme une onde progressive simple dans le vide. Il pourrait correspondre à une onde évanescente si k est imaginaire pur ($k=j\kappa$) ou si $\omega$ est imaginaire pur.
