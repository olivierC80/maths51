Exercice 5 :

Codage de Hamming pour transmettre 4 bits (m1..m4) avec détection/correction d'1 erreur. Mot de 7 bits (k1, k2, m1, k3, m2, m3, m4). Bits de contrôle k1, k2, k3 calculés pour parité paire:

  • t1 = parité(k1, m1, m2, m4) = 0
  • t2 = parité(k2, m1, m3, m4) = 0
  • t3 = parité(k3, m2, m3, m4) = 0

A la réception (K1..M4), on recalcule T1, T2, T3. Si T3T2T1 = 000, pas d'erreur. Sinon, T3T2T1 donne la position du bit erroné.

Exemple: Mot à transmettre: 1101 (m1=1, m2=1, m3=0, m4=1). Calculer k1, k2, k3. Mot émis. Supposer une erreur sur le 3ème bit. Mot reçu. Calculer T1, T2, T3. Corriger.

  1. Représenter un schéma synoptique de la transmission (émetteur, récepteur, correcteur).
  2. Proposer un schéma pour chacun des 3 modules (Calcul des ki, Calcul des Ti, Correction).
