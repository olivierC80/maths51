2.6 Fonctions booléennes non-simplifiables

Sur $\mathbb{B}^n$ (pour $n$ variables), il existe deux uniques fonctions $F$ et $G$ totalement insimplifiables! Ces deux fonctions vérifient les propriétés suivantes :

  • $F(x_1, \dots, x_n) = \overline{G(x_1, \dots, x_n)}$ (elles sont complémentaires l'une de l'autre)
  • $\text{card}(C_F) = \text{card}(C_G) = 2^{n-1}$ (elles valent 1 pour exactement la moitié des combinaisons d'entrée)
  • chaque produit (minterme) diffère au moins de deux variables avec tout autre produit (de la même fonction). Ceci implique qu'aucun regroupement de mintermes n'est possible.

Pour trois variables ($n=3$), nous obtenons :

$ F(a, b, c) = \bar{a}\bar{b}\bar{c} + \bar{a}b c + a\bar{b}c + a b\bar{c} $
$ G(a, b, c) = \bar{a}\bar{b}c + \bar{a}b\bar{c} + a\bar{b}\bar{c} + a b c $

(Ces fonctions correspondent à la parité paire (F=1 si nb pair de 1) et impaire (G=1 si nb impair de 1) des entrées, ou l'inverse selon la convention 0/1)

Pourtant il est quand même possible de leur donner une représentation condensée. Pour cela, on définit l'opérateur OU EXCLUSIF ($\oplus$) dont la table de vérité est :

Nous allons également nous intéresser à $\overline{x \oplus y}$ aussi appelé “comparateur binaire” (ou XNOR, équivalence) qui renvoit “1” si x et y sont égaux :

$ F(a, b, c) = \bar{a}(\bar{b}\bar{c} + bc) + a(\bar{b}c + b\bar{c}) $
$ F(a, b, c) = \bar{a}(\overline{b \oplus c}) + a(b \oplus c) $

$ F(a, b, c) = \bar{a}\bar{y} + ay $
Ceci correspond à $ \overline{a \oplus y} $.
Donc $ F(a, b, c) = \overline{a \oplus (b \oplus c)} $.

Comme $G = \bar{F}$, on a :

$ G(a, b, c) = a \oplus (b \oplus c) $

$ G(a, b, c) = a \oplus b \oplus c $
$ F(a, b, c) = \overline{a \oplus b \oplus c} $

3.1 Introduction

Nous avons vu dans le chapitre précédent l'importance des règles algébriques pour simplifier une fonction booléenne. Nous avons également vu qu'une simplification pouvait se faire suivant différentes voies et aboutir à des solutions différentes et équivalentes. Mais il faut garder à l'esprit qu'une mauvaise utilisation de ces règles peut conduire à une impasse, c'est à dire à une expression qui peut encore se simplifier à condition de redévelopper certains termes.

Considérons par exemple la fonction $f(x, y, z) = \bar{x}y + xyz$. L'expression associée à cette fonction semble être simplifiée au mieux. Pourtant :

$ \begin{aligned} \bar{x}y + xyz &= \bar{x}y(z+\bar{z}) + xyz && \text{(utiliser } A9: z+\bar{z}=1 \text{)} \\ &= \bar{x}yz + \bar{x}y\bar{z} + xyz \\ &= \bar{x}yz + xyz + \bar{x}y\bar{z} && \text{(réorganiser)} \\ &= (\bar{x}+x)yz + \bar{x}y\bar{z} && \text{(factoriser)} \\ &= 1 \cdot yz + \bar{x}y\bar{z} && \text{(utiliser } A9: \bar{x}+x=1 \text{)} \\ &= yz + \bar{x}y\bar{z} \end{aligned} $

(Note: Le PDF donne $\bar{x}y + \bar{y}z$. Vérifions $f=\bar{x}y+xyz$. Table de vérité: 010=1, 011=1, 111=1. La forme $yz+\bar{x}y\bar{z}$ donne 1 pour 011, 111 (yz) et 010 ($\bar{x}y\bar{z}$). C'est correct. La forme $\bar{x}y+\bar{y}z$ donne 1 pour 010, 011 ($\bar{x}y$) et 101 ($\bar{y}z$). Ce n'est pas la même fonction. L'exemple de simplification du PDF est erroné.)

De plus, la table de vérité n'est pas une représentation pratique pour simplifier une expression booléenne à cause des règles algébriques principales à utiliser :

  • Regroupement: $xy + x\bar{y} = x(y + \bar{y}) = x \cdot 1 = x$
  • Absorption: $xy + x = x(y+1) = x \cdot 1 = x$

Pour l'exemple, considérons la table de vérité suivante :
