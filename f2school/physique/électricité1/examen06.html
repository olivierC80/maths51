Exercice 2:

1) La force électrostatique est dirigée vers le haut puisque la charge sont négative la force électrostatique et champ électrostatique sont de sens opposés; le champ électrostatique est donc dirigé vers le bas. La plaque du haut chargée positivement celle du bas négativement.

2) a) la force compense l'effet de la gravitation est la force électrostatique.

b) à l'équilibre $\sum \vec{F} = 0 \implies m\vec{g} + q\vec{E} = \vec{0}$
projection sur l'axe dirigé vers le bas: $mg - |q|E = 0$.
par conséquent $|q| = \frac{mg}{E} = \frac{\rho \frac{4}{3}\pi R^3}{E}$.
$m=\rho V = \rho \frac{4}{3}\pi R^3$ parce que la gouttelette est sphérique.
or $E = U/d$. donc $|q| = \frac{\rho \frac{4}{3}\pi R^3}{(U/d)}$.

3) $|q| = \rho \frac{4}{3}\pi R^3 \frac{d}{U} = 900 \times \frac{4\pi}{3} (2.05 \times 10^{-6})^3 \times \frac{1.5 \times 10^{-2}}{3000} = 1.59 \times 10^{-19} C$ (Calcul à vérifier)
$|q| = 1.59 \times 10^{-19} \approx 1 \times 1.6 \times 10^{-19} C$.
La charge de l'électron $|e| = 1.6 \times 10^{-19} C$.
donc $|q| \approx 1 |e|$.

Exercice 3:

A1) La géométrie de la surface de gauss associée à ce fil est un cylindre.

A2) $\oint \vec{E} \cdot d\vec{S} = Q_{int}/\varepsilon_0$. Le champ est radial $\vec{E}(r)$, $\vec{E} \perp \vec{dS}_{bases}=0$. $\oint E(r) dS_L = Q_{int}/\varepsilon_0$. $E(r) (2\pi r L) = \lambda L / \varepsilon_0$. $\implies E(r) = \frac{\lambda}{2\pi\varepsilon_0 r}$. $\vec{E}(r) = \frac{\lambda}{2\pi\varepsilon_0 r} \vec{e}_r$.

B1) Le fil 1 est créé un champ $\vec{E}_1(M) = \frac{\lambda}{2\pi\varepsilon_0 r_1} \vec{e}_{r1}$ dans M.
Le fil 2 est créé un champ $\vec{E}_2(M) = \frac{\lambda}{2\pi\varepsilon_0 r_2} \vec{e}_{r2}$ dans M.
donc $\vec{E}_T(M) = \frac{\lambda}{2\pi\varepsilon_0}(\frac{\vec{e}_{r1}}{r_1} + \frac{\vec{e}_{r2}}{r_2})$.

B2) Le champ est nul lorsque $\vec{E}_1(M) + \vec{E}_2(M) = 0$.
$\frac{\lambda}{2\pi\varepsilon_0 r_1} \vec{e}_{r1} + \frac{\lambda}{2\pi\varepsilon_0 r_2} \vec{e}_{r2} = 0$.
Dans ce cas on a $\vec{e}_{r1} = -\vec{e}_{r2} = \vec{e}_r$.
$(\frac{\lambda}{2\pi\varepsilon_0 r_1} - \frac{\lambda}{2\pi\varepsilon_0 r_2})\vec{e}_r = 0$.
$\implies \frac{1}{r_1} = \frac{1}{r_2} \implies r_1=r_2$. Or $r_1+r_2=a$.
Le champ est nul pour $r_1=r_2$. Donc le champ est nul pour $r_1=r_2=a/2$.

Exercice II:

1) La résistance du fil: $R = U/I = 22 \times 10^{-3} / 750 \times 10^{-3} = 0.029 \Omega$.

2a) La résistivité: $R = \rho L/S \implies \rho = R S/L = 0.029 \times (\pi (1 \times 10^{-3})^2) / 5 = 1.82 \times 10^{-8} \Omega.m$.

2b) La conductivité: $\gamma = 1/\rho = 1 / (1.82 \times 10^{-8}) = 5.49 \times 10^7 \Omega^{-1}.m^{-1}$.

3) Densité de courant: $j = I/S = 750 \times 10^{-3} / (\pi (10^{-3})^2) = 2.38 \times 10^5 A/m^2$. (Calcul à vérifier)
$\vec{j} = \gamma \vec{E} \implies \vec{j} = j \vec{u}$. $|j| = \gamma |E| \implies |E| = |j|/\gamma = 2.38 \times 10^5 / 5.49 \times 10^7 = 4.3 \times 10^{-3} V/m$.(Calcul à vérifier)

4) Le champ à l'intérieur du fil: $\vec{E} = \vec{j}/\gamma$. $|E| = j/\gamma = 4.3 \times 10^{-3} V/m$. (Voir calcul 3)

5) Nombre d'électrons libres par unité de volume: $\vec{j} = n q \vec{v}_d = n (-e) \vec{v}_d$.
$|j| = n |e| v_d \implies n = |j| / (|e| v_d) = 2.38 \times 10^5 / (1.6 \times 10^{-19} \times 1.7 \times 10^{-5}) = 8.75 \times 10^{28} m^{-3}$. (Calcul à vérifier)

Exercice III:

1) Le champ électrostatique créé par un disque chargé: $E = \frac{\sigma}{2\varepsilon_0} (1 - \frac{z}{\sqrt{R^2+z^2}}) \vec{e}_z$.

2) L'expression du champ créé par l'ensemble de ces deux disques au point M situé entre les deux disques:
$E_1 = \frac{\sigma}{2\varepsilon_0} (1 - \frac{z}{\sqrt{R^2+z^2}}) \vec{e}_z$. $E_2 = \frac{\sigma}{2\varepsilon_0} (1 - \frac{a-z}{\sqrt{R^2+(a-z)^2}}) (-\vec{e}_z)$.
$E_T = E_1+E_2 = \frac{\sigma}{2\varepsilon_0} [ (1 - \frac{z}{\sqrt{R^2+z^2}}) - (1 - \frac{a-z}{\sqrt{R^2+(a-z)^2}}) ] \vec{e}_z $.

3) Oui, le champ résultant peut être nul dans le milieu d'espace interne aux deux disques.
En effet: $E_T=0 \implies (1 - \frac{z}{\sqrt{R^2+z^2}}) = (1 - \frac{a-z}{\sqrt{R^2+(a-z)^2}})$.
$\implies \frac{z}{\sqrt{R^2+z^2}} = \frac{a-z}{\sqrt{R^2+(a-z)^2}}$. $\implies z^2(R^2+(a-z)^2) = (a-z)^2(R^2+z^2)$.
$\implies z^2R^2 + z^2(a-z)^2 = (a-z)^2R^2 + (a-z)^2z^2$.
$\implies z^2 R^2 = (a-z)^2 R^2$.
$\implies z^2 = (a-z)^2$. $\implies z = a-z$ (car $z>0, a-z>0$). $\implies 2z=a \implies z=a/2$.
Le champ s'annule au milieu d'espace interne aux deux disques.

Exercice IV:

1) L'expression de la capacité de ce condensateur est: $C = \varepsilon_0 S / e$.
AN: $C = (8.84 \times 10^{-12}) \times (100 \times 10^{-6}) / (2 \times 10^{-3}) = 4.42 \times 10^{-13} F = 0.442 pF$. (Note: Le calcul manuscrit donne 4.42 pF, une erreur de facteur 10 dans l'un des calculs ou dans la conversion finale. 8.84e-12 * 100e-6 / 2e-3 = 4.42e-13 F = 0.442 pF)

2) Les charges placées sur chaque armature: $Q = CU = (0.442 \times 10^{-12}) \times 6 = 2.65 \times 10^{-12} C = 2.65 pC$.
Armature A (positive): $Q_A = +2.65 pC$. Armature B (négative): $Q_B = -2.65 pC$.

3) Le champ électrostatique dans l'espace entre les armatures est: $E = U/e$. AN: $E = 6 / (2 \times 10^{-3}) = 3000 V/m$.

4) L'énergie emmagasinée dans cet espace: $W = \frac{1}{2} C U^2 = \frac{1}{2} (0.442 \times 10^{-12}) \times 6^2 = 7.96 \times 10^{-12} J$.

5) Les expressions des nouvelles tension U' et énergie W':
Charge Q reste constante: $Q = 2.65 pC$. Nouvelle capacité $C' = \varepsilon_0 S / e' = \varepsilon_0 S / (e+0.25 \times 10^{-3}) = (8.84 \times 10^{-12}) \times 100 \times 10^{-6} / (2.25 \times 10^{-3}) = 3.93 \times 10^{-13} F = 0.393 pF$.
Nouvelle tension $U' = Q / C' = 2.65 \times 10^{-12} / 0.393 \times 10^{-12} = 6.74 V$.
Nouvelle énergie $W' = \frac{1}{2} C' U'^2 = \frac{1}{2} Q U' = \frac{1}{2} (2.65 \times 10^{-12}) \times 6.74 = 8.93 \times 10^{-12} J$.
Vérification: $W' = Q^2 / (2C') = (2.65 \times 10^{-12})^2 / (2 \times 0.393 \times 10^{-12}) = 8.93 \times 10^{-12} J$.
