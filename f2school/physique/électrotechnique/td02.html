EXERCICE N°1

Un transformateur triphasé possède les caractéristiques suivantes :

  • Tensions nominales : $U_{1n} = 20$KV / $U_{2n} = 380$V (tensions entre phases)
  • Puissance apparente nominale : $S_n = 200$KVA
  • Fréquence : $f = 50$Hz

1- On a réalisé les essais suivants :

  • Essai à vide (côté BT) : $U_{10} = U_{1n} = 20$KV ; $U_{20} = 400$V ; $P_0 = 680$W (pertes fer)
  • Essai en court-circuit (côté HT) : $I_{2cc} = I_{2n}/2$ ; $U_{1cc} = 400$V ; $P_{cc} = 680$W
  1. Calculer le courant secondaire nominal $I_{2n}$.
    La puissance apparente nominale est $S_n = \sqrt{3} U_{2n} I_{2n}$. Ici, $U_{2n}=380$V (tension nominale entre phases au secondaire). On isole $I_{2n}$: $I_{2n} = S_n / (\sqrt{3} U_{2n})$.
  2. Déterminer la tension secondaire entre phases $U_2$, quand le secondaire alimente une charge triphasée équilibrée de facteur de puissance $\cos(\phi) = 0.8$AR (Arrière = inductif) et débite un courant $I_2 = k \times I_{2n}/4$ avec $k \in \{1; 2; 4; 5\}$.
    Il faut d'abord déterminer la résistance $R_S$ et la réactance $X_S$ ramenées au secondaire à partir de l'essai en court-circuit. Les pertes mesurées $P_{cc}$ correspondent aux pertes Joule à $I_{2cc} = I_{2n}/2$: $P_{cc} = 3 R_S I_{2cc}^2 = 3 R_S (I_{2n}/2)^2$. On calcule $R_S$. Le rapport de transformation par phase $m$ dépend du couplage, mais on peut utiliser $m \approx U_{20}/U_{1n} = 400 / 20000$ pour les tensions entre phases. Pour les calculs d'impédance, il faut le rapport par phase. Assumons Yy pour simplifier $m_{ph} = (400/\sqrt{3}) / (20000/\sqrt{3}) = 400/20000 = 1/50$. L'impédance de court-circuit vue du primaire par phase est $Z_{1cc\_ph} = (U_{1cc}/\sqrt{3}) / I_{1cc}$. $I_{1cc} = m_{ph} I_{2cc}$. L'impédance ramenée au secondaire par phase est $Z_S = Z_{1cc\_ph} / m_{ph}^2$. $X_S = \sqrt{Z_S^2 - R_S^2}$. La chute de tension entre phases approximative est $\Delta U_2 \approx \sqrt{3} (R_S I_2 \cos \phi + X_S I_2 \sin \phi)$. Pour $\cos \phi = 0.8$, $\sin \phi = 0.6$. La tension secondaire $U_2 = U_{20} - \Delta U_2$. Calculer $U_2$ pour chaque valeur de $k$.
  3. Calculer le rendement $\eta$ dans les cas du b).
    Puissance utile : $P_2 = \sqrt{3} U_2 I_2 \cos \phi$. Utiliser $U_2$ calculé pour chaque $k$. $\cos \phi = 0.8$. Pertes fer : $P_{fer} = P_0 = 680$W. Pertes Joule à $I_2$ : $P_{Joule}(I_2) = 3 R_S I_2^2$. On peut aussi utiliser $P_{Joule}(I_2) = P_{cc} \times (I_2 / I_{2cc})^2 = P_{cc} \times (I_2 / (I_{2n}/2))^2$. Rendement : $\eta = P_2 / (P_2 + P_{fer} + P_{Joule})$. Calculer pour chaque $k$.

  1. Déterminer l'indice horaire.
    Analyse du schéma : Primaire (HT, A'B'C') : Les enroulements sont connectés entre A'-B', B'-C', C'-A'. C'est un couplage Triangle (D). Secondaire (BT, a'b'c') : Les enroulements sont connectés entre a et n, b et n, c et n. C'est un couplage Étoile (Y) avec neutre sorti (n). Phasage : La borne A' (début d'enroulement HT) correspond à la borne A (fin d'enroulement HT connectée à B'). La borne a (début d'enroulement BT) correspond à la borne a' (sortie phase BT). Le schéma montre que la tension simple $V_{an}$ (secondaire) est en phase avec la tension $V_{A'N'}$ (primaire si Y) ou déphasée par rapport à $U_{A'B'}$ (primaire si D). Dans un couplage Dyn standard, on compare $U_{A'B'}$ et $V_{an}$. Si A est connecté à a, B à b, C à c, on a souvent Dyn11 (BT en retard de 330° ou avance de 30°) ou Dyn5 (BT en retard de 150°). D'après le schéma (A-a, B-b, C-c), c'est un Dyn11. L'indice horaire est 11.
  2. La section de circuit magnétique est de $S_{fer} = 0.02$m² et l'induction maximale $B_{max} = 1.4$Tesla. Calculer le nombre de spires par phase au primaire ($N_1$) et au secondaire ($N_2$).
    Utiliser la formule de Boucherot par phase : $V_{phase} \approx 4.44 N_{phase} S_{fer} f B_{max}$. Primaire (D) : La tension par phase est la tension entre lignes $V_{1ph} = U_{1n} = 20000$V. $N_1 = V_{1ph} / (4.44 S_{fer} f B_{max})$. Secondaire (Y) : La tension par phase est $V_{2ph} = U_{20} / \sqrt{3} = 400 / \sqrt{3}$ V. $N_2 = V_{2ph} / (4.44 S_{fer} f B_{max})$.

EXERCICE N°2

Soient $T_1$ et $T_2$ deux transformateurs triphasés possédant les caractéristiques suivantes :

  • $T_1$: $S_{n1} = 100$KVA ; $20$KV / $220$V (tensions composées)
  • $T_2$: $S_{n2} = 60$KVA ; $20$KV / $220$V (tensions composées)

1- Les couplages de deux transformateurs sont donnés par les figures suivantes :

  1. Donner le type de couplage de chaque transformateur, déterminer leurs indices horaires.
    Analyse T2 : Primaire (A'B'C' vers ABCN) semble D. Secondaire (abcn vers a'b'c') semble Y. Couplage Dyn. Connexions (A-a, B-b, C-c). Indice horaire probable : 11 (Dyn11). Analyse T1 : Primaire (A'B'C' vers ABCN) semble Y. Secondaire (abcn vers a'b'c') semble Z (Zigzag). Couplage Yzn. Connexions (A-a, B-b, C-c). Indice horaire probable : 11 ou 5 selon la réalisation du zigzag (Yzn11 ou Yzn5).
  2. Déterminer pour chaque transformateur, le nombre de spires par colonne au secondaire $N_2$ en fonction de $N_1$ (on donne $U_{10}=20$KV, $U_{20}=230$V comme tensions entre phases à vide).
    Le rapport de transformation par colonne $N_2/N_1$ est approximativement égal au rapport des tensions par phase $V_{2ph}/V_{1ph}$. T2 (Dyn) : $V_{1ph} = U_{10} = 20000$V. $V_{2ph} = U_{20}/\sqrt{3} = 230/\sqrt{3}$V. $N_2/N_1 \approx V_{2ph}/V_{1ph} = (230/\sqrt{3}) / 20000$. T1 (Yzn) : $V_{1ph} = U_{10}/\sqrt{3} = 20000/\sqrt{3}$V. Pour le zigzag standard, la tension de phase $V_{2ph}$ est liée à la tension composée $U_{20}$ par $U_{20} = \sqrt{3} V_{2ph}$ ou $U_{20}=2V_{2ph}$ selon la configuration. Si $U_{20} = 230V$ est la tension composée, $V_{2ph}$ n'est pas simple. Cependant, si on considère le rapport des tensions *simples équivalentes*, on peut utiliser $N_2/N_1 \approx (U_{20}/\sqrt{3}) / (U_{10}/\sqrt{3}) = U_{20}/U_{10} = 230/20000$. Mais le rapport de spires par phase est $V_{2ph}/V_{1ph}$. Pour Yzn, $V_{2ph} = \frac{U_{20}}{2\sqrt{3}} \times \sqrt{3^2+1^2}$ ??? Non, plus simple: $V_{2ph} = U_{20}/\sqrt{3}$. Alors $N_2/N_1 \approx V_{2ph}/V_{1ph} = (230/\sqrt{3}) / (20000/\sqrt{3}) = 230/20000$.
  3. Peut-on coupler ces deux transformateurs en parallèle ?
    Conditions : Mêmes tensions nominales (20KV/220V - OK). Mêmes rapports de transformation (T1: $\approx 230/20000$, T2: $\approx (230/\sqrt{3})/20000$ - NON). Mêmes indices horaires (T1: Yzn?, T2: Dyn11 - NON). Mêmes tensions de court-circuit relatives ($u_{cc}\%$) (Inconnu). Conclusion: Ne peuvent pas être couplés en parallèle car rapports et indices horaires différents.

2- Sur chacun de deux transformateurs précédents on a réalisé les essais suivants :

  • Pour $T_1$ : essai en court circuit $I_{2cc}=I_{2n1}/2$ ; $U_{1cc1}=300$V et $P_{1cc1}=600$W
  • Pour $T_2$: essai en charge avec $\cos(\phi)=1$ ; $I_{22}=I_{2n2}$ ; $U_{22}=224$V
  • Pour $T_2$: essai en charge avec $\cos(\phi)=0.707$AR ; $I_{22}=I_{2n2}/3$ ; $U_{22}=226.75$V
  1. Calculer $R_{s1}$, $R_{s2}$, $X_{s1}$ et $X_{s2}$ (paramètres ramenés au secondaire, par phase).
    T1: $I_{2n1} = 100000 / (\sqrt{3} \times 220) \approx 262.4$A. $I_{2cc1}=I_{2n1}/2 \approx 131.2$A. $P_{1cc1} = 3 R_{s1} I_{2cc1}^2 \implies R_{s1} = P_{1cc1} / (3 I_{2cc1}^2)$. $m_{ph1} \approx (220/\sqrt{3})/(20000/\sqrt{3}) = 220/20000=0.011$. $U_{1cc1\_ph} = 300/\sqrt{3}$V (si Y). $Z_{s1} = (U_{1cc1\_ph}/m_{ph1}) / I_{2cc1}$. $X_{s1}=\sqrt{Z_{s1}^2-R_{s1}^2}$. T2: $I_{2n2} = 60000 / (\sqrt{3} \times 220) \approx 157.5$A. $U_{20} \approx 220$V. $V_{20ph} = 220/\sqrt{3}$. Cas 1 ($\cos\phi=1, I_2=I_{2n2}$): $\Delta V_{ph1} = V_{20ph} - (U_{22}/\sqrt{3}) = (220-224)/\sqrt{3}$ (Tension a augmenté? Impossible. $U_{20}$ doit être > 224V. Utilisons $U_{20}=230$V de l'essai précédent). $\Delta V_{ph1} \approx R_{s2} I_{2n2} \cos\phi + X_{s2} I_{2n2} \sin\phi = R_{s2} I_{2n2}$. $(230-224)/\sqrt{3} = R_{s2} \times 157.5$. Calculer $R_{s2}$. Cas 2 ($\cos\phi=0.707, \sin\phi=0.707, I_2=I_{2n2}/3$): $\Delta V_{ph2} = V_{20ph} - (U_{22}/\sqrt{3}) = (230-226.75)/\sqrt{3}$. $\Delta V_{ph2} \approx R_{s2} (I_{2n2}/3) \cos\phi + X_{s2} (I_{2n2}/3) \sin\phi = (R_{s2} + X_{s2}) \times (I_{2n2}/3) \times 0.707$. Remplacer $R_{s2}$ et $I_{2n2}$ pour trouver $X_{s2}$.
  2. Calculer la puissance apparente nominale dont on peut disposer avec l'ensemble de deux transformateurs.
    Comme ils ne peuvent pas être mis en parallèle, la puissance totale disponible est la somme de leurs puissances nominales utilisées séparément : $S_{totale} = S_{n1} + S_{n2} = 100 + 60 = 160$ KVA.
  3. Les deux transformateurs alimentent une charge commune constituée par 3 impédances $\bar{Z} = 0.36 e^{j32^\circ} \Omega$. Calculer le rendement global quand les deux transformateurs débitent leurs courants secondaires (courants nominaux $I_{2n1}, I_{2n2}$ ? Ou courant total $I_L = V_L/Z_L$ partagé entre eux?).
    Cette question est ambiguë car ils ne peuvent pas être mis en parallèle. Si l'on suppose qu'ils alimentent la charge *séparément* (ce qui est impossible pour une seule charge triphasée), ou si l'on ignore l'impossibilité de la mise en parallèle et qu'on calcule un rendement *hypothétique* : il faudrait calculer comment le courant total $I_L = U_2 / Z_{phase}$ se répartit entre T1 et T2 ($I_{21}, I_{22}$) selon leurs impédances $Z_{s1}, Z_{s2}$. Ensuite, calculer $P_{out} = 3 \times Re(Z) \times I_{phase}^2$. Pertes $P_{fer1}+P_{fer2} + 3R_{s1}I_{21}^2 + 3R_{s2}I_{22}^2$. $\eta = P_{out} / (P_{out} + \text{Pertes})$. Ceci est très complexe et basé sur une hypothèse erronée. Si la question signifie "calculer le rendement de chaque transformateur quand il débite son $I_{2n}$ dans une charge quelconque", alors $\eta_1 = P_{2n1} / (P_{2n1} + P_{fer1} + P_{J1n})$ et $\eta_2 = P_{2n2} / (P_{2n2} + P_{fer2} + P_{J2n})$.

EXERCICE N°3

Un transformateur triphasé, dont le primaire, branché en triangle (D) est alimenté par un système de tension triphasé $U_1 = 20$KV, $f=50$Hz. Le secondaire est branché en étoile (Y). Ce transformateur débite dans une installation fonctionnant en charge nominale sous une tension $U_2 = 380$V (entre phases) / $V_2 = 220$V (phase-neutre) et comprenant :

  • 12 moteurs triphasés identiques : $P_{u\_moteur} = 3$KW, rendement $\eta_m = 0.8$, facteur de puissance $\cos\phi_m = 0.82$.
  • 90 lampes de $P_{lampe} = 60$W, $V_{lampe}=220$V régulièrement reparties sur 3 phases (entre phase et neutre).

1- Pour réaliser l'essai à vide, on ne disposera pas d'une alimentation de 20KV. On l'alimente du coté secondaire sous 380V entre phases. On a relevé coté sortie (primaire) une tension entre phases de 19570V. Déduire de ces mesures :

  1. Le rapport de transformation par colonne ($m_c = N_2/N_1$).
    Essai à vide inversé. Alimentation secondaire Y ($V_{2ph} = 380/\sqrt{3}$V). Sortie primaire D ($U_{10} = 19570$V, $V_{1ph} = 19570$V). Rapport par colonne $m_c = N_2/N_1 = V_{2ph}/V_{1ph} = (380/\sqrt{3}) / 19570$.
  2. Le nombre des spires d'un enroulement primaire $N_1$ sachant que le secondaire comporte $N_2 = 60$ spires par phase.
    $N_1 = N_2 / m_c = 60 / m_c$.

2- Maintenant, le transformateur branché normalement (primaire sous 20KV entre phases), débite dans l'installation dont tous les appareils fonctionnent (charge normale) :

  1. Calculer l'intensité du courant dans un enroulement secondaire ($I_2$) et son déphasage ($\phi_2$) par rapport à la tension simple secondaire.
    Calcul de la charge totale : Moteurs : Puissance absorbée par moteur $P_{a\_moteur} = P_{u\_moteur} / \eta_m = 3000 / 0.8 = 3750$W. Puissance réactive par moteur $Q_{a\_moteur} = P_{a\_moteur} \tan(\acos(0.82))$. Total moteurs : $P_{tot\_m} = 12 \times P_{a\_moteur}$. $Q_{tot\_m} = 12 \times Q_{a\_moteur}$. Lampes : Puissance totale $P_{tot\_l} = 90 \times 60$W (réparties sur les 3 phases, donc $30 \times 60$W par phase). $Q_{tot\_l} = 0$. Charge totale : $P_{TOTAL} = P_{tot\_m} + P_{tot\_l}$. $Q_{TOTAL} = Q_{tot\_m}$. Puissance apparente totale : $S_{TOTAL} = \sqrt{P_{TOTAL}^2 + Q_{TOTAL}^2}$. Courant secondaire (courant de ligne car Y) : $I_2 = S_{TOTAL} / (\sqrt{3} U_2) = S_{TOTAL} / (\sqrt{3} \times 380)$. Déphasage global : $\cos \phi_2 = P_{TOTAL} / S_{TOTAL}$. $\phi_2 = \acos(P_{TOTAL} / S_{TOTAL})$ (angle du courant par rapport à la tension simple $V_2$).
  2. Calculer la chute de tension entre phases ($\Delta U_2$) au secondaire.
    La chute de tension $\Delta U_2 = U_{20} - U_2$. On a $U_2=380$V en charge. Il faut $U_{20}$. L'essai à vide inversé donne $U_{10}=19570$V pour $U_{2\_alim}=380$V. Donc si $U_1=20000$V, $U_{20} = (20000 / 19570) \times 380 \approx 388.3$V. La chute de tension est $\Delta U_2 = U_{20} - U_2 = 388.3 - 380 = 8.3$V. On peut aussi la calculer si $R_S, X_S$ étaient connues : $\Delta U_2 \approx \sqrt{3} (R_S I_2 \cos \phi_2 + X_S I_2 \sin \phi_2)$, mais ces paramètres ne sont pas donnés.

EXERCICE N°4

Les essais d'un transformateur triphasé Y-y ont donné les résultats suivants :

  • Essai à vide : $U_{10}=380$V ; $U_{20}=400$V
  • Essai en court-circuit : $U_{1cc}=19$V ; $I_{2cc}=4.5$A ; $P_{cc}=81$W
  1. Calculer $R_S$ et $X_S$ (ramenés au secondaire, par phase).
    Couplage Yy. Rapport de transformation par phase $m = V_{20ph}/V_{10ph} = (400/\sqrt{3}) / (380/\sqrt{3}) = 400/380 \approx 1.053$. Pertes Joule : $P_{cc} = 3 R_S I_{2cc}^2 \implies R_S = P_{cc} / (3 I_{2cc}^2) = 81 / (3 \times 4.5^2) \approx 1.333 \Omega$. Impédance : $U_{1cc\_ph} = 19/\sqrt{3}$V. $Z_{S} = (U_{1cc\_ph} / m) / I_{2cc} = ((19/\sqrt{3}) / (400/380)) / 4.5 \approx 2.308 \Omega$. Réactance : $X_S = \sqrt{Z_S^2 - R_S^2} = \sqrt{2.308^2 - 1.333^2} \approx 1.884 \Omega$.
  2. Le transformateur alimenté au primaire sous $U_1=380$V débite sur un récepteur triphasé de facteur de puissance $\cos(\phi)=0.8$AR, un courant $I_2=4.5$A. Quelle sera la tension entre phases au secondaire $U_2$?
    Tension à vide par phase : $V_{20ph} = U_{20}/\sqrt{3} = 400/\sqrt{3}$V. Chute de tension par phase : $\Delta V_{2ph} \approx R_S I_2 \cos \phi + X_S I_2 \sin \phi$. $\cos \phi = 0.8$, $\sin \phi = 0.6$. $I_2 = 4.5$A. $\Delta V_{2ph} \approx (1.333 \times 4.5 \times 0.8) + (1.884 \times 4.5 \times 0.6) \approx 4.80 + 5.09 \approx 9.89$V. Tension en charge par phase : $V_{2ph} = V_{20ph} - \Delta V_{2ph} = (400/\sqrt{3}) - 9.89 \approx 230.94 - 9.89 \approx 221.05$V. Tension entre phases en charge : $U_2 = \sqrt{3} V_{2ph} \approx \sqrt{3} \times 221.05 \approx 382.9$V. (Attention: on a trouvé U2 > U1 ! Cela peut arriver si le rapport m > 1. Vérifions : 400/380 > 1).
  3. Le secondaire est maintenant chargé par 3 résistances identiques ($R=180 \Omega$) couplées en triangle. La tension au primaire est toujours $U_1=380$V. Quelles sont les valeurs efficaces du courant de ligne ($I_2$) et de la tension entre phases ($U_2$) au secondaire?
    Charge en Triangle : $Z_{charge\_ph} = R = 180 \Omega$. $\cos \phi = 1$. Courant dans une résistance de la charge : $J_L = U_2 / R = U_2 / 180$. Courant de ligne secondaire (demandé par la charge) : $I_2 = \sqrt{3} J_L = \sqrt{3} U_2 / 180$. On a aussi la relation $U_2 = U_{20} - \Delta U_2$. $\Delta U_2 \approx \sqrt{3} (R_S I_2 \cos \phi + X_S I_2 \sin \phi) = \sqrt{3} R_S I_2$ (car $\cos\phi=1$). Donc $U_2 = U_{20} - \sqrt{3} R_S I_2$. Substituer $I_2$: $U_2 = U_{20} - \sqrt{3} R_S (\sqrt{3} U_2 / 180) = U_{20} - (3 R_S / 180) U_2 = U_{20} - (R_S / 60) U_2$. $U_2 (1 + R_S / 60) = U_{20}$. $U_2 = U_{20} / (1 + R_S / 60)$. $U_{20}=400$V, $R_S \approx 1.333 \Omega$. $U_2 = 400 / (1 + 1.333 / 60) \approx 400 / 1.0222 \approx 391.3$V. Courant de ligne secondaire : $I_2 = \sqrt{3} U_2 / 180 \approx \sqrt{3} \times 391.3 / 180 \approx 3.77$A.
