Acides et Bases selon Brönsted

Cliquez sur chaque question pour faire apparaître la réponse.

Question 1 : Quelle est la définition d'un acide selon Brönsted-Lowry ?

Un acide selon Brönsted-Lowry est une espèce chimique (molécule ou ion) capable de céder un ou plusieurs protons (ions H+).

Exemple : HCl dans l'eau cède un proton H+ à H2O.

Question 2 : Quelle est la définition d'une base selon Brönsted-Lowry ?

Une base selon Brönsted-Lowry est une espèce chimique capable de capter un ou plusieurs protons (ions H+).

Exemple : NH3 capte un proton H+ de H2O pour former NH4+.

Question 3 : Qu'appelle-t-on un couple acide/base conjugués ? Donner un exemple.

Un couple acide/base conjugués est constitué de deux espèces chimiques qui se transforment l'une en l'autre par gain ou perte d'un proton H+.

On le note Acide / Base.

Exemple : Le couple CH3COOH / CH3COO-. L'acide acétique (CH3COOH) est l'acide, et l'ion acétate (CH3COO-) est sa base conjuguée.

Relation : Acide ⇌ Base + H+

Question 4 : Identifier les deux couples acide/base mis en jeu dans la réaction suivante :
HNO2 + H2O ⇌ NO2- + H3O+

Dans cette réaction :

  • HNO2 (acide nitreux) cède un proton pour devenir NO2- (ion nitrite). Le couple est HNO2 / NO2-.
  • H2O (eau) capte un proton pour devenir H3O+ (ion hydronium). Le couple est H3O+ / H2O.

HNO2 est l'acide et H2O est la base dans le sens direct.

Question 5 : Qu'est-ce qu'une espèce amphotère (ou ampholyte) ? Citer un exemple courant.

Une espèce amphotère (ou ampholyte) est une espèce chimique qui peut se comporter à la fois comme un acide et comme une base selon Brönsted.

Exemple courant : L'eau (H2O).

  • Comme acide : H2O ⇌ OH- + H+ (couple H2O / OH-)
  • Comme base : H2O + H+ ⇌ H3O+ (couple H3O+ / H2O)

Autres exemples : l'ion hydrogénocarbonate (HCO3-), l'ion hydrogénosulfate (HSO4-), les acides aminés.

Question 6 : L'ion hydrogénocarbonate HCO3- est amphotère. Écrire les demi-équations protoniques montrant son caractère acide et son caractère basique.

L'ion hydrogénocarbonate (HCO3-) peut :

  • Agir comme un acide : Il cède un proton H+ pour former sa base conjuguée, l'ion carbonate (CO32-).
    Demi-équation : HCO3- ⇌ CO32- + H+
    Le couple est HCO3- / CO32-.
  • Agir comme une base : Il capte un proton H+ pour former son acide conjugué, l'acide carbonique (H2CO3, souvent en équilibre avec CO2,H2O).
    Demi-équation : HCO3- + H+ ⇌ H2CO3
    Le couple est H2CO3 / HCO3-.
Question 7 : Décrire le mécanisme général d'une réaction acido-basique selon Brönsted.

Une réaction acido-basique selon Brönsted est un transfert de proton H+ entre l'acide d'un couple (Acide 1 / Base 1) et la base d'un autre couple (Acide 2 / Base 2).

Réaction générale : Acide 1 + Base 2 ⇌ Base 1 + Acide 2

Exemple : NH3 (Base 2) réagit avec H2O (Acide 1, ici jouant le rôle d'acide faible).
H2O + NH3 ⇌ OH- + NH4+
(Acide 1) + (Base 2) ⇌ (Base 1) + (Acide 2)
Couples : H2O/OH- et NH4+/NH3.
